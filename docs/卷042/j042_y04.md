# 卷042 漢紀三十四 — 世祖光武皇帝中之上九年

九年〔注:癸巳(みずのとみ)の年、三三年〕。

①春、正月。潁陽成侯(えいようせいこう)の祭遵(さいじゅん)が軍中で亡くなった〔注:潁陽県は潁川郡に属する〕。帝は馮異(ふうい)に詔して、その軍営を併せ統率させた。祭遵という人は、清廉で倹約、慎み深く、私を抑えて公につくし、賜った褒美はすべて士卒に分け与えた。軍規はきびしく整っており、彼が駐屯する土地の役人や民は、そこに軍隊がいることにすら気づかぬほどであった。士を登用するにも儒術によって選び、酒宴を開いて音楽を奏でるときも、必ず雅(みやび)な歌をうたい、投壺(とうこ)で遊んだ〔注:雅歌とは雅(みやび)な詩を歌うこと。投壺は壺に矢を投げ入れ、入った数で勝敗を競い、負けた者に酒を飲ませる遊戯〕。臨終に際しては、薄葬(=質素な葬礼)にせよと遺言し、家のことを問われても、最後まで何も言わなかった。帝は祭遵の死をことのほか深く悼んだ。祭遵の柩(ひつぎ)が河南(かなん)に着くと、車駕(=帝)はみずから素服(=喪服)を着てこれに臨み、はるかに望んで哭(こく)し、激しく嘆き悲しんだ。帰るとき、城門に立ち寄って、通り過ぎる喪車(=葬列の車)を見送り、涙を流してとめられなかった。葬礼が済むと、さらにみずから太牢(たいろう=牛・羊・豕の三牲)を供えて祀(まつ)った。帝は大長秋(だいちょうしゅう)・謁者(えっしゃ)・河南尹(かなんいん)に詔して葬儀をとりしきらせ〔注:大長秋は皇后づきの官で、もとは「将行」といった秦の官。景帝の中六年に大長秋と改名した。後漢以降はもっぱら宦官を任じた〕、大司農(だいしのう)に費用を出させた。葬る段になると、車駕はふたたびこれに臨み、葬り終えると、また墓へ出向いて、その夫人や家族を見舞った。その後、朝会(ちょうかい)のたびに、帝はいつも嘆いて言った。「祭征虜(さいせいりょ)のように、国を憂え公につくす者を、どうしたら得られようか」〔注:祭遵は征虜将軍であった〕。衞尉(えいい)の銚期(ようき)が言った。「陛下はこの上なく仁徳のあるお方で、祭遵を哀れみ思うことやみませぬが、それゆえ群臣はみな、自分は祭遵に及ばぬと恥じ、おそれ入っております」〔注:帝が祭遵を慕ってたびたび口にするので、群臣は自分が祭遵に及ばぬのを恥じ、それぞれ恐縮しているという意〕。帝はそこで(祭遵を口にするのを)やめた。

②隗囂(かいごう)は、病んだうえに飢え、糗糒(きゅうび=干し飯)をかじりながら〔注:糗は炒った大豆と米の粉、糒は乾(ほ)し飯〕、怒りと無念のうちに死んだ。王元(おうげん)・周宗(しゅうそう)は、隗囂の末子である純(じゅん)を立てて王とし、兵をまとめて冀(き)に立てこもった。公孫述(こうそんじゅつ)は将の趙匡(ちょうきょう)・田弇(でんえん)を遣わして、純を助けさせた。帝は馮異を遣わしてこれを撃たせた。

③公孫述は、その翼江王(よくこうおう)田戎(でんじゅう)・大司徒(だいしと)任滿(じんまん)・南郡太守(なんぐんたいしゅ)程汎(ていはん)を遣わし、数万人を率いて江関(こうかん)を下らせ、馮駿(ふうしゅん)らの軍を撃ち破り、ついに巫(ふ)、および夷道(いどう)・夷陵(いりょう)を攻め落とし〔注:建武五年、岑彭(しんほう)は馮駿の軍を江州にとどめ、夷道・夷陵に分けて駐屯させていた。巫県もまた南郡に属する〕、そのまま荊門(けいもん)・虎牙(こが)に拠(よ)った〔注:荊門山は南にあり、上は合わさり下は開いて、その形は門に似る。虎牙山は北にあり、石壁は紅(あか)く、白い文(もよう)が牙に似るのでこの名がある。この二山は楚の西の要塞であった〕。江水を横切って浮橋(うきはし)と関楼(かんろう=やぐら)を築き、欑柱(さんちゅう=束ねた木の杭)を立てて水路をふさぎ〔注:「関楼」は范曄『後漢書』では「鬭楼」に作る。今の城上の敵楼(=やぐら)のようなもの。叢(む)らがる木を柱としたものを欑柱という〕、陣営を連ねて山にまたがらせ、陸路をふさいで、漢軍を防いだ。

④夏、六月、丙戌(ひのえいぬ)の日。帝は緱氏(こうし)に行幸し、轘轅(かんえん)に登った〔注:緱氏県は河南尹に属する。県内に緱氏山・轘轅山・轘轅坂があり、いずれも雒陽の東南にある〕。

⑤呉漢(ごかん)は、王常(おうじょう)ら四将軍の兵五万余人を率いて、盧芳(ろほう)の将である賈覽(からん)・閔堪(びんかん)を高柳(こうりゅう)で撃った〔注:高柳県は代郡に属する〕。匈奴(きょうど)がこれを救援し、漢軍は不利となった。こうして匈奴はいよいよ勢いを増し、略奪・暴虐が日ごとにひどくなった。帝は詔して、朱祜(しゅこ)を常山(じょうざん)に、王常を涿郡(たくぐん)に、破姦将軍(はかんしょうぐん)の侯進(こうしん)を漁陽(ぎょよう)に駐屯させ、討虜将軍(とうりょしょうぐん)の王霸(おうは)を上谷太守(じょうこくたいしゅ)として、匈奴に備えさせた。

⑥帝は来歙(らいきゅう)を遣わして、長安に駐屯する諸将をことごとく監督・統括させ、太中大夫(たいちゅうたいふ)の馬援(ばえん)をその副(=次官)とした。来歙は上書して言った。「公孫述は、隴西(ろうせい)・天水(てんすい)を防壁としているからこそ、いまだに命脈を保ち、息をついていられるのです。いま、この二郡を平らげてしまえば、公孫述の知恵も策も尽き果てましょう。よろしく兵馬をいっそう選りすぐり、物資・兵糧を蓄えておくべきです。いま西州(=西方の地)は破られたばかりで、兵も民も疲れ飢えております。もし財貨や穀物で招き寄せれば、その人々を集めることができましょう。国家が出すべきものは一つだけではなく、費用が足りぬのは臣も承知しております。しかし、これはやむをえぬことなのです」。帝はこれをもっともとした。そこで詔して、汧(けん)に六万斛(こく)の穀物を蓄えさせた。秋、八月、来歙は馮異ら五将軍を率いて、隗純(=隗囂の子の純)を天水で討った。

⑦驃騎将軍(ひょうきしょうぐん)の杜茂(ともう)が、賈覽と繁畤(はんし)で戦い〔注:繁畤県は鴈門郡に属する〕、杜茂の軍は大敗した。

⑧諸羌(しょきょう=もろもろの羌族)は、王莽(おうもう)の末年から塞内(=長城の内側)に入って住みつき、金城郡(きんじょうぐん)の属県の多くが彼らの手に落ちていた。隗囂はこれを討伐できず、そのまま手なずけて受け入れ、その兵を動員して漢に対抗させていた。司徒掾(しとえん)の班彪(はんぴょう)が上言して言った〔注:『続漢志』に、司徒の掾・属は三十一人。掾は比三百石、属は比二百石〕。「いま、涼州(りょうしゅう)の各郡には、どこにも降伏した羌族がおります。羌や胡(こ)は、髪を振り乱し、衣を左前に着る風俗で、漢人と入り混じって住んでおりますが、習俗が異なり、言葉も通じず、たびたび下級役人やずる賢い者どもに侵し奪われ、行き詰まって怒り、よりどころを失っております。それゆえ反乱を起こすのです。およそ蛮夷(ばんい)の侵し乱れは、みなこれが原因なのです。古い制度では、益州の管区には蛮夷騎都尉(ばんいきとい)を置き〔注:武帝が西南夷を開いたとき、一都尉を置いた〕、幽州の管区には領烏桓校尉(りょううがんこうい)を、涼州の管区には護羌校尉(ごきょうこうい)を置き、いずれも節(せつ)を持たせて統括・保護にあたらせ〔注:応劭(おうしょう)いわく、護烏桓・護羌校尉は比二千石で、節を擁(も)つ。長史一人・司馬二人を置き、いずれも六百石〕、彼らの怨みやもつれをおさめ、毎年めぐり歩いて、その苦しみを問わせました。また、たびたび使者や通訳を遣わして、その動静を通じ導き、塞外の羌・夷を役所の耳目(=情報源)とすれば、州や郡はそれによって警戒・備えができたのです。いまこそ、よろしく昔どおりに戻して、威光と防備を明らかにすべきです」。帝はこれに従い、牛邯(ぎゅうかん)を護羌校尉とした。

⑨盗賊が、陰貴人(いんきじん)の母である鄧氏(とうし)と、その弟の訢(きん)を殺した。帝はたいそう心を痛め、陰貴人の弟の就(しゅう)を宣恩侯(せんおんこう)に封じた〔注:帝は貴人の父である陰陸(いんりく)を追って宣恩哀侯(せんおんあいこう)の爵を贈り、就にその哀侯の跡を継がせた。後漢の旧制では、皇后の父だけが侯に封ぜられた。貴人はまだ皇后の位に正式についていないのに、その父に追爵したのは、旧制にないことである〕。帝はさらに、就の兄である侍中(じちゅう)の興(こう)を召して、これを封じようとし、印綬(いんじゅ)を目の前に置いた。興はかたく辞退して言った。「臣には、まっさきに城に登り敵陣を陥(おとしい)れたような功もないのに、一家のうち数人がそろって爵位と領地をいただけば、天下の人々に不満を抱かせてしまいます。まことに望むところではありませぬ」。帝はこれをよしとし、その志をまげさせなかった。貴人がそのわけを問うと、興は言った。「そもそも外戚(がいせき)の家というものは、困ったことに謙(へりくだ)って退くことを知らず、娘を嫁がせれば侯王に添わせようとし、嫁を取れば公主(=皇女)を流し目に狙う。愚かな私の心には、それがまことに不安なのです。富貴には極まりがあり、人は足るを知るべきで、おごり高ぶればなおさら世間の見聞きする者にそしられます」。貴人はその言葉に感じ入り、みずから深くへりくだって、ついに親族のために官位を求めることはしなかった。

⑩帝は寇恂(こうじゅん)を召し返し、漁陽太守(ぎょようたいしゅ)の郭伋(かくきゅう)を潁川太守とした。郭伋は山賊の趙宏(ちょうこう)・召吳(しょうご)ら数百人を招いて降伏させ、みな帰郷させて農籍(=農民の戸籍)に編入した〔注:附農とは農籍に編入すること〕。そのうえで、自分が独断専行したとして、みずからを弾劾した〔注:降伏した賊をほしいままに放免したことをさす〕が、帝はそれをとがめなかった。のち、趙宏・召吳の仲間たちは、郭伋の威信を聞き伝え、遠くは江南(こうなん)から、あるいは幽州(ゆうしゅう)・冀州(きしゅう)から、申し合わせたわけでもないのにそろって降伏し、その列はひきもきらなかった。

⑪莎車(さしゃ)の王である康(こう)が死に、弟の賢(けん)が立った。賢は拘彌(こうび)・西夜(せいや)の王を攻め殺し〔注:拘彌は前漢の杅冞(うび)。唐では寧彌(ねいび)という。西夜国は雒陽を去ること一万四千四百里〕、康の二人の子をそれぞれ王とした。

九年
春，正月，潁陽成侯祭遵薨於軍，詔馮異幷將其營。遵爲人，廉約小心，克己奉公，賞賜盡與士卒；約束嚴整，所在吏民不知有軍。取士皆用儒術，對酒設樂，必雅歌投壺。臨終，遺戒薄葬；問以家事，終無所言。帝愍悼之尤甚，遵喪至河南，車駕素服臨之，望哭哀慟；還，幸城門，閱過喪車，涕泣不能已；喪禮成，復親祠以太牢。詔大長秋、謁者、河南尹護喪事，大司農給費。至葬，車駕復臨之；旣葬，又臨其墳，存見夫人、室家。其後朝會，帝每歎曰：「安得憂國奉公如祭征虜者乎！」衞尉銚期曰：「陛下至仁，哀念祭遵不已，羣臣各懷慚懼。」帝乃止。
隗囂病且餓，餐糗糒，恚憤而卒。王元、周宗立囂少子純爲王，總兵據冀。公孫述遣將趙匡、田弇助純。帝使馮異擊之。
公孫述遣其翼江王田戎、大司徒任滿、南郡太守程汎將數萬人下江關，擊破馮駿等軍，遂拔巫及夷道、夷陵，因據荊門、虎牙，橫江水起浮橋、關樓，立欑柱以絕水道，結營跨山以塞陸路，拒漢兵。
夏，六月，丙戌，帝幸緱氏，登轘轅。
吳漢率王常等四將軍兵五萬餘人擊盧芳將賈覽、閔堪於高柳；匈奴救之，漢軍不利。於是匈奴轉盛，鈔暴日增。詔朱祜屯常山，王常屯涿郡，破姦將軍侯進屯漁陽，以討虜將軍王霸爲上谷太守，以備匈奴。
帝使來歙悉監護諸將屯長安，太中大夫馬援爲之副。歙上書曰：「公孫述以隴西、天水爲藩蔽，故得延命假息；今二郡平蕩，則述智計窮矣。宜益選兵馬，儲積資糧。今西州新破，兵人疲饉，若招以財穀，則其衆可集。臣知國家所給非一，用度不足，然有不得已也！」帝然之。於是詔於汧積穀六萬斛。秋，八月，來歙率馮異等五將軍討隗純於天水。
驃騎將軍杜茂與賈覽戰於繁畤，茂軍敗績。
諸羌自王莽末入居塞內，金城屬縣多爲所有。隗囂不能討，因就慰納，發其衆與漢相拒。司徒掾班彪上言：「今涼州部皆有降羌。羌胡被髮左衽，而與漢人雜處，習俗旣異，言語不通，數爲小吏黠人所見侵奪，窮恚無聊，故致反叛。夫蠻夷寇亂，皆爲此也。舊制，益州部置蠻夷騎都尉，幽州部置領烏桓校尉，涼州部置護羌校尉，皆持節領護，治其怨結，歲時巡行，問所疾苦。又數遣使譯，通導動靜，使塞外羌夷爲吏耳目，州郡因此可得警備。今宜復如舊，以明威防。」帝從之。以牛邯爲護羌校尉。
盜殺陰貴人母鄧氏及弟訢。帝甚傷之，封貴人弟就爲宣恩侯。復召就兄侍中興，欲封之，置印綬於前。興固讓曰：「臣未有先登陷陳之功，而一家數人，並蒙爵土，令天下觖望，誠所不願！」帝嘉之，不奪其志。貴人問其故，興曰：「夫外戚家苦不知謙退，嫁女欲配侯王，取婦眄睨公主，愚心實不安也。富貴有極，人當知足，夸奢益爲觀聽所譏。」貴人感其言，深自降挹，卒不爲宗親求位。
帝召寇恂還，以漁陽太守郭伋爲潁川太守。伋招降山賊趙宏、召吳等數百人，皆遣歸附農；因自劾專命，帝不以咎之。後宏、吳等黨與聞伋威信，遠自江南，或從幽、冀，不期俱降，駱驛不絕。
莎車王康卒，弟賢立，攻殺拘彌、西夜王，而使康兩子王之。
