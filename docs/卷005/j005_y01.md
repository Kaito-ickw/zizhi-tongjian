# 卷005 周紀五 — 赧王下四十三年

> 巻 5 / 294 ・ 周紀五 ・ 年号: 赧王下四十三年 ・ 西暦: 272 BCE

四十三年〔注:己丑(きちゅう)の年、紀元前二七二年〕。

楚は、左徒(さと)の黃歇(こうけつ)を、太子の完(かん)に付き添わせて、秦に人質として差し出した〔注:左徒は楚の官名である〕。

秦は南陽郡(なんようぐん)を置いた〔注:およそ山の南、川の北を「南陽」と呼ぶ。秦の南陽郡は、南山(なんざん)の南、漢水(かんすい)の北にあるので、こう名づけられた〕。

秦・魏・楚が手を組んで燕を攻めた。

燕の惠王(けいおう)が薨じ、その子の武成王(ぶせいおう)が立った。

赧王下
==四十三年==
楚以左徒黃歇侍太子完爲質於秦。
秦置南陽郡。
秦、魏、楚共伐燕。
燕惠王薨，子武成王立。
