# 卷002 周紀二 — 顯王二十六年

顯王二十六年〔注:戊寅(ぼいん)の年、紀元前三四三年〕。

周王が秦に伯(はく)の地位を授けた〔注:伯とは、周の二伯・九伯のように諸侯を統率する任である〕。諸侯はみな秦に祝賀した。秦の孝公(こうこう)は公子の少官(しょうかん)に軍を率いさせ、逢澤(ほうたく)で諸侯と会合させたうえで、ともに周王に朝見させた。

---

<details>
<summary>原文を表示</summary>

二十六年
王致伯于秦，諸侯皆賀秦。秦孝公使公子少官帥師會諸侯于逢澤以朝王。
