# 卷006 秦紀一 — 始皇帝上八年

> 巻 6 / 294 ・ 秦紀一 ・ 年号: 始皇帝上八年 ・ 西暦: 239 BCE

八年〔注:壬戌(じんじゅつ)の年、紀元前二三九年〕。

魏は鄴(ぎょう)の地を趙に与えた。韓(かん)の桓惠王(かんけいおう)が薨(こう)じ、子の安(あん)が即位した。

---

<details>
<summary>原文を表示</summary>

八年
魏與趙鄴。
韓桓惠王薨，子安立。
