# 卷006 秦紀一 — 始皇帝上十五年

十五年〔注:己巳(きし)の年、紀元前二三二年〕。

秦王は大軍を起こして趙(ちょう)を攻め、一軍は鄴(ぎょう)に、もう一軍は太原(たいげん)に迫り、狼孟(ろうもう)・番吾(ばご)を取った。だが李牧(りぼく)の軍に行き当たって、退いた〔注:秦軍は李牧を恐れ、戦おうとせずに引き返したのである〕。

これより前、燕(えん)の太子丹(たいしたん)は、かつて趙に人質となっていたことがあり、そのころ、まだ即位前の秦王と親しくしていた〔注:秦王の父・異人(いじん)も趙に人質に出されており、王は趙の邯鄲(かんたん)で生まれたのである〕。やがて秦王が即位すると、今度は丹が秦への人質となったが、王は彼を礼遇しなかった。丹は怒り、燕へ逃げ帰った〔注:のちに丹が荊軻(けいか)を遣わして秦王を刺させる事件の伏線である〕。

十五年
王大興師伐趙，一軍抵鄴，一軍抵太原，取狼孟、番吾；遇李牧而還。
初，燕太子丹嘗質於趙，與王善。王卽位，丹爲質於秦，王不禮焉。丹怒，亡歸。
