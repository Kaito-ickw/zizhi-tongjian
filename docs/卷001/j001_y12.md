# 卷001 周紀一 — 安王十二年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王十二年 ・ 西暦: 390 BCE

[← 巻インデックス](README.md)

安王十二年〔注:辛卯(しんぼう)の年、紀元前三九〇年〕。

秦と晋が武城(ぶじょう)で戦った。

斉が魏を攻め、襄陽(じょうよう)を奪い取った〔注:「襄陽」は「襄陵」の誤りとされる〕。

魯が斉の軍を平陸(へいりく)で破った。

---

<details>
<summary>原文を表示</summary>

十二年
秦、晉戰于武城。
齊伐魏，取襄陽。
魯敗齊師于平陸。
