# 卷003 周紀三 — 赧王上八年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上八年 ・ 西暦: 307 BCE

八年〔注:甲寅(きのえとら)の年、紀元前三〇七年〕。

①甘茂(かんぼう)は宜陽(ぎよう)を攻めたが、五か月たっても陥落させられなかった。樗里子(ちょりし)と公孫奭(こうそんせき)は、案の定これに異を唱えた。秦王は甘茂を呼び戻し、兵を引き上げさせようとした。甘茂は言った。「息壤(そくじょう)の盟がございましょう」〔注:先に息壤で結んだ盟約を持ち出したのである〕。王は言った。「あるとも。」そこで王は総力を挙げて兵を動員し、甘茂を後押しした〔注:佐とは助けること〕。秦軍は六万の首を斬り、ついに宜陽を陥落させた。韓の公仲侈(こうちゅうし)は秦に出向いて詫び、和議を請うた〔注:請平とは和を請うのと同じ〕。

②秦の武王は力比べの遊びを好んだ。任鄙(じんぴ)・烏獲(うかく)・孟說(もうえつ)といった力自慢の者たちが、みな高位の官に取り立てられた。八月、王は孟說と鼎(かなえ)を持ち上げたが、血脈が切れて薨(こう)じた〔注:この鼎とは九鼎のことである。脈は臓腑をつなぐもので、重い物を持ち上げて力が及ばないと脈が切れて死ぬ。『史記』では、武王は周に至って周で卒したとある〕。孟說は一族もろとも誅(ちゅう)された〔注:族とは、その一族を皆殺しにすること〕。武王には子がなく、異母弟の稷(しょく)が燕(えん)で人質となっていたので、国人がこれを迎えて立てた〔注:逆とは迎えること〕。これが昭襄王(しょうじょうおう)である。昭襄王の母は羋(び)氏の八子(はっし)で〔注:羋は楚の姓。八子は後宮の妃嬪(ひひん)の位の一つ〕、楚の女(むすめ)であった。これがのちの宣太后(せんたいごう)である。

③趙の武靈王(ぶれいおう)は北のかた中山(ちゅうざん)の地を攻め巡り〔注:略とは、軍を進めて土地を巡り従えること。その軍はすばやい〕、房子(ぼうし)に至り〔注:房子は常山郡に属する県〕、さらに代(だい)に至った。北は無窮(むきゅう)まで〔注:代の北の塞外には数千里の大砂漠が広がり、果てしないので無窮という〕、西は黃河まで及び、黃華(こうか)の頂きに登った〔注:黃華は黃河のほとりの山の名であろう〕。武靈王は肥義(ひぎ)と謀り、胡服(こふく)を着て馬上から弓を射る戦法(騎射)を民に教えようとして言った。「愚か者は笑うだろうが、賢者は道理を見抜くものだ。たとえ世じゅうが私を笑い者にしようとも、胡の地も中山も、私は必ず手に入れてみせる!」こうして自ら胡服を身につけた。

国人はみなこれを望まず、王の叔父にあたる公子成(こうしせい)は病と称して参朝しなかった。王は使者を遣わして頼んで言わせた。「家のことは親に従い〔注:親とは父母のこと〕、国のことは君に従うもの。いま私が新たな教えを立てて服装を改めようとしているのに、叔父上が従われなければ、私は天下が私を非難するのではないかと恐れるのです。国を治めるには定法があり、民の利を本とする。政を行うには筋道があり、命令が行き渡ることを第一とする。徳を明らかにするにはまず卑しい者から説き、政に従わせるにはまず貴い者の信を得る〔注:徳は下々にまで及ぶのが望ましいから、まず卑賤の者に説く。卑賤の者がその徳に感じ入れば、徳の及ぶ広さが知れる。法は貴く君に近い者から行われるべきだから、まず貴い者の信を得る。貴近の者が法を守れば、法が必ず行われることが知れる〕。だからこそ、叔父上の(進んで従うという)義によって、胡服の事業を成し遂げたいのです。」公子成は再拝して額を地につけ、言った。「私が聞きますに、中国(中原)というものは、聖賢が教化を施す地であり、礼楽が用いられる地であり、遠方の者があこがれて訪れる地であり、蠻夷(ばんい)が手本とする地でございます。いま王はこれを捨てて、遠方の異民族の服を真似て着〔注:襲とは衣を重ねて着ること〕、いにしえの道を変え、人々の心に逆らおうとなさる。どうか王よ、よくよくお考えください!」〔注:孰は古い「熟」の字に同じで、よく考える意〕。使者がこれを王に報告した。王はみずから出向いて頼み、言った。「我が国は東に齊(せい)と中山があり〔注:趙の都の邯鄲(かんたん)は東で齊に接し、中山はその東北にある〕、北に燕(えん)と東胡(とうこ)があり、西に樓煩(ろうはん)・秦・韓との境を抱えている〔注:林胡(りんこ)・樓煩は嵐(らん)・勝(しょう)より北の地で、趙の西辺である〕。いま騎射の備えがなければ、何をもってこれらを守れようか? かつて中山は齊の強兵を後ろ盾にして我が地を侵し荒らし、我が民を捕らえて連れ去り、水を引いて鄗(こう)を囲んだ。社稷(しゃしょく)の神霊のご加護がなければ、鄗はあやうく守りきれぬところであった〔注:鄗は趙の邑。後漢の光武帝が高邑(こうゆう)と改めた〕。亡き先君はこれを恥とされた〔注:趙国の恥としたのである〕。だから私は服を改めて騎射を取り入れ、四方の境の難に備え、中山への遺恨を晴らそうとしているのだ。それなのに叔父上は中原の習俗に従い、服を改めるという評判を嫌い、鄗での屈辱を忘れようとしておられる。これは私の望むところではない!」公子成は命に従った。そこで王は彼に胡服を賜り、翌日、公子成は胡服を着て参朝した。こうしてはじめて胡服の令を発し、騎射の兵を募ったのである。

八年
甘茂攻宜陽，五月而不拔。樗里子、公孫奭果爭之。秦王召甘茂，欲罷兵。甘茂曰：「息壤在彼。」王曰：「有之。」因大悉起兵以佐甘茂，斬首六萬，遂拔宜陽。韓公仲侈入謝於秦以請平。
秦武王好以力戲，力士任鄙、烏獲、孟說皆至大官。八月，王與孟說舉鼎，絕脈而薨；族孟說。武王無子，異母弟稷爲質於燕，國人逆而立之，是爲昭襄王。昭襄王母羋八子，楚女也，實宣太后。
趙武靈王北略中山之地，至房子，遂至代，北至無窮，西至河，登黃華之上。與肥義謀胡服騎射以敎百姓，曰：「愚者所笑，賢者察焉。雖驅世以笑我，胡地、中山，吾必有之！」遂胡服。
國人皆不欲，公子成稱疾不朝。王使人請之曰︰「家聽於親，國聽於君。今寡人作敎易服而公叔不服，吾恐天下議己也。制國有常，利民爲本；從政有經，令行爲上。明德先論於賤，而從政先信於貴，故願慕公叔之義以成胡服之功也。」公子成再拜稽首曰：「臣聞中國者，聖賢之所敎也，禮樂之所用也，遠方之所觀赴也，蠻夷之所則效也。今王舍此而襲遠方之服，變古之道，逆人之心，臣願王孰圖之也！」使者以報。王自往請之，曰：「吾國東有齊、中山，北有燕、東胡，西有樓煩、秦、韓之邊。今無騎射之備，則何以守之哉？先時中山負齊之強兵，侵暴吾地，係累吾民，引水圍鄗；微社稷之神靈，則鄗幾於不守也。先君醜之，故寡人變服騎射，欲以備四境之難，報中山之怨。而叔順中國之俗，惡變服之名，以忘鄗事之醜，非寡人之所望也！」公子成聽命，乃賜胡服；明日服而朝。於是始出胡服令而招騎射焉。
