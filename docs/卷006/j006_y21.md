# 卷006 秦紀一 — 始皇帝上十二年

十二年〔注:丙寅(へいいん)の年、紀元前二三五年〕。

文信侯(ぶんしんこう=呂不韋)は酖毒(ちんどく)の酒をあおって死に〔注:酖は南方に産する鴆(ちん)という鳥で、蝮(まむし)を食らう。その羽で酒をかき混ぜて飲ませると、たちどころに死ぬ〕、ひそかに葬られた。その舍人(しゃじん=食客)で葬儀に参列して哭(な)いた者は、みな追放され遠方へ移された〔注:臨とは哭くこと〕。そのうえ王は言った。「今より後、嫪毐(ろうあい)や不韋(ふい)のように道に背いて国事をほしいままにする者があれば、その一門を財産没収のうえ取り潰す。これを見せしめとせよ。」

〔注:揚子(ようし)の『法言(ほうげん)』は言う。ある人が問うた。「呂不韋は智者だったのではないか。一人の人間を元手に大きな利を手に入れたのだから」と。答えて言う。「誰が不韋を智者などと言えようか。国を手に入れた代わりに一族を失ったのだ。呂不韋のあの盗みっぷりは、壁を穿(うが)って忍び込む盗賊の親玉とでもいうべきものよ〔注:穿(せん)は壁に穴をあけること、窬(ゆ)は塀をくり抜くこと〕。こそ泥なら、小甕(こがめ)や升目(ますめ)ほどのわずかな盗みを働く者を見たことはあるが〔注:擔(たん)は儋とも書き、斉(せい)の人は小甕を儋と呼ぶ。石は斗石(ますめ)のこと〕、洛陽(らくよう)まるごとを盗み取った者など、ついぞ見たことがない。」〕

六月から雨が降らず、八月まで続いた。

四郡(しぐん)の兵を発して、魏(ぎ)を助けて楚(そ)を伐(う)たせた〔注:函谷関(かんこくかん)以東の四郡の兵を発したのである〕。

十二年
文信侯飲酖死，竊葬。其舍人臨者，皆逐遷之。且曰︰「自今以來，操國事不道如嫪毐、不韋者，籍其門，視此！」
揚子《法言》曰︰或問︰「呂不韋其智矣乎？以人易貨。」曰︰「誰謂不韋智者歟！以國易宗。呂不韋之盜，穿窬之雄乎！穿窬也者，吾見擔石矣，未見雒陽也。」
自六月不雨，至于八月。
發四郡兵助魏伐楚。
