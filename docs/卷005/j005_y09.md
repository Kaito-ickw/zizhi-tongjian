# 卷005 周紀五 — 赧王下五十一年

五十一年〔注:丁酉(ていゆう)の年、紀元前二六四年〕。

秦の武安君(ぶあんくん)が韓を伐ち、九つの城を抜き、五万の首を斬った。

田單(でんたん)が趙の相(しょう)となった。

==五十一年==
秦武安君伐韓，拔九城，斬首五萬。
田單爲趙相。
