# 卷003 周紀三 — 赧王上元年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上元年 ・ 西暦: 314 BCE

[← 巻インデックス](README.md)

赧王(なんおう)上〔注:赧とは深く恥じ入ることをいう。王室は微弱で危うく、東周・西周のあいだに身を寄せて間借りするありさまで、まことに赧(あか)らむほど恥ずべき状態だったので、この号がつけられた。本来、諡(おくりな)の法には赧の字はない〕。

元年〔注:丁未(ひのとひつじ)の年、紀元前三一四年〕。

秦の人が義渠(ぎきょ)〔注:義渠は戎の国の名である〕に侵攻し、二十五城を得た。

魏の人が秦に背いた。秦の人は魏を討ち、曲沃(きょくよく)を取って、その住民は(魏に)返した。さらに韓を岸門(がんもん)で破り、韓の太子の倉(そう)が人質として秦に入って和睦した。

燕では子之(しし)が王となって三年、国内は大いに乱れていた。将軍の市被(しひ)が太子の平(へい)と謀って子之を攻めようとした。齊王が使者を遣わして太子に言わせた。「わたくしは、太子が君臣の義を正し、父子の(あるべき)地位を明らかにしようとしていると聞いた〔注:君臣の義を正すとは、太子平が子之の王を僭称した罪を正そうとしていること、父子の位を明らかにするとは、太子平が父の噲(かい)の位を継ぐべきだということをいう〕。わたくしの国は、太子の御命じになるままに(従いましょう)。」太子はそこで一党と結び兵を集め、市被に子之を攻めさせたが、勝てなかった。市被は逆に太子を攻めた。戦乱は数か月に及び、死者は数万人にのぼり、民は苦しみ恐れおののいた。齊王は章子(しょうし)に命じて五都(ごと)の兵を率いさせ、北地(ほくち)の兵力をも合わせて燕を討たせた〔注:北地は齊の北の境である〕。燕の兵士は戦わず、城門も閉じなかった。齊の人は子之を捕らえて肉醤(にくびしお)にし〔注:醢は肉醤のことである〕、ついで燕王の噲を殺した。

齊王が孟子(もうし)に問うた。「ある者はわたくしに燕を取るなと言い、ある者は取れと言う。万乗(ばんじょう)の国(=齊)で万乗の国(=燕)を討ち〔注:いにしえは天子の領土が方千里で、戦車一万乗を出した。戦国の七国は併呑によって強大となり、当時はみな万乗の国であった〕、五十日で攻め取った。これは人の力(だけ)で及ぶことではない(=天意である)。取らなければ、きっと天の災いがあろう。これを取るのはどうだろうか。」孟子は答えて言った。「取って燕の民が喜ぶのなら取りなさい。いにしえの人でそうした者がおります、武王がそれです。取って燕の民が喜ばぬのなら取ってはなりません。いにしえの人でそうした者がおります、文王がそれです。万乗の国で万乗の国を討つにあたり、(燕の民が)竹の器に飯を盛り壺に飲み物を入れて王の軍を迎えたのは〔注:簞は竹の器、食はたきあがった食べ物、漿は飲み水のことである〕、ほかでもありません、水火の苦しみから逃れようとしたのです。もし(齊の支配で)水がいっそう深く、火がいっそう熱くなるようなら、(民の心は)他へと移っていくだけのことです〔注:燕の民が他国へと移っていくであろうということ〕。」

諸侯が燕を救おうと相談しはじめた。齊王が孟子に言った。「諸侯にはわたくしを討とうと謀る者が多い。どう対処すればよいか。」孟子は答えた。「わたくしはこう聞いております、七十里(四方の小国)から出て天下に政(まつりごと)を行った者がいる、湯王(とうおう)がそれです。千里(の大国)を持ちながら人を恐れた者がいるとは、まだ聞いたことがありません。『書経』にこうあります、『我が君を待ち望む、君が来られればわれらは生き返る』と〔注:これは『書経』仲虺之誥(ちゅうきのこう)の句である。徯は待つ、后は君のことである〕。いま燕がその民を虐げているところへ、王が出向いてこれを征伐すれば、民は自分たちを水火の苦しみのなかから救い出してくれるものと思い、竹の器に飯を盛り壺に飲み物を入れて王の軍を迎えたのです。それなのに、もし(王が)その父兄を殺し、その子弟を縛りあげ〔注:係累とは縛りつなぐことである〕、その宗廟(そうびょう)を壊し、その宝器を運び去るようでは、どうしてうまくいきましょうか。天下はもとより齊の強大さを恐れているのです。それにいま領土を倍にしたうえ〔注:齊は燕を併せれば領土が以前の倍になる〕、仁政を行わないとあっては、これは天下の兵を動かす(=諸侯を敵に回す)ことになります。王は速やかに令を出して、(捕らえた)老人と子どもを返し〔注:旄は年老いた者、倪は幼い者のことである〕、(運び去ろうとした)宝器を留め置き、燕の民衆と相談して(新しい)君主を立て、そのうえで燕から引き上げなさい。そうすればまだ(諸侯の攻撃を)止めるのに間に合いましょう。」しかし齊王は聞き入れなかった。

ほどなく燕の人々が(齊に)背いた。齊王は言った。「わたしは孟子に対してまことに恥ずかしい。」陳賈(ちんか)が言った。「王よ、ご心配なさいますな。」そして孟子に会って言った。「周公(しゅうこう)とはどういう人物でしょうか。」孟子は言った。「いにしえの聖人です。」陳賈は言った。「周公は管叔(かんしゅく)に商(=殷)を監視させましたが〔注:いにしえは殷と商を通称した。商は最初の封地を国号とし、殷は都の亳(はく)を国号としたもの。孟子では陳賈は『殷を監視させた』と言っているが、いま『通鑑』が『商を監視させた』とするのは、宋の廟の諱(いみな)を避けたためである〕、管叔は商をかかえて反乱を起こしました。周公はその反乱を起こすことを知ったうえで彼に監視させたのでしょうか。」孟子は言った。「知らなかったのです。」陳賈は言った。「それでは聖人にも過ちがあるということですか。」孟子は言った。「周公は弟、管叔は兄です。周公の過ちは、もっともなことではありませんか。それに、いにしえの君子は過ちがあれば改めましたが、いまの君子は過ちがあればそれを押し通します。いにしえの君子の過ちは、日食や月食のようなもので、民はみなそれを目にし、改めればみなそれを仰ぎ見ました。ところがいまの君子は、ただ過ちを押し通すだけでなく、あまつさえそれに言い訳まで重ねるのです。」

この年、齊の宣王が薨(こう)じ、子の湣王(びんおう)地(ち)が立った。

赧王上
元年
秦人侵義渠，得二十五城。
魏人叛秦。秦人伐魏，取曲沃而歸其人。又敗韓於岸門，韓太子倉入質于秦以和。
燕子之爲王三年，國內大亂。將軍市被與太子平謀攻子之。齊王令人謂太子曰：「寡人聞太子將飭君臣之義，明父子之位，寡人之國唯太子所以令之。」太子因要黨聚衆，使市被攻子之，不克。市被反攻太子。搆難數月，死者數萬人，百姓恫恐。齊王令章子將五都之兵，因北地之衆以伐燕。燕士卒不戰，城門不閉。齊人取子之，醢之，遂殺燕王噲。
齊王問孟子曰：「或謂寡人勿取燕，或謂寡人取之。以萬乘之國伐萬乘之國，五旬而舉之，人力不至於此；不取，必有天殃。取之何如？」孟子對曰：「取之而燕民悅則取之，古之人有行之者，武王是也。取之而燕民不悅則勿取，古之人有行之者，文王是也。以萬乘之國伐萬乘之國，簞食壺漿以迎王師，豈有他哉？避水火也。如水益深，如火益熱，亦運而已矣！」
諸侯將謀救燕。齊王謂孟子曰：「諸侯多謀伐寡人者，何以待之？」對曰：「臣聞七十里爲政於天下者，湯是也；未聞以千里畏人者也。《書》曰：『徯我后，后來其蘇。』今燕虐其民，王往而征之，民以爲將拯己於水火之中也，簞食壺漿以迎王師。若殺其父兄，係累其子弟，毀其宗廟，遷其重器。如之何其可也！天下固畏齊之強也，今又倍地而不行仁政，是動天下之兵也。王速出令，反其旄倪，止其重器，謀於燕衆，置君而後去之，則猶可及止也。」齊王不聽。
已而燕人叛。王曰：「吾甚慚於孟子。」陳賈曰：「王無患焉。」乃見孟子，曰：「周公何人也？」曰：「古聖人也。」陳賈曰：「周公使管叔監商，管叔以商畔也。周公知其將畔而使之與？」曰：「不知也。」陳賈曰：「然則聖人亦有過與？」曰：「周公，弟也，管叔，兄也，周公之過不亦宜乎！且古之君子，過則改之；今之君子，過則順之。古之君子，其過也如日月之食，民皆見之；及其更也，民皆仰之。今之君子，豈徒順之，又從爲之辭！」
是歲，齊宣王薨，子湣王地立。
