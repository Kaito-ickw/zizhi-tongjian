# 卷048 漢紀四十 — 孝和皇帝下六年

六年〔注:甲午(きのえうま)の年、紀元九四年〕。

春、正月、皇甫稜(こうほりょう)が免じられ、執金吾(しつきんご)の朱徽(しゅき)を行(こう)度遼将軍(=度遼将軍代行)とした。当時、(南)単于(=安国)と中郎将(ちゅうろうしょう、=使匈奴中郎将)の杜崇(とすう)は折り合いが悪く、(安国は)上書して杜崇を訴えた。杜崇は西河(せいが)太守にそれとなく示して、単于の上奏文を遮らせたので〔注:中郎将とは使匈奴中郎将(匈奴に派遣された中郎将)のこと。単于は西河の美稷(びしょく)に居たので、(杜崇は)太守にそれとなく命じてその上奏文を遮らせ、(帝の)耳に達しないようにしたのである〕、単于は自分の言い分を(帝に)聞いてもらう手立てがなくなった。杜崇はそこで朱徽とともに上言した。「南単于の安国は、古くからの胡(=旧来の部衆)を疎んじ遠ざけ、新たに降(くだ)った者を親しみ近づけ、左賢王(さけんおう)の師子(しし)および左臺且渠(さだいしょきょ)の劉利(りゅうり)らを殺そうとしております。また、右部(うぶ)の降った者たちは、こぞって安国を迫り脅(おど)して兵を起こさせ、背(そむ)かせようと謀(はか)っております。どうか西河・上郡(じょうぐん)・安定(あんてい)に命じて、これに備えさせてください」。帝はこれを公卿(こうけい)に下して議論させたところ、みなこう考えた。「蛮夷(ばんい)は心変わりが激しく、その動きを測り知るのは難しいとはいえ、大軍が集結すれば、必ずやあえて軽々しくは動きますまい。いまは方略(はかりごと)にすぐれた使者を単于の庭(陣営)に遣わし、杜崇・朱徽および西河太守と力を合わせ、その動静をうかがわせるのがよろしいでしょう。もし格別の異変がなければ、杜崇らに安国のもとへ赴かせ、その左右の大臣を集め、横暴をはたらいて辺境の害となっている部衆を責め、ともに罪を裁いて誅(ちゅう)させればよいのです〔注:ともに集まってその罪を公平に処し、誅すべき者は誅すること〕。もし命に従わぬなら、臨機の方略をとらせ、事が終わったのちに、功に応じて加減して賞賜(しょうし)を行えば〔注:李賢いわく、賜う物を加減して、多くは与えないこと〕、これもまた諸々の蛮夷への見せしめとして十分でしょう」。そこで朱徽・杜崇はとうとう兵を発して単于の庭へ進軍した。安国は夜、漢軍が来たと聞いて大いに驚き、帳(=幕屋)を捨てて逃げ去り〔注:帳は単于の住まいで、これを穹廬(きゅうろ)といい、また廬帳(ろちょう)ともいう〕、そのまま兵を挙げて師子を誅そうとした。師子はあらかじめこれを知り、廬落(ろらく、部族)をことごとく率いて曼柏城(まんぱくじょう)に入った〔注:曼柏県は五原(ごげん)郡に属する〕。安国は追って城下まで来たが、門は閉ざされ、入ることができなかった。朱徽は吏を遣わして(両者を)諭(さと)し和解させようとしたが、安国は聞き入れなかった。城がついに陥(お)ちなかったので、(安国は)兵を率いて五原に駐屯した。杜崇・朱徽は諸郡の騎兵を発して急ぎこれを追ったので、(安国の)部衆はみな大いに恐れ、安国の舅(おじ)である骨都侯(こつとこう)の喜爲(きい)は、自分たちもまきぞえで誅されることを案じ、そこで安国を撃ち殺し、師子を立てて亭独尸逐侯鞮単于(ていどくしちくこうていぜんう)とした。

己卯(つちのとう)の日、司徒の丁鴻(ていこう)が薨(こう)じた。

二月、丁未(ひのとひつじ)の日、司空の劉方(りゅうほう)を司徒に、太常(たいじょう)の張奮(ちょうふん)を司空とした。

夏、五月、城陽懐王(じょうようかいおう)の淑(しゅく)が薨じた。子が無く、国は廃された。

秋、七月、京師(けいし、都)で干ばつがあった。

西域都護(せいいきとご)の班超(はんちょう)は、龜茲(きじ)・鄯善(ぜんぜん)ら八国の兵あわせて七万余人を動員して焉耆(えんき)を討ち、その城下に至り、焉耆王の広(こう)・尉犂王(いりおう)の汎(はん)らを陳睦(ちんぼく)の故城(旧城)におびき寄せ、これを斬って、その首を京師へ送り届けた。そのまま兵を放って略奪させ、五千余の首を斬り、捕虜一万五千人を獲(え)、あらためて焉耆の左侯(さこう)の元孟(げんもう)を焉耆王に立てた〔注:焉耆国には左右の将・左右の侯がある〕。班超は焉耆に半年とどまり、これを慰撫(いぶ)した。こうして西域の五十余国はことごとく人質を差し出して(漢に)内属(ないぞく)し、海浜(かいひん、西の海べ)に至るまで〔注:西海の浜である。いわゆる条支(じょうし)・大秦(たいしん)・蒙奇(もうき)・兜勒(とうろく)の諸国である〕、四万里のかなたから、みな言葉を幾重にも通訳して貢ぎ物を献じた〔注:班超が西域の功を成しえたわけは、匈奴が衰え疲弊して、その力が西域にまで及ばなかったからである〕。

南単于の師子が立つと、降(くだ)った胡(こ)の五、六百人が夜に師子を襲った。安集掾(あんしゅうえん)の王恬(おうてん)が衛護(えいご)の士(兵)を率いてこれと戦い、これを破った〔注:使匈奴中郎将は掾(えん)を置き、事に応じて員数を定めた。安集掾とは、匈奴を安んじ集める役を称したものである〕。そこで降った胡はついにたがいに動揺し、十五部・二十余万人がみな反(そむ)き、先代の単于・屯屠何(とんとか)の子である薁鞮日逐王(いくていじつちくおう)の逢侯(ほうこう)を脅して立て、単于とした。そして官吏や民を殺し略奪し、郵亭(ゆうてい、宿駅)や廬帳(ろちょう、テント)を焼き払い、輜重(しちょう、荷駄)を引き連れて朔方(さくほう)へ向かい、幕北(ばくほく、ゴビ砂漠の北)へ渡ろうとした。九月、癸丑(みずのとうし)の日、光禄勳(こうろくくん)の鄧鴻(とうこう)に行車騎将軍事(=車騎将軍代行)を任じ、越騎校尉(えっきこうい)の馮柱(ふうちゅう)・行度遼将軍の朱徽とともに、左右の羽林(うりん)・北軍の五校(ごこう)の兵および郡国の迹射(せきしゃ)・辺境ぞいの兵を率いさせ〔注:李賢いわく、漢には迹射の士があり、(獣などの)足跡をたどって射ることをいう〕、烏桓校尉(うがんこうい)の任尚(じんしょう)には烏桓・鮮卑(せんぴ)あわせて四万人を率いさせて、これを討たせた。当時、南単于(=師子)と中郎将の杜崇は牧師城(ぼくしじょう)に駐屯しており〔注:漢の辺郡には牧師菀(ぼくしえん、牧場)があって馬を養った。これはその牧師菀の城で、西河郡の美稷県の境にあろう〕、逢侯は一万余騎を率いてこれを攻め囲んだ。冬、十一月、鄧鴻らが美稷に至ると、逢侯は囲みを解いて去り、満夷谷(まんいこく)へ向かった。南単于は子を遣わして一万騎を率いさせ、これに杜崇の率いる四千騎を加え、鄧鴻らとともに大城塞(だいじょうさい)で逢侯を追撃し〔注:大城県はもと西河郡に属し、『郡国志』では朔方郡に属する〕、四千余の首を斬った。任尚は鮮卑・烏桓を率いて満夷谷で逢侯を迎え撃ち、ふたたび大いにこれを破り、前後あわせて一万七千余の首を斬った。逢侯はとうとう部衆を率いて塞を出、漢兵は追いきれずに引き返した。

大司農(だいしのう)の陳寵(ちんちょう)を廷尉とした。陳寵は生まれつき情け深く哀れみがあり、たびたび疑わしい裁判を論じるにあたって、つねに経典(けいてん)に照らし、努めて寛大さに従ったので、(法を)むごく刻(きざ)む風潮は、これによっていくらか衰えた。

帝は尚書令(しょうしょれい)で江夏(こうか)の黄香(こうこう)を東郡(とうぐん)太守としようとしたが、黄香は辞退して言った。「郡を治め政(まつりごと)をとることは、私の才の及ぶところではありません。どうか朝廷に留めて宂官(じょうかん、閑職)の末に加えていただき〔注:宂(じょう)は散(さん)の意で、定員外の閑職〕、(人を)監督し責(せ)める小さな職をお与えのうえ、宮中や尚書台の煩雑な事務にお任せください」〔注:宮は宮中、臺は尚書台のこと。尚書は王命を出納するので、宮臺の煩事という〕。帝はそこでふたたび黄香を留めて尚書令とし、秩(ちつ)を二千石に増し〔注:『百官志』によれば、尚書令の秩は千石。いま特に秩を二千石に増したのは、黄香が尚書にあること久しく、また郡守への任を辞して拝さなかったので、ふたたび留めて尚書令とし、郡守の禄を給したのである〕、たいそう親しみ重んじられた。黄香もまた職務をつつしみ勤め、公(おおやけ)を憂えること(自分の)家のようにした。

六年
春，正月，皇甫稜免，以執金吾朱徽行度遼將軍。時單于與中郎將杜崇不相平，乃上書告崇；崇諷西河太守令斷單于章，單于無由自聞。崇因與朱徽上言：「南單于安國，疏遠故胡，親近新降，欲殺左賢王師子及左臺且渠劉利等；又，右部降者，謀共迫脅安國起兵背畔，請西河、上郡、安定爲之儆備。」帝下公卿議，皆以爲：「蠻夷反覆，雖難測知，然大兵聚會，必未敢動搖。今宜遣有方略使者之單于庭，與杜崇、朱徽及西河太守幷力，觀其動靜。如無他變，可令崇等就安國會其左右大臣，責其部衆橫暴爲邊害者，共平罪誅。若不從命，令爲權時方略，事畢之後，裁行賞賜，亦足以示百蠻。」於是徽、崇遂發兵造其庭。安國夜聞漢軍至，大驚，棄帳而去，因舉兵欲誅師子。師子先知，乃悉將廬落入曼柏城；安國追到城下，門閉，不得入。朱徽遣吏譬和之，安國不聽；城旣不下，乃引兵屯五原。崇、徽因發諸郡騎追赴之急，衆皆大恐，安國舅骨都侯喜爲等慮幷被誅，乃格殺安國，立師子爲亭獨尸逐侯鞮單于。
己卯，司徒丁鴻薨。
二月，丁未，以司空劉方爲司徒，太常張奮爲司空。
夏，五月，城陽懷王淑薨，無子，國除。
秋，七月，京師旱。
西域都護班超發龜茲、鄯善等八國兵合七萬餘人討焉耆，到其城下，誘焉耆王廣、尉犂王汎等於陳睦故城，斬之，傳首京師；因縱兵鈔掠，斬首五千餘級，獲生口萬五千人，更立焉耆左侯元孟爲焉耆王。超留焉耆半歲，慰撫之。於是西域五十餘國悉納質內屬，至于海濱，四萬里外，皆重譯貢獻。
南單于師子立，降胡五六百人夜襲師子，安集掾王恬將衞護士與戰，破之。於是降胡遂相驚動，十五部二十餘萬人皆反，脅立前單于屯屠何子薁鞮日逐王逢侯爲單于，遂殺略吏民，燔燒郵亭、廬帳，將車重向朔方，欲度幕北。九月，癸丑，以光祿勳鄧鴻行車騎將軍事，與越騎校尉馮柱、行度遼將軍朱徽將左右羽林、北軍五校士及郡國迹射、緣邊兵，烏桓校尉任尚將烏桓、鮮卑合四萬人討之。時南單于及中郎將杜崇屯牧師城，逢侯將萬餘騎攻圍之。冬，十一月，鄧鴻等至美稷，逢侯乃解圍去，向滿夷谷。南單于遣子將萬騎及杜崇所領四千騎，與鄧鴻等追擊逢侯於大城塞，斬首四千餘級。任尚率鮮卑、烏桓要擊逢侯於滿夷谷，復大破之，前後凡斬萬七千餘級。逢侯遂率衆出塞，漢兵不能追而還。
以大司農陳寵爲廷尉。寵性仁矜，數議疑獄，每附經典，務從寬恕，刻敝之風，於此少衰。
帝以尚書令江夏黃香爲東郡太守，香辭以：「典郡從政，才非所宜，乞留備宂官，賜以督責小職，任之宮臺煩事。」帝乃復留爲尚書令，增秩二千石，甚見親重。香亦祗勤物務，憂公如家。
