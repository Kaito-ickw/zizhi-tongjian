# 卷003 周紀三 — 愼靚王元年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 愼靚王元年 ・ 西暦: 320 BCE

[← 巻インデックス](README.md)

---

愼靚王(しんせいおう)元年〔注:辛丑(しんちゅう)の年、紀元前三二〇年。愼靚王は名を定(てい)といい、顯王の子である〕。

衞(えい)が、その称号をさらに格下げして「君(くん)」と称することになった〔注:衞はすでに顯王二十三年に称号を「侯」に下げていた。秦と魏のあいだに挟まれて国は日に日に削られ弱まっていったため、今またその号を引き下げて「君」としたのである〕。

---

<details>
<summary>原文を表示</summary>

愼靚王
元年
衞更貶號曰君。
