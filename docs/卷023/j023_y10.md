# 卷023 漢紀十五 — 孝昭皇帝上四年

四年（前七七年）

①春、正月、丁亥、帝が元服を加えた（成人の冠をつける儀礼を行った）〔注：「元服」の「元」は首（頭）の意で、冠は頭につけるものなので「元服」という〕。

②甲戌、富民定侯の田千秋が薨去した〔注：「民を安んじ深く慮る」を「定」という諡とする〕。当時、政務はすべて大将軍の霍光が決裁しており、千秋は丞相の地位にありながら、ただ慎み深く実直に身を保つばかりであった。

③夏、五月、丁丑、孝文帝の廟の正殿が火災に遭った〔注：人の起こした火を「火」という〕。帝と群臣はみな素服（白く質素な装い）をまとい、中二千石の官を動員し、五校（将作大匠配下の工兵部隊）を率いて修復にあたらせ〔注：五校令は五校の工兵を統べる〕、六日で完成した。太常および廟令・丞・郎・吏らは、みな大不敬の罪で弾劾された。たまたま大赦があり、太常で轑陽侯の徳は庶人に落とされた〔注：轑陽侯の食邑は清河にある。「轑」は「料」または「聊」の音〕。

④六月、天下に大赦を行った。

⑤かつて、杅冞（うび）国が太子の頼丹を人質として龜茲（きじ）に送っていた〔注：龜茲国の都は延城〕。貳師将軍が大宛を攻めて帰還した際〔注：この件は第二十一巻・武帝太初元年に見える〕、頼丹を連れて都（長安）に入った。霍光は桑弘羊のかつての建議を採用し、頼丹を校尉に任じ、兵を率いて輪臺で屯田させた〔注：桑弘羊が輪臺での屯田を建議した件は第二十二巻・征和元年に見える〕。龜茲の貴人・姑翼がその王に言った。「頼丹はもともと我が国に臣属していた身でありながら、今や漢の印綬を帯びてやって来て、我が国に迫って屯田しています。きっと害をなすに違いありません」。王はそこで頼丹を殺し、上書して漢に謝罪した。

楼蘭（ろうらん）王が死ぬと、匈奴がいち早くこれを聞きつけ、人質に取っていた楼蘭の王子・安帰を送り返し、安帰は王に立つことができた〔注：『西域伝』では名を「常帰」に作るが、ここでは『昭紀』および『傅介子伝』に従う〕。漢は使者を遣わして新王に入朝するよう詔したが、王は断って来なかった。楼蘭国は西域の最も東のはずれに位置し〔注：西域の東のはずれである〕、漢に近く、白龍堆（はくりゅうたい）に面していた〔注：白龍堆は土の龍のような形で、頭がなく尾があり、高いものは三、四丈、低いものでも一丈余り、いずれも東北を向いて連なっている〕。水も草も乏しいため、楼蘭は常に道案内を出し、水を背負い糧食を担いで漢の使者を送り迎えする役を担っていた。さらにたびたび漢の役人や兵士に略奪されて懲りており、漢と通じることを快く思わなかった〔注：「艾」は「乂（こりる）」と読む〕。その後はまた匈奴の反間（間諜）となり、たびたび漢の使者を遮って殺した。

楼蘭王の弟・尉屠耆（いとき）が漢に降り、その実情をくわしく報告した。駿馬監で北地出身の傅介子が大宛に使いした際〔注：駿馬監は太僕の属官。北地郡は涼州刺史に属する〕、詔によってついでに楼蘭と龜茲を譴責するよう命じられた。介子は楼蘭・龜茲に至り、それぞれの王を譴責すると、王たちはみな謝罪して服した。介子は大宛から帰る途中、龜茲に着いたところ、ちょうど烏孫から来た匈奴の使者が龜茲に滞在していた。介子はそこで自分の吏士を率い、ともに匈奴の使者を討ち斬った。帰還して事の次第を奏上すると、詔によって介子は中郎に任ぜられ、さらに平樂監に遷された〔注：平樂監は平樂観を管理する官〕。

介子は大将軍の霍光に言った。「楼蘭と龜茲はたびたび背反を繰り返しているのに、誅伐しなければ、懲らしめになりません。私が龜茲を通ったとき、その王は人に気安く近寄ってくるので、容易に討てそうでした。ぜひ行って刺し殺し、その威を諸国に示したいのです」。大将軍は言った。「龜茲は道のりが遠い。まずは楼蘭で試してみよ」。そこで霍光はこれを天子に申し上げて介子を派遣させた。介子は士卒とともに黄金や絹織物を携え、「外国に下賜するため」という名目を声高に唱えて楼蘭に至った。楼蘭王は介子に親しもうとする気がなかったので、介子はうわべは立ち去るふりをし、楼蘭の西の境まで来ると、通訳に命じてこう言わせた〔注：この通訳は楼蘭国の通訳である〕。「漢の使者は黄金や錦繍を携え、諸国に下賜して回っているのだ。王が受け取りに来ないなら、私は西方の国へ行ってしまうぞ」。そして金や絹を取り出して通訳に見せた。通訳が戻って王に報告すると、王は漢の品々を欲しがり、使者に会いに来た。

介子は王と並んで酒を酌み交わし、品々を並べて見せ、酒を飲んで一同みな酔った。介子は王に言った。「天子が私に、王へ内密にお伝えすることを託されました」〔注：ひそかに語るべきことがあるということ〕。王が立ち上がり、介子に従って天幕の中に入り、人払いをして二人だけで語ろうとすると〔注：人を遠ざけて二人きりで語ること〕、壮士二人が背後から王を刺し、刃が胸で交わって、王はその場で絶命した〔注：「匈」は「胸」に同じ〕。王の貴臣や側近たちはみな散り散りに逃げ去った。介子は人々に王が漢に背いた罪を告げ諭して言った。「天子は私を遣わして王を誅させた。これからは漢にいる王の弟・尉屠耆を新たな王に立てる。漢の軍がまさに到着しようとしている。みだりに動くな。動けば自ら国を滅ぼすことになるぞ」。介子はそのまま王・安帰の首を斬り、駅伝の早馬を駆って都へ送り、その首を北闕の下に懸けた〔注：「縣」は古くは「懸」の字に通じる〕。

そして尉屠耆を王に立て、その国の名を鄯善（ぜんぜん）と改め、新たに印章を刻んで与えた。また宮女を賜って夫人とし、車馬や輜重（物資輸送の車）を整えてやった〔注：「鄯」は「上扇」の音〕。丞相は百官を率いて横門の外まで見送り、旅立ちの祭りである祖道を行って彼を送り出した〔注：横門は長安城の北面、西寄りの第一の門。「祖」は祖道（旅立ちの祭り）のこと〕。王（尉屠耆）は自ら天子に願い出て言った。「私は長く漢に滞在しており、今帰国しても後ろ盾が乏しく孤立しています。しかも前王の子が国に残っており、その者に殺されはしないかと恐れております。国内に伊循城という土地があり、そこは地味が肥沃です。どうか漢が一人の将軍を派遣して屯田し穀物を蓄えてくださいますよう。そうすれば私はその威光を頼みにできます」。そこで漢は司馬一人と吏士四十人を遣わし、伊循で屯田させて鄯善を鎮め安んじた〔注：「塡」は「鎭（しずめる）」と読む〕。

秋、七月、乙巳、范明友を平陵侯に封じ〔注：烏桓を破った功を賞したもの。平陵侯の食邑は南陽の武當にある〕、傅介子を義陽侯に封じた〔注：義陽侯の食邑は南陽の平氏にある〕。

【臣光曰（司馬光の論評）】
王者が戎狄（異民族）に対するときは、背けば討ち、服従すれば赦して放っておくものだ〔注：「舍」は「捨（ゆるす・そのままにする）」と読む〕。今、楼蘭王はすでにその罪を認めて服したのに、それをさらに追って誅殺してしまった。これでは後に背く者が出ても、もはや手なずけて心服させることはできまい。どうしても罪ありとして討つというのなら、軍を整え将兵に布告し〔注：「鞠」は告げること。開戦の日に軍勢を並べ、誓いを立てて告げるのである〕、堂々と罰を加えるべきであった。それを今、使者を遣わして金や絹で誘い出して殺すとは。これでは後に諸国へ使いする者が出ても、はたして信用されようか。そもそも大いなる漢の強大さをもってして、蛮夷に対し盗賊まがいの謀略を弄するとは、なんとも恥ずべきことではないか。論者の中には介子を称えて「奇功（目覚ましい手柄）」とする者もいるが、それは誤りである。

四年
春，正月，丁亥，帝加元服。
甲戌，富民定侯田千秋薨。時政事壹決大將軍光；千秋居丞相位，謹厚自守而已。
夏，五月，丁丑，孝文廟正殿火。上及羣臣皆素服，發中二千石將五校作治，六日，成。太常及廟令丞、郎、吏，皆劾大不敬；會赦，太常轑陽侯德免爲庶人。
六月，赦天下。
初，杅冞遣太子賴丹爲質於龜茲；貳師擊大宛還，將賴丹入至京師。霍光用桑弘羊前議，以賴丹爲校尉，將軍田輪臺。龜茲貴人姑翼謂其王曰：「賴丹本臣屬吾國，今佩漢印綬來，迫吾國而田，必爲害。」王卽殺賴丹而上書謝漢。
樓蘭王死，匈奴先聞之，遣其質子安歸歸，得立爲王。漢遣使詔新王令入朝，王辭不至。樓蘭國最在東垂，近漢，當白龍堆，乏水草，常主發導，負水擔糧，送迎漢使；又數爲吏卒所寇，懲艾，不便與漢通。後復爲匈奴反間，數遮殺漢使。其弟尉屠耆降漢，具言狀。駿馬監北地傅介子使大宛，詔因令責樓蘭、龜茲。介子至樓蘭、龜茲，責其王，皆謝服。介子從大宛還，到龜茲，會匈奴使從烏孫，在龜茲，介子因率其吏士共誅斬匈奴使者。還，奏事，詔拜介子爲中郎，遷平樂監。
介子謂大將軍霍光曰：「樓蘭、龜茲數反覆，而不誅，無所懲艾。介子過龜茲時，其王近就人，易得也；願往刺之以威示諸國！」大將軍曰：「龜茲道遠，且驗之於樓蘭。」於是白遣之。介子與士卒俱齎金幣，揚言以賜外國爲名，至樓蘭。樓蘭王意不親介子，介子陽引去，至其西界，使譯謂曰：「漢使者持黃金、錦繡行賜諸國。王不來受，我去之西國矣。」卽出金、幣以示譯。譯還報王，王貪漢物，來見使者。介子與坐飲，陳物示之，飲酒皆醉。介子謂王曰：「天子使我私報王。」王起，隨介子入帳中屛語，壯士二人從後刺之，刃交匈，立死；其貴臣、左右皆散走。介子告諭以王負漢罪，「天子遣我誅王，當更立王弟尉屠耆在漢者。漢兵方至，毋敢動，自令滅國矣！」介子遂斬王安歸首，馳傳詣闕，縣首北闕下。
乃立尉屠耆爲王，更名其國爲鄯善，爲刻印章；賜以宮女爲夫人，備車騎、輜重。丞相率百官送至橫門外，祖而遣之。王自請天子曰：「身在漢久，今歸單弱，而前王有子在，恐爲所殺。國中有伊循城，其地肥美，願漢遣一將屯田積穀，令臣得依其威重。」於是漢遣司馬一人、吏士四十人田伊循以塡撫之。
秋，七月，乙巳，封范明友爲平陵侯，傅介子爲義陽侯。
臣光曰：王者之於戎狄，叛則討之，服則舍之。今樓蘭王旣服其罪，又從而誅之，後有叛者，不可得而懷矣。必以爲有罪而討之，則宜陳師鞠旅，明致其罰。今乃遣使者誘以金幣而殺之，後有奉使諸國者，復可信乎！且以大漢之強而爲盜賊之謀於蠻夷，不亦可羞哉！論者或美介子以爲奇功，過矣！
