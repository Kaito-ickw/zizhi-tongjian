# 卷002 周紀二 — 顯王三十九年

> 巻 2 / 294 ・ 周紀二 ・ 年号: 顯王三十九年 ・ 西暦: 330 BCE

[← 巻インデックス](README.md)

顯王三十九年〔注:辛卯(かのとう)の年、紀元前三三〇年〕。

秦が魏を攻め、焦(しょう)と曲沃(きょくよく)を包囲した〔注:いずれも弘農郡(こうのうぐん)陜県(せんけん)あたりの地〕。魏は少梁(しょうりょう)と河西(かせい)の地を秦に引き渡した〔注:顯王二十九年に魏はすでに河西を秦に献じて和睦していたが、ここでようやく実際にその地を割き渡した〕。

---

<details>
<summary>原文を表示</summary>

三十九年
秦伐魏，圍焦、曲沃。魏入少梁、河西地於秦。
