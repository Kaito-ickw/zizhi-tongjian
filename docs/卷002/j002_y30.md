# 卷002 周紀二 — 顯王三十七年

> 巻 2 / 294 ・ 周紀二 ・ 年号: 顯王三十七年 ・ 西暦: 332 BCE

顯王三十七年〔注:己丑(つちのとうし)の年、紀元前三三二年〕。

秦の惠王(けいおう)が犀首(さいしゅ)を遣わして齊(せい)・魏(ぎ)を欺き、彼らと結んでともに趙(ちょう)を攻め、合従(がっしょう)の盟約を破ろうとした。趙の肅侯(しゅくこう)が蘇秦(そしん)を責めると、蘇秦は恐れ、燕(えん)への使者となることを願い出て、「必ず齊に仕返ししてみせます」と約した。蘇秦が趙を去ると、合従の盟約はすべて解けてしまった。趙の人々は黄河の水を決壊させて齊・魏の軍に注ぎ込み、齊・魏の軍はそこでようやく退いた。

魏は陰晉(いんしん)を割いて秦と和睦した。これは実は華陰(かいん)のことである〔注:華陰はもとの陰晉。秦の惠文王五年に寧秦(ねいしん)と改名し、漢の高帝のとき華陰県と改めた。華山の北(陰)にあるのでこの名がある〕。

齊王が燕を攻め、十城を奪ったが、まもなくこれを返した。

三十七年
秦惠王使犀首欺齊、魏，與共伐趙，以敗從約，趙肅侯讓蘇秦，蘇秦恐，請使燕，必報齊。蘇秦去趙而從約皆解。趙人決河水以灌齊、魏之師，齊、魏之師乃去。
魏以陰晉爲和於秦，實華陰。
齊王伐燕，取十城；已而復歸之。
