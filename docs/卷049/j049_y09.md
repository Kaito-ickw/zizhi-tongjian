# 卷049 漢紀四十一 — 孝安皇帝上元初元初元年

元初元年〔注:甲寅(きのえとら)の年、紀元一一四年〕。

春、正月、甲子(きのえね)の日、改元した(元号を「元初(げんしょ)」と改めた)。

二月、乙卯(きのとう)の日、日南(にちなん)郡で地が裂け、その裂け目は百里あまりにも及んだ〔注:『東観記(とうかんき)』によれば、裂け目は長さ百八十二里、幅五十六里に達したという〕。

三月、癸亥(みずのとい)の日、日食があった。

詔(みことのり)を下して兵を遣わし、河内(かだい)の谷あいを抜ける要衝(ようしょう、軍事上の要所)三十六か所に屯営(とんえい)させ、いずれにも塢壁(おへき、土塁をめぐらした砦)を築かせ、鳴鼓(めいこ、合図の太鼓)を設けて、羌(きょう、西方の異民族)の侵攻に備えさせた〔注:太行(たいこう)山脈の北から恒山(こうざん)にかけては并州(へいしゅう)と冀州(きしゅう)を隔てているが、その間には谷づたいの道が多く通じている。そこでいま、要衝の地に砦を築いて羌の侵入に備えたのである〕。

夏、四月、丁酉(ひのととり)の日、天下に大赦(たいしゃ)を行った。

京師(けいし、みやこ)と五つの郡国で旱(ひでり)が起こり、蝗(いなご)の害があった。

五月、先零(せんれい)の羌が雍城(ようじょう、右扶風の雍県の城)を侵した。

蜀(しょく)郡の夷(い、異民族)が蠶陵(さんりょう)を侵し、県令(けんれい、県の長官)を殺した。

九月、乙丑(きのとうし)の日、太尉(たいい)の李脩(りしゅう)が罷免(ひめん)された。

羌の豪族(ごうぞく)である號多(ごうた)が、もろもろの部族とともに武都(ぶと)・漢中(かんちゅう)・巴郡(はぐん)を荒らし略奪した。板楯蛮(はんじゅんばん、巴の地に住む勇猛な異民族)が救援に駆けつけ〔注:板楯蛮は、かつて白虎を射殺した功により租税を免じられた巴の人々で、生まれつき強く勇ましく、たびたび敵陣を突き破ったという〕、漢中の五官掾(ごかんえん、郡の属官)である程信(ていしん)が郡兵を率い、この蛮人らとともに賊を撃ち破った。號多は逃げ帰り、隴道(ろうどう、隴山を越える道)を断ち切って、零昌(れいしょう)と合流した。侯霸(こうは)・馬賢(ばけん)がこれと枹罕(ふかん)で戦い、撃ち破った。

辛未(かのとひつじ)の日、大司農(だいしのう、農政をつかさどる長官)である山陽(さんよう)の司馬苞(しばほう)を太尉とした。

冬、十月、戊子(つちのえね)の朔(ついたち)、日食があった。

涼州刺史(りょうしゅうしし、涼州の監察長官)の皮楊(ひよう)が羌を狄道(てきどう)で撃ったが、大敗し、戦死者は八百人あまりに上った。

この年、十五の郡国で地震があった。

元初元年
春，正月，甲子，改元。
二月，乙卯，日南地坼，長百餘里。
三月，癸亥，日有食之。
詔遣兵屯河內通谷衝要三十六所，皆作塢壁，設鳴鼓，以備羌寇。
夏，四月，丁酉，赦天下。
京師及郡國五旱，蝗。
五月，先零羌寇雍城。
蜀郡夷寇蠶陵，殺縣令。
九月，乙丑，太尉李脩罷。
羌豪號多與諸種鈔掠武都、漢中、巴郡，板楯蠻救之，漢中五官掾程信率郡兵與蠻共擊破之。號多走還，斷隴道，與零昌合，侯霸、馬賢與戰於枹罕，破之。
辛未，以大司農山陽司馬苞爲太尉。
冬，十月，戊子朔，日有食之。
涼州刺史皮楊擊羌於狄道，大敗，死者八百餘人。
是歲，郡國十五地震。
