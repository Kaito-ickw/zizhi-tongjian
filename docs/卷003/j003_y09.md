# 卷003 周紀三 — 赧王上三年

三年〔注:己酉(つちのととり)の年、紀元前三一二年〕。

春、秦の軍が楚の軍と丹陽(たんよう)で戦い〔注:この丹陽は丹水(たんすい)の北岸をいう〕、楚の軍は大敗した。秦は甲冑をつけた兵八万を斬り、屈匄(くつかい)を捕らえ、さらに列侯(れっこう)や執圭(しっけい)〔注:執圭は楚の爵位で、圭を手にして君主に拝謁する者である〕の位にある者七十余人を捕虜とし、そのまま漢中郡(かんちゅうぐん)を奪い取った。楚王は国内の兵をすべて動員してふたたび秦を襲い、藍田(らんでん)で戦ったが〔注:藍田は秦の地で、楚はここまで深く攻め入った〕、楚の軍はまた大敗した。韓と魏は楚が苦境にあると聞き、南下して楚を襲い、鄧(とう)にまで攻め込んだ。楚の人々はこれを聞くと、兵を引いて帰り、二つの城を割いて秦に和睦を願い出た。

燕の人々はともに太子の平(へい)を立てて君主とした。これが昭王(しょうおう)である。昭王は燕が破られたあと、死者を弔い遺児を見舞い、民と苦楽をともにし、身を低くして手厚い礼物を用意し、賢者を招き寄せようとした。そして郭隗(かくかい)に言った。「齊はわが国の内乱に乗じて燕を襲い破った。わたしは燕が小さく力が乏しく、報復するには足りないことをよく承知している。だが、まことに賢士を得てともに国を治め、先王の恥をすすぐことができれば、それがわたしの願いだ。先生はふさわしい人物を見つけてくれ。わたしはみずからその者に仕えよう。」郭隗が言った。「むかし、ある君主が涓人(けんじん)〔注:涓人とは、君主の身辺で掃除など清潔の用を司る、近しい側近である〕に千金を持たせて千里の馬を求めさせたところ、馬はすでに死んでいたのに、その首を五百金で買って帰ってきました。君主が激怒すると、涓人は言いました。『死んだ馬でさえ買うのです。まして生きた馬ならなおさら。馬はじきに参りましょう。』はたして一年とたたぬうちに、千里の馬が三頭も集まりました。いま王がどうしても士を招きたいとお望みなら、まずこの隗(かい)から始めてください。わたしより賢い者であれば、千里の道を遠いと思って来ないことなどありましょうか。」そこで昭王は隗のために新たに屋敷を築き直し、これを師として仕えた。すると士たちは先を争って燕に向かった。樂毅(がくき)は魏から、劇辛(げきしん)は趙からやって来た。昭王は樂毅を亞卿(あけい)に任じ、国政をゆだねた。

韓の宣惠王(せんけいおう)が薨(こう)じ、子の襄王(じょうおう)倉(そう)が立った。

三年
春，秦師及楚戰于丹陽，楚師大敗；斬甲士八萬，虜屈匄及列侯、執七十餘人，遂取漢中郡。楚王悉發國內兵以復襲秦，戰於藍田，楚師大敗。韓、魏聞楚之困，南襲楚，至鄧。楚人聞之，乃引兵歸，割兩城以請平于秦。
燕人共立太子平，是爲昭王。昭王於破燕之後，弔死問孤，與百姓同甘苦，卑身厚幣以招賢者。謂郭隗曰：「齊因孤之國亂而襲破燕，孤極知燕小力少，不足以報；然誠得賢士與共國，以雪先王之恥，孤之願也。先生視可者，得身事之！」郭隗曰：「古之人君有以千金使涓人求千里馬者，馬已死，買其首五百金而返。君大怒，涓人曰：『死馬且買之，況生者乎！馬今至矣。』不期年，千里之馬至者三。今王必欲致士，先從隗始，況賢於隗者，豈遠千里哉！」於是昭王爲隗改築宮而師事之。於是士爭趣燕：樂毅自魏往，劇辛自趙往。昭王以樂毅爲亞卿，任以國政。
韓宣惠王薨，子襄王倉立。
