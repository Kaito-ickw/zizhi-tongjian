# 卷002 周紀二 — 顯王三年

顯王三年〔注:乙卯(いつぼう)の年、紀元前三六六年〕。

魏と韓が宅陽(たくよう)で会合した。

秦が魏軍と韓軍を洛陽(らくよう)で打ち破った。

---

<details>
<summary>原文を表示</summary>

三年
魏、韓會于宅陽。
秦敗魏師、韓師于洛陽。
