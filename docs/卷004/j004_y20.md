# 卷004 周紀四 — 赧王中三十七年

赧王三十七年〔注:癸未(きび)の年、紀元前二七八年〕。

秦の大良造(だいりょうぞう)である白起(はくき)が楚を攻め、その都の郢(えい)を攻め落とし、夷陵(いりょう)を焼いた。楚の襄王(じょうおう)の軍勢はちりぢりになり、ついに二度と戦うことなく、都を東北の陳(ちん)へ移した〔注:陳は、いにしえの陳国(ちんこく)のあった地である〕。秦は郢を南郡(なんぐん)とし、白起を武安君(ぶあんくん)に封じた〔注:武安という号は、よく軍士を養い、戦えば必ず勝ち、民を安んじ集めることができる、という意味で付けられたものである〕。

三十七年
秦大良造白起伐楚，拔郢，燒夷陵。楚襄王兵散，遂不復戰，東北徙都於陳。秦以郢爲南郡，封白起爲武安君。
