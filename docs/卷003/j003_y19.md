# 卷003 周紀三 — 赧王上十三年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上十三年 ・ 西暦: 302 BCE

[← 巻インデックス](README.md)

---

十三年〔注:己未(つちのとひつじ)の年、紀元前三〇二年〕。

①秦王・魏王と韓の太子嬰(えい)が臨晉(りんしん)で会した。韓の太子は咸陽(かんよう)まで赴いてから帰った。秦は魏に蒲阪(ほはん)を返した〔注:去年、秦は魏の蒲阪を取っていた〕。

②秦の大夫(たいふ)に、ひそかに楚の太子と争った者がいた。太子はこれを殺し、逃げ帰った。

十三年
秦王、魏王、韓太子嬰會于臨晉，韓太子至咸陽而歸；秦復與魏蒲阪。
秦大夫有私與楚太子鬬者，太子殺之，亡歸。
