# 卷001 周紀一 — 安王十六年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王十六年 ・ 西暦: 386 BCE

[← 巻インデックス](README.md)

安王十六年〔注:乙未(いつび)の年、紀元前三八六年〕。

はじめて斉の大夫である田和(でんか)を、正式な諸侯に取り立てた〔注:田氏は、これより斉の国を完全にわが物とした。田和が太公(たいこう)である〕。

趙の公子・朝(ちょう)が反乱を起こし、魏へ亡命した。そして魏とともに邯鄲(かんたん)を襲ったが、攻め落とせなかった。

---

<details>
<summary>原文を表示</summary>

十六年
初命齊大夫田和爲諸侯。
趙公子朝作亂，奔魏；與魏襲邯鄲，不克。

</details>
