# 卷002 周紀二 — 顯王十九年

顯王十九年〔注:辛未(しんび)の年、紀元前三五〇年〕。

秦の商鞅(しょうおう)が咸陽(かんよう)に冀闕(きけつ)〔注:宮門の左右に対になって立つ物見やぐら〕と宮殿・朝廷を築き、そこへ都を移した。民のうち、父子・兄弟が同じ部屋で起き伏しする習わしを禁じた。いくつもの小さな村里をまとめて一つの縣(けん)とし、縣ごとに令(れい)と丞(じょう)を置いて、全部で三十一縣とした。井田(せいでん)の制を廃止し、田畑のあぜ道(阡陌(せんぱく))を開いた〔注:南北の道を阡、東西の道を陌という〕。斗(と)・桶(よう)・権(けん)・衡(こう)・丈(じょう)・尺(しゃく)といった升目や重さ・長さの基準を統一した。

秦と魏が彤(とう)で会った。

趙の成侯(せいこう)が亡くなり、公子の緤(せつ)が太子と君位を争った。緤は敗れて韓へ逃げ込んだ。

十九年
秦商鞅築冀闕宮庭於咸陽，徙都之。令民父子、兄弟同室內息者爲禁。幷諸小鄕聚，集爲一縣，縣置令、丞，凡三十一縣。廢井田，開阡陌。平斗、桶、權、衡、丈、尺。
秦、魏遇于彤。
趙成侯薨，公子緤與太子爭立；緤敗，奔韓。
