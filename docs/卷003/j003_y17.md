# 卷003 周紀三 — 赧王上十一年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上十一年 ・ 西暦: 304 BCE

[← 巻インデックス](README.md)

十一年〔注:丁巳(ひのとみ)の年、紀元前三〇四年〕。

①秦王と楚王が黃棘(こうきょく)で盟(ちか)いを結んだ。秦は楚に上庸(じょうよう)を返した〔注:三年前、秦は楚軍を破って屈匄(くつかい)を捕らえ、楚の上庸を奪っていた〕。

---

<details>
<summary>原文を表示</summary>

十一年
秦王、楚王盟于黃棘；秦復與楚上庸。
