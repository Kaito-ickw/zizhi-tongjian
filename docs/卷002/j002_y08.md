# 卷002 周紀二 — 顯王十一年

顯王十一年〔注:癸亥(きがい)の年、紀元前三五八年〕。

秦が韓の軍を西山(せいざん)〔注:宜陽(ぎよう)の熊耳(ゆうじ)山から東は嵩高(すうこう)山に連なり、南は魯陽(ろよう)に至るまでの一帯で、いずれも韓の西方の山地をいう〕で打ち破った。

十一年
秦敗韓師于西山。
