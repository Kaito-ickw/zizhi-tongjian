# 卷006 秦紀一 — 始皇帝上九年

九年〔注:癸亥(きがい)の年、紀元前二三八年〕。

秦は魏を攻め、垣(えん)と蒲(ほ)を取った。夏四月、寒波が来て凍死する民が出た。王は雍(よう)に一泊した。己酉(きゆう)の日、王は冠礼(かんれい)を行い〔注:成人の式のこと〕、剣を帯びた。楊端和(ようたんわ)が魏を伐(う)ち、衍氏(えんし)を取った〔注:衍氏は鄭州にあった〕。

そもそもの経緯はこうである。王が即位したころはまだ年少であり、太后(たいこう)はそのころ文信侯(ぶんしんこう=呂不韋)と密かに通じていた。王がしだいに成長するにつれ、文信侯は事が露見して自分に禍がふりかかることを恐れ、舎人(しゃじん)の嫪毐(ろうあい)を偽りの宦官(かんがん)として太后のもとに送り込んだ。太后は毐を寵愛し、二人の子をもうけた。毐は長信侯(ちょうしんこう)に封じられ、太原(たいげん)が毐の領国(りょうごく)とされ、政事はすべて毐が決するようになった。毐の食客になろうとする者がおびただしく集まった。王の側近に毐と口論した者があり、毐が実は宦官でないことを告げると、王は官吏に命じて毐を取り調べさせた。毐は恐れをなし、王の御璽(ぎょじ)を偽って兵を発し、蘄年宮(きねんきゅう)〔注:秦の恵公が造った宮殿で雍にある〕を攻めて乱を起こそうとした。王は相国(しょうこく)の昌平君(しょうへいくん)・昌文君(しょうぶんくん)に命じて兵を出させて毐を攻め、咸陽(かんよう)で戦って数百の首を斬り、毐は敗れて逃げたが捕らえられた。秋九月、毐の三族(さんぞく)を夷滅(いめつ)し〔注:三族とは父母・兄弟・妻子のこと〕、一党はみな車裂(くるまざき)にして一族を根絶やしにした。舎人で罪の軽い者は蜀(しょく)に流され、合わせて四千余家に及んだ。太后は雍の萯陽宮(ほいようきゅう)〔注:秦の文王が造った宮殿〕に移され、その二子は殺された。王はまた令を下した。「太后のことについて諫(いさ)める者があれば、戮(りく)して殺し、四肢を断ち切って宮門の前に積み重ねよ。」これにより死者は二十七人に上った。

斉の客人、茅焦(もうしょう)が名を通じて諫めを申し出た〔注:「上謁」とは、名を通じて面会を求めること〕。王は使者を遣わして言った。「お前は宮門の前に積み重なっているものが見えないのか。」茅焦は答えた。「臣が聞くところでは、天には二十八宿(にじゅうはっしゅく)があります〔注:角・亢・氐など二十八の星宿〕。いま死者は二十七人、臣が参ったのはその数を満たすためです。臣は死を恐れる者ではありません。」使者が急いで王に報告すると、茅焦の同郷で同じ食卓の者たちは、みな衣物をかついで逃げ出した。王は大いに怒って言った。「この男はわざわざ余に逆らいに来たか。さっさと呼んで大釜(おおがま)で煮てしまえ〔注:鑊(かく)はいわゆる鍋のこと〕。宮門の前に積み重ねるどころではないわ。」王は剣の柄(つか)を握って座り、口からは涎(よだれ)が垂れていた〔注:沫(まつ)は涎のこと〕。使者が茅焦を呼び入れると、茅焦はゆっくりと進み出て、再び拝礼してから立ち上がり、こう申し上げた。「臣が聞くところでは、命を大切にする者は死を忌まず、国を持つ者は亡国を忌みません。死を忌む者は生を得ることができず、亡国を忌む者は国を保てません。死生存亡は、聖主がすぐにも聞きたいと望まれることです。陛下〔注:「陛下」は、天子に直接呼びかけることをはばかって、階段の下に控える者に向けて奏上する形式から生まれた尊称〕、お聞きになりますか。」王が「何が言いたいのか」と問うと、茅焦は言った。「陛下には狂悖(きょうはい)の行いがございますが、ご自身では気づいておられないのでしょうか。仮父(かふ=嫪毐)〔注:嫪毐を指す〕を車裂きにし、二人の弟を袋に詰めて打ち殺し〔注:袋に人を入れて打ち叩いて殺すこと〕、母を雍に追い、諫める者を残虐に殺す——桀(けつ)・紂(ちゅう)の行いもここまでは至りませんでした。いま天下の人々がこれを聞いて、ことごとく心が離れ、秦に向く者がいなくなっております。臣は密かに陛下のためを危(あや)ぶんでおります。臣の言はこれまでです。」茅焦は衣を脱いで処刑台の前に伏した〔注:質(鑕)とは鉄の刃台のこと〕。王は殿を下り、みずから手をとって言った。「先生、どうか起き上がって衣(ころも)をつけてください。今こそ先生のお教えを賜りたい。」王は茅焦を上卿(じょうけい)に取り立てた。そして王みずから車を御(ぎょ)し、左の席を空けて太后を迎えに行き、咸陽に連れ帰った。母と子は元のような関係に戻った。

楚(そ)の考烈王(こうれつおう)には子がなく、春申君はそれを憂えて、子をもうけやすい女性を多く求めて王に献上したが、結局子はできなかった。趙の人、李園(りえん)は妹を楚王に差し上げようとしたが、楚王に子ができにくいと聞いて、長く寵愛を受けられないことを恐れた。そこで春申君の食客になることを求めた。やがて暇を申し出て帰ろうとしたが〔注:謝暇して帰省すること〕、わざと期限に遅れて戻った〔注:春申君に問わせるための仕掛けであった〕。春申君が問うと、李園は言った。「斉王が人を遣わして臣の妹を求め、その使者と共に飲んでいたため、期限を過ぎてしまいました。」春申君が「正式な聘礼(へいれい)は入りましたか」と問うと〔注:婚礼の財物が届けられたかを問うもの〕、「まだです」との答えだった。春申君はそこで李園の妹を自らの妾として納めた。やがて妊娠した。李園は妹に春申君を説かせてこう言わせた。「楚王があなたを愛して大切にする心は、ご兄弟をしのぐほどです。あなたは楚の宰相を二十余年お続けです〔注:周赧王五十三年に春申君が楚の相になってよりここまで二十余年〕。それなのに王には子がおられません。王が亡くなれば〔注:「百歳後」とは死後のこと〕、兄弟の誰かが後を継ぎ、その方もそれぞれ昔からの親しい者を重んじるでしょう。あなたはどうしてこの寵愛を長く保てましょうか。それだけではありません〔注:それだけに止まらないということ〕。あなたは地位が高く久しく権勢をふるってきた分、王の兄弟方に失礼を重ねたことも多い。その方が即位すれば、禍があなた自身に及ぶでしょう。いま妾(わらわ)には子がありますが、まだ誰も知りません。妾があなたの寵を受けてまだ日が浅い今のうちに、あなたの威光で妾を楚王に差し上げれば、王は必ず妾を寵愛されるでしょう。そして天の恵みで男の子が生まれれば、それはあなたの子が王になるということ。楚の国はそっくりあなたのものになります。わが身に降りかかる予測もつかぬ禍と引き換えにできましょうか。」春申君はこれをもっともと思い、李園の妹をそっと別に設けた館舎(やかたや)に住まわせ〔注:謹舍とは別に館舎を設けて丁重に奉仕警護すること〕、楚王に話をつけた。王が妹を召し入れて寵愛すると、やがて男の子が生まれ、太子に立てられた。

李園の妹は王后(おうこう)となり、李園も権勢を振るうようになった。しかし李園は春申君がこの秘密を漏らすことを恐れ、ひそかに死士(しし=命知らずの刺客)を養い、春申君を殺して口を封じようとしていた。国人の中にこれを知る者も少なくなかった。楚王が病気になると、朱英(しゅえい)が春申君に言った。「世には思いがけない幸運があり、また思いがけない禍もあります〔注:無望とは、望まないのに突然やって来ること〕。いまあなたは思いがけないことの多い世〔注:生死が無常であること〕にあって、思いがけないことの多い君主〔注:喜怒の節度がないこと〕に仕えておられる。どうして思いがけないことをしてくれる人なしで済ませられましょうか〔注:吉凶がにわかに現れること〕。」春申君は「思いがけない幸運とは何ですか」と問うた。朱英は答えた。「あなたは楚の宰相を二十余年お務めです。名は宰相ながら実は王も同然。王がいまご病気で、いつ薨じてもおかしくない。薨じれば、あなたが幼い君主を補佐して国政を担い、君主が成長して政権を返す時が来ても、そのまま南面(なんめん)して一国の主と称するに至るかもしれません。これが思いがけない幸運というものです。」「では思いがけない禍とは何ですか。」「李園は国政を担ってはいないがあなたの仇敵です。兵を持たずして死士を養うこと久しい。王が薨じれば、李園は必ず先に入って権を握り、あなたを殺して口を封じるでしょう。これが思いがけない禍というものです。」「思いがけないことをしてくれる人とは何ですか。」「あなたが臣を郎中(ろうちゅう)〔注:宮門の守備に当たる近衛の官〕にお任じください。王が薨じて李園が先に入ろうとするとき、臣があなたのために李園を殺します。これが思いがけないことをしてくれる人というものです。」春申君は答えた。「先生、それは置いておいてください。李園は気の弱い人物で、私とも親しくしています。どうしてそんなことになりましょうか。」朱英は自分の言が用いられないと知り、身の危険を感じて逃げ去った。それから十七日後、楚王が薨じた。李園はやはり先に入り、棘門(きょくもん)〔注:壽春(じゅしゅん)の城門の名〕の内側に死士を潜ませていた。春申君が入ろうとすると、死士たちが両側から斬りかかり、その首を棘門の外に投げ捨てた。そうして役人たちに春申君の一族をすべて捕らえて誅滅させた。太子が即位し、これを幽王(ゆうおう)という。

〔注:揚子(ようし)の『法言(ほうげん)』は言う。「信陵君・平原君・孟嘗君・春申君の四君は世の益となったかと問われれば」、「君主が政を失い、奸臣が国命を盗んだのだ。なんの益になりましょうか」〕

秦王は、文信侯(ぶんしんこう=呂不韋)が先王に大きな功績があったことを思い、誅殺するに忍びなかった〔注:その事跡は前巻の周赧王五十八年に見える〕。

九年
伐魏，取垣、蒲。
夏，四月，寒，民有凍死者。
王宿雍。
己酉，王冠，帶劍。
楊端和伐魏，取衍氏。
初，王卽位，年少，太后□時與文信侯私通。王益壯，文信侯恐事覺，禍及己，乃詐以舍人嫪毐爲宦者，進於太后。太后幸之，生二子，封毐爲長信侯，以太原爲毐國，政事皆決於毐；客求爲毐舍人者甚衆。王左右有與毐爭言者，告毐實非宦者，王下吏治毐。毐懼，矯王御璽發兵，欲攻蘄年宮爲亂。王使相國昌平君、昌文君發卒攻毐，戰咸陽，斬首數百，毐敗走，獲之。秋，九月，夷毐三族，黨與皆車裂滅宗；舍人罪輕者徙蜀，凡四千餘家。遷太后於雍萯陽宮，殺其二子。下令曰︰「敢以太后事諫者，戮而殺之，斷其四支，積於闕下！」死者二十七人。齊客茅焦上謁請諫。王使謂之曰︰「若不見夫積闕下者邪？」對曰︰「臣聞天有二十八宿，今死者二十七人，臣之來固欲滿其數耳，臣非畏死者也！」使者走入白之。茅焦邑子同食者，盡負其衣物而逃。王大怒曰︰「是人也，故來犯吾，趣召鑊烹之，是安得積厥下哉！」王按劍而坐，口正沫出。使者召之入，茅焦徐行至前，再拜謁起，稱曰︰「臣聞有生者不諱死，有國者不諱亡，諱死者不可以得生，諱亡者不可以得存。死生存亡，聖主所欲急聞也，陛下欲聞之乎！」王曰︰「何謂也？」茅焦曰︰「陛下有狂悖之行，不自知邪？車裂假父，囊撲二弟，遷母於雍，殘戮諫士；桀、紂之行不至於是矣！今天下聞之，盡瓦解，無嚮秦者，臣竊爲陛下危之！臣言已矣！」乃解衣伏質。王下殿，手自接之曰︰「先生起就衣，今願受事！」乃爵之上卿。王自駕，虛左方，往迎太后，歸於咸陽，復爲母子如初。
楚考烈王無子，春申君患之，求婦人宜子者甚衆，進之，卒無子。趙人李園持其妹欲進諸楚王，聞其不宜子，恐久無寵，乃求爲春申君舍人。已而謁歸，故失期而還。春申君問之，李園曰︰「齊王使人求臣之妹，與其使者飲，故失期。」春申君曰︰「聘入乎？」曰︰「未也。」春申君遂納之。旣而有娠，李園使其妹說春申君曰︰「楚王貴幸君，雖兄弟不如也。今君相楚二十餘年而王無子，卽百歲後將更立兄弟，彼亦各貴其故所親，君又安得常保此寵乎！非徒然也。君貴，用事久，多失禮於王之兄弟，兄弟立，禍且及身矣。今妾有娠而人莫知，妾幸君未久，誠以君之重，進妾於王，王必幸之。妾賴天而有男，則是君之子爲王也。楚國盡可得，孰與身臨不測之禍哉！」春申君大然之。乃出李園妺，謹舍而言諸楚王。王召入，幸之，遂生男，立爲太子。
李園妹爲王后，李園亦貴用事，而恐春申君泄其語，陰養死士，欲殺春申君以滅口；國人頗有知之者。楚王病，朱英謂春申君曰︰「世有無望之福，亦有無望之禍。今君處無望之世，事無望之主，安可以無無望之人乎！」春申君曰︰「何謂無望之福？」曰︰「君相楚二十餘年矣，雖名相國，其實王也。王今病，旦暮薨，薨而君相幼主，因而當國，王長而反政，不卽遂南面稱孤，此所謂無望之福也。」「何謂無望之禍？」曰︰「李園不治國而君之仇也，不爲兵而養死士之日久矣。王薨，李園必先入，據權而殺君以滅口，此所謂無望之禍也。」「何謂無望之人？」曰︰「君置臣郎中，王薨，李園先入，臣爲君殺之，此所謂無望之人也。」春申君曰︰「足下置之。李園，弱人也，僕又善之。且何至此！」朱英知言不用，懼而亡去。後十七日，楚王薨，李園果先入，伏死士於棘門之內。春申君入，死士俠刺之，投其首於棘門之外；於是使吏盡捕誅春申君之家。太子立，是爲幽王。
揚子《法言》曰︰「或問信陵、平原、孟嘗、春申益乎？」曰︰「上失其政，姦臣竊國命，何其益乎！」
王以文信侯奉先王功大，不忍誅。
