# 卷002 周紀二 — 顯王四十年

> 巻 2 / 294 ・ 周紀二 ・ 年号: 顯王四十年 ・ 西暦: 329 BCE

顯王四十年〔注:壬辰(みずのえたつ)の年、紀元前三二九年〕。

秦が魏を攻め、黄河を渡って汾陰(ふんいん)・皮氏(ひし)を奪い、焦(しょう)を攻め落とした〔注:汾陰・皮氏はいずれも河東郡(かとうぐん)の地。皮氏はもとの耿(こう)国〕。

楚の威王(いおう)が薨(こう)じ、子の懷王(かいおう)・槐(かい)が立った〔注:諡法(しほう)では「慈仁にして短命なるを懷という」〕。

宋公・剔成(てきせい)の弟の偃(えん)が剔成を襲撃した。剔成は齊へ逃れ、偃が自立して君主となった。

四十年
秦伐魏，渡河，取汾陰、皮氏，拔焦。
楚威王薨，子懷王槐立。
宋公剔成之弟偃襲攻剔成；剔成奔齊，偃自立爲君。
