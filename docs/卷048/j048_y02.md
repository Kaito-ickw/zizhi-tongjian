# 卷048 漢紀四十 — 孝和皇帝下五年

五年〔注:癸巳(みずのとみ)の年、紀元九三年〕。

春、正月、乙亥(きのとい)の日、明堂(めいどう)で宗祀(そうし)し、霊台(れいだい)に登り、天下を赦した。

戊子(つちのえね)の日、千乗貞王(せんじょうていおう)の劉伉(りゅうこう)が薨(こう)じた〔注:『諡法(しほう)』に、直道(ちょくどう)にして撓(たわ)まざるを「貞」といい、君に事(つか)えて疑い無きを「貞」といい、清白にして節を守るを「貞」という、とある〕。

辛卯(かのとう)の日、皇弟(こうてい)の劉萬歲(りゅうばんさい)を広宗王(こうそうおう)に封じた〔注:広宗県は鉅鹿(きょろく)郡に属する〕。

甲寅(きのえとら)の日、太傅(たいふ)の鄧彪(とうひょう)が薨じた。

戊午(つちのえうま)の日、隴西(ろうせい)で地震があった。

夏、四月、壬子(みずのえね)の日、阜陵殤王(ふりょうしょうおう)の兄である劉魴(りゅうほう)を、(その跡を)継がせて阜陵王に封じた〔注:『諡法』に、未だ家(か)せずして若死にするを「殤(しょう)」という。阜陵殤王の劉沖(りゅうちゅう)は質王(しつおう)劉延(りゅうえん)の子で、(永元)元年に跡を継いで封ぜられ、三年に薨じて嗣子(しし)が無かったので、いま劉魴をもって(その後を)継いで封じたのである〕。

九月、辛酉(かのととり)の日、広宗殤王(こうそうしょうおう)の劉萬歲が薨じた。子が無く、国は廃された。

当初、竇憲(とうけん)はすでに於除鞬(おじょけん)を立てて北単于(ほくぜんう)とし、これを助けて北庭(ほくてい、匈奴の本拠)に帰らせようとしていたが〔注:この事は前巻の(永元)三年に見える〕、ちょうど竇憲が誅されたので(その計画は)取りやめになった。於除鞬はみずから(漢に)背いて北へ帰った。(帝は)詔(みことのり)して、将兵長史(しょうへいちょうし)の王輔(おうほ)に千余騎を率いさせ、任尚(じんしょう)とともに追討させ、これを斬って、その軍勢を撃ち破った。

耿夔(こうき)が北匈奴を破ったとき〔注:この事は前巻の(永元)三年に見える〕、鮮卑(せんぴ)はこれに乗じて移り住み、その地を占拠した〔注:拓跋(たくばつ)氏が北の荒野から南へ移ったのは、思うにこのときであろう〕。匈奴の余種(残党)で(その地に)留まった者は、なお十余万落(らく、世帯)あったが、みなみずから鮮卑と号した。鮮卑はこれによってしだいに盛んになった。

冬、十月、辛未(かのとひつじ)の日、太尉の尹睦(いんぼく)が薨じた。

十一月、乙丑(きのとうし)の日、太僕(たいぼく)の張酺(ちょうほ)を太尉とした。張酺は尚書の張敏(ちょうびん)らとともに、「射声校尉(しゃせいこうい)の曹褒(そうほう)は、ほしいままに漢の礼を制定し、聖人の学術を破り乱しました。刑罰を加えて誅すべきです」と奏上した。上奏文はあわせて五たびに及んだ。帝は、張酺が(自分の)家学(かがく)を固守して融通がきかぬことを承知しており〔注:(張酺が)その家伝の学を固守したことをいう〕、その上奏は取り上げなかったものの、漢の礼はとうとう施行されなかった〔注:曹褒が礼を制定した事は、前巻の章帝の章和元年に見える〕。

この年、武陵郡(ぶりょうぐん)の兵が叛(そむ)いた蠻(ばん)を破り、これを降伏させた。

梁王(りょうおう)の劉暢(りゅうちょう)は、従官(じゅうかん、侍従)の卞忌(べんき)とともに祭祀(さいし)して福を求めた〔注:『姓譜』によれば、卞(べん)氏は周の曹叔振鐸(そうしゅくしんたく)の後裔(こうえい)で、曹の支子が卞に封ぜられ、それを氏としたという〕。卞忌らは(劉暢に)おもねり媚(こ)びて、「神は、王(あなた)はまさに天子となるべしと告げております」と言った。劉暢もこれに応じてやり取りをした。(これが)有司(役人)に上奏され、(劉暢を)召し出して詔獄(しょうごく)に下すよう請うた。帝は許さず、ただ(劉暢の領地から)成武(せいぶ)・單父(ぜんぽ)の二県を削っただけであった〔注:成武・單父の二県は、もと山陽(さんよう)に属し、のち済陰(さいいん)に属したが、章帝が(これを)梁国に加増したものである〕。劉暢は恥じ恐れて、上疏(じょうそ)してみずからを深く責めて言った。「私は生まれつき愚かで、(身を)防ぎ慎むことを知らず、みずから死罪に陥りました。(本来なら)おおやけの誅(ちゅう)に伏(ふく)すべき身です。陛下は聖徳をもって、法を枉(ま)げて(私の罪を)寛大に裁き〔注:李賢いわく、曲平とは、法を曲げて恩を施し、その罪を公平に処すること〕、なみはずれて私をお赦(ゆる)しになり、(そのために)汚(けが)れをこうむられました〔注:汙(お)は悪の意。天下は劉暢を赦したことを汚れを受け入れたものと見なした。これが「(陛下が)劉暢のために汚れをこうむる」ということ〕。私は、これほど大きなお赦(ゆる)しは二度とは得られぬと承知しております。みずから誓って身を慎み妻子を引き締め、二度と法度(規律)を踏みはずすような出入りはせず、二度と勝手な浪費はいたしません。(領地の)収入には余りがありますので、どうか睢陽(すいよう)・穀熟(こくじゅく)・虞(ぐ)・蒙(もう)・寧陵(ねいりょう)の五県だけを食(は)む(領有する)ことにお裁(さば)きいただき、残りの食んでいる四県はお返しいたします〔注:四県とは、下邑(かゆう)・尉氏(いし)・薄(はく)・郾(えん)である〕。私(劉暢)の側室(小妻)は三十七人おりますが〔注:およそ正室でない者は、みな小妻(=側室)である〕、そのうち子の無い者は本家へ帰すことを願います。みずから謹厳(きんげん)な奴婢(ぬひ)二百人を選び、そのほかに賜った虎賁(こほん)・官騎(かんき)および諸々の工技(こうぎ、工人)・鼓吹(こすい、楽人)・倉頭(そうとう)・奴婢、武器や弩(いしゆみ)、厩(うまや)の馬は、みなもとの役所へお返しいたします〔注:虎賁士は虎賁中郎将に属し、官騎は騶騎(すうき)で、王家では官騎と名づけ、厩馬とともに太僕に属する。工技は尚方(しょうほう)に、鼓吹は黄門(こうもん)に、倉頭・奴婢は永巷(えいこう)・御府(ぎょふ)・奚官(けいかん)などの令に、武器や弩は考工令(こうこうれい)に属し、それぞれもとの役所がある〕。私(劉暢)は、骨肉の近親でありながら、聖(天子)の教化を乱し、清流(清らかな世)を汚しました。すでに生き永らえることができた以上、まことに(あつかましくも)凶悪なこの身で、ふたたび大きな宮殿に住み、大国を食み、官属(役人)を構え、雑物(さまざまな品)を蓄えるなどという面目(めんぼく)は、毛頭ございません〔注:李賢いわく、古え軍が行(こう)するとき、二伍(=十人)を「什(じゅう)」とし、食器の類を必ず共用したので「什物(じゅうぶつ)」という。いまの人は生活の道具をすべて什物という〕。どうか陛下、恩を加えてお許しくださいますよう」。帝は手厚い詔をもって(辞退を)聞き入れなかった。

護羌校尉(ごきょうこうい)の貫友(かんゆう)が〔注:貫(かん)は姓で、漢の初めに趙の相であった貫高(かんこう)がいる〕、通訳の使者を遣わして諸羌(の連携)を分断させ、財貨で誘ったので、これによって(羌は)解散した。そこで兵を遣わして塞を出させ、大榆谷(だいゆこく)・小榆谷(しょうゆこく)に迷唐(めいとう)を攻め、首級・捕虜八百余人を獲(え)、麦(むぎ)数万斛(こく)を取り、そのまま逢留大河(ほうりゅうたいが)を挟んで城塢(じょうう、とりで)を築き〔注:この大河とは黄河のこと。河水はここに至って「逢留」の名があり、(大・小の)二榆谷の北にある〕、大きな航(渡し舟)を作り、河橋(かきょう、舟橋)を造って、兵を渡して迷唐を撃とうとした〔注:酈道元(れきどうげん)の『水経注』に、河の狭まったところに橋を架けた、とある〕。迷唐は部落(部族)を率いて遠くへ移り、賜支河曲(ししかきょく)に拠った〔注:『西羌伝』に、賜支とは『禹貢』にいう析支(せきし)のこと。羌は河関(かかん)の西南に居り、賜支から河首(かしゅ、黄河の源)に至るまで、地は千里にわたる、とある。河水は屈曲して東北に流れ、析支の地を通る。これが河曲である〕。

単于(ぜんう)の屯屠何(とんとか)が死に、(先代の)単于・宣(せん)の弟である安国(あんこく)が立った。安国ははじめ左賢王(さけんおう)であったが、(これといった)誉れもなかった。安国が単于となると、(先代の)単于・適(てき)の子で右谷蠡王(うろくれいおう)であった師子(しし)が、順序に従って左賢王に転じた。師子はもとより勇敢で悧巧(りこう)、知恵が多く、先代の単于・宣および屯屠何は、みなその気概と決断力を愛し、たびたび兵を率いて塞を出させ、北庭(ほくてい、北匈奴)を急襲させた。(師子は)帰っては賞賜(しょうし)を受け、天子もまた格別の(恩賞を)加えた。これによって国中(の者)はこぞって師子を敬い、安国にはなびかなかった。安国は師子を殺そうとした。新たに降(くだ)った諸々の胡(こ)は、もと塞外にいたころ、たびたび師子に駆り立てられ略奪された者たちで〔注:塞外とは、(彼らが)以前に北部(北匈奴)に属していたときをいう〕、多くは師子を怨んでいた。安国は(これら)降った者たちに(ひそかに)はかりごとを委(ゆだ)ね、ともに謀議を巡らした。師子はその謀(はかりごと)を察し、そこで別に五原(ごげん)の境に居を移し、龍庭(りゅうてい)で会議があるたびに〔注:匈奴の龍庭はもと塞外にあったが、このとき南単于は塞内に住んでおり、その住む所もまた龍庭と呼んだ〕、師子はそのつど病と称して出向かなかった。度遼将軍(どりょうしょうぐん)の皇甫稜(こうほりょう)はこれを知って、やはり(師子を)かばって(会議に)行かせなかった。単于(安国)は憤りを抱くことますます甚だしかった。

五年
春，正月，乙亥，宗祀明堂，登靈臺，赦天下。
戊子，千乘貞王伉薨。
辛卯，封皇弟萬歲爲廣宗王。
甲寅，太傅鄧彪薨。
戊午，隴西地震。
夏，四月，壬子，紹封阜陵殤王兄魴爲阜陵王。
九月，辛酉，廣宗殤王萬歲薨，無子，國除。
初，竇憲旣立於除鞬爲北單于，欲輔歸北庭，會憲誅而止。於除鞬自畔還北，詔遣將兵長史王輔以千餘騎與任尚共追討，斬之，破滅其衆。
耿夔之破北匈奴也，鮮卑因此轉徙據其地。匈奴餘種留者尚有十餘萬落，皆自號鮮卑；鮮卑由此漸盛。
冬，十月，辛未，太尉尹睦薨。
十一月，乙丑，太僕張酺爲太尉。酺與尚書張敏等奏「射聲校尉曹褒，擅制漢禮，破亂聖術，宜加刑誅。」書凡五奏。帝知酺守學不通，雖寢其奏，而漢禮遂不行。
是歲，武陵郡兵破叛蠻，降之。
梁王暢與從官卞忌祠祭求福，忌等諂媚云：「神言王當爲天子。」暢與相應答，爲有司所奏，請徵詣詔獄。帝不許，但削成武、單父二縣。暢慙懼，上疏深自刻責曰：「臣天性狂愚，不知防禁，自陷死罪，分伏顯誅。陛下聖德，枉法曲平，橫赦貸臣，爲臣受汙。臣知大貸不可再得，自誓束身約妻子，不敢復出入失繩墨，不敢復有所橫費，租入有餘，乞裁食睢陽、穀熟、虞、蒙、寧陵五縣，還餘所食四縣。臣暢小妻三十七人，其無子者，願還本家，自選擇謹敕奴婢二百人，其餘所受虎賁、官騎及諸工技、鼓吹、倉頭、奴婢，兵弩、廐馬，皆上還本署。臣暢以骨肉近親，亂聖化，汙清流，旣得生活，誠無心面目以凶惡復居大宮，食大國，張官屬，藏雜物，願陛下加恩開許。」上優詔不聽。
護羌校尉貫友遣譯使構離諸羌，誘以財貨，由是解散。乃遣兵出塞，攻迷唐於大、小榆谷，獲首虜八百餘人，收麥數萬斛，遂夾逢留大河築城塢，作大航，造河橋，欲度兵擊迷唐。迷唐率部落遠徙，依賜支河曲。
單于屯屠何死，單于宣弟安國立。安國初爲左賢王，無稱譽；及爲單于，單于適之子右谷蠡王師子以次轉爲左賢王。師子素勇黠多知，前單于宣及屯屠何皆愛其氣決，數遣將兵出塞，掩擊北庭，還，受賞賜，天子亦加殊異。由是國中盡敬師子而不附安國，安國欲殺之；諸新降胡，初在塞外數爲師子所驅掠，多怨之。安國委計降者，與同謀議。師子覺其謀，乃別居五原界，每龍庭會議，師子輒稱病不往。度遼將軍皇甫稜知之，亦擁護不遣，單于懷憤益甚。
