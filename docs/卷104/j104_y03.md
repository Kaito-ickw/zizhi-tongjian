# 卷104 晉紀二十六 — 烈宗孝武皇帝上之中三年

①春、二月、乙巳の日、新しい宮殿の造営を始め、帝(孝武帝)は会稽王(かいけいおう)の邸に移り住んだ。

②秦王苻堅は征南大将軍・都督征討諸軍事・守尚書令・長楽公の苻丕(ふひ)、武衛将軍の苟萇、尚書の慕容暐(ぼようい)に歩兵・騎兵七万を率いさせて襄陽に侵攻させた。荊州刺史の楊安(ようあん)に樊(はん)・鄧(とう)の兵を率いさせて先鋒とし、征虜将軍で始平の石越(せきえつ)に精鋭騎兵一万を率いさせて魯陽関(ろようかん)から出させ、京兆尹の慕容垂・揚武将軍の姚萇に兵五万を率いさせて南郷から出させ、領軍将軍の苟池・右将軍の毛当(もうとう)・強弩将軍の王顕(おうけん)に兵四万を率いさせて武当から出させ、合流して襄陽を攻めさせた。夏、四月、秦の軍は沔水(べんすい)の北に到達した。梁州刺史の朱序は、秦には船がないだろうと油断して備えをしていなかった。ところが石越が騎兵五千を率いて漢水を渡ったので、朱序は驚き恐れ、中城(なかじろ)を固く守った。石越はその外郭を陥落させ、船百余艘を手に入れて残りの軍を渡らせた。長楽公の苻丕は諸将を督励して中城を攻めた。

朱序の母の韓氏(かんし)は、秦の軍が間もなく到着すると聞くと、自ら城壁に登って見回り、西北の隅に至って、そこが堅固でないと見抜き、百余人の婢女(はしため)と城中の成年女性を率いて、その内側に斜めの城壁(邪城)を築かせた。秦の軍が到着すると、案の定西北の隅が崩れたので、人々は移ってこの新しい城を守った。襄陽の人々はこれを「夫人城(ふじんじょう)」と呼んだ。

桓沖は上明にあって七万の兵を擁していたが、秦の軍の強さを恐れて、あえて進もうとしなかった。

苻丕は急いで襄陽を攻めようとしたが、苟萇は「我らの兵は敵の十倍もあり、乾飯や兵糧は山のように積まれています。ただ漢水・沔水の民を徐々に許(きょ)・洛(らく)に移し、彼らの輸送路を塞ぎ、援軍を断ってしまえば、まるで網の中の鳥のようなもの。どうして捕らえられぬことを心配しましょうか。それなのに、なぜ多くの将兵を死なせてまで急いで成功を求める必要がありましょうか」と言った。苻丕はこれに従った。慕容垂は南陽を陥落させ、太守の鄭裔(ていえい)を捕らえ、苻丕と襄陽で合流した。

③秋、七月、新しい宮殿が完成し、辛巳の日、帝はここに入って住んだ。

④秦の兗州刺史の彭超(ほうちょう)は、彭城(ほうじょう)で沛郡太守の戴𨔵(たいとん)を攻めることを願い出て、さらに「どうかさらに重臣の将を遣わし、淮南の諸城を攻めさせ、征南(苻丕)の軍と碁の劫(こう、挟み撃ちの形勢)〔注:碁で右を攻めれば敵がこれに応じ、その隙に左を攻め取ることを「劫」という。ここでは碁の形勢を兵の形勢にたとえている〕を成し、東西から並び進めば、丹陽(たんよう、建康)を平らげるのも難しくありません」と言った。秦王苻堅はこれに従い、都督東討諸軍事とし、後将軍の俱難(くなん)・右禁将軍の毛盛(もうせい)・洛州刺史の邵保(しょうほ)に歩兵・騎兵七万を率いさせて淮陽(わいよう)・盱眙(くい)に侵攻させた。彭超は越の弟であり、邵保は邵羌(しょうきょう)の従弟である。八月、彭超は彭城を攻めた。詔を下し、右将軍の毛虎生(もうこせい)に兵五万を率いさせて姑孰(こじゅく)に鎮させ、秦の軍を防がせた。

秦の梁州刺史の韋鍾(いしょう)は、魏興太守の吉挹(きつゆう)を西城(せいじょう)で包囲した。

⑤九月、秦王苻堅は群臣と酒を飲み、秘書監の朱肜(しゅじゅう)を酒正(しゅせい、酒宴の取り仕切り役)とし、それぞれに酔いつぶれるまで飲ませることを定めとした。秘書侍郎の趙整(ちょうせい)は「酒徳の歌(しゅとくのうた)」を作って言った――「地には酒泉の郡が連なり、天には酒旗の星が垂れる〔注:「池」はもと「旗」の誤りとされる〕。杜康(とこう)は酒造りの妙を極め、儀狄(ぎてき)はいち早くその術を知った。紂王(ちゅうおう)は酒に溺れて殷の国を喪い、桀王(けつおう)は酒に溺れて夏の国を傾けた。これらのことから言えば、前の人の危うさこそ、後の人の戒めとなるものだ」。苻堅は大いに喜び、趙整に命じてこれを書き記させ、酒の戒めとした。これより後、群臣と宴を催すときは、礼にかなう程度の飲酒にとどめるようになった。

⑥秦の涼州刺史の梁熙は使者を西域に遣わし、秦の威徳を広く知らしめた。冬、十月、大宛(だいえん)が汗血馬(かんけつば)を献上した。秦王苻堅は「私はかねてより漢の文帝の人となりを慕っている。千里の馬など何の役に立とうか」と言い、群臣に命じて「馬を止める詩」を作らせ、この馬を送り返させた〔注:送り返すだけでよいものを、なぜわざわざ詩まで作らせたのか、これもまた名声を求める行き過ぎである、という史官の評〕。

⑦巴西(はせい)の人・趙宝(ちょうほう)が涼州で兵を起こし、自ら晋の西蛮校尉・巴郡太守と称した〔注:蜀の人々が晋を慕っていたことを物語る〕。

⑧秦の豫州刺史・北海公の苻重(ふちょう)は洛陽に鎮していたが、謀反を企てた。秦王苻堅は「長史の呂光(りょこう)は忠実で正しい人物だから、必ずやこれに与しないだろう」と言い、ただちに呂光に命じて苻重を捕らえさせ、檻車(おりぐるま)で長安に送らせた。だが苻重を赦し、公の身分のまま邸宅にとどめさせた。苻重は苻洛の兄である。

⑨十二月、秦の御史中丞の李柔(りじゅう)は弾劾して上奏した――「長楽公苻丕らは十万の兵を擁しながら、小城を攻め囲むだけで、一日に万金を費やし、久しく成果がありません。どうか召還して廷尉に下されますよう」。秦王苻堅は「苻丕らは多くの費用を費やしながら成果がないのは、まことに処罰すべきである。だが軍はすでに長く駐留しており、むなしく引き返させるわけにもいかない。特にこれを許し、功を成すことで罪を贖わせよう」と言い、黄門侍郎の韋華(いか)に持節を持たせて苻丕らを厳しく叱責させ、苻丕に剣を賜って「来春までに勝てなければ、お前は自ら命を絶つがよい。二度と顔を見せて私に会おうとするな」と言わせた。

⑩周虓(しゅうこう)は秦にあって、ひそかに桓沖に書を送り、秦の秘密の策謀を告げた。またひそかに逃げて漢中に奔ろうとしたが、秦の人々に捕らえられ、それでも赦された。

三年
春，二月，乙巳，作新宮，帝移居會稽王邸。
秦王堅遣征南大將軍‧都督征討諸軍事‧守尚書令‧長樂公丕、武衞將軍苟萇、尚書慕容暐帥步騎七萬寇襄陽，以荊州刺史楊安帥樊、鄧之衆爲前鋒，征虜將軍始平石越帥精騎一萬出魯陽關，京兆尹慕容垂、揚武將軍姚萇帥衆五萬出南鄕，領軍將軍苟池、右將軍毛當、強弩將軍王顯帥衆四萬出武當，會攻襄陽。夏，四月，秦兵至沔北，梁州刺史朱序以秦無舟檝，不以爲虞。旣而石越帥騎五千浮渡漢水，序惶駭，固守中城；越克其外郭，獲船百餘艘以濟餘軍。長樂公丕督諸將攻中城。
序母韓氏聞秦兵將至，自登城履行，至西北隅，以爲不固，帥百餘婢及城中女丁築邪城於其內。及秦兵至，西北隅果潰，衆移守新城，襄陽人謂之夫人城。
桓沖在上明擁衆七萬，憚秦兵之強，不敢進。
丕欲急攻襄陽，苟萇曰：「吾衆十倍於敵，糗糧山積，但稍遷漢、沔之民於許、洛，塞其運道，絕其援兵，譬如網中之禽，何患不獲，而多殺將士，急求成功哉！」丕從之。慕容垂拔南陽，執太守鄭裔，與丕會襄陽。
秋，七月，新宮成；辛巳，帝入居之。
秦兗州刺史彭超請攻沛郡太守戴𨔵於彭城，且曰：「願更遣重將攻淮南諸城，爲征南棊劫之勢，東西並進，丹陽不足平也！」秦王堅從之，使都督東討諸軍事；後將軍俱難、右禁將軍毛盛、洛州刺史邵保帥步騎七萬寇淮陽、盱眙。超，越之弟；保，羌之從弟也。八月，彭超攻彭城。詔右將軍毛虎生帥衆五萬鎭姑孰以禦秦兵。
秦梁州刺史韋鍾圍魏興太守吉挹於西城。
九月，秦王堅與羣臣飲酒，以祕書監朱肜爲正，人以極醉爲限。祕書侍郎趙整作《酒德之歌》曰：「地列酒泉，天垂酒池，杜康妙識，儀狄先知。紂喪殷邦，桀傾夏國，由此言之，前危後則。」堅大悅，命整書之以爲酒戒，自是宴羣臣，禮飲而已。
秦涼州刺史梁熙遣使入西域，揚秦威德。冬，十月，大宛獻汗血馬。秦王堅曰：「吾嘗慕漢文帝之爲人，用千里馬何爲！」命羣臣作《止馬之詩》而反之。
巴西人趙寶起兵涼州，自稱晉西蠻校尉、巴郡太守。
秦豫州刺史北海公重鎭洛陽，謀反；秦王堅曰：「長史呂光忠正，必不與之同。」卽命光收重，檻車送長安，赦之，以公就第。重，洛之兄也。
十二月，秦御史中丞李柔劾奏：「長樂公丕等擁衆十萬，攻圍小城，日費萬金，久而無效，請徵下廷尉。」秦王堅曰：「丕等廣費無成，實宜貶戮；但師已淹時，不可虛返，其特原之，令成功贖罪。」使黃門侍郎韋華持節切讓丕等，賜丕劍曰：「來春不捷，汝可自裁，勿復持面見吾也！」
周虓在秦，密與桓沖書，言秦陰計；又逃奔漢中，秦人獲而赦之。
