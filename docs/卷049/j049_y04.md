# 卷049 漢紀四十一 — 孝安皇帝上三年

三年〔注:己酉(つちのととり)の年、紀元一〇九年〕。

春、正月、庚子(かのえね)の日、皇帝が元服(げんぷく、成人の冠礼)を加え、天下を赦した。

騎都尉(きとい)の任仁(じんじん)を遣わして、諸郡の屯営の兵を監督させ、三輔(さんぽ)を救援させた。任仁はたびたび戦って利あらず、当煎(とうせん)・勒姐(ろくし)の羌(きょう)が破羌県(はきょうけん)を攻め落とし〔注:破羌県は金城(きんじょう)郡に属する〕、鍾羌(しょうきょう)が臨洮県(りんとうけん)を攻め落として、隴西南部都尉(ろうせいなんぶとい)を捕らえた〔注:臨洮県は隴西南部都尉の治所(役所の所在地)である〕。

三月、京師は大飢饉(だいききん)となり、民は互いに食(は)み合った。壬辰(みずのえたつ)の日、公卿(こうけい)が宮闕(きゅうけつ)に参って謝罪した。帝は詔して「政(まつりごと)の過ちを改め善に復(かえ)すことに努めて、私の行き届かぬところを助けよ」と言った〔注:「変」は改めるの意。過ちを改めて善に復することである〕。

壬寅(みずのえとら)の日、司徒の魯恭(ろきょう)が罷免された。魯恭は二度三公の位にあり〔注:和帝の永元十二年、魯恭は呂蓋(りょがい)に代わって司徒となり、永初元年、また梁鮪(りょうい)に代わって司徒となった〕、府の属官のうち成績優秀な者を選び招いて、列卿(れっけい)や郡守(ぐんしゅ)にまで至らせた者が数十人いた〔注:これは魯恭の府の掾属(えんぞく、属官)で成績優秀な者をいう〕。ところが門下の年老いた書生(耆生)〔注:「耆」は老いるの意〕の中には、推薦にあずかれぬ者もいて、怨み言を抱く者まで出た。魯恭はこれを聞いて言った。「学問が身に講(おさ)まらぬこと、それこそが私の心配の種だ〔注:『論語』に載る孔子の言葉である〕。書生たちには、郷里からの推挙(郷挙)というものがあるではないか」〔注:李賢いわく、人は学問が身につかぬことこそを患(うれ)えるべきで、もしよく学び究めれば、おのずと郷里からの推挙があるはず、どうして三公の招き(辟召)を待つ必要があろうか、という意〕。魯恭は結局(推挙について)何も言わず、またその不満について彼らと議論を交わすこともしなかった。学問を受ける者には、必ず徹底して疑問を突き詰め問いただし、その道(学業)が成ってから、礼を尽くして送り出した。学者たちは言った。「魯公に送り出してもらい、議論を交わしてもらうことは、うわべだけで得られるものではない」。

夏、四月、丙寅(ひのえとら)の日、大鴻臚(だいこうろ)で九江(きゅうこう)出身の夏勤(かきん)を司徒とした。

三公は、国家の財政(国用)がまだ足りないことから、官吏や民が銭や穀物を納めれば、その額に応じて、関内侯(かんだいこう)・虎賁(こほん)・羽林郎(うりんろう)・五官(ごかん)・大夫(たいふ)・官府の吏・緹騎(ていき)・営士(えいし)になれるよう、それぞれ等級に差を設けて(売官を)許すよう奏上した〔注:「五官」もまた郎(ろう)である。「大夫」とは光禄大夫(こうろくたいふ)・太中大夫(たいちゅうたいふ)・中散大夫(ちゅうさんたいふ)・諫議大夫(かんぎたいふ)のこと。「官府の吏」とは諸官府で勤務する者。李賢いわく、『続漢志』に、執金吾(しつきんご)には緹騎二百人がある、とある。「緹(てい)」は赤黄色(あかきいろ)。「営士」とは五校(ごこう)の営の士のこと〕。

甲申(きのえさる)の日、清河愍王(せいがびんおう)の虎威(こい)が薨(こう)じた。子がなかった。五月、丙申(ひのえさる)の日、楽安王(らくあんおう)の寵(ちょう)の子の延平(えんぺい)を清河王に封じ、清河孝王の後(あと)を継がせた。

六月、漁陽(ぎょよう)の烏桓(うがん)が、右北平(ゆうほくへい)の胡(こ)千人あまりとともに、代郡(だいぐん)・上谷(じょうこく)を侵した〔注:司馬光の考証——『帝紀』には涿郡(たくぐん)もあるが、『列伝』にはない。ここでは『列伝』に従う〕。

漢人の韓琮(かんそう)が、匈奴の南単于(なんぜんう)に従って入朝(にゅうちょう)していたが〔注:漢人は匈奴と入り混じって住んでおり、韓琮はそうした縁で南単于に仕えていた〕、南単于のもとへ帰ると、南単于に説いて言った。「関東(かんとう)は水害つづきで、人民は飢えてことごとく死に絶えました。攻撃なさるべきです」。単于はその言葉を信じ、ついに漢に叛(そむ)いた。

秋、七月、海賊の張伯路(ちょうはくろ)らが海沿いの九郡を侵し、二千石(にせんせき)の太守や、県令(けんれい)・県長(けんちょう)を殺した。侍御史(じぎょし)で巴郡(はぐん)出身の龐雄(ほうゆう)を遣わして州郡の兵を監督させてこれを撃たせ、張伯路らは降伏を乞うたが、まもなくまた屯(たむろ)して集まった。

九月、鴈門(がんもん)の烏桓の率衆王(そっしゅうおう)である無何允(むかいん)が、鮮卑(せんぴ)の大人(たいじん)の丘倫(きゅうりん)ら、および南匈奴の骨都(こっと)とともに、あわせて七千騎で五原(ごげん)を侵し、太守と高渠谷(こうきょこく)で戦い〔注:李賢いわく、『東観記』では九原(きゅうげん)の高梁谷(こうりょうこく)で戦ったとする。「渠」と「梁」は字形が似ており、きっとどちらかが誤りであろう〕、漢の兵は大敗した。

南単于は中郎将の耿种(こうちゅう)を美稷(びしょく)で包囲した〔注:中郎将とは使匈奴中郎将(しきょうどちゅうろうしょう)のこと。「种」の音は「ちゅう」〕。冬、十一月、大司農(だいしのう)で陳国(ちんこく)出身の何熙(かき)に車騎将軍の事を行(代行)させ、中郎将の龐雄をその副(ふく)とし、五営および辺郡の兵あわせて二万人あまりを率いさせ、また詔して遼東(りょうとう)太守の耿夔(こうき)に鮮卑および諸郡の兵を率いさせて、ともにこれを撃たせた。梁慬(りょうきん)には度遼将軍(どりょうしょうぐん)の事を行(代行)させた。龐雄・耿夔は南匈奴の薁鞬日逐王(いくけんじつちくおう)を撃ち、これを破った。

十二月、辛酉(かのととり)の日、九つの郡国で地震があった。

乙亥(きのとい)の日、天苑(てんえん)に彗星(すいせい、星孛)が現れた〔注:『晋書』天文志に、天苑十六星は昴(ぼう)・畢(ひつ)の南にあり、天子の苑囿(えんゆう、庭園)で、獣を養う所である、とある〕。

この年、京師および四十一の郡国で水害があり、幷州(へいしゅう)・涼州(りょうしゅう)二州は大飢饉となり、人が互いに食み合った。

太后は、陰陽が和せず、軍事(軍旅)がたびたび起こったことから、詔して、年末に衛士(えいし)をねぎらって送り出す際に、余興の戯(ぎ、雑技)を設けず楽を奏でぬこととし〔注:前漢(西都)の制度では、年の暮れに衛卒(えいそつ)が交代し、天子が臨んで饗応(きょうおう)し、ねぎらって送り出した。『続漢志』に、もとの衛士をねぎらって送り出す儀礼が記される——百官が会し、位が定まると、謁者(えっしゃ)が節(せつ)を持って端門(たんもん)から旧衛士を導き入れ、衛司馬(えいしば)が幡(はた)と鉦(かね)を執って行列を護(まも)る。整列すると、侍御史が節を持ってねぎらい、詔の恩をもって苦しみを問い、その上奏したいことを受ける。終わると、饗応し賜物を与えて楽を奏で、角抵(かくてい、相撲のような技)を観(み)せる。楽が終わると、ねぎらって送り出し、農桑(のうそう)に励むよう勧めた、とある〕、疫病を追い払う侲子(しんし)を半分に減らした〔注:李賢いわく、「侲子」とは疫病を追い払う役の者で、音は「しん」。『続漢書』に、大儺(たいだ、追儺の儀式)では、黄門の子弟で十歳以上十二歳以下の者百二十人を選んで侲子とし、みな赤い幘(さく、頭巾)に黒い衣を着け、大きな鞉(とう、振り鼓)を執る、とある〕。

三年
春，正月，庚子，皇帝加元服，赦天下。
遣騎都尉任仁督諸郡屯兵救三輔。仁戰數不利，當煎、勒姐羌攻沒破羌縣，鍾羌攻沒臨洮縣，執隴西南部都尉。
三月，京師大饑，民相食。壬辰，公卿詣闕謝；詔「務思變復，以助不逮。」
壬寅，司徒魯恭罷。恭再在公位，選辟高第至列卿、郡守者數十人，而門下耆生或不蒙薦舉，至有怨望者。恭聞之，曰：「學之不講，是吾憂也，諸生不有鄕舉者乎！」終無所言，亦不借之議論。學者受業，必窮核問難，道成，然後謝遣之。學者曰：「魯公謝與議論，不可虛得。」
夏，四月，丙寅，以大鴻臚九江夏勤爲司徒。
三公以國用未足，奏令吏民入錢穀得爲關內侯、虎賁、羽林郎、五官、大夫、官府吏、緹騎、營士各有差。
甲申，清河愍王虎威薨，無子。五月，丙申，封樂安王寵子延平爲清河王，奉孝王後。
六月，漁陽烏桓與右北平胡千餘寇代郡、上谷。
漢人韓琮隨匈奴南單于入朝，旣還，說南單于云：「關東水潦，人民飢餓死盡，可擊也。」單于信其言，遂反。
秋，七月，海賊張伯路等寇濱海九郡，殺二千石、令、長；遣侍御史巴郡龐雄督州郡兵擊之，伯路等乞降，尋復屯聚。
九月，鴈門烏桓率衆王無何允與鮮卑大人丘倫等及南匈奴骨都合七千騎寇五原，與太守戰于高渠谷，漢兵大敗。
南單于圍中郎將耿种於美稷。冬，十一月，以大司農陳國何熙行車騎將軍事，中郎將龐雄爲副，將五營及邊郡兵二萬餘人，又詔遼東太守耿夔率鮮卑及諸郡共擊之。以梁慬行度遼將軍事。雄、夔擊南匈奴薁鞬日逐王，破之。
十二月，辛酉，郡國九地震。
乙亥，有星孛于天苑。
是歲，京師及郡國四十一雨水，幷、涼二州大飢，人相食。
太后以陰陽不和，軍旅數興，詔歲終饗遣衞士勿設戲作樂，減逐疫侲子之半。
