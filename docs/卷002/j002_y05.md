# 卷002 周紀二 — 顯王七年

顯王七年〔注:己未(きび)の年、紀元前三六二年〕。

魏が韓軍と趙軍を澮(かい)で打ち破った。

秦と魏が少梁(しょうりょう)で戦い、魏軍は大敗した〔注:敗績とは、軍が総崩れになることをいう〕。魏の公孫痤(こうそんざ)が捕らえられた。

衞(えい)の聲公(せいこう)が薨じ、子の成侯・速(そく)が立った。

燕(えん)の桓公(かんこう)が薨じ、子の文公(ぶんこう)が立った。

秦の獻公(けんこう)が薨じ、子の孝公(こうこう)〔注:名は渠梁(きょりょう)〕が立った。孝公はこのとき二十一歳であった。当時、黄河と西方の山地より東には強国が六つ〔注:韓・魏・趙・齊・楚・燕〕、淮水(わいすい)と泗水(しすい)の間には小国が十余り〔注:宋・魯・鄒・滕・薛などの国々〕あり、楚と魏が秦と国境を接していた。魏は長城を築き、鄭(てい)から洛水(らくすい)沿いに北へと上郡(じょうぐん)の地を領していた。楚は漢中(かんちゅう)を起点に、南は巴(は)・黔中(けんちゅう)を領していた。魏と楚はともに秦を夷狄(いてき)あつかいして排斥し、そのため秦は中原(ちゅうげん)諸国の会盟(かいめい)に加わることができなかった。そこで孝公は発憤し、徳を施し政治を整えて、秦を強国にしようと志した。

七年
魏敗韓師、趙師于澮。
秦、魏戰于少梁，魏師敗績；獲魏公孫痤。
衞聲公薨，子成侯速立。
燕桓公薨，子文公立。
秦獻公薨，子孝公立。孝公生二十一年矣。是時河、山以東強國六，淮、泗之間小國十餘，楚、魏與秦接界。魏築長城，自鄭濱洛以北有上郡；楚自漢中，南有巴、黔中：皆以夷翟遇秦，擯斥之，不得與中國之會盟。於是孝公發憤，布德修政，欲以強秦。
