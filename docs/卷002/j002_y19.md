# 卷002 周紀二 — 顯王二十三年

> 巻 2 / 294 ・ 周紀二 ・ 年号: 顯王二十三年 ・ 西暦: 346 BCE

[← 巻インデックス](README.md)

顯王二十三年〔注:乙亥(いつがい)の年、紀元前三四六年〕。

齊が自国の大夫(たいふ)である牟(ぼう)を処刑した。

魯の康公(こうこう)が亡くなり、その子の景公(けいこう)偃(えん)が跡を継いで立った。

衞(えい)は称号をさらに引き下げて「侯(こう)」とあらため、三晉(さんしん)〔注:韓・趙・魏〕に臣従した〔注:周の成王が康叔を衞侯に封じ、その子孫は爵位を進めて「公」と称していたが、いまや次第に弱小となったため、号を引き下げて「侯」としたのである〕。

---

<summary>原文を表示</summary>

二十三年
齊殺其大夫牟。
魯康公薨，子景公偃立。
衞更貶號曰侯，服屬三晉。
