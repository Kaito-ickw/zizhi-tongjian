# 卷002 周紀二 — 顯王十五年

> 巻 2 / 294 ・ 周紀二 ・ 年号: 顯王十五年 ・ 西暦: 354 BCE

[← 巻インデックス](README.md)

---

顯王十五年〔注:丁卯(ていぼう)の年、紀元前三五四年〕。

秦が元里(げんり)で魏の軍を打ち破り、首級を七千挙げ〔注:秦の法では、戦って敵の首を一つ斬った者には爵位を一級与えた。それでこれを「級」と呼ぶ〕、少梁(しょうりょう)を奪い取った。

魏の惠王(けいおう)が趙を攻め、邯鄲(かんたん)を包囲した。楚王は景舍(けいしゃ)を遣わして趙を救援させた。

---

<summary>原文を表示</summary>

十五年
秦敗魏師于元里，斬首七千級，取少梁。
魏惠王伐趙，圍邯鄲。楚王使景舍救趙。
