# 卷004 周紀四 — 赧王中十八年

赧王(たんおう)十八年〔注:甲子(こうし)の年、紀元前二九七年〕。

楚の懷王(かいおう)が(秦から)逃げ帰ろうとした。秦人はこれに気づき、楚へ通じる道を塞いだ〔注:楚へ帰る道を遮ったのである〕。懷王は抜け道をたどって趙へ逃げ込もうとした。だが趙の主父(しゅほ)は代(だい)におり、趙の者たちは懷王を受け入れようとしなかった。懷王は今度は魏へ逃れようとしたが、秦人に追いつかれ、(秦へ)連れ戻された。

魯の平公(へいこう)が薨じ、その子の緡公(びんこう)賈(か)が立った。

赧王中
十八年
楚懷王亡歸。秦人覺之，遮楚道。懷王從間道走趙。趙主父在代，趙人不敢受。懷王將走魏，秦人追及之，以歸。
魯平公薨，子緡公賈立。
