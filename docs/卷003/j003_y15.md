# 卷003 周紀三 — 赧王上九年

九年〔注:乙卯(きのとう)の年、紀元前三〇六年〕。

①秦の昭王(昭襄王)は向壽(しょうじゅ)を遣わして宜陽(ぎよう)を鎮め〔注:平とは、宜陽の境界を正し、その民を安んじること〕、樗里子(ちょりし)と甘茂(かんぼう)には魏を伐たせた。甘茂は王に進言して、武遂(ぶすい)を韓に返させた〔注:武遂はもと韓に属し、平陽に近い。去年、秦は宜陽を抜いたとき黃河を渡って武遂に城を築いたが、今これを韓に返したのである〕。向壽と公孫奭(こうそんせき)はこれに反対したが聞き入れられず、このことで甘茂を恨み、讒言(ざんげん)するようになった。甘茂は身を案じ、魏の蒲阪(ほはん)攻めを途中でやめて出奔した〔注:蒲阪は河東郡に属する〕。樗里子は魏と和を結んで兵を引いた〔注:講とは和すること〕。甘茂は齊(せい)へ亡命した。

②趙王(武靈王)は中山(ちゅうざん)の地を攻め巡り、寧葭(ねいか)に至った。西へ胡の地を攻め巡って、榆中(ゆちゅう)に至った。林胡(りんこ)の王が馬を献上した〔注:林胡は、草木の茂み(林薄)に拠って住む胡の一族なので林胡という〕。王は(中山から)帰還すると、樓緩(ろうかん)を秦に、仇液(きゅうえき)を韓に、王賁(おうほん)を楚に〔注:この王賁は趙人で、秦将の王賁とは別人〕、富丁(ふてい)を魏に、趙爵(ちょうしゃく)を齊に遣わした。代(だい)の相である趙固(ちょうこ)に胡の地を統轄させ、その兵を集めさせた〔注:致とは、至らせること〕。

③楚王は齊・韓と合従(がっしょう)を結んだ〔注:楚は齊・韓と合従したものの、まもなくこれに背くことになり、かえって齊・韓の兵を招き寄せるだけであった〕。

九年
秦昭王使向壽平宜陽，而使樗里子、甘茂伐魏。甘茂言於王，以武遂復歸之韓。向壽、公孫奭爭之，不能得，由此怨讒甘茂。茂懼，輟伐魏蒲阪，亡去。樗里子與魏講而罷兵。甘茂奔齊。
趙王略中山地，至寧葭；西略胡地，至榆中。林胡王獻馬，歸，使樓緩之秦，仇液之韓，王賁之楚，富丁之魏，趙爵之齊；代相趙固主胡，致其兵。
楚王與齊、韓合從。
