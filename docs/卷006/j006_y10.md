# 卷006 秦紀一 — 始皇帝上元年

始皇帝(しこうてい)上〔注:諱(いみな)は政(せい)、莊襄王の子。のちに天下を併せると、自らの徳は三皇を兼ね功は五帝にもまさると考え、自ら「皇帝」と号した。さらに帝位を一世から万世まで伝えようと願って諡法(しほう)を廃し、「始皇帝」と号した〕。

元年〔注:乙卯(いつぼう)の年、紀元前二四六年〕。

蒙驁(もうごう)が(反いた晉陽を)攻めてこれを平定した〔注:晉陽を撃って平定したのである〕。

韓は秦を疲弊させて東方への侵攻をやめさせようと企て、そこで水利技師の鄭国(ていこく)を間者(かんじゃ)として秦に送り込んだ。(鄭国の進言により)秦は仲山(ちゅうざん)のあたりから涇水(けいすい)を引いて運河を掘り、北山に沿わせて東のかた洛水(らくすい)へ注がせる工事を始めた〔注:涇水は安定郡涇陽県西の幵頭山に発し、東へ流れて馮翊陽陵県で渭水に注ぐ。仲山は九嵏(きゅうそう)の東の仲山である〕。工事が半ばに差しかかったとき、(韓の)企てが発覚し、秦人は鄭国を殺そうとした〔注:中作とは工事が中途で事業が未完成なこと、覺とは露見することで、韓の謀略が露れたのである〕。鄭国は言った。「私は韓のために(秦の東進を遅らせて)数年の延命を図ったにすぎません。しかしこの運河が完成すれば、それは秦にとっても万世の利となりましょう。」そこで(秦は)ついにこれを完成させた。淤(よど)んだ泥水を引いて塩気の多い荒れ地四万余頃(けい)に灌漑したところ、収穫はどこも一畝あたり一鍾(いっしょう)に達した〔注:師古いわく、濁った淤泥(おでい)の水を引いて塩分の多い土地に注ぎ、肥沃に変えたので、一畝の収穫が六斛四斗(=一鍾)に達した〕。関中(かんちゅう)はこれによっていっそう豊かになった。

始皇帝上
元年
蒙驁擊定之。
韓欲疲秦人，使無東伐，乃使水工鄭國爲間於秦，鑿涇水自仲山爲渠，並北山，東注洛。中作而覺，秦人欲殺之。鄭國曰︰「臣爲韓延數年之命，然渠成，亦秦萬世之利也。」乃使卒爲之。注塡閼之水漑舄鹵之地四萬餘頃，收皆畝一鍾，關中由是益富饒。
