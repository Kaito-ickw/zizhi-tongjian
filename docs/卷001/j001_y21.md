# 卷001 周紀一 — 安王二十三年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王二十三年 ・ 西暦: 379 BCE

[← 巻インデックス](README.md)

安王二十三年〔注:壬寅(じんいん)の年、紀元前三七九年〕。

趙が衞(えい)を襲ったが、攻め落とせなかった。

斉の康公(こうこう)が薨じたが、跡継ぎの子がなく、田氏がついに斉を併呑して我がものとした〔注:これによって、もとの斉の君主であった姜氏(きょうし)の家系は絶えた〕。

この年、斉の桓公(かんこう)もまた薨じ、子の威王(いおう)因斉(いんせい)が立った〔注:この桓公は田午(でんご)、威王はその子の田因斉(でんいんせい)。いずれも田氏の斉の君主〕。

二十三年
趙襲衞，不克。
齊康公薨，無子，田氏遂幷齊而有之。
是歲，齊桓公亦薨，子威王因齊立。
