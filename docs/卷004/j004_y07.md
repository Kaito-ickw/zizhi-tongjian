# 卷004 周紀四 — 赧王中二十四年

赧王二十四年〔注:庚午(こうご)の年、紀元前二九一年〕。

秦が韓を攻め、宛(えん)を抜いた〔注:宛はもと申伯(しんぱく)の国で、南陽郡に属する〕。

秦の燭壽(しょくじゅ)が罷免された。魏冉(ぎぜん)が再び丞相となり、穰(じょう)と陶(とう)に封じられて、穰侯(じょうこう)と呼ばれた。また公子巿(こうしふつ)を宛に、公子悝(こうしかい)を鄧(とう)に封じた。

二十四年
秦伐韓，拔宛。
秦燭壽免。魏冉復爲丞相，封於穰與陶，謂之穰侯。又封公子巿於宛，公子悝於鄧。
