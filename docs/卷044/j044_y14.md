# 卷044 漢紀三十六 — 顯宗孝明皇帝上三年

永平三年〔注:庚申(かのえさる)の年、西暦六〇年〕。

① 春、二月、甲寅(こういん)の日、太尉(たいい)の趙熹(ちょうき)と司徒(しと)の李訢(りきん)が免じられた。丙辰(へいしん)の日、左馮翊(さひょうよく)の郭丹(かくたん)を司徒とした。己未(きび)の日、南陽太守(なんようたいしゅ)の虞延(ぐえん)を太尉とした。

② 甲子(こうし)の日、貴人(きじん)の馬氏(ばし)を皇后に立て、皇子の炟(たつ)を太子とした。

皇后は、馬援(ばえん)の娘である。光武帝の世に、選ばれて太子(=のちの明帝)の宮に入り、よく陰皇后(いんこうごう)に仕え、周りの同輩たちにも如才なく接し、礼儀作法を整え行き届かせたので、上の者にも下の者にも安んじられ、ついにとりわけ寵愛されるようになった。帝(明帝)が即位すると、貴人となった。当時、皇后の前母(ぜんぼ、父の先妻)の姉の娘である賈氏(かし)もまた選ばれて入っており、皇子の炟を生んだ。帝は皇后に子がなかったので、炟を養うよう命じ、皇后にこう言った。「人は必ずしも自分で子を生まねばならぬわけではない。ただ、慈しみ育てることが行き届かぬのを心配すればよいのだ」。皇后はそこで心を尽くして炟を養い育て、その苦労は実の子に対する以上であった。太子(炟)もまた孝心が純朴で篤く、母子の慈しみ合う情には、初めから終わりまで、ほんのわずかな隙間もなかった〔注:「纖介」とは細やかなこと、「間」とは隙間のことである〕。皇后はつねづね、皇統の世継ぎがまだ多くないことを気にかけ、(他の女性を)帝のお側に推し挙げて取り次ぎ、まるで手遅れになるのを恐れるかのようであった。後宮で帝にお目通りする者があれば、そのたびにねぎらって受け入れ、もしたびたび寵愛を受けて召される者があれば、いっそう手厚くもてなした。

担当の役人が長秋宮(ちょうしゅうきゅう、皇后)を立てることを上奏したとき〔注:皇后の宮を長秋宮という〕、帝はまだ何も言わなかったが、皇太后(光武帝の陰皇后)が言った。「馬貴人は徳が後宮の中で第一です。まさにその人(皇后にふさわしい者)です」。皇后は宮中で正位(皇后の位)に就くと、いよいよ自ら謙虚につつましくふるまい、書物を読むことを好んだ。つねに粗い絹(きぬ)の衣を着て〔注:「大練」とは大帛(たいはく)、すなわち厚地の絹である〕、裳(も)には縁飾りもつけなかった。朔(ついたち)と望(もちづき、十五日)に、もろもろの姫君や公主たちが朝見(ちょうけん)に参じ、皇后の袍衣(ほうい)が目も粗く地味なのを遠目に見て、(かえって上等の)綺縠(きこく、あや織りの絹)だと思い込み、近寄って見て(その質素さに気づき)思わず笑った。皇后は言った。「この絹はとりわけ染め色がよく映えるので、これを使っているだけです」。群臣が奏上する案件で、決着のつけにくい難しいものがあると〔注:「平」とは決すること。「難平」とは決しがたいもの〕、帝はたびたびこれを皇后に試しに尋ねた。皇后はそのつど事を分析して筋道を解きほぐし、それぞれの実情を言い当てたが、それでいて一度も身内の私情を政事に差しはさむことはなかった。帝はこのために皇后を寵愛し敬うこと、初めから終わりまで衰えることがなかった。

③ 帝は中興の功臣たちを思い起こし、二十八将(にじゅうはちしょう)を南宮(なんきゅう)の雲台(うんだい)に描かせた。鄧禹(とうう)を筆頭に、次いで馬成(ばせい)・呉漢(ごかん)・王梁(おうりょう)・賈復(かふく)・陳俊(ちんしゅん)・耿弇(こうえん)・杜茂(とぼう)・寇恂(こうじゅん)・傅俊(ふしゅん)・岑彭(しんほう)・堅鐔(けんたん)・馮異(ふうい)・王覇(おうは)・朱祜(しゅこ)・任光(じんこう)・祭遵(さいじゅん)・李忠(りちゅう)・景丹(けいたん)・万脩(ばんしゅう)・蓋延(こうえん)・邳肜(ひゆう)・銚期(ようき)・劉植(りゅうしょく)・耿純(こうじゅん)・臧宮(ぞうきゅう)・馬武(ばぶ)・劉隆(りゅうりゅう)と続けた。さらに王常(おうじょう)・李通(りつう)・竇融(とうゆう)・卓茂(たくぼう)を加えて、合わせて三十二人とした。馬援(ばえん)は椒房(しょうぼう、皇后)の親戚であったため、ただ一人この中に加えられなかった〔注:「與」は「預(あずか)る」と読む〕。

④ 夏、四月、辛酉(しんゆう)の日、皇子の建(けん)を千乗王(せんじょうおう)に、羨(せん)を広平王(こうへいおう)に封じた。

⑤ 六月、丁卯(ていぼう)の日、天船(てんせん)星の北に星孛(せいはい、彗星)が現れた。

⑥ 帝は北宮(ほっきゅう)を大々的に造営しはじめた。このとき天は旱(ひでり)であったので、尚書僕射(しょうしょぼくや)で会稽(かいけい)の人である鍾離意(しょうりい)が、闕(けつ、宮門)に参じ、冠を脱いで(罪を待つ姿で)、上疏(じょうそ)して言った。「昔、成湯(せいとう、殷の湯王)は旱に遭ったとき、六つの事がらをもって自らを責めて言いました。『政(まつりごと)が節度を欠いているのか。民を使役して苦しめているのか。宮室を造りすぎているのか。女のとりなし(後宮の口出し)が盛んなのか。賄賂(わいろ)が横行しているのか。讒言(ざんげん)する者がはびこっているのか』と。ひそかに見ますに、北宮の大工事のために、民は農作業の時期を失っております。古来、宮室が小さく狭いことが苦しみだったためしはなく、ただ民が安らげないことこそが心配なのです。どうか今はこの工事をおやめになり、天の心におこたえください」。帝は策詔(さくしょう、ふだに記した詔)で答えて言った。「湯王は六つの事を引いて、その咎(とが)はすべて自分一人にあるとした。そなたは冠と履(くつ)をもとどおりにつけよ。謝るには及ばぬ」〔注:策詔とは、詔をふだに書きつけたものである〕。さらに大匠(だいしょう、将作大匠)に命じて諸宮の工事を止めさせ、急ぎでないものを減らし省いた。詔の中では、(湯王にならって)あわせて公卿・百官にも自らの非を詫び、するとちょうどその時に恵みの雨が降りそそいだ〔注:「澍」とは、万物をうるおし注ぐ雨のことである〕。

鍾離意は全椒(ぜんしょう)の長(ちょう)である劉平(りゅうへい)を推挙し、劉平は詔によって召されて議郎(ぎろう)に任じられた〔注:全椒県は九江郡に属す〕。劉平は全椒にあって、その治政が恩恵に満ちていたので、民の中には、自ら財産を多めに申告して進んで税を納める者や、年齢を少なめに申告して進んで労役に就く者さえいた。刺史(しし)や太守(たいしゅ)が管内を巡視しても、獄(ひとや)につながれた囚人はおらず、人々はそれぞれ満ち足りていて、何を問えばよいかもわからず、ただ詔書(しょうしょ)を布告して去るばかりであった。

帝は生まれつき性質がせまく、こまかいことまで詮索し、人の耳目の及ばぬ隠れた事を暴き出すことを聡明さと思っていた〔注:「隠」とは私(ひそ)かなこと。帝は耳目をもって人の隠れた事をのぞき、これを暴くのを好んだのである〕。公卿や大臣はたびたびそしられ、側近の尚書(しょうしょ)以下の者でさえ、引きずり回されるありさまであった〔注:「提・曳」とは、物を投げつけて打ち、また引きずることをいう〕。あるとき、帝はある件で郎(ろう)の薬崧(やくすう)に腹を立て〔注:「薬」は姓、「崧」は名である〕、杖で打ちかかった。薬崧が寝台の下に逃げ込むと、帝はひどく怒り、激しい口調で言い放った。「郎(=薬崧)、出てこい!」。薬崧はそこで言った。「『天子は穆穆(ぼくぼく)としておごそかに、諸侯は皇皇(こうこう)として堂々と』と申します〔注:『礼記』曲礼の文。穆穆は麗(うるわ)しいさま、皇皇は盛んなさまをいう〕。君主みずから立ち上がって郎を打つなどということは、いまだ聞いたことがございません」。帝はそこで彼を赦した。

このとき、朝廷の者で恐れおののかぬ者はなく、誰もが厳しく細かく振る舞って誅責(ちゅうせき)を避けようとしたが、ただ鍾離意だけがあえて諫(いさ)め争い、たびたび詔書を(承服せず)封じたまま突き返し、臣下に過失があれば、そのつどとりなして救った。たまたま続けざまに天変地異があったので、鍾離意は上疏して言った。「陛下は鬼神(きしん)を敬い畏(おそ)れ、民をいたわり憐れんでおられます。それなのに天の気が和らがず、寒暑(かんしょ)が時節をたがえるのは、その咎は群臣にあります。群臣が教化をひろめて職務をまっとうできず、むごく厳しいことを世のならいとし、百官には互いを親しむ心がなく、役人も民もやわらぎ睦(むつ)む志を持たぬために、ついには和の気を損ない逆らって、天災を招いているのです。民は徳によってこそ従わせられるもので、力で押さえつけるのは難しいものです。『鹿鳴(ろくめい)』の詩がかならず宴(うたげ)の楽しみをうたうのは〔注:『鹿鳴』は『詩経』小雅の篇で、群臣をもてなす宴の詩である〕、人と神の心が和らぎ合って、はじめて天の気も和らぐからです。どうか陛下は聖徳を垂れ、刑罰をゆるめ、時節の気にしたがって陰陽を調えてくださいますよう」。帝は、すぐにはこの言を用いることができなかったものの、その至誠を知り、最後まで鍾離意を愛し厚遇した。

⑦ 秋、八月、戊辰(ぼしん)の日、詔して太楽官(たいがくかん)の名を太予(たいよ)と改めた。これは讖文(しんもん、予言の書)によったものである〔注:『尚書璇璣鈐(しょうしょせんきけん)』に「帝漢(ていかん)出づる有り、徳あまねくして楽を作り、これを予(よ)と名づく」とあり、これにもとづいて改めたのである〕。

⑧ 壬申(じんしん)の晦(みそか、月末)、日食があった。詔して言った。「昔、楚の荘王(そうおう)は災いが無くとも、かえって自ら戒め畏れた。魯の哀公(あいこう)は禍(わざわい)が大きかったのに、天は譴(せ)めを下さなかった。いま起こったこの異変は、もしかするとまだ救えるかもしれぬ。担当の役人たちは、つとめてその職分を思い、徳なきこの私を正してくれよ」。

⑨ 冬、十月、甲子(こうし)の日、帝は皇太后に従って章陵(しょうりょう)に行幸した〔注:光武帝の建武六年、舂陵郷(しょうりょうきょう)を章陵県と改めた〕。荊州刺史(けいしゅうしし)の郭賀(かくが)は、その治政にとりわけ優れた功績があったので〔注:荊州は南陽・南郡・江夏・零陵・桂陽・武陵・長沙などの郡を統べる〕、帝は三公(さんこう)の服である黼黻(ほふつ、刺繍をほどこした礼服)と冕旒(べんりゅう、玉だれのついた礼冠)を賜い、管内を巡視するときには車の前のとばりを取り払うよう命じ、民にその姿と装束を見させて、徳ある者であることを世に明らかにした〔注:「襜帷(せんい)」とは、車の前のとばりである〕。戊辰(ぼしん)の日、章陵から還った。

⑩ この年、都および七つの郡国(ぐんこく)で大水(おおみず)があった。

⑪ 莎車(さしゃ)の王である賢(けん)は、軍事力をもって于窴(うてん)・大宛(だいえん)・嬀塞(きそく)の王国を脅して奪い取り〔注:嬀塞国は、塞種(そくしゅ)のうち嬀水(きすい)のほとりに住む者たちが、これを国名としたものである〕、自分の将を派遣してそれらを守らせた。于窴の人々はその将である君徳(くんとく)を殺し、有力者の休莫霸(きゅうばくは)を立てて王とした。賢は諸国の兵数万を率いてこれを攻めたが、休莫霸にさんざんに打ち破られ、命からがら逃げ帰った。休莫霸は進んで莎車を包囲したが、流れ矢に当たって死んだ。于窴の人々は改めてその兄の子の広徳(こうとく)を立てて王とし、広徳は弟の仁(じん)に命じて賢を攻めさせた。広徳の父は以前から莎車に捕らえられていたが、賢はその父を送り返し、自分の娘を広徳に嫁がせて、和親(わしん)を結んだ。

三年
春，二月，甲寅，太尉趙熹、司徒李訢免。丙辰，以左馮翊郭丹爲司徒。己未，以南陽太守虞延爲太尉。
甲子，立貴人馬氏爲皇后，皇子炟爲太子。
后，援之女也，光武時，以選入太子宮，能奉承陰后，傍接同列，禮則脩備，上下安之，遂見寵異；及帝卽位，爲貴人。時后前母姊女賈氏亦以選入，生皇子炟；帝以后無子，命養之，謂曰：「人未必當自生子，但患愛養不至耳！」后於是盡心撫育，勞悴過於所生。太子亦孝性淳篤，母子慈愛，始終無纖介之間。后常以皇嗣未廣，薦達左右，若恐不及。後宮有進見者，每加慰納；若數所寵引，輒加隆遇。
及有司奏立長秋宮，帝未有所言，皇太后曰：「馬貴人德冠後宮，卽其人也。」后旣正位宮闈，愈自謙肅，好讀書。常衣大練，裙不加緣；朔望諸姬主朝請，望見后袍衣疏粗，以爲綺縠，就視，乃笑。后曰：「此繒特宜染色，故用之耳。」羣臣奏事有難平者，帝數以試后，后輒分解趣理，各得其情，然未嘗以家私干政事。帝由是寵敬，始終無衰焉。
帝思中興功臣，乃圖畫二十八將於南宮雲臺，以鄧禹爲首，次馬成、吳漢、王梁、賈復、陳俊、耿弇、杜茂、寇恂、傅俊、岑彭、堅鐔、馮異、王霸、朱祜、任光、祭遵、李忠、景丹、萬脩、蓋延、邳肜、銚期、劉植、耿純、臧宮、馬武、劉隆；又益以王常、李通、竇融、卓茂，合三十二人。馬援以椒房之親，獨不與焉。
夏，四月，辛酉，封皇子建爲千乘王，羨爲廣平王。
六月，丁卯，有星孛於天船北。
帝大起北宮。時天旱，尚書僕射會稽鍾離意詣闕、免冠、上疏曰：「昔成湯遭旱，以六事自責曰：『政不節邪？使民疾邪？宮室營邪？女謁盛邪？苞苴行邪？讒夫昌邪？』竊見北宮大作，民失農時；自古非苦宮室小狹，但患民不安寧，宜且罷止，以應天心。」帝策詔報曰：「湯引六事，咎在一人，其冠、履，勿謝！」又敕大匠止作諸宮，減省不急。詔因謝公卿百僚，遂應時澍雨。
意薦全椒長劉平，詔徵拜議郎。平在全椒，政有恩惠，民或增貲就賦，或減年從役。刺史、太守行部，獄無繫囚，人自以得所，不知所問，唯班詔書而去。
帝性褊察，好以耳目隱發爲明，公卿大臣數被詆毀，近臣尚書以下至見提曳。常以事怒郎藥崧，以杖撞之；崧走入牀下，帝怒甚，疾言曰：「郎出！」崧乃曰：「天子穆穆，諸侯皇皇，未聞人君，自起撞郎。」帝乃赦之。
是時朝廷莫不悚慄，爭爲嚴切以避誅責，唯鍾離意獨敢諫爭，數封還詔書，臣下過失，輒救解之。會連有變異，上疏曰：「陛下敬畏鬼神，憂恤黎元，而天氣未和，寒暑違節者，咎在羣臣不能宣化治職，而以苛刻爲俗，百官無相親之心，吏民無雍雍之志，至於感逆和氣，以致天災。百姓可以德勝，難以力服，《鹿鳴》之詩必言宴樂者，以人神之心洽，然後天氣和也。願陛下垂聖德，緩刑罰，順時氣以調陰陽。」帝雖不能時用，然知其至誠，終愛厚之。
秋，八月，戊辰，詔改太樂官曰太予，用讖文也。
壬申晦，日有食之。詔曰：「昔楚莊無災，以致戒懼，魯哀禍大，天不降譴。今之動變，儻尚可救，有司勉思厥職，以匡無德！」
冬，十月，甲子，車駕從皇太后幸章陵。荊州刺史郭賀，官有殊政，上賜以三公之服，黼黻、冕旒；敕行部去襜帷，使百姓見其容服，以章有德。戊辰，還自章陵。
是歲，京師及郡國七大水。
莎車王賢以兵威逼奪于窴、大宛、嬀塞王國，使其將守之。于窴人殺其將君德，立大人休莫霸爲王，賢率諸國兵數萬擊之，大爲休莫霸所敗，脫身走還。休莫霸進圍莎車，中流矢死，于窴人復立其兄子廣德爲王，廣德使其弟仁攻賢。廣德父先拘在莎車，賢乃歸其父，以女妻之，與之和親。
