# 卷004 周紀四 — 赧王中三十九年

赧王三十九年〔注:乙酉(きのととり)の年、紀元前二七六年〕。

秦の武安君(ぶあんくん)が魏を攻め、二つの城を攻め落とした。

楚王が東方の地の兵を集めると十万あまりを得て、ふたたび西へ向かい、江南(こうなん)の十五の邑(むら)を取った〔注:東方の地とは、楚の東の境にあたる淮水(わいすい)・汝水(じょすい)のあたりの地である〕。

魏の安釐王(あんきおう)が、その弟の無忌(むき)を信陵君(しんりょうくん)に封じた。

三十九年
秦武安君伐魏，拔兩城。
楚王收東地兵，得十餘萬，復西取江南十五邑。
魏安釐王封其弟無忌爲信陵君。
