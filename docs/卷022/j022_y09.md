# 卷022 漢紀十四 — 世宗孝武皇帝下之下三年

征和三年（前九〇年、辛卯）

① 春、正月、武帝は雍（よう）に行幸し、安定郡・北地郡まで足を延ばした。

② 匈奴が五原郡・酒泉郡に侵入し、二人の都尉を殺した。三月、武帝は李廣利に七万の兵を率いさせて五原から、商丘成に二万を率いさせて西河から、馬通に四万騎を率いさせて酒泉から出撃させ、匈奴を攻撃した。

③ 夏、五月、天下に大赦した。

④ 匈奴の単于は、漢軍が大挙して出撃したと聞くと、輜重（しちょう、物資）をすべて北の郅居水（しっきょすい）のほとりへ移した。〔注：「邸」は「至る」の意。「郅」は「しつ」と読む。〕左賢王は配下の民を駆り立てて余吾水を渡らせ、六、七百里も離れた兜銜山（とかんざん）に移らせた。単于みずからは精兵を率いて姑且水（こしょすい）を渡った。商丘成の軍が到着し、近道（邪径）づたいに匈奴を追ったが、敵の姿は見えず、引き返した。〔注：師古によれば、近道を通って追ったが、匈奴に出会えずに帰った、の意。〕匈奴は大将と李陵に三万余騎を率いさせて漢軍を追わせ、九日にわたって転戦し、蒲奴水（ほどすい）まで至ったが、匈奴側が不利となって引き上げた。馬通の軍が天山に至ると、匈奴は大将の偃渠（えんきょ）に二万余騎を率いさせて漢軍を待ち伏せさせたが、漢軍が強盛なのを見て引き下がり、馬通は戦果も損害もなかった。このとき漢は、車師（しゃし）が馬通軍の進路を遮るのを恐れ、開陵侯・成娩（せいばん）に楼蘭・尉犂・危須など六国の兵を率いさせて、ともに車師を包囲させ、その王と民をことごとく捕らえて帰還した。〔注：危須国は危須城を都とし、長安から七千二百九十里、西は焉耆まで百里。〕貳師将軍（李廣利）が塞を出ると、匈奴は右大都尉と衛律に五千騎を率いさせ、夫羊句山（ふようこうざん）の谷あいで漢軍を待ち伏せして攻撃させたが、貳師はこれを撃破し、勝ちに乗じて敗走する敵を范夫人城（はんふじんじょう）まで追った。〔注：服虔によれば「夫羊」は地名。師古によれば「句山」は西の山。応劭によれば、范夫人城はもと漢の将が築いた城で、その将が亡びる際、妻が残兵を率いて守り抜いたので、その名がついた。張晏によれば、范氏は胡（匈奴）を呪詛できる女であった。〕匈奴は逃げ散り、あえて立ち向かう者はいなかった。

さきに貳師将軍が出征する際、丞相の劉屈氂（りゅうくつもう）が餞別の祭り（祖道）を行い、渭橋まで見送った。〔注：「祖」は道の神を祭る軷（はつ）祭。出立にあたり車馬を整え、酒や干し肉を供えて旅立ちの始まりとする。師古によれば、祖とは送行の祭りで、それにあわせて宴を設ける。昔、黄帝の子・累祖（るいそ）が遠出を好んで道中で死んだので、これを行神（旅の神）として祭るようになったという。〕李廣利は言った。「どうか君侯（あなた）が、早く昌邑王を皇太子に立てるよう願い出てください。もし昌邑王が帝位に立てば、君侯はこの先、何の心配もいりますまい」。〔注：当時、列侯はみな「君侯」と呼ばれ、これは尊称である。〕劉屈氂は承諾した。昌邑王とは、貳師将軍の妹である李夫人の子であり、貳師の娘は劉屈氂の息子の妻になっていた。それゆえ二人はともに昌邑王を立てたいと望んだのである。

ちょうどそのころ、内者令の郭穰（かくじょう）が、「丞相の夫人が武帝を呪詛し、貳師将軍とともに祈祷を行い、昌邑王を帝に立てようとしている」と告発した。〔注：班固の百官表によれば、内者令は少府に属す。また昭帝紀によれば内謁者令の郭穰がいる。内者・謁者にはそれぞれ令と丞があり、いずれも少府に属す。〕取り調べたところ、罪は大逆不道に当たった。六月、詔が下り、劉屈氂を厨車（食事を運ぶ車）に乗せて市中を引き回し〔注：師古によれば、厨車は食べ物を載せる車。「徇」は引き回して見せしめにすること。〕、東市で腰斬（腰から斬る刑）に処し、妻子は華陽街で梟首（さらし首）にした。〔注：「要」は「腰」に同じ。長安城内には八つの大通りがあり、華陽はその一つ。〕貳師将軍の妻子も捕らえられた。貳師はこれを聞いて憂え恐れた。彼の属吏である胡亞夫（こあふ）もまた罪を逃れて従軍していたが、貳師にこう説いた。「奥方もご一家も、みな役人の手に落ちています。もしお帰りになれば、思いどおりになるどころか、ちょうど牢獄に直行となりましょう。さすれば、この郅居水より北の地を、二度と再び目にすることなどできましょうか」。〔注：如淳によれば、いったん誅殺されてしまえば、その後で匈奴に降ろうとしても、もはやかなわぬ、の意。〕貳師はこれによって疑い迷い、敵地深くに攻め入って功を立てようとし、ついに北の郅居水のほとりまで至った。匈奴はすでに退いていたので、貳師は護軍に二万騎を率いさせて郅居水を渡らせた。すると左賢王と左大将が率いる二万騎と出くわし、漢軍と一日戦って、漢軍は左大将を討ち取り、匈奴の死傷者は非常に多かった。軍の長史が、決眭都尉（けっすいとい）の煇渠侯（きょきょこう）と謀って言った。〔注：晋灼によれば、決眭都尉は匈奴の官名という。ただし胡三省は、功臣表で煇渠侯に封じられた雷電が匈奴から降った者ではないことなどから、決眭都尉は匈奴の官ではないと考証する。「煇」は「き（輝）」と読む。〕「将軍は異心を抱き、兵を危険にさらしてまで手柄を求めようとしている。これでは必ず敗れるに違いない」。そして共謀して貳師を捕らえようとした。貳師はこれを聞きつけ、長史を斬り、兵を引き返して燕然山（えんぜんざん）まで至った。〔注：匈奴伝によれば、燕然山は匈奴領内の速邪烏（そくやう）の地にある。〕単于は漢軍が疲れ切っているのを知り、自ら五万騎を率いて貳師を迎え撃ち、双方に多くの死傷者が出た。夜になると、匈奴は漢軍の前方に深さ数尺の塹壕を掘り、後方から急襲したので、漢軍は大混乱に陥った。貳師将軍はついに降伏した。単于はかねてから李廣利が漢の大将であることを知っていたので、自分の娘を娶せ〔注：「妻」はここで「めあわせる（嫁がせる）」の意。〕、その地位を衛律より上に置いて尊び寵愛した。（一方、漢に残った李廣利の）一族はついに族滅された。

⑤ 秋、いなごの害（蝗害）があった。

⑥ 九月、もと城父（じょうほ）の県令であった公孫勇（こうそんゆう）が、食客の胡倩（こせん）らと謀反を企てた。〔注：班固の地理志によれば、城父県は沛郡に属す。「父」は「ほ（甫）」と読む。〕胡倩は光祿大夫を詐称し、盗賊を取り締まる命を受けた使者だと偽った。淮陽太守の田廣明（でんこうめい）がこれを察知し、兵を発して捕らえ斬った。〔注：高祖十一年に淮陽国を置いた。当時は郡で、兗州に属した。唐代の陳州の地。〕公孫勇は繡衣（刺繍入りの官服）を着て四頭立ての馬車に乗り、圉（ぎょ）県まで来たが、圉県の守尉である魏不害（ぎふがい）らがこれを誅殺した。〔注：師古は陳留の圉県とするが、胡三省は地理志により圉県は淮陽に属すとする。〕武帝は魏不害ら四人を侯に封じた。〔注：魏不害は当塗侯、江德は轑陽侯、蘇昌は蒲侯、圉県の小吏は関内侯となり、圉の遺郷を食邑とした。〕

⑦ 役人や民が巫蠱（呪術）のかどで互いに告発し合ったが、取り調べてみると、多くは事実無根であった。武帝は、太子が（江充に追い詰められて）恐れおののいただけで他意はなかったことを、しだいに悟っていた。〔注：江充に迫られ、おびえて身の証も立てられず、やむなく兵を挙げて江充を殺しただけで、他意はなかった、の意。〕ちょうどそのころ、高廟の守衛の郎（高寢郎）である田千秋（でんせんしゅう）が緊急の上奏（急変）をして、太子の冤罪を訴えて言った。〔注：師古によれば、高廟の衛寝の郎。通常でないことを告げるので「急変」という。〕「子が父の兵を勝手に動かしたとて、罪はせいぜい笞打ち程度。天子の子が誤って人を殺めたとて、いったい何の罪に当たりましょうか。私はかつて、白髪の老翁が夢に現れ、こう告げよと教えてくれたのです」。武帝はここで大いに心を打たれて目が覚める思いがし、田千秋を召し出して言った。「父子のあいだのことは、人が口を出しにくいものだ。それをそなただけが、太子に罪のなかったことを明らかにしてくれた。これは高廟の神霊が、そなたを使わして私を諭させたのに違いない。そなたこそ、私の輔佐となるべき人物だ」。そして即座に田千秋を大鴻臚に任じた。〔注：師古によれば、（上奏を聞いて）その場ですぐ任命したので、時を移さなかったことをいう。〕一方で江充の一族を族滅し、蘇文（そぶん）を横橋の上で焼き殺した。〔注：すなわち横門の外の渭橋。「橫」は「こう（光）」と読む。〕また、泉鳩里で太子に刃を向けた者は、初め北地太守に取り立てられたが、のちに族滅された。武帝は太子が無実の罪で死んだことを哀れみ、思子宮（子を思う宮）を建て、湖県に「歸来望思の台（帰り来たれと望み思う台）」を築いた。〔注：師古によれば、自分が望み見て太子を思い、せめてその魂が帰り来るようにと願う意。その台は今の湖城県の西、閺郷県の東にあり、土台が今も残っている。〕天下の人々はこれを聞いて悲しんだ。

三年
春，正月，上行幸雍，至安定、北地。
匈奴入五原、酒泉，殺兩都尉。三月，遣李廣利將七萬人出五原，商丘成將二萬人出西河，馬通將四萬騎出酒泉，擊匈奴。
夏，五月，赦天下。
匈奴單于聞漢兵大出，悉徙其輜重北邸郅居水；左賢王驅其人民度余吾水六七百里，居兜銜山；單于自將精兵渡姑且水。商丘成軍至，追邪徑，無所見，還。匈奴使大將與李陵將三萬餘騎追漢軍，轉戰九日，至蒲奴水；虜不利，還去。馬通軍至天山，匈奴使大將偃渠將二萬餘騎要漢兵，見漢兵強，引去；通無所得失。是時，漢恐車師遮馬通軍，遣開陵侯成娩將樓蘭、尉犂、危須等六國兵共圍車師，盡得其王民衆而還。貳師將軍出塞，匈奴使右大都尉與衞律將五千騎要擊漢軍於夫羊句山陿，貳師擊破之，乘勝追北至范夫人城；匈奴奔走，莫敢距敵。
初，貳師之出也，丞相劉屈氂爲祖道，送至渭橋。廣利曰：「願君侯早請昌邑王爲太子；如立爲帝，君侯長何憂乎！」屈氂許諾。昌邑王者，貳師將軍女弟李夫人子也；貳師女爲屈氂子妻，故共欲立焉。會內者令郭穰告「丞相夫人祝詛上及與貳師共禱祠，欲令昌邑王爲帝」，按驗，罪至大逆不道。六月，詔載屈氂廚車以徇，要斬東市，妻子梟首華陽街；貳師妻子亦收。貳師聞之，憂懼，其掾胡亞夫亦避罪從軍，說貳師曰：「夫人、室家皆在吏，若還，不稱意適與獄會，郅居以北，可復得見乎！」貳師由是狐疑，深入要功，遂北至郅居水上。虜已去，貳師遣護軍將二萬騎度郅居之水，逢左賢王、左大將將二萬騎，與漢兵合戰一日，漢軍殺左大將，虜死傷甚衆。軍長史與決眭都尉煇渠侯謀曰：「將軍懷異心，欲危衆求功，恐必敗。」謀共執貳師。貳師聞之，斬長史，引兵還至燕然山。單于知漢軍勞倦，自將五萬騎遮擊貳師，相殺傷甚衆；夜，塹漢軍前，深數尺，從後急擊之，軍大亂；貳師遂降。單于素知其漢大將，以女妻之，尊寵在衞律上。宗族遂滅。
秋，蝗。
九月，故城父令公孫勇與客胡倩等謀反，倩詐稱光祿大夫，言使督盜賊；淮陽太守田廣明覺知，發兵捕斬焉。公孫勇衣繡衣、乘駟馬車至圉；圉守尉魏不害等誅之。封不害等四人爲侯。
吏民以巫蠱相告言者，案驗多不實。上頗知太子惶恐無他意，會高寢郎田千秋上急變，訟太子冤曰：「子弄父兵，罪當笞。天子之子過誤殺人，當何罪哉！臣嘗夢一白頭翁敎臣言。」上乃大感寤，召見千秋，謂曰：「父子之間，人所難言也，公獨明其不然。此高廟神靈使公敎我，公當遂爲吾輔佐。」立拜千秋爲大鴻臚，而族滅江充家，焚蘇文於橫橋上；及泉鳩里加兵刃於太子者，初爲北地太守，後族。上憐太子無辜，乃作思子宮，爲歸來望思之臺於湖，天下聞而悲之。
