# 卷007 秦紀二 — 始皇帝下三十一年

三十一年〔注:乙酉(きのととり)の年、紀元前二一六年〕。

人民に、自分の田地を自己申告させた〔注:始皇は二十六年に民の呼び名を「黔首(けんしゅ)」=黒い布で頭を覆う者、と改めていた〕。

---

<details>
<summary>原文を表示</summary>

三十一年
使黔首自實田。
