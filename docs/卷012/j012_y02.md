# 卷012 漢紀四 — 太祖高皇帝下九年

九年〔注:癸卯(みずのとう)の年、紀元前一九八年〕。

冬。主上は、庶民の家から娘をもらい受けてこれを長公主と名づけ〔注:外の庶民の家から女をもらって、これを公主と名づけたのである〕、単于(冒頓)に嫁がせた。劉敬(りゅうけい)を遣わして、和親の盟約を結ばせた。

臣光(司馬光)が論じる――建信侯(けんしんこう=劉敬)は、冒頓は残忍非道な者で仁義では説き伏せられぬと言いながら、その同じ口で、冒頓と婚姻を結ぼうとした。なんと前と後とで言うことが食い違っていることか。骨肉の情(肉親の情愛)や、尊卑の序(けじめ)というものは、ただ仁義をわきまえた人だけが理解できるものだ。それなのに、どうしてこんなもので冒頓を従わせようなどというのか。そもそも上代の帝王が夷狄(いてき=異民族)を御(ぎょ)するには、服従すれば徳をもってこれを懐(なつ)け、背けば威をもってこれを震え上がらせたのであって、婚姻を結んだなどとは聞いたことがない。それに冒頓は、自分の父を禽獣(きんじゅう)のように見なして狩り殺した男だ。妻の父(舅〈しゅうと〉)ごときを、何とも思うはずがない。建信侯の策は、もとよりお粗末きわまる。まして魯元(ろげん=魯元公主)はすでに趙の后(きさき)となっているのに、それをまた奪い取ってよいものか。

劉敬は匈奴から帰ってくると、こう進言した。「匈奴の河南(かなん)にいる白羊王(はくようおう)・樓煩王(ろうはんおう)は、長安から近いところでは七百里。身軽な騎兵なら一日一夜で秦中(しんちゅう)に達することができます〔注:秦中とは関中(かんちゅう)をいう。もとの秦の地〕。関中は戦乱で破れたばかりで〔注:新破とは、戦乱を経て、まだ豊かに満ちていないこと〕、民が少なく、土地は肥えております。さらに人を移して充実させるべきです。そもそも諸侯がはじめて挙兵したときには、斉の田氏(でんし)一族、楚の昭氏(しょうし)・屈氏(くつし)・景氏(けいし)でなければ、立ち上がることはできませんでした〔注:斉の王族は諸田、楚の王族は昭・屈・景。いずれも二国の有力な一族〕。いま陛下は関中を都とされてはいますが、実は民が少なく、東には六国の強大な氏族がおります。ひとたび異変が起これば、陛下とて枕を高くして眠ることはできますまい。臣は願わくは、陛下が六国の王の子孫や、豪傑(ごうけつ)・名家を関中に移り住まわせますよう。事なきときは胡(こ=匈奴)への備えとなり、諸侯に異変があれば、彼らを率いて東を伐つこともできます。これこそ本(もと)を強くして末(すえ)を弱める術でございます」。主上は「よし」と言った。十一月、斉・楚の大族である昭氏・屈氏・景氏・懷氏(かいし)・田氏の五族と、豪傑たちを関中に移し、便利のよい田地・屋敷を与えた〔注:都合のよい田畑・屋敷をいう〕。あわせて十余万人であった。

十二月、主上は洛陽(らくよう)へ行幸した。

貫高(かんこう)を恨む家の者が、その陰謀を知って、朝廷に異変として密告した〔注:謀とは、主上を弑(しい)そうとする企てをいう。この件のはじまりは前の巻の七年にある。変とは尋常でないこと、すなわち尋常ならざる事を訴え出ること〕。そこで主上は、趙王(張敖)および謀反に加わった者たちを、みな逮捕させた。趙午(ちょうご)ら十余人は、われ先にと自ら頸(くび)を切って死のうとした。だが貫高だけは怒って罵った。「誰がお前たちにそうしろと言った。いま王はまことに何も企んではおられぬのに、王までいっしょに捕らえられたのだ。お前たちがみな死んでしまえば、誰が王の無実を明かすのだ」。そこで彼は、囲い車(かこいぐるま)に密封されて閉じ込められ〔注:轞車(かんしゃ)とは、車を檻(おり)の形にして四方を板で囲い、中が見えないようにしたもの。膠致とは、すきまなく密封して開けられないようにし、都へ送り届けること〕、趙王とともに長安へ護送された。貫高は取り調べに答えて言った。「やったのは我ら一味だけのこと。王はまことに何もご存じない」。役人が責め問い、笞(むち)で数千回も打ち、錐(きり)で刺し〔注:剟(てつ)も刺すこと。鉄で刺すのである〕、体じゅう打てる場所がなくなるほどであったが、貫高はとうとう、それ以上は何も言わなかった。呂后(りょこう)はたびたび「張王は公主(=魯元公主)の夫なのですから、こんなことをするはずがありませぬ」と言った。だが主上は怒って言った。「もし張敖が天下を握ったとて、お前の娘くらいの女に事欠こうか」。そう言って聞き入れなかった。

廷尉(ていい)が貫高の供述を主上に報告した。主上は言った。「立派な男だ。誰かあの男を知る者はおらぬか。私的なつてで尋ねさせよう」〔注:貫高と平素から親しい者を求め、私的に問わせようとしたのである〕。中大夫(ちゅうたいふ)の泄公(せつこう)が申し出た〔注:泄(せつ)は姓〕。「あの者は私と同郷の出で、かねてよく知っております。あれはもともと趙国で、義によって身を立て、人から侵されることをよしとせず、ひとたび諾(うべな)ったことは必ず守る男です」。主上は泄公に節(せつ)を持たせて遣わし、箯輿(べんよ)に横たわる貫高のもとへ行って問わせた〔注:箯輿とは、竹や木を編んで輿(こし)の形にしたもの。貫高は責め苦で弱りきっていたので、これに乗せられていた〕。泄公は貫高とねぎらい合い、旧知の仲のように打ち解けて〔注:相労とは、互いにその苦労を問うこと〕、そのうえで尋ねた。「張王には、はたして謀(はかりごと)があったのか」。貫高は言った。「人の情として、誰しも自分の父母や妻子をいとおしまぬ者があろうか。いまわしは三族(さんぞく)がみな罪に問われて死を科せられている〔注:罪に問われて死を抵(あ)てられること〕。それでも、王を自分の肉親以上にいとおしむことがあろうか。ただ、王にはまことに謀反の心はなく、やったのは我らだけなのだ」。そう言って、ことの本(もと)の趣旨と、王が何も知らなかったいきさつを、つぶさに語った。そこで泄公は宮中に入り、すべてを主上に報告した。春、正月、主上は趙王張敖を赦し、廃して宣平侯(せんぺいこう)とし、代王(だいおう)の如意(じょい)を移して趙王とした。

主上は貫高の人となりを立派だと思い、泄公にこと細かに告げさせた。「張王はすでに釈放された」。そうして貫高を赦した。貫高は喜んで言った。「わが王は、本当に出られたのか」。泄公は「そうだ」と答えた。泄公は言った。「主上はそなたを高く買っておられる。それゆえそなたを赦されたのだ」。貫高は言った。「わしが死なず、この身に傷の残らぬところまで責め苦を受けながらも生きながらえてきたのは、張王の無実を明かすためだ。いま王はすでに出られた。わしの責めは果たした。死んでも思い残すことはない。それに、人の臣下でありながら、主君を弑(しい)し位を簒(うば)おうとした汚名を負った身。どの面目(めんぼく)があって、ふたたび主上にお仕えできようか。たとえ主上がわしを殺さずとも、わしの心が恥じずにおられようか」。そうして、仰向(あおむ)けに喉(のど)の頸(くび)の大脈を断ち切って死んだ〔注:亢(こう)とは頸(くび)の大脈、すなわち喉のこと〕。

荀悅(じゅんえつ)が論じる――貫高は、はじめに乱の謀(はかりごと)をめぐらした、主君を弑(しい)そうとする賊である。たとえ自分の王の無実を証し立てられたとしても、小さな信義(=主君を守った忠)は、大逆(たいぎゃく)を埋め合わせることはできず、私的な善行は、公(おおやけ)の罪を贖(あがな)うことはできない。『春秋』の義(大義)は「居正(きょせい)を大なりとする(正道に居ることを最も重んじる)」のであって〔注:大居正とは、正に居ることを大とすること〕、その罪は赦さなくてよいのだ。

臣光(司馬光)が論じる――高祖は、驕(おご)って臣下(の心)を失い、貫高は、強情(ごうじょう)によって主君(の国)を亡ぼした。貫高に謀反を起こさせたのは、高祖の過ちである。張敖に国を失わせたのは、貫高の罪である。

詔(みことのり)して言った。「丙寅(ひのえとら)の日より前の罪は、殊死(しゅし=斬刑)より下のものは、みな赦す」。

二月、主上は洛陽から都へ帰り着いた。

これより前、主上は詔を下していた。「趙の群臣・賓客で、あえて張王に付き従う者は、みな一族もろとも誅す」。ところが郎中(ろうちゅう)の田叔(でんしゅく)・孟舒(もうじょ)は、みずから髡鉗(こんけん=髪を剃り、首かせをはめられた罪人)の姿となって、趙王家の奴隷となり、王に付き従った。張敖が罪を免れると、主上は田叔・孟舒らを立派だと思った。召し出して語り合ってみると、漢の朝廷の臣下で、彼らに勝る者はいなかった〔注:古くは右を尊しとした。才能の右に出る者がない、という意〕。主上は彼らをことごとく郡守(ぐんしゅ)や諸侯の相(しょう)に任じた。

夏、六月の晦日(みそか)、日食があった。

あらためて、丞相(じょうしょう)の蕭何(しょうか)を相国(しょうこく)とした〔注:丞相から相国に進むのは、相国の位が丞相より尊いからである〕。

九年
冬，上取家人子名爲長公主，以妻單于。使劉敬往結和親約。
臣光曰︰建信侯謂冒頓殘賊，不可以仁義說，而欲與爲婚姻，何前後之相違也！夫骨肉之恩，尊卑之敍，唯仁義之人爲能知之；柰何欲以此服冒頓哉！蓋上世帝王之御夷狄也，服則懷之以德，叛則震之以威，未聞與爲婚姻也。且冒頓視其父如禽獸而獵之，奚有於婦翁！建信侯之術，固已疏矣；況魯元已爲趙后，又可奪乎！
劉敬從匈奴來，因言︰「匈奴河南白羊、樓煩王，去長安近者七百里，輕騎一日一夜可以至秦中。秦中新破，少民，地肥饒，可益實。夫諸侯初起時，非齊諸田、楚昭、屈、景莫能興。今陛下雖都關中，實少民，東有六國之強族；一日有變，陛下亦未得高枕而臥也。臣願陛下徙六國後及豪桀、名家居關中；無事可以備胡，諸侯有變，亦足率以東伐。此強本弱末之術也。」上曰︰「善！」十一月，徙齊、楚大族昭氏、屈氏、景氏、懷氏、田氏五族及豪桀於關中，與利田、宅，凡十餘萬口。
十二月，上行如洛陽。
貫高怨家知其謀，上變告之。於是上逮捕趙王及諸反者。趙午等十餘人皆爭自剄；貫高獨怒罵曰︰「誰令公爲之？今王實無謀，而幷捕王。公等皆死，誰白王不反者？」乃轞車膠致，與王詣長安。高對獄曰︰「獨吾屬爲之，王實不知。」吏治，搒笞數千，刺剟，身無可擊者；終不復言。呂后數言︰「張王以公主故，不宜有此。」上怒曰︰「使張敖據天下，豈少而女乎！」不聽。
廷尉以貫高事辭聞。上曰︰「壯士！誰知者？以私問之。」中大夫泄公曰︰「臣之邑子，素知之，此固趙國立義不侵、爲然諾者也。」上使泄公持節往問之箯輿前。泄公與相勞苦，如生平驩，因問︰「張王果有計謀不？」高曰︰「人情寧不各愛其父母、妻子乎？今吾三族皆以論死，豈愛王過於吾親哉？顧爲王實不反，獨吾等爲之。」具道本指所以爲者、王不知狀。於是泄公入，具以報上。春，正月，上赦趙王敖，廢爲宣平侯，徙代王如意爲趙王。
上賢貫高爲人，使泄公具告之曰︰「張王已出。」因赦貫高。貫高喜曰︰「吾王審出乎？」泄公曰︰「然。」泄公曰︰「上多足下，故赦足下。」貫高曰︰「所以不死、一身無餘者，白張王不反也。今王已出，吾責已塞，死不恨矣。且人臣有篡弑之名，何面目復事上哉！縱上不殺我，我不愧於心乎！」乃仰絕亢，遂死。
荀悅論曰︰貫高首爲亂謀，殺主之賊；雖能證明其王，小亮不塞大逆，私行不贖公罪。《春秋》之義大居正，罪無赦可也。
臣光曰︰高祖驕以失臣，貫高狠以亡君。使貫高謀逆者，高祖之過也；使張敖亡國者，貫高之罪也。
詔︰「丙寅前有罪，殊死已下，皆赦之。」
二月，行自洛陽至。
初，上詔︰「趙羣臣賓客敢從張王者，皆族。」郎中田叔、孟舒皆自髡鉗爲王家奴以從。及張敖旣免，上賢田叔、孟舒等。召見，與語，漢廷臣無能出其右者。上盡拜爲郡守、諸侯相。
夏，六月晦，日有食之。
更以丞相何爲相國。
