# 卷001 周紀一 — 安王八年

安王八年〔注:丁亥(ていがい)の年、紀元前三九四年〕。

斉が魯を攻め、最(さい)を奪い取った。

鄭の負黍(ふしょ)が背いて、ふたたび韓のものとなった〔注:負黍はもと韓の地で、鄭の繻公十六年に韓を破って奪っていた〕。

---

<details>
<summary>原文を表示</summary>

八年
齊伐魯，取最。
鄭負黍叛，復歸韓。
