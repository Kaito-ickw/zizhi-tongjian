# 卷007 秦紀二 — 始皇帝下二十四年

二十四年〔注:戊寅(つちのえとら)の年、紀元前二二三年〕。

王翦(おうせん)と蒙武(もうぶ)が楚王の負芻(ふすう)を捕らえ、その地に楚郡(そぐん)を置いた〔注:楚はこうして滅んだ。秦の三十六郡に楚郡は無く、これは楚を滅ぼしたときに一時的に置いただけで、のちに九江(きゅうこう)・鄣(しょう)・會稽(かいけい)の三郡に分けられた〕。

---

<summary>原文を表示</summary>

二十四年
王翦、蒙武虜楚王負芻，以其地置楚郡。
