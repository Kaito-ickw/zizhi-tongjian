# 卷004 周紀四 — 赧王中二十二年

赧王二十二年〔注:戊辰(ぼしん)の年、紀元前二九三年〕。

韓の公孫喜(こうそんき)と魏の軍が、秦を攻めた〔注:魏については将軍の名を記さず単に「魏人」とあるのは、その将の地位が低かったためである〕。(秦の)穰侯(じょうこう)は、左更(さこう)の白起(はくき)を秦王に推薦し、向壽(しょうじゅ)に代えて軍を率いさせた。白起は魏・韓の軍を伊闕(いけつ)で破り、二十四万の首を斬り、公孫喜を捕らえ、五つの城を抜いた。秦王は白起を国尉(こくい)に任じた〔注:戦国の世には国尉と郡尉があった〕。

秦王は楚王に書状を送って言った。「楚は秦に背いた。秦は諸侯を率いて楚を討とうとしている。願わくは王よ、士卒を整え、一度心ゆくまで戦おうではないか」〔注:樂(らく)は快(こころよ)いの意で、一戦して気を晴らそうというのである〕。楚王はこれを憂え、そこで再び秦と和親を結んだ。

二十二年
韓公孫喜、魏人伐秦。穰侯薦左更白起於秦王以代向壽將兵，敗魏師、韓師於伊闕，斬首二十四萬級，虜公孫喜，拔五城。秦王以白起爲國尉。
秦王遺楚王書曰：「楚倍秦，秦且率諸侯伐楚，願王之飭士卒，得一樂戰！」楚王患之，乃復與秦和親。
