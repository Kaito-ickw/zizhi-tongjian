# 卷003 周紀三 — 赧王上十年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上十年 ・ 西暦: 305 BCE

[← 巻インデックス](README.md)

---

十年〔注:丙辰(ひのえたつ)の年、紀元前三〇五年〕。

①彗星(すいせい)が現れた。

②趙王は中山(ちゅうざん)を伐ち、丹丘(たんきゅう)・爽陽(そうよう)・鴻(こう)の塞(とりで)を取り、さらに鄗(こう)・石邑(せきゆう)・封龍(ほうりゅう)・東垣(とうえん)を取った。中山は四つの邑(まち)を献上して和を請うた。

③秦の宣太后(せんたいこう)には、父違いの弟に穰侯(じょうこう)魏冉(ぎぜん)、同じ父の弟に華陽君(かようくん)羋戎(びじゅう)がいた。王(昭王)の同母の弟には高陵君(こうりょうくん)・涇陽君(けいようくん)がいた。なかでも魏冉が最も賢く、惠王(惠文王)・武王の時代から官職に就いて政務を執っていた〔注:惠王とは惠文王、昭王とは昭襄王のことである〕。武王が薨(こう)じると、弟たちが王位を争ったが、昭王を擁立する力があったのは魏冉だけであった。昭王が即位すると、魏冉を将軍として咸陽(かんよう)を守らせた。この年、庶長(しょちょう)の壯(そう)および大臣や公子たちが反乱を企てたので、魏冉はこれを誅殺した。惠文后(けいぶんこう)に至るまで皆まともな死に方ができず〔注:惠文后は昭王の嫡母である。天命をまっとうして死ぬことを良死という〕、悼武王后(とうぶおうこう)は秦を出て魏に身を寄せた〔注:悼武王后とは秦の武王の后(きさき)で、昭王の兄嫁にあたる〕。王の兄弟で好ましからぬ者は、魏冉がことごとく滅ぼした。王が幼かったので、宣太后みずから政務を執り、魏冉を任じて政(まつりごと)をとらせ、その威勢は秦の国じゅうを震わせた。

十年
彗星見。
趙王伐中山，取丹丘、爽陽、鴻之塞，又取鄗、石邑、封龍、東垣。中山獻四邑以和。
秦宣太后異父弟曰穰侯魏冉，同父弟曰華陽君羋戎；王之同母弟曰高陵君、涇陽君。魏冉最賢，自惠王、武王時，任職用事。武王薨，諸弟爭立，唯魏冉力能立昭王。昭王卽位，以魏冉爲將軍，衞咸陽。是歲，庶長壯及大臣、諸公子謀作亂，魏冉誅之；及惠文后皆不得良死，悼武王后出居于魏，王兄弟不善者，魏冉皆滅之。王少，宣太后自治事，任魏冉爲政，威震秦國。
