# 卷034 漢紀二十六 — 孝哀皇帝中三年

三年〔注:丁卯(ひのとう)の年、紀元前四年〕。

春正月、広徳夷王(こうとくいおう)の弟・劉広漢(りゅうこうかん)を広平王(こうへいおう)に立てた〔注:広徳夷王・劉雲客(りゅううんかく)は成帝の鴻嘉(こうか)二年に封ぜられたが、二年後に死んで跡継ぎがなかった。いま広漢を立てて、中山靖王(ちゅうざんせいおう)の祭祀を継がせたのである〕。

帝太太后(ていたいたいこう)の住む桂宮(けいきゅう)の正殿が火災になった。

帝は使者を遣わして丞相の平当(へいとう)を召し、これを列侯に封じようとした。だが平当は病が重く、召しに応じなかった。家の者が平当に言った。「無理にでも起き上がって侯の印を受けられぬのですか。子孫のためにも」。平当は言った。「わしは高い位にありながら、すでに『素餐(そさん=禄を食(は)むばかりで功のないこと)』の責めを負っておる。いま起き上がって侯の印を受け、また臥(ふ)せって死んだなら、死んでもなお余りある罪というもの。いま起き上がらぬのは、ほかでもない、子孫のためなのだ」。そして上書して辞職を願い出たが、帝は許さなかった。三月、己酉(つちのととり)の日、平当は薨(こう)じた。

彗星が河鼓(かこ)の星座に現れた〔注:河鼓は牽牛の北にある星座である〕。

夏四月、丁酉(ひのととり)の日、王嘉(おうか)を丞相とし、河南太守(かなんたいしゅ)の王崇(おうすう)を御史大夫とした。王崇は、もと京兆尹(けいちょういん)であった王駿(おうしゅん)の子である。王嘉は、当節の政治が苛酷でせわしなく、郡国の太守・国相がたびたび更迭されるのを憂え、上疏(じょうそ)して言った。「私の聞くところ、聖王の功業は人材を得ることにございます。孔子も『材(=賢才)は得がたい。まことにそのとおりではないか』と申しました。ですから『世を継いで諸侯を立てるのは、その先祖の賢にあやかる(象(かたど)る)ためだ』というのです。たとえその子孫がみな賢いとは限らずとも、天子が彼らのために臣を選び、命卿(めいけい=天子が任命する卿)を立てて補佐させてきました。その国に居て、代々その地位を尊び重んじてこそ、士や民の多くが付き従い、それによって教化が行きわたり、治政の功が立つのです。いまの郡守(=太守)は、古(いにしえ)の諸侯よりも重い任です。昔は賢材を極力えりすぐりました。賢材は得がたいので、用いるに足る者を抜擢するには、ときに囚人の中から登用することさえありました。昔、魏尚(ぎしょう)は事件に座(かか)わって獄につながれましたが、文帝は馮唐(ふうとう)の言葉に感じ入り、使者に節(せつ)を持たせてその罪を赦し、雲中太守(うんちゅうたいしゅ)に任じました。匈奴は彼を恐れはばかったものです。武帝は韓安国(かんあんこく)を囚人の中から抜擢し、梁内史(りょうないし)に任じました。それによって骨肉(=梁孝王)の身も安泰となったのです〔注:この一件は本来、景帝の時のことで、『武帝』は『景帝』の誤りである〕。張敞(ちょうしょう)が京兆尹であったとき、罪があって罷免されるはずでしたが、ずる賢い小役人がそれを知って張敞にたてついたので、張敞はこれを捕らえて殺しました。その役人の家が冤罪を訴え出て、使者が事件を再調査し、張敞が人を不当に殺した(賊殺)と弾劾しました。張敞の逮捕を求める上奏がなされましたが、天子はその上奏を取り上げず、ちょうどその折に張敞は別件で罷免されました。逃亡して十数日のうちに、宣帝は張敞を召し出して冀州刺史(きしゅうしし)に任じ、ついにその働きを得たのです。先の世が、この三人(魏尚・韓安国・張敞)をえこひいきしたのではありません。その才覚が公(おおやけ)のために役立つことを求めたのです。孝文帝の時には、官に居る役人が、その地位のままで子や孫を育て上げ〔注:役人の中には、その官職をそのまま氏(うじ)とする者もあった〕、『倉氏(そうし)』『庫氏(こし)』というのは、倉(くら)や庫(くら)の役人の子孫が官職を姓にしたものです。二千石の長吏(高位の役人)も、その官に安んじ職を楽しみました。そうしてこそ、上下が互いに信頼し合い、その場しのぎ(苟且)の心を持つ者がいなくなるのです。その後、しだいに制度が変わり、公卿以下が次々と互いをせきたて、また政事をたびたび改めるようになりました。司隷校尉や部刺史が、些細なことまで取り立てて弾劾し、人の隠しごと(陰私)を暴き立てました。役人は、ときに数か月で官を退き、去る者を送り新たな者を迎えるため、道は人の行き来でごった返しました。中ほどの才の者は、ただ無難にとりつくろって身を全うしようとし、部下を強く統率できず、下の才の者は、罪を恐れてわが身のことばかり考え、ひたすら私利を図る者が多くなりました。二千石の高官はますます軽んじられ、役人も民も彼らをあなどり、ときにそのわずかな過失を取り上げて、ふくらませて罪に仕立て、司隷・刺史に告げ、あるいは上書して告発しました。人々は、高官たちが簡単に失脚させられると知り、少しでも意に染まぬことがあれば、背こうとする心を抱くようになりました。先に、山陽(さんよう)の逃亡囚・蘇令(それい)らがほしいままに暴れ回ったとき、役人や兵士は危難に臨んでも、だれ一人として節義に殉じて死のうとはしませんでした。それは、太守や国相の威権が、もとから奪われていたからです。孝成皇帝はこれを悔いて詔を下し、二千石の高官を『故縦(ことさらに罪人を見逃した)』のかどでは咎めぬこととし(=寛大に扱い)、使者を遣わして金を賜い、その心を手厚く慰めました。まことに、国家に急があれば二千石の高官にこそ事を委ねるのであり、二千石が重んじられて簡単には危うくされぬからこそ、部下を使いこなせるのです。孝宣皇帝は、よく民を治める役人を大切にされ、その役人を弾劾する上奏があっても宮中にとどめおいて取り上げず、たまたま恩赦があればいっさいを不問にされました。先例では、尚書はめったに告発の上奏文を担当官へ下しませんでした。それは、百姓をわずらわせ、取り調べのために証人を引き合いに出して拘束し、あげくに獄中で死ぬ者まで出るからです。告発の文に必ず『あえてこれを告げる(敢告之)』という文言があってはじめて、これを下したのです。どうか陛下、賢者を選ぶことに心をとどめ、善(よ)い点を覚えて過ちを忘れ、臣下を寛容に受け入れ、完全(備)を求めて責めないでください。二千石・部刺史・三輔(さんぽ)の県令で、才あって職にたえる者は、人情として過失がないとは限りません。大目に見て(闊略)、力を尽くす者が励みを得られるようにすべきです。これこそ、いま差し迫った急務であり、国家の利益です。先に蘇令の乱が起きたとき、大夫を遣わして賊を追わせ、その状況を調べさせようとしましたが、その時、現任の大夫には使いに立てられる者がおらず、盩厔(ちゅうちつ)の県令・尹逢(いんほう)を召して諫大夫に任じ、これを遣わしました。いま大夫の中で才能ある者は非常に少なうございます。あらかじめ、ものになりそうな者を養い育てておくべきです。そうすれば、士は危難に駆けつけて、その死を惜しまなくなります。事に臨んでにわかに求めるのは、朝廷に人材がいることを示すゆえんではありません」。王嘉はそこで、儒者の公孫光(こうそんこう)・満昌(まんしょう)と、有能な役人の蕭咸(しょうかん)・薛脩(せつしゅう)を推薦した。みな、もと二千石で名声のある者であった。天子はこれを受け入れて用いた。

六月、魯頃王(ろけいおう)の子で郚郷侯(ごきょうこう)の劉閔(りゅうびん)を魯王に立てた〔注:本文の「部郷」は「郚郷」が正しい〕。

帝は病が床について癒えなかったため、冬十一月、壬子(みずのえね)の日、太皇太后に詔を下させて、甘泉(かんせん)の泰畤(たいじ)と汾陰(ふんいん)の后土(こうど)の祠を復活させ、(都の)南郊・北郊の祭祀を廃した。帝もまた、みずから甘泉や河東(かとう)へ赴くことができず、担当の官を遣わして事を執り行わせ、礼にのっとって祀(まつ)らせた。

無塩(むえん)の危山(きざん)で、土がひとりでに盛り上がって草を覆い、まるで馳道(ちどう=天子の通る道)のような形になった〔注:無塩県は東平国に属する〕。また、瓠山(こざん)で石がひとりでに転がって立ち上がった。東平王(とうへいおう)劉雲(りゅううん)と、その后(きさき)である謁(えつ)とが、みずからその石のある所へ行って祭った。さらに宮中で石を加工して瓠山の立石をかたどり、倍草(ばいそう=黄倍草)を束ねて、あわせてこれを祀った。河内(かだい)の息夫躬(そくふきゅう)と、長安の孫寵(そんちょう)とが相談してこれを告発しようと謀り、言った。「これは侯に封ぜられるための策になるぞ」。そこで中郎(ちゅうろう)の右師譚(ゆうしたん)とともに、中常侍(ちゅうじょうじ)の宋弘(そうこう)を介して、謀反の変事として上奏し、これを告発した。このとき帝は病にかかっており、不吉に思って忌み嫌うことが多かったので、この件を担当官に下し、王の后である謁を捕らえて獄に下し、取り調べさせた。謁は罪を認めた。「祭祀をして帝を呪い、劉雲のために天子の位を求めました。石が立ち上がったのは、かつて宣帝が微賤の身から身を起こしたしるしの再来だと考えたのです」〔注:石が立つことを帝位の前ぶれとする話は、卷二十三、昭帝の元鳳三年に見える〕。担当官は東平王の処刑を願い出たが、詔が下り、死罪は免じて王位を廃し、房陵(ぼうりょう)へ流すこととなった。劉雲は自殺した。謁と、その母方の伯父である伍宏(ごこう)、それに成帝の母方の伯父・安成共侯(あんせいきょうこう)王崇の夫人である放(ほう)は、みな市(いち)で処刑された(棄市)。この件は御史大夫の王崇にも累が及び、王崇は大司農(だいしのう)に左遷された。孫寵を抜擢して南陽太守(なんようたいしゅ)とし、右師譚を潁川都尉(えいせんとい)とし、宋弘・息夫躬はいずれも光禄大夫(こうろくたいふ)・左曹(さそう)・給事中(きゅうじちゅう)とした。

三年
春，正月，立廣德夷王弟廣漢爲廣平王。
帝太太后所居桂宮正殿火。
上使使者召丞相平當，欲封之；當病篤，不應。室家或謂當：「不可強起受侯印爲子孫邪？」當曰：「吾居大位，已負素餐責矣；起受侯印，還臥而死，死有餘罪。今不起者，所以爲子孫也！」遂上書乞骸骨，上不許。三月，己酉，當薨。
有星孛于河鼓。
夏，四月，丁酉，王嘉爲丞相，河南太守王崇爲御史大夫。崇，京兆尹駿之子也。嘉以時政苛急，郡國守相數有變動，乃上疏曰：「臣聞聖王之功在於得人，孔子曰：『材難，不其然與！』故『繼世立諸侯，象賢也。』雖不能盡賢，天子爲擇臣、立命卿以輔之。居是國也，累世尊重，然後士民之衆附焉，是以敎化行而治功立。今之郡守重於古諸侯，往者致選賢材，賢材難得，拔擢可用者，或起於囚徒。昔魏尚坐事繫，文帝感馮唐之言，遣使持節赦其罪，拜爲雲中太守；匈奴忌之。武帝擢韓安國於徒中，拜爲梁內史；骨肉以安。張敞爲京兆尹，有罪當免，黠吏知而犯敞，敞收殺之，其家自冤，使者覆獄，劾敞賊殺人，上逮捕不下，會免；亡命十數日，宣帝徵敞拜爲冀州刺史，卒獲其用。前世非私此三人，貪其材器有益於公家也。孝文時，吏居官者或長子孫，倉氏、庫氏則倉庫吏之後也；其二千石長吏亦安官樂職，然後上下相望，莫有苟且之意。其後稍稍變易，公卿以下傳相促急，又數改更政事，司隸、部刺史舉劾苛細，發揚陰私，吏或居官數月而退，送故迎新，交錯道路。中材茍容求全，下材懷危內顧，壹切營私者多。二千石益輕賤，吏民慢易之，或持其微過，增加成罪，言於司隸、刺史，或上書告之；衆庶知其易危，小失意則有離畔之心。前山陽亡徒蘇令等縱橫，吏士臨難，莫肯伏節死義，以守、相威權素奪也。孝成皇帝悔之，下詔書，二千石不爲故縱，遣使者賜金，尉厚其意，誠以爲國家有急，取辦於二千石；二千石尊重難危，乃能使下。孝宣皇帝愛其善治民之吏，有章劾事留中，會赦壹解。故事：尚書希下章，爲煩擾百姓，證驗繫治，或死獄中，章文必有『敢告之』字乃下。唯陛下留神於擇賢，記善忘過，容忍臣子，勿責以備。二千石、部刺史、三輔縣令有材任職者，人情不能不有過差，宜可闊略，令盡力者有所勸。此方今急務，國家之利也。前蘇令發，欲遣大夫使逐問狀，時見大夫無可使者，召盩厔令尹逢，拜爲諫大夫遣之。今諸大夫有材能者甚少，宜豫畜養可成就者，則士赴難不愛其死；臨事倉卒乃求，非所以明朝廷也。」嘉因薦儒者公孫光、滿昌及能吏蕭咸、薛脩，皆故二千石有名稱者，天子納而用之。
六月，立魯頃王子部鄕侯閔爲王。
上以寢疾未定，冬，十一月，壬子，令太皇太后下詔復甘泉泰畤、汾陰后土祠，罷南、北郊。上亦不能親至甘泉、河東，遣有司行事而禮祠焉。
無鹽危山土自起覆草，如馳道狀；又，瓠山石轉立。東平王雲及后謁自之石所祭；治石象瓠山立石，束倍草，幷祠之。河內息夫躬、長安孫寵相與謀共告之，曰：「此取封侯之計也！」乃與中郎右師譚共因中常侍宋弘上變事，告焉。是時上被疾，多所惡，事下有司，逮王后謁下獄驗治；服「祠祭詛祝上，爲雲求爲天子，以爲石立，宣帝起之表也。」有司請誅王，有詔，廢徙房陵。雲自殺，謁及舅伍宏及成帝舅安成共侯夫人放，皆棄市。事連御史大夫王崇，左遷大司農。擢寵爲南陽太守，譚潁川都尉，弘、躬皆光祿大夫、左曹、給事中。
