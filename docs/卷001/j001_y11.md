# 卷001 周紀一 — 安王十一年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王十一年 ・ 西暦: 391 BCE

[← 巻インデックス](README.md)

安王十一年〔注:庚寅(こういん)の年、紀元前三九一年〕。

秦が韓の宜陽(ぎよう)を攻め、六つの邑(ゆう)を奪い取った〔注:この「邑」は『周礼』にいう「四井を一邑とする」の邑、すなわち集落の単位〕。

さかのぼれば、斉の田常(でんじょう)が襄子盤(じょうし・はん)を生み、盤が荘子白(そうし・はく)を生み、白が太公和(たいこう・わ)を生んだ〔注:田常は『春秋左氏伝』にいう陳成子恒(ちんせいし・こう)のこと。司馬光が宋の仁宗の諱(いみな)「恒」を避けて「常」と書きかえた。陳の公子完(かん)が斉へ亡命してから五代目の常の代に、国政の実権を握った〕。この年、斉の田和は、斉の康公を海辺へ移し、一城だけを与えて、その先祖の祭祀を続けさせた。

十一年
秦伐韓宜陽，取六邑。
初，田常生襄子盤，盤生莊子白，白生太公和。是歲，齊田和遷齊康公於海上，使食一城，以奉其先祀。
