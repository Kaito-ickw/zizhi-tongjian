# 卷006 秦紀一 — 始皇帝上十年

> 巻 6 / 294 ・ 秦紀一 ・ 年号: 始皇帝上十年 ・ 西暦: 237 BCE

十年〔注:甲子(こうし)の年、紀元前二三七年〕。

冬十月、文信侯(ぶんしんこう=呂不韋)は相国(しょうこく)の職を解かれ、都を出て自分の領国へ赴いた〔注:文信侯の領国は河南の洛陽(らくよう)にあった〕。

宗室(そうしつ=王族)や大臣たちが議論して言った。「諸侯の国から来て秦に仕えている者は、みなそれぞれの主君のために秦の君臣を離間させる回し者にすぎません。どうか一律に追放してください〔注:遊説して秦の君臣の仲を裂くことを言う〕。」そこで大々的に捜索を行い、他国出身の客(かく)を追放することになった。

客卿(かくけい)である楚(そ)の人、李斯(りし)もこの追放の対象に入っていた。立ち去る道すがら、李斯は上書してこう述べた。「昔、穆公(ぼくこう)は人材を求め、西は戎(じゅう)から由余(ゆうよ)を取り、東は宛(えん)で百里奚(ひゃくりけい)を得、宋(そう)から蹇叔(けんしゅく)を迎え、晋(しん)から丕豹(ひひょう)・公孫支(こうそんし)を招きました〔注:公孫支は子桑(しそう)のこと〕。こうして二十国を併せ、ついに西戎(せいじゅう)の覇者となりました。孝公(こうこう)は商鞅(しょうおう)の法を用い、諸侯は親しんで服従し、その治績は今に至るまで国を強く保っています。恵王(けいおう)は張儀(ちょうぎ)の計を用いて六国の合従(がっしょう)を散じ、彼らを秦に服従させました。昭王(しょうおう)は范睢(はんしょ)を得て、公室(こうしつ=王室)を強め、私門(しもん=有力家臣の権勢)を塞ぎました。この四君は、いずれも客(他国出身の人材)の働きによって功を成したのです。こうして見れば、客が秦に何の負い目がありましょうか。そもそも美女・音楽・真珠・玉は秦の産ではないのに、王が身にまとい用いておられるものは数多くあります〔注:色とは女色のこと〕。ところが人を用いるとなると話は別で、可否も問わず、是非も論ぜず、秦の生まれでない者は去らせ、客である者は追放する。これでは重んじているのが美女・音楽・真珠・玉であり、軽んじているのが人民だということになります。臣の聞くところでは、泰山(たいざん)は土塊(つちくれ)を拒まないからこそその大きさを成し、河海(かかい)は細流(さいりゅう)をえり好みしないからこそその深さに達し、王者は民衆を退けないからこそその徳を明らかにできる、と申します。これこそ五帝(ごてい)・三王(さんおう)が無敵であった理由です。いま黔首(けんしゅ=人民)〔注:秦では民を黔首、すなわち黒い頭、と呼んだ〕を棄てて敵国を富ませ、賓客(ひんかく)を退けて諸侯の功業を助けるのは、いわゆる敵に兵器を貸し与え、盗賊に食糧を持たせてやるようなものです〔注:藉(しゃく)は貸す意、齎(さい)は持たせてやる意〕。」王はそこで李斯を召し返してその官を元に戻し、逐客(ちくかく)の令を取り消した。李斯は驪邑(りゆう)まで行ったところで引き返した〔注:驪邑は秦の地で、漢の高帝七年に新豊(しんぽう)と改称された〕。王は結局、李斯の謀(はかりごと)を用い、ひそかに弁士に金玉を持たせて諸侯のもとへ遊説させた。諸侯の名士で財によって籠絡(ろうらく)できる者には手厚く贈って手なずけ、応じない者は鋭い剣で刺し殺し、その君臣の仲を裂く計略をめぐらせた。そのうえで良将にその後を続かせ、数年のうちに、ついに天下を併合した。

十年
冬，十月，文信侯免相，出就國。
宗室大臣議曰︰「諸侯人來仕者，皆爲其主遊間耳，請一切逐之。」於是大索，逐客。客卿楚人李斯亦在逐中，行，且上書曰︰「昔穆公求士，西取由余於戎，東得百里於宛，迎蹇叔於宋，求丕豹、公孫支於晉，幷國二十，遂霸西戎。孝公用商鞅之法，諸侯親服，至今治強。惠王用張儀之計，散六國之從，使之事秦。昭王得范睢，強公室，杜私門。此四君者，皆以客之功。由此觀之，客何負於秦哉！夫色、樂、珠、玉不產於秦而王服御者衆；取人則不然，不問可否，不論曲直，非秦者去，爲客者逐。是所重者在乎色、樂、珠、玉，而所輕者在乎人民也。臣聞太山不讓土壤，故能成其大；河海不擇細流，故能就其深；王者不卻衆庶，故能明其德；此五帝、三王之所以無敵也。今乃棄黔首以資敵國，卻賓客以業諸侯，所謂藉寇兵，齎盜糧者也。」王乃召李斯，復其官，除逐客之令。李斯至驪邑而還。王卒用李斯之謀，陰遣辯士齎金玉遊說諸侯，諸侯名士可下以財者厚遺結之，不肯者利劍刺之，離其君臣之計，然後使良將隨其後，數年之中，卒兼天下。
