# 卷050 漢紀四十二 — 孝安皇帝中六年

六年〔注:己未(つちのとひつじ)の年、紀元一一九年〕。

春、二月、乙巳(きのとみ)の日、京師(けいし)および郡国四十二か所で地震があった。

夏、四月、沛国(はいこく)・勃海(ぼっかい)で大風が吹き、雹(ひょう)が降った。

五月、京師で旱(ひでり)があった。

六月、丙戌(ひのえいぬ)の日、平原哀王(へいげんあいおう)の得(とく)が薨(こう)じた。子がなかった。

秋、七月、鮮卑(せんぴ)が馬城塞(ばじょうさい)を侵し、長吏(ちょうり)を殺した〔注:馬城県は代郡(だいぐん)に属する。李賢いわく、『捜神記』に——昔、秦人が武周塞(ぶしゅうさい)に城を築いて胡(こ)に備えたが、完成しかけては幾度も崩れた。一頭の馬が走り回ってあちこち往き来したので、父老(ふろう)がこれを異(あや)しみ、その(馬のめぐった)跡に沿って城を築いたところ崩れなくなり、そこで「馬城」と名づけた、とある。胡三省が按ずるに、『捜神記』のいうのは鴈門(がんもん)郡の馬邑(ばゆう)のことで、ここは代郡の馬城であるから、李賢の誤りである〕。度遼将軍(どりょうしょうぐん)の鄧遵(とうじゅん)と中郎将(ちゅうろうしょう)の馬続(ばしょく)が南単于(なんぜんう)を率いて追撃し、大いに破った。

九月、癸巳(みずのとみ)の日、陳懐王(ちんかいおう)の竦(しょう)が薨じた。子がなく、国は除かれた〔注:竦は陳王羨(せん)の孫である〕。

冬、十二月、戊午(つちのえうま)の朔(ついたち)、日食があり、皆既(かいき)であった。

郡国八か所で地震があった。

この年、太后は、和帝(わてい)の弟である済北王(せいほくおう)の寿(じゅ)・河間王(かかんおう)の開(かい)の子の男女で五歳以上の者四十余人、および鄧氏(とうし)の近親の子孫三十余人を召し集め、いずれにも邸第(ていだい、屋敷)を設けて〔注:李賢いわく、『蒼頡篇(そうけつへん)』に「邸とは舎(やしき)のこと」とある〕、経書(けいしょ)を教え学ばせ、みずから試験を監督した。(太后は)従兄(いとこ)の河南尹(かなんいん)の豹(ひょう)・越騎校尉(えっきこうい)の康(こう)らに詔(みことのり)して言った。「末世(まっせ)の貴戚(きせき)で禄(ろく)を食(は)む家は、暖かな衣を着て美食をとり、堅牢な車に乗り良馬を駆りながら〔注:李賢いわく、「堅」とは良い車、「良」とは良い馬のこと。胡三省が按ずるに、この語は『史記』范蠡(はんれい)伝に出る〕、学問にかけては壁に顔を向けて立つように何も見えず、(物事の)善し悪し(臧否)もわきまえない〔注:『尚書』に「学ばざれば牆(かべ)に面(めん)す」とある。まっすぐ壁に向かって立てば、何も見えぬということ〕。これこそが禍(わざわい)や敗亡の起こるもとなのだ」。

豫章(よしょう)で芝草(しそう、めでたい霊芝)が生えた。太守の劉祗(りゅうし)はこれを(瑞祥として朝廷に)献上しようとし、郡の人である唐檀(とうだん)に尋ねたところ、唐檀は言った。「いま外戚(がいせき)が勢いを盛んにし、君主の道は弱まっています。これがどうしてめでたい瑞兆(ずいちょう)でありましょうか」。劉祗はそこで(献上を)取りやめた。

益州刺史(えきしゅうしし)の張喬(ちょうきょう)は、従事(じゅうじ)の楊竦(ようしょう)を遣わして兵を率いさせ、楪榆(ちょうゆ)に至り〔注:楪榆県は武帝が開いて置いた県で、益州郡に属する。県の東に葉榆沢(ちょうゆたく)があるのでそう名づけた。明帝のとき分けて永昌郡に属させた。「楪」は「葉」と同じ〕、封離(ほうり)らを撃って大いに破り、首三万余級を斬り、捕虜(生口)千五百人を獲(え)た。封離らは恐れおののき、その共謀者である渠帥(きょすい、頭目)を斬って、楊竦のもとに降伏を乞うた。楊竦は手厚くこれを慰(なぐさ)めて受け入れ、残りの三十六種族もみな降伏して帰服した。楊竦はそこで、長吏でよこしまでずる賢く(姦猾)、蛮夷(ばんい)を侵犯した者九十人を奏上し、(これらは)みな死一等を減じる扱いとなった。

さきに、西域(せいいき)の諸国は漢との関係が絶えていたが〔注:この事は前巻の永初元年に見える〕、北匈奴(きたきょうど)がふたたび武力でこれを服属させ〔注:使役して臣下として服属させること〕、ともに辺境を侵す寇(あだ)となった。敦煌(とんこう)太守の曹宗(そうそう)はこれを憂え、上奏して行長史(こうちょうし、長史代行)の索班(さくはん)を遣わし、千余人を率いて伊吾(いご)に駐屯させ、(西域諸国を)招き撫(なだ)めさせた〔注:「索」は敦煌に出る姓。また『左伝』に商人(殷人)の七族に索氏がある。「上遣」とは、上奏して遣わすこと。「行長史」とは、長史の職務を代行する者で、まだ正式の長史ではないことをいう〕。こうして車師前王(しゃしぜんおう)および鄯善王(ぜんぜんおう)が、ふたたび(漢に)降ってきた。

さきに、疏勒(そろく)王の安国(あんこく)が死に、子がなかったので、国人(くにびと)はその舅(おじ)の子の遺腹(いふく)を立てて王とした。遺腹の叔父である臣磐(しんばん)は月氏(げっし)におり、月氏は臣磐を受け入れて(疏勒王に)立てた〔注:『西域伝』にいう——元初年間、安国は舅の臣磐に罪があったため月氏へ移したが、月氏王は臣磐を親愛していた。遺腹が(疏勒王に)立てられると、月氏は兵を遣わして臣磐を疏勒へ送り返した。国人はもとより臣磐を敬い愛しており、また月氏を畏(おそ)れはばかったので、ただちにそろって遺腹の印綬(いんじゅ)を奪い、臣磐を迎えて王に立てた〕。後に莎車(さしゃ)が于窴(うてん)に叛(そむ)いて疏勒に属(つ)き〔注:明帝の永平四年以来、莎車は于窴に属していた〕、疏勒はこうして強盛となり、亀茲(きじ)・于窴と肩を並べる敵国(てきこく、対抗する国)となった。

六年
春，二月，乙巳，京師及郡國四十二地震。
夏，四月，沛國、勃海大風，雨雹。
五月，京師旱。
六月，丙戌，平原哀王得薨，無子。
秋，七月，鮮卑寇馬城塞，殺長吏，度遼將軍鄧遵及中郎將馬續率南單于追擊，大破之。
九月，癸巳，陳懷王竦薨，無子，國除。
冬，十二月，戊午朔，日有食之，旣。
郡國八地震。
是歲，太后徵和帝弟濟北王壽、河間王開子男女年五歲以上四十餘人，及鄧氏近親子孫三十餘人，並爲開邸第，敎學經書，躬自監試。詔從兄河南尹豹、越騎校尉康等曰：「末世貴戚食祿之家，溫衣美飯，乘堅驅良，而面牆術學，不識臧否，斯故禍敗之所從來也。」
豫章有芝草生，太守劉祗欲上之，以問郡人唐檀，檀曰：「方今外戚豪盛，君道微弱，斯豈嘉瑞乎！」祗乃止。
益州刺史張喬遣從事楊竦將兵至楪榆，擊封離等，大破之，斬首三萬餘級，獲生口千五百人。封離等惶怖，斬其同謀渠帥，詣竦乞降。竦厚加慰納，其餘三十六種皆來降附。竦因奏長吏姦猾，侵犯蠻夷者九十人，皆減死論。
初，西域諸國旣絕於漢，北匈奴復以兵威役屬之，與共爲邊寇。敦煌太守曹宗患之，乃上遣行長史索班將千餘人屯伊吾以招撫之。於是車師前王及鄯善王復來降。
初，疏勒王安國死，無子，國人立其舅子遺腹爲王；遺腹叔父臣磐在月氏，月氏納而立之。後莎車畔于窴，屬疏勒，疏勒遂強，與龜茲、于窴爲敵國焉。
