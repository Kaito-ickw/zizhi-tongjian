# 卷103 晉紀二十五 — 烈宗孝武皇帝上之上三年

①春、正月、辛亥の日、大赦した。

②夏、五月、丙午の日、藍田献侯(らんでんけんこう)の王坦之(おうたんし)が卒(しゅっ)した。臨終に際して謝安(しゃあん)・桓沖(かんちゅう)へ書(手紙)を送ったが、そこにはただ国家を案じる言葉だけがあり、私事には一言も触れなかった。

③桓沖は、謝安がかねてより重い名望を得ていたことから、揚州(の刺史の任)を彼に譲り、自分は地方へ出ることを望んだ。桓氏一族はこれを得策でないとして、腕を扼(にぎ)って固く諫めない者はなく〔注:揚州は都に近い畿内を統べ、その職権は重く要となる地位であったため、一族はこぞって桓沖を思いとどまらせようとしたのである〕、郗超(ちちょう)もまた強くこれを止めたが、桓沖はどちらの言も聞き入れず、平然とかまえていた。甲寅の日、詔(みことのり)して桓沖を都督徐豫兗青揚五州諸軍事・徐州刺史とし、京口を鎮めさせ、謝安には揚州刺史を領(りょう、兼任)させ、両人に侍中を加えた。

④六月、秦の清河武侯(せいがぶこう)・王猛(おうもう)が病床に臥した。秦王・苻堅(ふけん)はみずから彼のために南郊・北郊および宗廟・社稷に祈り、また侍臣を分かち遣わして黄河や華山(かざん)など諸々の神にあまねく祈らせた〔注:「河・嶽」とは黄河と華山の神を指し、四嶽すべてに遍(あまね)く祈ったわけではない〕。王猛の病はやや癒えたので、苻堅はこれを機に死罪以下の罪をことごとく赦した〔注:殊死とは身体を斬り分ける死刑のことで、それ以下の罪をすべて赦したということ〕。王猛は上疏して言った――「陛下がわたくし一人の命のために天地の徳をお損ないになろうとは、思いもよりませんでした。陛下の威武の勢いは八方の果てまで震わせ、教化の誉れは天地四方に光り輝いております。九州百郡のうち、すでに十の七をお持ちで、燕を平らげ蜀を定めたのも、地に落ちた芥(あくた)を拾うほどたやすいことでした。しかし物事をよく始める者が必ずしもよく成し遂げるとは限らず、よく始める者が必ずしもよく終えられるとは限りません。それゆえ昔の聖王たちは、大業の成し難さを知り、深い谷に臨むかのように戦々兢々として身を慎んだのです。伏してお願い申し上げます、陛下には昔の聖王の跡を追われますよう。そうなれば天下にとってこれ以上の幸せはございません」。苻堅はこれを読んで深く悲しんだ。秋、七月、苻堅はみずから王猛の邸(やしき)に赴いて病を見舞い、後事を尋ねた。王猛は言った――「晋は江南の片隅に僻在しているとはいえ、正統の暦を受け継いでおり、上下ともに安んじ和らいでおります〔注:王猛は秦に仕えながらも、正統が江南(晋)にあることを認めていた。徐光の見方は誤りである〕。わたくしが没した後は、どうか晋を討とうなどとお考えになりませぬよう。鮮卑と西羌こそがわが仇敵であり、いずれ必ず国の患いとなりましょう〔注:後にはたして王猛の言葉どおりになった〕。徐々にこれを除くのが、社稷のためかと存じます」。言い終えると息を引き取った。苻堅は納棺までに三度みずから臨んで哭し、太子の宏(こう)に言った――「天はわたしに天下を統一させたくないというのか。どうしてこうも早く景略(けいりゃく、王猛の字)を奪うのか」。そして王猛を漢の霍光(かくこう)の故事にならって葬った。

⑤八月、癸巳の日、(晋は)王氏を皇后に立て、大赦した。皇后は王濛(おうもう)の孫である〔注:王濛は清談(せいだん、老荘思想に基づく哲学的談論)に秀で、劉惔(りゅうたん)と並び称された人物である〕。皇后の父で晋陵太守の王蘊(おううん)を光禄大夫とし、五兵尚書(ごへいしょうしょ)を領ねさせ、建昌侯に封じたが、王蘊は固く辞退して受けなかった。

⑥九月、(孝武)帝は『孝経』を講じ、初めて典籍を披(ひら)いて読み、儒学の士を招いた。謝安は東莞(とうかん)の徐邈(じょばく)を推薦して中書舎人に補した。徐邈は帝から意見を求められるたびに、多くの益をもたらした。帝はときに宴を催し、酒宴が酣(たけなわ)になった後、手ずから詔(みことのり)や詩文を書いて侍臣に賜うのを好んだが、その文辞は時に軽率で、述べるところが卑俗で雑駁(ざっぱく)なものもあった。徐邈はそのつど回収して中書省に持ち帰り、削り整えて見られるものに仕立て直し、帝が改めて目を通してから、それを外に出させた。当時の世論はこのことで徐邈を高く評価した。

⑦冬、十月、癸酉の朔(ついたち)、日食があった。

⑧秦王・苻堅は詔を下して言った――「新たに賢明な補佐役を喪(うしな)い、諸官庁もあるいはわが意にかなわぬことがあろう。未央殿の南に聴訟観(ちょうしょうかん、訴訟を聴くための建物)を置くがよい。わたしは五日に一度そこへ臨んで、民の隠れた苦しみを汲み取ろう。今、天下はまだ大いに定まってはいないが、しばらくは武を偃(や)め文を脩(おさ)め、武侯(ぶこう)の雅(みやび)やかな志にかなうこととしよう〔注:武侯とは王猛の諡(おくりな)である〕。儒教をますます崇(あが)め、老子・荘子および図讖(としん、予言の書)の学を禁じ、犯す者は棄市(きし、市中での処刑と晒し)とせよ」。そして学生を厳選し、太子および公・侯・百官の子弟はみな学に就いて業を受けさせ、宮中・宮外の四禁・二衛・四軍(しきん・にえい・しぐん)の長上(ちょうじょう)の将士にもみな学を受けさせた〔注:秦には中軍・外軍の将軍のほか、前禁・後禁・左禁・右禁の将軍を四禁といい、左衛・右衛の将軍を二衛といい、衛軍・撫軍・鎮軍・冠軍の将軍を四軍といった。長上とは、宮中に長く宿衛する将士のことである〕。二十人につき一人の経生(けいせい、経典を教える師)をあて、読み方と句読を教えさせ、後宮には典学(てんがく)を置いて掖庭(えきてい、後宮の女官たち)を教えさせ、聡明な宦官や女隷(じょれい)を選んで博士のもとへ遣わし経典を授けさせた〔注:女隷とは官の奴婢に落とされた女たちで、いわゆる奚官女(けいかんじょ)である〕。尚書郎の王佩(おうはい)が図讖を読んでいたため、苻堅はこれを誅殺し、以後、図讖を学ぶ者は絶えた。

三年
春，正月，辛亥，大赦。
夏，五月，丙午，藍田獻侯王坦之卒；臨終與謝安、桓沖書，惟以國家爲憂，言不及私。
桓沖以謝安素有重望，欲以揚州讓之，自求外出。桓氏族黨皆以爲非計，莫不扼腕固諫，郗超亦深止之，沖皆不聽，處之澹然。甲寅，詔以沖都督徐‧豫‧兗‧青‧揚五州諸軍事、徐州刺史，鎭京口；以安領揚州刺史，並加侍中。
六月，秦清河武侯王猛寢疾，秦王堅親爲之祈南、北郊及宗廟、社稷，分遣侍臣徧禱河、嶽諸神。猛疾少瘳，爲之赦殊死以下。猛上疏曰：「不圖陛下以臣之命而虧天地之德，威烈振乎八荒，聲敎光乎六合，九州百郡，十居其七，平燕定蜀，有如拾芥。夫善作者不必善成，善始者不必善終，是以古先哲王，知功業之不易，戰戰兢兢，如臨深谷。伏惟陛下，追蹤前聖，天下幸甚。」堅覽之悲慟。秋，七月，堅親至猛第視疾，訪以後事。猛曰：「晉雖僻處江南，然正朔相承，上下安和，臣沒之後，願勿以晉爲圖。鮮卑、西羌，我之仇敵，終爲人患，宜漸除之，以便社稷。」言終而卒。堅比斂，三臨哭，謂太子宏曰：「天不欲使吾平壹六合邪，何奪吾景略之速也？」葬之如漢霍光故事。
八月，癸巳，立皇后王氏，大赦。后，濛之孫也。以后父晉陵太守蘊爲光祿大夫，領五兵尚書，封建昌侯；蘊固辭不受。
九月，帝講《孝經》，始覽典籍，延儒士。謝安薦東莞徐邈補中書舍人，每被顧問，多所匡益。帝或宴集，酣樂之後，好爲手詔詩章以賜侍臣，或文詞率爾，所言穢雜；邈應時收斂還省刊削，皆使可觀，經帝重覽，然後出之。時議以此多邈。
冬，十月，癸酉朔，日有食之。
秦王堅下詔曰：「新喪賢輔，百司或未稱朕心，可置聽訟觀於未央南，朕五日一臨，以求民隱。今天下雖未大定，權可偃武脩文，以稱武侯雅旨。其增崇儒敎；禁老、莊、圖讖之學，犯者棄市。」妙簡學生，太子及公侯百僚之子皆就學受業；中外四禁、二衞、四軍長上將士，皆令受學。二十人給一經生，敎讀音句，後宮置典學以敎掖庭，選閹人及女隸敏慧者詣博士授經。尚書郎王佩讀讖，堅殺之；學讖者遂絕。
