# 卷001 周紀一 — 安王十七年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王十七年 ・ 西暦: 385 BCE

安王十七年〔注:丙申(へいしん)の年、紀元前三八五年〕。

秦の庶長(しょちょう)であった改(かい)が、河西(かせい)で獻公(けんこう)を迎えて擁立した〔注:庶長は秦の爵位・官位の一つ。「改」は人名。獻公はかつて即位を阻まれ、黄河の西の地に逃れていた〕。そして出子(しゅっし)とその母を殺し、淵のほとりに沈めた。

斉が魯を攻めた。

韓が鄭を攻めて陽城(ようじょう)を奪い、さらに宋を攻めて宋公を捕らえた。

斉の太公が薨じ、子の桓公午(かんこうご)が立った。

十七年
秦庶長改逆獻公于河西而立之；殺出子及其母，沈之淵旁。
齊伐魯。
韓伐鄭，取陽城；伐宋，執宋公。
齊太公薨，子桓公午立。
