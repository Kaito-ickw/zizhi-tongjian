# 卷054 漢紀四十六 — 孝桓皇帝上之下六年

六年〔注:癸卯(みずのとう)の年、紀元一六三年〕。

春、二月、戊午(つちのえうま)の日、司徒の种暠(ちゅうこう)が薨(こう)じた。

三月、戊戌(つちのえいぬ)の日、天下を赦(ゆる)した。

衛尉(えいい)で潁川(えいせん)の許栩(きょく)を司徒とした。

夏、四月、辛亥(かのとい)の日、康陵(こうりょう)の東署(とうしょ)が火災に遭った。

五月、鮮卑(せんぴ)が遼東属国(りょうとうぞっこく)を侵した。

秋、七月、甲申(きのえさる)の日、平陵(へいりょう)の園寝(えんしん、陵園の御殿)が火災に遭った〔注:平陵は昭帝の陵(みささぎ)〕。

桂陽(けいよう)の賊である李研(りけん)らが郡境を侵し、武陵(ぶりょう)の蛮がふたたび反乱したが、太守の陳挙(ちんきょ)〔注:校勘——一本では「挙」を「奉」に作る〕がこれを討ち平らげた。宦官(かんがん)はかねてより馮緄(ふうこん)を憎んでいたので、八月、馮緄は、軍が(都に)帰還したのちに盗賊がふたたび蜂起したことを罪に問われて、免ぜられた。

冬、十月、丙辰(ひのえたつ)の日、帝は広成(こうせい)で校猟(こうりょう、軍を整えての狩り)をし、そのまま函谷関(かんこくかん)・上林苑(じょうりんえん)に行幸した。光禄勲(こうろくくん)の陳蕃(ちんばん)が上疏(じょうそ)して諫(いさ)めて言った。「安らかで太平の時でさえ、遊猟(遊畋)には節度があるべきです。まして今は『三空(さんくう)』の厄(やく)があるではありませんか。田野(=田畑)は空(むな)しく、朝廷は空しく、倉庫は空しい。加えて兵戎(へいじゅう、=戦乱)はまだ収まらず、四方の(民)は離散しております。これは陛下が心を焦がし顔(おもて)を曇らせ(毀顔)、坐(ざ)したまま夜明けを待つ(=寝ずに憂える)べき時なのです〔注:毀顔とは、顔に憂いの色があり、臣民の上に臨(のぞ)んでも、(平静な)顔を保てないことをいう〕。どうして旗を揚げ武を曜(かがや)かし、輿馬(よば、車馬)の見もの(=狩猟)に心を馳(は)せてよいものでしょうか。また前の秋には雨が多く、民はようやく麦を種(ま)き始めたばかりですのに、いまその種まきを勧(すす)めるべき時を失わせ、(民に)禽(とり)を駆り立て道を掃き清める(除路)労役を課(=給)させるのは、賢聖(=賢く聖なる君)が民を恤(あわ)れむ本意ではございません」。(陳蕃の)書(=上疏)が奏上されたが、容(い)れられなかった。

十一月、司空の劉寵(りゅうちょう)が免ぜられた。十二月、衛尉の周景(しゅうけい)を司空とした。周景は周栄(しゅうえい)の孫である。当時、宦官はまさに勢いを増していた。周景は太尉の楊秉(ようへい)とともに上言した。「内外の吏(=官)の職には、その任にふさわしからぬ者が多くおります。旧来の典(=制度)では、中臣(=宦官)の子弟は、(顕)職に居て権勢を握ってはならぬとされておりました。しかるに今は、(宦官の)枝葉(=一族)〔注:枝葉とは、宦官の一族・親戚をいう〕や賓客(=取り巻き)が職署(=官庁)〔注:署とは官舎(=役所)のこと〕に居並び、あるいは年若い凡庸な者が、守宰(しゅさい、=郡県の長官)の(任を)つかさどっております。(そのため)上下は怒り憂え、四方(の民)は愁(うれ)い苦しんでおります。旧来の典章(=制度)に遵(したが)って、貪(むさぼ)り害(そこな)う(貪残な)者を退け、災いと誹謗(を招くもと)を塞(ふさ)ぐべきです。どうか司隸校尉(しれいこうい)・中二千石(ちゅうにせんせき、=列卿)・城門校尉(じょうもんこうい)・五営校尉(ごえいこうい)・北軍中候(ほくぐんちゅうこう)に(命を)下し、それぞれの管轄(所部)を実際に調べ(実覈)させ〔注:司隸校尉は三輔・三河・弘農を、中二千石はそれぞれその属官を率い、城門校尉は十二城門の司馬・門候を、五営校尉は屯騎・越騎・歩兵・長水・射声(の五営)を、北軍中候は五営の監督をつかさどる〕、斥(しりぞ)け罷(や)めさせるべき者を、自ら(その)実状(状)をもって三府(さんぷ、三公の府)に申し述べさせ、(さらに)調べて(廉察)漏(も)れがあれば、続けて上申させてください」。帝はこれに従った。そこで楊秉は、(各地の)牧(ぼく)・守(=長官)や青州刺史の羊亮(ようりょう)ら五十人あまりを条(箇条書き)にして奏上し、ある者は死し、ある者は免ぜられ、天下(の人々)で粛然(しゅくぜん)としない者はなかった。

(帝は)詔(みことのり)して皇甫規(こうほき)を召し、度遼将軍(どりょうしょうぐん)に任じた。これより先、張奐(ちょうかん)は梁冀(りょうき)の元の属吏(故吏)であったことを罪に問われ、官を免ぜられ禁錮(きんこ、=任官を禁じられる)に処されており、(かつての)交友・旧知の者はみな、あえて(彼のために)弁護しようとはしなかった。ただ皇甫規だけが(張奐を)推挙し、前後七度にわたって上奏したので、これによって(張奐は)武威(ぶい)太守に任じられた。皇甫規が度遼将軍となり、営(=軍営)に着いて数か月して、上書して張奐を推挙した。「(張奐は)才略がともに優れており、元帥(=度遼将軍)〔注:元帥とは、度遼将軍をいう〕の正官に就(つ)けて、衆望に従(かな)うべきです。もしなお、愚臣(=私)こそ事を挙(あ)げ行う(=任を全うする)にふさわしいと(陛下が)お考えなら、(臣は)願わくは冗官(じょうかん、閑職)を乞い、張奐の副(=副将)にしていただきたく存じます」。朝廷はこれに従った。張奐を皇甫規に代えて度遼将軍とし、皇甫規を使匈奴中郎将(しきょうどちゅうろうしょう)とした。

西州(=涼州方面)の吏民で、闕(けつ、宮門)に詰めかけて、前の護羌校尉(ごきょうこうい)である段熲(だんけい)のために冤(えん、=ぬれぎぬ)を訴える者がたいそう多かった。おりしも滇那(てんな)ら諸種族の羌がますます勢いを増し、涼州は危うく失われかけたので、そこでふたたび段熲を護羌校尉とした。

尚書(しょうしょ)の朱穆(しゅぼく)は宦官のほしいままな横暴(恣横)を憎(にく)み、上疏して言った。「漢の故事(=旧例)を按(かんが)えますに、中常侍(ちゅうじょうじ)は士人(しじん、=学識ある士)も選んで(任に)与(あずか)らせておりましたが、建武(=光武帝の世)以後は、ことごとく宦者(=宦官)を用いるようになりました。延平(=殤帝の世)以来、(宦官は)しだいにいよいよ貴(とうと)く盛んになり、貂璫(ちょうとう、貂(てん)の尾と金の飾り)の装いを借り(=身につけ)、常伯(じょうはく、=侍中)の任に処(つ)き〔注:李賢いわく、璫は金で作り、冠の前に金の蝉(せみ)を付けたもの。『漢官儀』に、中常侍は秦の官で、漢が興ると士人を用いることもあって銀璫(ぎんとう)に左貂(さちょう)、光武帝以後はもっぱら宦者を任じて右貂(うちょう)金璫とした、とある。常伯とは侍中のこと〕、天朝(=朝廷)の政事(まつりごと)は、ひとたびその(宦官の)手を経(へ)る(=握られる)ようになりました。(その)権勢は海内(=天下)を傾け、寵愛(ちょうあい)と貴さは際限なく、(宦官の)子弟・親戚も、そろって栄(さか)えた任(=顕職)を荷(にな)い、ほしいままに驕(おご)りあふれ、誰も禁じ止めることができず、天下を窮(きわ)め破り、小民(=庶民)を搾り尽くしております。愚臣(=私)が思いますに、(宦官を)ことごとく罷(や)め省(はぶ)き、往初(おうしょ、=昔のはじめ)の(あり方)に遵(したが)い復(かえ)って、改めて海内の清(きよ)く淳(あつ)い士で国体(=国家の体制)に明達(=精通)した者を選び、その(宦官の)処(=地位)に補(あ)てれば、兆庶(ちょうしょ)・黎萌(れいぼう、=万民)も、聖化(=聖天子の徳化)を蒙(こうむ)りましょう」。帝は容れなかった。のちに朱穆は、(帝に)拝謁する機会(進見)に乗じて、ふたたび口頭で(意見を)陳(の)べた。「臣の聞きますに、漢家の旧典では、侍中・中常侍を各一人置いて尚書の事を省(み=管理)させ〔注:李賢いわく、省は覧(み)る意〕、黄門侍郎(こうもんじろう)一人を(置いて)書奏(=文書)の取り次ぎ・伝達をさせ〔注:李賢いわく、伝は通(つう)じる意〕、いずれも姓族(=名門の士人)〔注:李賢いわく、族望(=名門の声望)ある士人を引き用いた〕を用いておりました。和熹太后(わきたいごう)が女主(=女性の君主)として(政を)称制(しょうせい、=代行)して以来、公卿に接しなかったため、そこで閹人(えんじん、=宦官)を常侍とし、小黄門(しょうこうもん)に両宮(りょうきゅう、=帝と太后)の命(めい)を通じさせました。これ以来、(宦官の)権勢は人主(=君主)を傾け、天下を窮(きわ)め困(くる)しめております。みな罷(や)め退けるべきで、広く耆儒(きじゅ、=老練の儒者)・宿徳(しゅくとく、=徳望ある老成の人)を選んで、(彼らに)政事に与(あずか)らせるべきです」。帝は怒って、応(こた)えなかった。朱穆は(地に)伏したまま起き上がろうとせず、左右(の近臣)が「出(=退出せよ)」と(声を)伝えた〔注:李賢いわく、声を伝えて退出を命じた〕。しばらくして、(朱穆は)ようやく小走りに去った。これより、中官(=宦官)はたびたび事にかこつけて詔と称して朱穆を詆毀(ていき、=そしり貶(おとし)めること)した。朱穆はもとより剛直で、意を得ず(=不本意のうちに)、それからいくらも経たぬうちに、憤懣(ふんまん)のあまり疽(そ、悪性の腫れ物)を発して死んだ。

六年
春，二月，戊午，司徒种暠薨。
三月，戊戌，赦天下。
以衞尉潁川許栩爲司徒。
夏，四月，辛亥，康陵東署火。
五月，鮮卑寇遼東屬國。
秋，七月，甲申，平陵園寢火。
桂陽賊李硏等寇郡界，武陵蠻復反；太守陳舉討平之。宦官素惡馮緄，八月，緄坐軍還，盜賊復發免。
冬，十月，丙辰，上校獵廣成，遂幸函谷關、上林苑。光祿勳陳蕃上疏諫曰：「安平之時，遊畋宜有節，況今有三空之戹哉！田野空，朝廷空，倉庫空。加之兵戎未戢，四方離散，是陛下焦心毀顏，坐以待旦之時也，豈宜揚旗曜武，騁心輿馬之觀乎！又前秋多雨，民始種麥，今失其勸種之時，而令給驅禽除路之役，非賢聖恤民之意也。」書奏，不納。
十一月，司空劉寵免。十二月，以衞尉周景爲司空。景，榮之孫也。
時宦官方熾，景與太尉楊秉上言：「內外吏職，多非其人。舊典，中臣子弟，不得居位秉勢；而今枝葉賓客，布列職署，或年少庸人，典據守宰；上下忿患，四方愁毒。可遵用舊章，退貪殘，塞災謗。請下司隸校尉、中二千石、城門、五營校尉、北軍中候，各實覈所部；應當斥罷，自以狀言三府，廉察有遺漏，續上。」帝從之。於是秉條奏牧、守青州刺史羊亮等五十餘人，或死或免，天下莫不肅然。
詔徵皇甫規爲度遼將軍。初，張奐坐梁冀故吏，免官禁錮，凡諸交舊，莫敢爲言；唯規薦舉，前後七上，由是拜武威太守。及規爲度遼，到營數月，上書薦奐，「才略兼優，宜正元帥，以從衆望。若猶謂愚臣宜充舉事者，願乞宂官，以爲奐副。」朝廷從之。以奐代規爲度遼將軍，以規爲使匈奴中郎將。
西州吏民守闕爲前護羌校尉段熲訟冤者甚衆；會滇那等諸種羌益熾，涼州幾亡，乃復以熲爲護羌校尉。
尚書朱穆疾宦官恣橫，上疏曰：「按漢故事，中常侍參選士人，建武以後，乃悉用宦者。自延平以來，浸益貴盛，假貂璫之飾，處常伯之任，天朝政事，一更其手；權傾海內，寵貴無極，子弟親戚，並荷榮任，放濫驕溢，莫能禁禦，窮破天下，空竭小民。愚臣以爲可悉罷省，遵復往初，更選海內清淳之士明達國體者，以補其處，卽兆庶黎萌，蒙被聖化矣！」帝不納。後穆因進見，復口陳曰：「臣聞漢家舊典，置侍中、中常侍各一人，省尚書事；黃門侍郎一人，傳發書奏；皆用姓族。自和熹太后以女主稱制，不接公卿，乃以閹人爲常侍，小黃門通命兩宮。自此以來，權傾人主，窮困天下，宜皆罷遣，博選耆儒宿德，與參政事。」帝怒，不應。穆伏不肯起，左右傳「出」！良久，乃趨而去。自此中官數因事稱詔詆毀之。穆素剛，不得意，居無幾，憤懣發疽卒。
