# 卷005 周紀五 — 赧王下四十七年

四十七年〔注:癸巳(きし)の年、紀元前二六八年〕。

秦王は范睢(はんすい)のはかりごとを用い、五大夫(ごたいふ)の綰(わん)に魏を攻めさせ、懷(かい)を抜いた〔注:懷県は河內郡(かだいぐん)に属する〕。

==四十七年==
秦王用范睢之謀，使五大夫綰伐魏，拔懷。
