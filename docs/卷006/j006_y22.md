# 卷006 秦紀一 — 始皇帝上十三年

十三年〔注:丁卯(ていぼう)の年、紀元前二三四年〕。

桓齮(かんき)が趙(ちょう)を攻め、趙の将軍扈輒(こちょう)を平陽(へいよう)で破り、十万の首を斬って扈輒を殺した。趙王は李牧(りぼく)を大将軍(だいしょうぐん)に任じ、ふたたび宜安(ぎあん)・肥下(ひか)で戦った。秦軍は大敗を喫し〔注:大崩れすることを敗績(はいせき)という〕、桓齮は本国へ逃げ帰った。趙は李牧を武安君(ぶあんくん)に封じた。

十三年
桓齮伐趙，敗趙將扈輒於平陽，斬首十萬，殺扈輒。趙王以李牧爲大將軍，復戰於宜安、肥下，秦師敗績，桓齮奔還。趙封李牧爲武安君。
