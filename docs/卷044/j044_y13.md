# 卷044 漢紀三十六 — 顯宗孝明皇帝上二年

永平二年〔注:己未(つちのとひつじ)の年、西暦五九年〕。

① 春、正月、辛未(しんび)の日、明堂(めいどう)で光武皇帝を宗祀(そうし、尊んで祀ること)した〔注:「宗」とは尊ぶことで、尊んで祀り、上帝(じょうてい、天)に配して合わせ祀るのである〕。帝および公卿(こうけい)・列侯(れっこう)は、このとき初めて冠冕(かんべん、礼冠)と玉佩(ぎょくはい、腰の玉飾り)を身に着けて儀式を執り行った〔注:明帝はこのとき初めて古礼にならい、冕(べん)の冠と衣裳(いしょう)の制を整え、天地・明堂の祭祀に用いた〕。儀式が終わると、霊台(れいだい)に登り、雲物(うんぶつ、空の雲気の色)を望み見た〔注:雲物とは雲気の色合いのことで、これを見て吉凶や災異の前兆を占った〕。そして天下に大赦を行った。

② 三月、辟雍(へきよう)に臨み、初めて大射(たいしゃ)の礼を行った〔注:大射の礼とは、王が祭祀にあたり、助祭の士を選ぶために行う射の儀礼である〕。

冬、十月、壬子(じんし)の日、上(かみ、明帝)は辟雍に行幸し、初めて養老(ようろう)の礼を行った。李躬(りきゅう)を三老(さんろう)、桓栄(かんえい)を五更(ごこう)とした。三老は都紵(とちょ、上等の麻布)の大袍(たいほう)を着て、進賢冠(しんけんかん)をかぶり、玉杖(ぎょくじょう)をついた〔注:玉杖は長さ七尺で、先端を鳩(はと)の鳥の形で飾る。鳩はものを喉に詰まらせない鳥であり、老人がむせないようにとの願いを込めたものである〕。五更も同じいでたちだが、杖はつかない。

乗輿(じょうよ、明帝)は辟雍の礼殿(れいでん)に到り、御座(ぎょざ)を東の廂(ひさし)の間にもうけ、使者を遣わして安車(あんしゃ、ゆったり座れる車)で三老・五更を太学(たいがく)の講堂から迎えさせた。天子(明帝)は門の屛(へい)のところで自ら出迎え、互いに礼を交わし、東の阼階(そかい、主人が昇る階)から先導した。三老は西の賓階(ひんかい、客が昇る階)から昇った。階のところに至ると、天子は礼にのっとって揖(ゆう、会釈)した。三老が昇って東向きに座ると、三公(さんこう)が几(き、ひじかけ)をすえ、九卿(きゅうけい)が履(くつ)を正してそろえた。天子は自ら肌脱ぎになって牲(いけにえ)を割き、醬(ひしお)を手にして勧め(注:醬は味つけの主役なので、これを手にして食事を勧めるのである)、爵(さかずき)を手にして酳(いん、食後に口をすすぐ酒)を勧めた。さらに、(老人が)骨を喉に詰まらせぬよう祈る者が前に、食べ物を喉に詰まらせぬよう祈る者が後ろに控えた。五更は南向きに座り、三公が供物を進め、礼はやはり三老のときと同様にした。

礼が終わると、桓栄とその弟子たちを堂(どう)に引き上げ、上(明帝)が自ら言葉を下して講説した。諸儒(しょじゅ、儒学者たち)は経書を手に、御前で疑問をぶつけて論じ合った。冠帯(かんたい)を締めた縉紳(しんしん、官人)たちで、橋門(きょうもん)をぐるりと取り囲んで見聞きする者は、億万を数えるほどであった〔注:辟雍は四方の門の外に水をめぐらせて見物人の入りを区切り、門の外にはみな橋が架かっている。見物人は水の外側にいるので、「橋門を囲む」と言うのである〕。

そこで詔(みことのり)を下し、桓栄に関内侯(かんだいこう)の爵を賜った。三老・五更には、みな二千石(にせんせき)の俸禄をもって、その身が終わるまで生活を支えた。また天下の三老には酒を賜い、一人につき酒一石(いっこく)、肉四十斤(きん)とした。

上(明帝)は、まだ太子であったころから桓栄に『尚書(しょうしょ)』を授かって学んでおり、帝位に即いてからも、なお桓栄を師の礼で尊んだ。あるとき太常府(たいじょうふ)に行幸して桓栄を東向きに座らせ、几(ひじかけ)と杖をすえ、百官および桓栄の門生(もんせい、弟子)数百人を集め、上は自ら経書を手にして教えを受けた。諸生(しょせい、学生たち)の中には席を立って疑問を発する者もあったが、上はへりくだって「太師(たいし)がここにおられる(私が出る幕ではない)」と言った。会が終わると、太官(たいかん、宮中の食膳係)の膳の道具一式を、すべて太常(桓栄)の家に賜った。

桓栄が病にかかるたびに、帝はそのつど使者を遣わして見舞わせ、太官や太医(たいい、宮廷医)の使いが道に絶え間なく続いた。病が篤(あつ)くなると、桓栄は上疏(じょうそ)して恩に謝し、爵位と封地を返上したいと願い出た。帝は桓栄の家に行幸して容態を尋ね、街(まち)に入ると車を下り、経書を抱えて歩み寄り、桓栄をなでてなみだを流し、寝床の敷物・とばり・刀剣・衣服や夜着を賜って、ずいぶん長く過ごしてから去った。これ以後、諸侯(しょこう)・将軍(しょうぐん)・大夫(たいふ)で見舞いに来る者は、二度と車に乗ったまま門まで乗りつけようとはせず、みな寝床の下で拝礼した。桓栄が亡くなると、帝は自ら喪服に着替えて喪に臨み、葬列を見送り、首山(しゅざん)の南に墓所を賜った〔注:首山は首陽山(しゅようざん)のことで、いまの偃師県(えんしけん)の西北にある〕。子の桓郁(かんいく)が跡を継ぐはずであったが、桓郁は兄の子である桓汎(かんはん)に爵を譲ろうとした。帝はこれを許さず、桓郁はそこで封を受けたが、その封地からの収入はすべて桓汎に与えた。帝は桓郁を侍中(じちゅう)とした。

③ 上(明帝)は、中山王(ちゅうざんおう)の焉(えん)が郭太后(かくたいこう)の末子で、太后がとりわけ彼を可愛がっていたことから、彼だけを都に留めおいていた。ここに至って初めて、ほかの諸王とともに自分の封国へ赴かせ、虎賁(こほん)・官騎(かんき、王家づきの騎兵)を賜った〔注:このとき北軍の胡騎(こき)百人を賜った。いずれも兵に長け、弓の上手な者たちであった〕。その恩寵はとりわけ厚く、焉だけは都への往き来を許された。帝は陰氏(いんし、生母の一族)と郭氏(かくし)の両家を礼遇し、何事につけ必ず公平に扱い、両家ともたびたび賞賜を受けて、恩寵はどちらにも等しく厚かった。

④ 甲子(こうし)の日、上(明帝)は長安(ちょうあん)に行幸した。十一月、甲申(こうしん)の日、使者を遣わして中牢(ちゅうろう、羊と豕(ぶた)の供え物)で蕭何(しょうか)と霍光(かくこう)を祀らせ、帝自身もその墓のかたわらを通るとき、車上で軾(しょく)に手をかけて敬礼した。さらに進んで河東(かとう)に行幸し、癸卯(きぼう)の日、宮に還った。

⑤ 十二月、護羌校尉(ごきょうこうい)の竇林(とうりん)が、人をあざむき欺いた罪と、不正に財貨をためこんだ罪に問われ、獄に下されて死んだ〔注:このとき羌(きょう)の滇吾(てんご)が背き、滇岸(てんがん)が降ってきたので、竇林は上奏して滇岸を大豪(だいごう、有力な首領)とした。のちに滇吾も降ると、竇林はまた上奏して滇吾を第一の豪とした。帝は一つの種族に二人の豪がいるのを怪しんで竇林を問いただし、調べ尽くしてその不正を知り、怒って竇林を免官した。さらに涼州刺史(りょうしゅうしし)が竇林の蓄財の罪を上奏したため、ついに獄に下されて死んだのである〕。竇林は、竇融(とうゆう)の従兄(いとこの兄)の子であった。

このころ竇氏(とうし)は、一人の公・二人の侯・三人の公主(こうしゅ)を娶った家柄・四人の二千石(にせんせき)の官が一時に並び立ち〔注:一公とは大司空(だいしくう)、二侯とは安豊侯(あんぽうこう)と顕親侯(けんしんこう)、四人の二千石とは衛尉(えいい)・城門校尉(じょうもんこうい)・護羌校尉(ごきょうこうい)・中郎将(ちゅうろうしょう)である〕、祖父の代から孫の代に至るまで、役所や邸宅が都じゅうに立ち並び、親戚や功臣の中でも肩を並べられる者はいなかった。竇林が誅されると、帝はたびたび詔を下して竇融を厳しく責めた。竇融は恐れおののいて骸骨(がいこつ)を乞い(辞職を願い出て)、詔によって邸宅に帰り、療養することを命じられた。

⑥ この年、初めて五郊(ごこう、東西南北と中央の郊外)で迎気(げいき)の祭祀を行った〔注:季節ごとにその方位の郊外で気を迎える祭りである。立春には東郊で青帝を、立夏には南郊で赤帝を、立秋には西郊で白帝を、立冬には北郊で黒帝を、また立秋に先立っては中央で黄帝を迎え、それぞれ車馬・衣服の色や歌・舞をその方位にそろえた〕。

⑦ 新陽侯(しんようこう)の陰就(いんしゅう)の子である陰豊(いんほう)が、酈邑公主(れきゆうこうしゅ)を妻に迎えた。公主は驕(おご)り高ぶって嫉妬深く、陰豊は彼女を殺したために誅され、その父母もみな自殺した〔注:この公主は光武帝の娘である。酈県(れきけん)は南陽郡(なんようぐん)に属す〕。

⑧ 南単于(なんぜんう)の汗(かん)が死に、(初代南単于である)比(ひ)の子の適(てき)が立って、䤈僮尸逐侯鞮単于(けいどうしちくこうていぜんう)となった。

二年
春，正月，辛未，宗祀光武皇帝於明堂，帝及公卿列侯，始服冠冕、玉佩以行事。禮畢，登靈臺，望雲物。赦天下。
三月，臨辟雍，初行大射禮。
冬，十月，壬子，上幸辟雍，初行養老禮；以李躬爲三老，桓榮爲五更。三老服都紵大袍，冠進賢，扶玉杖；五更亦如之，不杖。乘輿到辟雍禮殿，御坐東廂，遣使者安車迎三老、五更於太學講堂，天子迎于門屛，交禮；道自阼階，三老升自賓階；至階，天子揖如禮。三老升，東面，三公設几，九卿正履，天子親袒割牲，執醬而饋，執爵而酳，祝鯁在前，祝饐在後。五更南面，三公進供，禮亦如之。禮畢，引桓榮及弟子升堂，上自爲下說，諸儒執經問難於前，冠帶縉紳之人圜橋門而觀聽者，蓋億萬計。於是下詔賜榮爵關內侯；三老、五更皆以二千石祿養終厥身。賜天下三老酒，人一石，肉四十斤。
上自爲太子，受《尚書》於桓榮，及卽帝位，猶尊榮以師禮。嘗幸太常府，令榮坐東面，設几杖，會百官及榮門生數百人，上親自執業；諸生或避位發難，上謙曰：「太師在是。」旣罷，悉以太官供具賜太常家。榮每疾病，帝輒遣使者存問，太官、太醫相望於道。及篤，上疏謝恩，讓還爵土。帝幸其家問起居，入街，下車，擁經而前，撫榮垂涕，賜以牀茵、帷帳、刀劍、衣被，良久乃去。自是諸侯、將軍、大夫問疾者，不敢復乘車到門，皆拜牀下。榮卒，帝親自變服臨喪送葬，賜冢塋于首山之陽。子郁當嗣，讓其兄子汎；帝不許，郁乃受封，而悉以租入與之。帝以郁爲侍中。
上以中山王焉，郭太后少子，太后尤愛之，故獨留京師，至是始與諸王俱就國，賜以虎賁、官騎，恩寵尤厚，獨得往來京師。帝禮待陰、郭，每事必均，數受賞賜，恩寵俱渥。
甲子，上行幸長安。十一月，甲申，遣使者以中牢祠蕭何、霍光，帝過，式其墓。進幸河東；癸卯，還宮。
十二月，護羌校尉竇林坐欺罔及臧罪，下獄死。林者，融之從兄子也。於是竇氏一公、兩侯、三公主、四二千石相與並時，自祖及孫，官府邸第相望京邑，於親戚功臣中莫與爲比。及林誅，帝數下詔切責融，融惶恐乞骸骨，詔令歸第養病。
是歲，初迎氣於五郊。
新陽侯陰就子豐尚酈邑公主。公主驕妬，豐殺之，被誅，父母皆自殺。
南單于汗死，單于比之子適立，爲䤈僮尸逐侯鞮單于。
