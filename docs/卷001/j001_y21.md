# 卷001 周紀一 — 安王二十三年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王二十三年 ・ 西暦: 379 BCE

[← 巻インデックス](README.md)

安王二十三年〔注:壬寅(じんいん)の年、紀元前三七九年〕。

趙が衛(えい)を襲ったが、攻め落とせなかった。

斉の康公(こうこう)が薨じたが、跡継ぎの子がなく、田氏はついに斉を併合して、これをすべてわが物とした〔注:姜氏(きょうし)の斉は、ここに滅んだ〕。

この年、(田氏の)斉の桓公もまた薨じ、子の威王(いおう)・因斉(いんせい)が立った。

---

二十三年
趙襲衞，不克。
齊康公薨，無子，田氏遂幷齊而有之。
是歲，齊桓公亦薨，子威王因齊立。
