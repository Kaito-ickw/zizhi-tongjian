# 卷004 周紀四 — 赧王中二十六年

赧王二十六年〔注:壬申(じんしん)の年、紀元前二八九年〕。

秦の大良造(だいりょうぞう)である白起(はくき)と、客卿(かくけい)の錯(さく)が魏を攻め、軹(し)まで進んで、大小あわせて六十一の城を奪い取った〔注:大良造は、秦の爵位である大上造(だいじょうぞう)のうち優れた者のこと。大上造は秦の第十六の爵である〕。

---

<details>
<summary>原文を表示</summary>

二十六年
秦大良造白起、客卿錯伐魏，至軹，取城大小六十一。
