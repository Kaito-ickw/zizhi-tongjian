# 卷006 秦紀一 — 昭襄王五十四年

> 巻 6 / 294 ・ 秦紀一 ・ 年号: 昭襄王五十四年 ・ 西暦: 253 BCE

五十四年〔注:戊申(ぼしん)の年、紀元前二五三年〕。

秦王(昭襄王)が雍(よう)で天帝を郊祭(こうさい)した〔注:雍は扶風に属する地で、秦の惠公がここに都し、天地や五帝を祭る五畤(ごし)があった。そこでここで上帝を郊祭し、天子の礼を行おうとしたのである〕。

楚が鉅陽(きょよう)へ都を遷した〔注:楚はかつて赧王三十七年に郢(えい)から東北の陳(ちん)へ遷っており、いま陳から鉅陽へ遷ったもの。ただしこの時もまだ陳の地を離れたわけではない〕。

五十四年
王郊見上帝於雍。
楚遷于鉅陽。
