# 卷104 晉紀二十六 — 烈宗孝武皇帝上之中二年

①春、高句麗(こうくり)・新羅(しらぎ)〔注:新羅は弁韓の末裔で漢の楽浪郡の地に居した〕・西南夷はみな使者を遣わして秦に朝貢した。

②趙(後趙)の旧将作功曹の熊邈(ゆうばく)は、しばしば秦王苻堅に石氏(後趙の石氏一族)の宮室や調度品の贅を尽くした様を語ったので、苻堅は熊邈を将作長史とし将作丞を兼任させ、船や兵器を大々的に造らせ、金銀で飾らせて、たいそう精巧を極めた。慕容農(ぼようのう)はひそかに慕容垂に言った――「王猛が死んでからというもの、秦の法制は日に日に頽廃してきています。いままた奢侈が重なれば、災いが訪れることでしょう。図讖(ずしん、予言の書)の言葉も、いずれ的中するはずです〔注:慕容農の見立ては慕容紹・慕容楷と同じであった〕。大王はよろしく英傑を集めて味方につけ、天意にお応えになるべきです。この好機を逃してはなりません」。慕容垂は笑って「天下の事は、お前の及ぶところではない」と言った。

③桓豁(かんかつ)は兗州刺史の朱序を梁州刺史に推挙し、襄陽に鎮させた。

④秋、七月、丁未の日、尚書僕射の謝安を司徒としたが、謝安は辞退して拝命しなかった。改めて侍中を加え、揚・豫・徐・兗・青の五州諸軍事の都督とした。

丙辰の日、征西大将軍・荊州刺史の桓豁が没した。冬、十月、辛丑の日、桓沖を江・荊・梁・益・寧・交・広の七州諸軍事の都督とし、荊州刺史を兼任させた。桓沖の子の桓嗣(かんし)を江州刺史とした。また五兵尚書の王蘊を都督江南諸軍事とし、徐州刺史を兼任させ〔注:江南諸軍とは晋陵(しんりょう)の諸軍を指す〕、征西司馬で南郡相を兼ねる謝玄を兗州刺史とし、広陵相を兼任させて、江北諸軍事を監督させた。

桓沖は秦が強盛であることから、江南〔注:ここでの江南とは上明を指す〕を頼みに拠点を移そうと考え、江陵から上明(じょうめい)に鎮を移すことを上奏し、冠軍将軍の劉波に江陵を守らせ、諮議参軍の楊亮に江夏を守らせた。

王蘊は徐州刺史の任を固く辞退したが、謝安は「そなたは皇后の父という重い立場にあるのだから、みだりに卑下して、その時々の恩遇を損なうべきではない」〔注:時遇とは、その時々の恩遇を言う〕と言った。そこで王蘊は命を受けた。

これより先、中書郎の郗超(ちちょう)は、自分の父の郗愔(ちいん)は本来ならば地位や処遇が謝安の上位にあるべきだと考えていたが、謝安が朝廷の実権を掌握する一方で、郗愔は閑職に留め置かれていた〔注:郗愔は徐州・兗州二州の刺史から会稽に鎮を移された〕。郗超はそのことに常に憤懣を抱き、それを言葉や顔つきに表していたため、これによって謝氏との間に隙ができていた。当時、朝廷はちょうど秦の侵攻を憂えており、北方を鎮め防ぐことのできる文武の良将を求める詔を出したところ、謝安は兄の子の謝玄をこれに応じさせて推挙した。郗超はこれを聞いて嘆じて言った――「謝安の明察さは、衆人の意に反してまで身内を推挙できるほどのものだ。謝玄の才は、その推挙に恥じないだけのものがある」。人々はみなそうは思わなかった。郗超は「私はかつて謝玄とともに桓公(桓温)の幕府にいたことがあり、彼が人材を用いるところを見たが、履(かわぐつ)や屐(木ぐつ)のようなささいな物事の間でさえ、その人に適した役目を与えないことはなかった。だから私はそれを知っているのだ」と言った。

謝玄は勇猛な士を募り、彭城の劉牢之(りゅうろうし)ら数人を得た。劉牢之を参軍とし、常に精鋭を率いて先鋒とさせたところ、戦えば勝たないことはなかった。当時「北府兵(ほくふへい)」〔注:晋の人々は京口を北府と呼んだ。謝玄が俱難らを破ってから徐州刺史を兼ねるようになったので、史書はまとめてこう呼ぶ〕と呼ばれ、敵はこれを恐れた。

⑤壬寅の日、護軍将軍・散騎常侍の王彪之(おうひょうし)が没した。これより先、謝安が宮室を増築しようとしたとき、王彪之は「中興(東晋建国)の初め、東府(とうふ)をそのまま宮殿としましたが、たいそう質素で見劣りするものでした。蘇峻(そしゅん)の乱の際、成帝は蘭台(御史台)の官が会する場所に留まりましたが、ほとんど寒暑をしのげぬほどでした。そこで改めて新しい宮殿を造営したのです。漢・魏の宮殿と比べれば質素なものですが、江を渡ったばかりの当初と比べれば贅沢と言えましょう。いま敵はまさに強盛なのですから、どうして大がかりな土木の役を起こし、百姓を苦しめ煩わせてよいものでしょうか」と言った。謝安は「宮室がみすぼらしく見劣りしていては、後世の人に無能と言われよう」と言った。王彪之は「およそ天下の重責を担う者は、国を保ち家を安んじ、政事を明らかにすることをこそ務めとすべきです〔注:「緝」は続けるの意、「熙」は広げるの意〕。それを、家屋を修築することが能とでも言うのですか」と言った。謝安はその議論を覆すことができず、そのため王彪之が世を去るまで、造営は行われなかった。

⑥十二月、臨海太守の郗超(ちちょう)が没した。これより先、郗超は桓氏の一派であったが、父の郗愔が朝廷に忠実であったため、そのことを知らせずにいた。病が重くなるに及んで、一つの書箱を門下生に渡して言った――「父上はご高齢だ。私が死んだのち、もし悲しみのあまり寝食を損なうようなことがあれば、この箱を差し上げよ。そうでなければ、そのまま焼き捨てよ」。やがて郗愔は案の定、悲しみのあまり病を発したので、門下生は箱を差し出した。中身はすべて桓温とのやり取りの密議であった。郗愔は大いに怒って「この馬鹿息子め、死ぬのが遅すぎたわ」と言い、それきり泣くのをやめた。

二年
春，高句麗、新羅、西南夷皆遣使入貢于秦。
趙故將作功曹熊邈屢爲秦王堅言石氏宮室器玩之盛，堅以邈爲將作長史，領將作丞，大脩舟艦、兵器，飾以金銀，頗極精巧。慕容農私言於慕容垂曰：「自王猛之死，秦之法制，日以頹靡，今又重之以奢侈，殃將至矣，圖讖之言，行當有驗。大王宜結納英傑以承天意，時不可失！」垂笑曰：「天下事非爾所及！」
桓豁表兗州刺史朱序爲梁州刺史，鎭襄陽。
秋，七月，丁未，以尚書僕射謝安爲司徒，安讓不拜；復加侍中、都督揚‧豫‧徐‧兗‧青五州諸軍事。
丙辰，征西大將軍、荊州刺史桓豁卒。冬，十月，辛丑，以桓沖都督江、荊、梁、益、寧、交、廣七州諸軍事，領荊州刺史；以沖子嗣爲江州刺史。又以五兵尚書王蘊都督江南諸軍事，領徐州刺史；征西司馬領南郡相謝玄爲兗州刺史，領廣陵相，監江北諸軍事。
桓沖以秦人強盛，欲移阻江南，奏自江陵徙鎭上明，使冠軍將軍劉波守江陵，諮議參軍楊亮守江夏。
王蘊固讓徐州，謝安曰：「卿居后父之重，不應妄自菲薄，以虧時遇。」蘊乃受命。
初，中書郎郗超自以其父愔位遇應在謝安之右，而安入掌機權，愔優遊散地，常憤邑形於辭色，由是與謝氏有隙。是時朝廷方以秦寇爲憂，詔求文武良將可以鎭禦北方者，謝安以兄子玄應詔。超聞之，歎曰：「安之明，乃能違衆舉親；玄之才，足以不負所舉。」衆咸以爲不然。超曰：「吾嘗與玄共在桓公府，見其使才，雖履屐間未嘗不得其任，是以知之。」
玄募驍勇之士，得彭城劉牢之等數人。以牢之爲參軍，常領精銳爲前鋒，戰無不捷。時號「北府兵」，敵人畏之。
壬寅，護軍將軍、散騎常侍王彪之卒。初，謝安欲增脩宮室，彪之曰：「中興之初，卽東府爲宮，殊爲儉陋。蘇峻之亂，成帝止蘭臺都坐，殆不蔽寒暑，是以更營新宮。比之漢、魏則爲儉，比之初過江則爲侈矣。今寇敵方強，豈可大興功役，勞擾百姓邪！」安曰：「宮室弊陋，後人謂人無能。」彪之曰：「凡任天下之重者，當保國寧家，緝熙政事，乃以脩室屋爲能邪！」安不能奪其議，故終彪之之世，無所營造。
十二月，臨海太守郗超卒。初，超黨於桓氏，以父愔忠於王室，不令知之。及病甚，出一箱書授門生曰：「公年尊，我死之後，若以哀惋害寢食者，可呈此箱；不爾，卽焚之。」旣而愔果哀惋成疾，門生呈箱，皆與桓溫往反密計。愔大怒曰：「小子死已晚矣！」遂不復哭。
