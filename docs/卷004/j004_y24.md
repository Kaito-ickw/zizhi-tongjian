# 卷004 周紀四 — 赧王中四十一年

> 巻 4 / 294 ・ 周紀四 ・ 年号: 赧王中四十一年 ・ 西暦: 274 BCE

赧王四十一年〔注:丁亥(ていがい)の年、紀元前二七四年〕。

魏がふたたび斉と合従(がっしょう)の同盟を結んだ。秦の穰侯(じょうこう)が魏を攻め、四つの城を攻め落とし、四万の首を斬った。

魯の湣公(びんこう)が亡くなり、その子の頃公(けいこう)讎(しゅう)が立った〔注:「頃」という諡(おくりな)は、心を奮い起こし、おそれつつしむさまをいう、ともいわれる〕。

---

<summary>原文を表示</summary>

四十一年
魏復與齊合從。秦穰侯伐魏，拔四城，斬首四萬。
魯湣公薨，子頃公讎立。
