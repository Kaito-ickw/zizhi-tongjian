# 卷001 周紀一 — 安王二十四年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王二十四年 ・ 西暦: 378 BCE

安王二十四年〔注:癸卯(きぼう)の年、紀元前三七八年〕。

狄(てき)が魏の軍を澮(かい)で破った。

魏・韓・趙が斉を攻め、靈丘(れいきゅう)まで進軍した。

晋の孝公が薨じ、子の靖公俱酒(せいこうぐしゅ)が立った。

---

<details>
<summary>原文を表示</summary>

二十四年
狄敗魏師于澮。
魏、韓、趙伐齊，至靈丘。
晉孝公薨，子靖公俱酒立。

</details>
