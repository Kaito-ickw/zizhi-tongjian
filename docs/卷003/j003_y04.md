# 卷003 周紀三 — 愼靚王四年

四年〔注:甲辰(こうしん)の年、紀元前三一七年〕。

秦が韓の軍を脩魚(しゅうぎょ)で破り、八万の首級を斬り、韓の将である䱸(しゅう)と申差(しんさ)を濁沢(だくたく)で捕らえた。諸侯はみな震えあがった。

齊の大夫が蘇秦(そしん)と(王の)寵を争い、刺客を放って蘇秦を刺させ、これを殺した。

張儀(ちょうぎ)が魏の襄王を説いて言った。「梁(=魏)の領土は四方千里にも届かず、兵は三十万を超えません。土地は四方とも平坦で、要害となる名だたる山も大河もなく、兵は楚・韓・齊・趙との国境を守りに出ています。亭(てい)や障(しょう)〔注:亭は人々が安んじて泊まる宿駅、障は敵を遮り防ぐための砦〕を守る兵は十万を超えません。梁の地勢は、もとより戦場になるしかない土地なのです。そもそも諸侯が結んだ合従(がっしょう)の盟約は、洹水(かんすい)のほとりで誓いを立て、兄弟の契りを結んで結束を固めたものでした〔注:このことは前の巻、顯王二十六年に見える〕。しかし、同じ父母から生まれた実の兄弟ですら、なお金銭をめぐって争い、傷つけ殺し合うものです。それなのに、裏切りを繰り返す蘇秦の使い残しの策謀をあてにしようとしても、それが成り立たぬことは火を見るより明らかです。大王が秦に従わなければ、秦は兵を繰り出して河外(かがい)を攻め、卷(けん)・衍(えん)・酸棗(さんそう)を押さえ〔注:このうち卷と酸棗は黄河の渡し場として要となる地である〕、衞(えい)を脅し、陽晉(ようしん)を奪うでしょう。そうなれば趙は南下できず、趙が南下できなければ梁は北上できず、梁が北上できなければ合従の道は断たれ、合従の道が断たれれば、大王の国が危うくならずにすむことなど、もはやかなわぬのです。ですからどうか大王には、よくよく計略をお定めいただきたい。そのうえで、わたくしめにはお暇を賜りますよう〔注:臣下が君に身を委ねれば、その身はもはや自分のものではない。だから退任を願い出ることを『骸骨を乞う』という〕」。そこで魏王は合従の盟約に背き、張儀を仲立ちにして秦に和睦を申し入れた。張儀は(秦へ)帰り、ふたたび秦の宰相となった。

魯の景公が薨じ、子の平公旅(へいこうりょ)が立った。

四年
秦敗韓師于脩魚，斬首八萬級，虜其將䱸、申差于濁澤。諸侯振恐。
齊大夫與蘇秦爭寵，使人刺秦，殺之。
張儀說魏襄王曰：「梁地方不至千里，卒不過三十萬，地四平，無名山大川之限，卒戍楚、韓、齊、趙之境，守亭、障者不過十萬，梁之地勢固戰場也。夫諸侯之約從，盟於洹水之上，結爲兄弟以相堅也。今親兄弟同父母，尚有爭錢財相殺傷，而欲恃反覆蘇秦之餘謀，其不可成亦明矣。大王不事秦，秦下兵攻河外，據卷、衍、酸棗，劫衞，取陽晉，則趙不南，趙不南則梁不北，梁不北則從道絕，從道絕則大王之國欲毋危不可得也。故願大王審定計議，且賜骸骨。」魏王乃倍從約，而因儀以請成于秦。張儀歸，復相秦。
魯景公薨，子平公旅立。
