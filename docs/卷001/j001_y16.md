# 卷001 周紀一 — 安王十七年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 安王十七年 ・ 西暦: 385 BCE

[← 巻インデックス](README.md)

---

安王十七年〔注:丙申(へいしん)の年、紀元前三八五年〕。

秦の庶長(しょちょう)である改(かい)が、河西(かせい)にいた献公(けんこう)を迎え入れて君主に立てた〔注:「庶長」は秦の官位の名。河西は黄河の西の地で、献公はかつて位を継げずこの地に身を寄せていた〕。そして出子(しゅっし)とその母を殺し、その亡骸を淵のほとりに沈めた。

斉が魯を攻めた。

韓が鄭を攻め、陽城(ようじょう)を奪い取った。さらに宋を攻め、宋公(そうこう)を捕らえた。

斉の太公(田和)が薨じ、子の桓公・午(ご)が立った。

十七年
秦庶長改逆獻公于河西而立之；殺出子及其母，沈之淵旁。
齊伐魯。
韓伐鄭，取陽城；伐宋，執宋公。
齊太公薨，子桓公午立。
