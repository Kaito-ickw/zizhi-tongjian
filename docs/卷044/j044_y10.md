# 卷044 漢紀三十六 — 世祖光武皇帝下中元中元元年

中元元年〔注:『続漢書』志はこの年を「建武三十二年を建武中元元年と改めた」と記すが、本書は袁宏(えんこう)『後漢紀』・范曄(はんよう)『後漢書』に従って、簡明に「中元元年」とする〕。

① 春、正月、淮陽王(わいようおう)が入朝(にゅうちょう)した。第五倫(だいごりん)も淮陽王の官属に従って、(帝に)お目通りすることができた。帝が政事について問うと、第五倫はこれに応じて受け答えをした。帝はおおいに喜び、翌日、改めて特別に召し入れ、夕方まで語り合った。帝は第五倫に言った。「聞くところでは、お前は役人であったころ、妻の父を笞(むち)打ち、従兄(いとこ)の家を訪ねても飯も食わなかったというが、本当にそんなことがあったのか」。第五倫は答えた。「私は三度妻を娶(めと)りましたが、いずれの妻にも父はおりませんでした(笞打つべき妻の父などおりません)。また若いころに飢饉(ききん)と戦乱に遭い、まことにみだりに他人の家で食事をすることなど、いたしませんでした。人々は私を愚かで世事に疎(うと)い者と見て、こんな話をこしらえたのでございましょう」。帝はおおいに笑った。そして第五倫を扶夷(ふい)の長に任じたが、彼がまだ任地に到着しないうちに、追って会稽(かいけい)太守に任じた。第五倫は政治を清廉に行い、しかも恵み深かったので、民は彼を慕った。

② 帝は『河図会昌符(かずかいしょうふ)』を読んだ。そこには「赤い劉氏の九代目が、岱宗(たいそう、泰山)に会して天命を受ける」とあった〔注:岱(たい)は泰山の別名で、岱宗ともいう。岱は「始まり」、宗は「長(おさ)」の意で、泰山は五岳の長とされる〕。帝はこの一文に心を動かされ、虎賁中郎将(こほんちゅうろうしょう)の梁松(りょうしょう)らに詔(みことのり)して、『河図』『洛書(らくしょ)』の讖文(しんもん、予言の書)を調べさせた。すると「(劉氏の)九代目が封禅(ほうぜん)を行うべきだ」とする箇所が、全部で三十六件もあった。そこで張純(ちょうじゅん)らが改めて封禅を行うよう奏上し〔注:泰山の頂に土を盛って壇を築き天を祭るのを「封」、泰山のふもとの小山で地を払い清めて地を祭るのを「禅」という〕、帝もついにこれを許した。帝は担当の役人に、元封(げんぽう)の故事を調べさせた〔注:元封の故事とは、漢の武帝が封禅を行ったときの先例である〕。それによれば、方形の石を二重に積み重ね、玉の檢(けん、文書を納める覆い)と金泥(きんでい)を用いるという。帝は石を加工する工事が難しく仕上げにくいと考え、武帝が昔(封禅で)用いた封石をそのまま使い、その中に玉牒(ぎょくちょう、祭文を刻んだ玉の板)を納めようとした。だが梁松らが、それはよくないと強く反対したので、帝は石工に命じて、傷のない青石を取らせることにし、必ずしも五色の石にはこだわらないこととした〔注:旧来の制度では石を用い、それぞれ方角の色に合わせた五色の石を用いていた〕。

丁卯(ていぼう)の日、帝の車駕(しゃが)は東方へ巡幸した。二月の己卯(きぼう)の日、魯(ろ)に行幸し、さらに進んで泰山に行幸した。辛卯(しんぼう)の日の早朝、燎(かがりび)を焚き、泰山のふもとの南方で天を祭った。もろもろの神々もみな合わせて祭られ、南郊(なんこう)の祭祀と同じ音楽を用いた。儀式が終わり、食事どき(午前なかば)になると、天子は輦(れん、てぐるま)に乗って山を登り、正午すぎに山頂に到り、衣を改めた。晡時(ほじ、午後の遅い時刻)になって、壇に登り北を向いて立った。尚書令(しょうしょれい)が玉牒を納めた檢を捧げ持ち、天子は一寸二分の璽(じ、印)で自らこれを封じた。それが済むと、太常(たいじょう)が騶騎(すうき、騎馬の供回り)二千余人に命じて壇上の方石を持ち上げさせ、尚書令が玉牒をその下に納め終えると、また石を元どおりにかぶせ、尚書令は五寸の印で石の檢を封じた。儀式が終わると、天子は二度拝礼した。群臣は万歳を唱え、それから来た道を通って下山した。真夜中をすぎてようやく帝は山のふもとに着き、百官は翌朝になってやっと(下山を)終えた。甲午(こうご)の日、梁父(りょうほ)山の北側で地を祭る禅(ぜん)の儀を行い、高后(こうこう、高祖の后)を配して祭り、山川のもろもろの神々も合わせて祭った。これは元始(げんし)年間の北郊の祭祀の先例にならったものである。

③ 三月の戊辰(ぼしん)の日、司空の張純が亡くなった。

④ 夏、四月の癸酉(きゆう)の日、帝の車駕は宮に還った。己卯(きぼう)の日、天下に大赦を行い、元号を改めた(中元元年とした)。

⑤ 帝は長安(ちょうあん)に行幸した。五月の乙丑(いっちゅう)の日、宮に還った。

⑥ 六月の辛卯(しんぼう)の日、太僕(たいぼく)の馮魴(ふうほう)を司空とした。

⑦ 乙未(いつび)の日、司徒の馮勤(ふうきん)が亡くなった。

⑧ 都(みやこ)に醴泉(れいせん、甘い泉)が湧き出し、また水辺に赤草(せきそう、朱草)が生え〔注:朱草は、一日に一葉ずつ生え、十五日を過ぎると一日に一葉ずつ落ちる、めでたい草とされる〕、各地の郡国からはたびたび甘露(かんろ)が献上された。群臣が奏上して言った。「めでたい瑞祥(ずいしょう)の品々がしきりに現れております。太史(たいし、史官の長)に命じて集め記録させ、後の世に伝えるべきです」。だが帝は受け入れなかった。帝は自ら「私には徳がない」と謙遜し、郡国から献上される瑞祥については、いつも抑えて取り合わなかった。それゆえ史官も、これらをほとんど記録できなかったのである。

⑨ 秋、三つの郡国に蝗(いなご)の害があった。

⑩ 冬、十月の辛未(しんび)の日、司隷校尉(しれいこうい)で東莱(とうらい)の李訢(りきん)を司徒とした。

⑪ 甲申(こうしん)の日、司空を遣わして高廟(こうびょう、高祖の廟)に告げ祭らせ、薄太后(はくたいこう)に「高皇后」の尊号を奉り、地祇(ちぎ、土地の神)に配して祭った。一方、呂太后(りょたいこう)の廟主(びょうしゅ、位牌)は園(その墓域)に移し〔注:呂太后はかつて劉氏の天下を危うくしたためである〕、四季ごとにそこで祭ることにした。

⑫ 十一月の甲子(こうし)の晦日(みそか)、日食があった。

⑬ この年、明堂(めいどう)・霊台(れいだい)・辟雍(へきよう)を建て〔注:明堂は天子が政教を行う殿堂、辟雍は周囲に水をめぐらせた天子の学宮、霊台は天文を観測する台である〕、図讖(としん、予言の書)を天下に公布した。

さかのぼって、帝は『赤伏符(せきふくふ)』(という讖文)によって帝位に即(つ)いた〔注:この経緯は本書巻四十・建武元年に見える〕。そのため讖文を信用し、多くの疑わしい事柄をこれで決めていた。給事中(きゅうじちゅう)の桓譚(かんたん)が上疏(じょうそ)して諫(いさ)めて言った。

「およそ人の情(じょう)というものは、目の前のありふれた事をおろそかにし、珍しい異聞をありがたがるものです。いにしえの聖王の記したところを見ますと、いずれも仁義と正道を根本としており、奇怪で根拠のないでたらめな話などありません。そもそも天道や性命(せいめい、人の本性と運命)は、聖人ですら語るのが難しいもので、孔子の高弟の子貢(しこう)以下の者でさえ聞くことができませんでした〔注:『論語』に、子貢が「先生が性と天道について語られるのは、聞くことができなかった」と言ったとある〕。まして後世の浅はかな儒者に、これが究められましょうか。いま、こざかしい小才の者や、方術・数術を扱う輩(やから)が、図書(としょ、讖緯や符命のたぐい)に勝手な付け足しをし、讖記(しんき)だと偽って称し、それで欲深く邪(よこしま)な者をたぶらかし、君主を誤らせております。どうしてこれを退け遠ざけずにいられましょうか。私こと譚(たん)がうけたまわるところ、陛下は方士(ほうし)の黄白の術〔注:薬によって金銀を作り出すという術〕を徹底して論破なさったとのこと、まことに賢明であられます。それなのに讖記だけは聞き入れようとなさる。これまた何たる誤りでしょうか。讖がたまに当たることがあっても、それは占いの数がたまたま合うようなもの(まぐれ当たり)にすぎません。陛下はどうか明らかな徳を垂れ、聖なるお心を発(はっ)して、つまらぬ者どもの曲がった説を退け、『五経』の正しい教えを述べられますように」。

この上疏が奉られると、帝は不機嫌になった。ちょうど霊台をどこに置くかを評議していたとき、帝は桓譚に言った。「私はこれを讖で決めたいと思うが、どうか」。桓譚は黙りこみ、しばらくしてから言った。「私は讖を読みません」。帝がそのわけを問うと、桓譚はまたしても、讖は経書(けいしょ)などではない、と言い立てた。帝は激怒して言った。「桓譚は聖人をそしり、法を無視する者だ。引き下ろして斬れ!」。桓譚は頭を地に打ちつけて血を流し、長いことそうしてようやく罪を解かれた。そして六安郡(りくあんぐん)の丞(じょう)に左遷され、赴任の途中で病死した〔注:桓譚が六安郡の丞に出されたのは実はこの年ではないが、通鑑は霊台の一件にちなんで、ここにまとめて記したのである〕。

范曄(はんよう)の論評。「桓譚は讖を善(よ)しとしなかったために流浪して死に、鄭興(ていこう)はへりくだった言葉でかろうじて罪を免れた。賈逵(かき)は讖に文章をこじつけて取り入ることができたので、最も身分が高く顕(あら)われた〔注:賈逵は明帝の永平年間に、『春秋左氏伝』が図讖と合致し、劉氏が聖王・堯(ぎょう)の後裔(こうえい)であることを明かす、と上言して帝に嘉(よみ)せられ、侍中などに昇った〕。世の君主がこんな調子で学問を論じるとは、何と悲しいことか!」。

⑭ 南匈奴の単于(ぜんう)である比(ひ)が死に、弟で左賢王(さけんおう)の莫(ばく)が立って、丘浮尤鞮単于(きゅうふゆうてい・ぜんう)となった。帝は使者を遣わし、璽書(じしょ、天子の印を押した文書)を持たせて璽綬(じじゅ、印と組みひも)を授け、衣冠(いかん)や繒綵(そうさい、絹織物)を賜った。これ以後、これが慣例となった。

中元元年
春，正月，淮陽王入朝，倫隨官屬得會見。帝問以政事，倫因此酬對，帝大悅；明日，復特召入，與語至夕。帝謂倫曰：「聞卿爲吏，篣婦公，不過從兄飯，寧有之邪？」對曰：「臣三娶妻，皆無父。少遭饑亂，實不敢妄過人食。衆人以臣愚蔽，故生是語耳。」帝大笑。以倫爲扶夷長，未到官，追拜會稽太守；爲政清而有惠，百姓愛之。
上讀《河圖會昌符》曰：「赤劉之九，會命岱宗。」上感此文，乃詔虎賁中郎將梁松等按察《河》、《雒》《讖文》，言九世當封禪者凡三十六事。於是張純等復奏請封禪，上乃許焉。詔有司求元封故事，當用方石再累，玉檢、金泥。上以石功難就，欲因孝武故封石，置玉牒其中，梁松等爭以爲不可，乃命石工取完青石，無必五色。
丁卯，車駕東巡；二月，己卯，幸魯，進幸泰山。辛卯，晨，燎，祭天於泰山下南方，羣神皆從，用樂如南郊。事畢，至食時，天子御輦登山，日中後，到山上，更衣。晡時，升壇北面，尚書令奉玉牒檢，天子以寸二分璽親封之，訖，太常命騶騎二千餘人發壇上方石，尚書令藏玉牒已，復石覆訖，尚書令以五寸印封石檢。事畢，天子再拜。羣臣稱萬歲，乃復道下。夜半後，上乃到山下，百官明旦乃訖。甲午，禪祭地於梁陰，以高后配，山川羣神從，如元始中北郊故事。
三月，戊辰，司空張純薨。
夏，四月，癸酉，車駕還宮；己卯，赦天下，改元。
上行幸長安；五月，乙丑，還宮。
六月，辛卯，以太僕馮魴爲司空。
乙未，司徒馮勤薨。
京師醴泉湧出，又有赤草生於水崖，郡國頻上甘露。羣臣奏言：「靈物仍降，宜令太史撰集，以傳來世。」帝不納。帝自謙無德，于郡國所上，輒抑而不當，故史官罕得記焉。
秋，郡國三蝗。
冬，十月，辛未，以司隸校尉東萊李訢爲司徒。
甲申，使司空告祠高廟，上薄太后尊號曰高皇后，配食地祇。遷呂太后廟主于園，四時上祭。
十一月，甲子晦，日有食之。
是歲，起明堂、靈臺、辟雍，宣布圖讖於天下。
初，上以《赤伏符》卽帝位，由是信用讖文，多以決定嫌疑。給事中桓譚上疏諫曰：「凡人情忽於見事而貴於異聞。觀先王之所記述，咸以仁義正道爲本，非有奇怪虛誕之事。蓋天道性命，聖人所難言也，自子貢以下，不得而聞，況後世淺儒，能通之乎！今諸巧慧小才、伎數之人，增益圖書，矯稱讖記，以欺惑貪邪，詿誤人主，焉可不抑遠之哉！臣譚伏聞陛下窮折方士黃白之術，甚爲明矣；而乃欲聽納讖記，又何誤也！其事雖有時合，譬猶卜數隻偶之類。陛下宜垂明德，發聖意，屛羣小之曲說，述《五經》之正義。」疏奏，帝不悅。會議靈臺所處，帝謂譚曰：「吾欲以讖決之，何如？」譚默然，良久曰：「臣不讀讖。」帝問其故，譚復極言讖之非經。帝大怒曰：「桓譚非聖無法，將下，斬之！」譚叩頭流血，良久，乃得解。出爲六安郡丞，道病卒。
范曄論曰：桓譚以不善讖流亡，鄭興以遜辭僅免；賈逵能傅會文致，最差貴顯；世主以此論學，悲哉！
南單于比死，弟左賢王莫立，爲丘浮尤鞮單于。帝遣使齎璽書拜授璽綬，賜以衣冠及繒綵，是後遂以爲常。
