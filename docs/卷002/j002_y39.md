# 卷002 周紀二 — 顯王四十七年

顯王四十七年〔注:己亥(つちのとい)の年、紀元前三二二年〕。

秦の張儀が齧桑から帰ると、秦の宰相を免ぜられて魏の宰相となった〔注:秦の相を免じられて魏の相となったのである〕。張儀は、魏にまず率先して秦に仕えさせ、ほかの諸侯にもそれに倣わせようと図った。だが魏王は聞き入れなかった。そこで秦王は魏を攻め、曲沃(きょくよく)・平周(へいしゅう)を奪った〔注:この曲沃は河東の地で、晉の桓叔(かんしゅく)が封ぜられた邑(むら)。漢の武帝のとき聞喜(ぶんき)と改名された〕。そして秦は、ひそかに張儀を以前にもまして手厚く遇した。

四十七年
秦張儀自齧桑還而免相，相魏。欲令魏先事秦而諸侯效之；魏王不聽。秦王伐魏，取曲沃、平周，復陰厚張儀益甚。
