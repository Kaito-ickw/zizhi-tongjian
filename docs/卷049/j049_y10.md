# 卷049 漢紀四十一 — 孝安皇帝上二年

二年〔注:乙卯(きのとう)の年、紀元一一五年〕。

春、護羌校尉(ごきょうこうい)の龐参(ほうしん)が、恩と信義をもって諸羌(しょきょう、羌の諸部族)を招き寄せると、號多(ごうた)らは人々を率いて降伏した。龐参は號多を都(みやこ)の朝廷に出頭させ、號多に侯(こう)の印を授けて帰らせた。龐参はようやく治所(ちしょ、役所の本拠)を令居(れいきょ)に戻し〔注:張掖(ちょうえき)から令居へ移し戻したのである〕、河西(かせい)へ通じる道を開いた。

零昌(れいしょう)が兵を分けて益州(えきしゅう)を侵したので、中郎将(ちゅうろうしょう)の尹就(いんしゅう)を遣わしてこれを討たせた。

夏、四月、丙午(ひのえうま)の日、貴人(きじん、後宮の高位の妃)である滎陽(けいよう)の閻氏(えんし)を立てて皇后とした〔注:閻后の母は、鄧弘(とうこう)の妻と母を同じくする間柄(同腹のきょうだい)であり、それゆえ立后がかなったのである〕。后は生まれつき嫉妬深く、後宮の李氏(りし)が皇子の保(ほ)を生むと、后は李氏を鴆(ちん、毒酒)で殺した〔注:これがのちに保が廃される伏線となる。保はのちに立てられ、これが順帝(じゅんてい)である〕。

五月、京師(けいし、みやこ)で旱(ひでり)があり、河南(かなん、すなわち京師の地)および十九の郡国で蝗(いなご)の害があった。

六月、丙戌(ひのえいぬ)の日、太尉(たいい)の司馬苞(しばほう)が薨(こう)じた。

秋、七月、辛巳(かのとみ)の日、太僕(たいぼく、天子の車馬をつかさどる長官)である泰山(たいざん)の馬英(ばえい)を太尉とした。

八月、遼東(りょうとう)の鮮卑(せんぴ)が無慮(むりょ)を包囲した。九月、さらに夫犂(ふり)の営(えい、とりで)を攻め、県令(けんれい)を殺した。

壬午(みずのえうま)の晦(みそか)、日食があった。

尹就が羌の一味である呂叔都(りょしゅくと)らを撃った。蜀(しょく)の人である陳省(ちんせい)・羅橫(らおう)が募(つの)りに応じて叔都を刺し殺し、ともに侯に封ぜられ、銭を賜った。

詔(みことのり)を下して、屯騎校尉(とんきこうい)の班雄(はんゆう)を三輔(さんぽ、長安周辺の三つの行政区)に駐屯させた。雄は班超(はんちょう)の子である。左馮翊(さひょうよく)の司馬鈞(しばきん)を征西将軍(せいせいしょうぐん)の代行とし、関中(かんちゅう)の諸郡の兵八千人あまりを統率させた。龐参は羌・胡(こ)の兵七千人あまりを率い、鈞と道を分けてともに零昌を撃った。龐参の軍は勇士(ゆうし)県の東に至ったところで〔注:勇士県は天水(てんすい)郡に属する。ただしこのころ天水はすでに漢陽(かんよう)郡と改められていた〕、杜季貢(ときこう)に敗れて退いた。鈞らは単独で進み、丁奚城(ていけいじょう)を攻め落としたが、杜季貢は手勢を率いて偽りの逃走をした。鈞は右扶風(ゆうふふう)の仲光(ちゅうこう)らに命じて羌の穀物を刈り取らせたが、光らは鈞の指図(さしず)に背いて兵を散らし、深入りした。羌はそこで伏兵を設けて待ち伏せ、これを迎え撃った。鈞は城中にあって、怒って救援しなかった。冬、十月、乙未(きのとひつじ)の日、光らの軍は敗れ、光ら自身もともに討ち死にして、戦死者は三千人あまりに上った。鈞はそこで逃げ帰った。龐参はすでに(合流の)期日に遅れていたので、病と称して兵を引き戻した。いずれも罪に問われて召喚され、獄に下された。鈞は自殺した。このとき度遼将軍(とりょうしょうぐん)の梁慬(りょうきん)も事に連座して罪に問われた。校書郎中(こうしょろうちゅう、宮中の書物を校訂する官)である扶風(ふふう)の馬融(ばゆう)が上書し、龐参と梁慬の知略・才能を称えて、過ちを許して功で償わせるべきだと説いた。詔が下って龐参らを赦(ゆる)し、馬賢(ばけん)を龐参に代えて護羌校尉を兼ねさせ、また任尚(じんしょう)を中郎将として、班雄に代えて三輔に駐屯させた。

懷令(かいれい、懐県の長官)の虞詡(ぐく)が任尚に説いて言った。「兵法に『弱きは強きを攻めず、走る者は飛ぶ者を追わず』とあります。これは自然の道理です。いま敵はみな騎馬で、一日に数百里を行き、来るさまは風雨のごとく、去るさまは切れた弓弦(ゆづる)のごとくです。歩兵でこれを追っても、勢いから追いつけません。だからこそ、二十万あまりの兵を駐屯させていながら、日を空費して功がないのです。あなた(使君〔しくん、地方の長官への敬称〕)のために計るなら、諸郡の兵をいったん解散させ、それぞれに数千銭を出させ、二十人で一頭の馬を共同で買わせるに越したことはありません。そうして一万騎の軍勢で、数千の敵を追い、その背後に食らいついて挟み撃ちにすれば、敵の進む道は自然と行き詰まります。民にも都合よく、事も有利に運び、大功が立つでしょう」。任尚はただちに上奏してその計略を用い、軽騎(けいき、身軽な騎兵)を遣わして杜季貢を丁奚城に撃ち、これを破った。

太后は、虞詡に将帥(しょうすい、将軍)の器量があると聞いて、武都(ぶと)太守(たいしゅ)とした。数千の羌が虞詡を陳倉(ちんそう)の崤谷(こうこく)で阻んだ〔注:この崤谷は陳倉県の境にあり、いまの大散関(だいさんかん)であって、弘農(こうのう)澠池(べんち)県の崤山(こうざん)ではない〕。虞詡はただちに軍を止めて進まず、「上書して援兵を求めた。それが到着しだい出発する」と言いふらした。羌はこれを聞いて、兵を分けて近隣の県を荒らしに行った。虞詡は敵の兵が散ったすきに、昼夜を分かたず道を進み、一日に百里あまりの強行軍をした。そして役人や兵士めいめいに竈(かまど)を二つずつ作らせ、その数を日ごとに倍に増やしていったので、羌はあえて近づかなかった。ある者が尋ねた。「孫臏(そんぴん)は竈を減らしたのに、あなたは増やしておられる。兵法では一日の行軍は三十里を超えず、不測の事態に備えるものですが、いまは一日に二百里近くも進む。なぜですか」。虞詡は言った。「敵は数が多く、わが兵は少ない。ゆっくり進めば追いつかれやすく、速く進めば敵には測りがたい。敵はわが竈が日ごとに増えるのを見れば、きっと郡の援兵が迎えに来て、人数が多く進みも速いと思い、われらを追うのをためらうだろう。孫臏は弱さを見せたが、わたしはいま強さを見せている。状況が違うからだ」。

武都の郡に着いてみると、兵は三千にも満たず、羌の軍勢は一万あまりで、赤亭(せきてい)を攻め囲むこと数十日に及んだ。虞詡はそこで軍中に命じ、強弩(きょうど、強い石弓)は射(い)させず、ひそかに小さな弩を射させた。羌は矢の威力が弱くて届かないと思い、兵を集めて激しく攻めてきた。虞詡はそこで二十の強弩で一人を狙って射させると、放つ矢はことごとく命中した。羌は大いに動揺し、退いた。虞詡はそのすきに城を出て奮戦し、多くを傷つけ殺した。翌日、全軍をすべて並べ、東郭門(とうかくもん、東の外城の門)から出て北郭門(ほっかくもん)から入らせ、衣服を着替えさせて何周も巡らせた。羌はその数がわからず、たがいに恐れ騒いだ。虞詡は敵が退くはずだと見て、ひそかに五百人あまりを浅瀬に伏せさせ、敵の逃げ道を見張らせた〔注:虞詡は、敵が退くとき川では浅瀬を踏んで渡るに違いないと察し、そこに伏兵を置いて待ったのである〕。敵は案の定どっと逃げ出し、そこを襲い撃って大いに破り、討ち取り捕らえた者はきわめて多く、敵はこれによって敗れ散った。虞詡はそこで地勢を見定め、営壁(えいへき、とりで)を百八十か所に築き、流亡した人々を呼び戻し、貧しい民に施しを与え、水運を開き通じさせた。虞詡が郡に着いた当初は、穀物は一石につき千銭、塩は一石につき八千銭、現存する戸数は一万三千であった。三年間政務を執ると、米は一石につき八十銭、塩は一石につき四百銭、民は四万戸あまりに増え、人々は満ち足り家ごとに不足なく、郡全体がこうして安定した。

十一月、庚申(かのえさる)の日、十の郡国で地震があった。

十二月、武陵(ぶりょう)の澧中(れいちゅう)の蛮(ばん、異民族)が反乱を起こしたが、州と郡が討って平定した。

己酉(つちのととり)の日、司徒(しと)の夏勤(かきん)が罷免された。

庚戌(かのえいぬ)の日、司空(しくう)の劉愷(りゅうがい)を司徒とし、光禄勲(こうろくくん)の袁敞(えんしょう)を司空とした。敞は袁安(えんあん)の子である。

もとの虎賁中郎将(こほんちゅうろうしょう)である鄧弘(とうこう)が亡くなった〔注:弘は母の喪に遭って官を辞し、奉朝請(ほうちょうせい、朝廷に参列する資格)の身であったので「もとの」という〕。弘は生まれつき倹(つま)しく質素で、欧陽(おうよう)氏の『尚書(しょうしょ)』を学び、宮中で帝に教授していた。担当の役人が、弘に驃騎将軍(ひょうきしょうぐん)を追贈し、特進(とくしん)の位を与え、西平侯(せいへいこう)に封ずるよう奏上した。太后は弘のつねづねの(質素を尊ぶ)意向を重んじ、追贈の位や衣服は加えず、ただ銭一千万・布一万匹を賜うにとどめた。弘の兄の鄧騭(とうしつ)らは、これもまた辞退して受けなかった。詔して弘の子の広徳(こうとく)を西平侯に封じた。葬ろうとするとき、担当の役人がふたたび、五営(ごえい、近衛の五部隊)の軽車(けいしゃ)や騎士を出し、儀礼を霍光(かくこう)の先例どおりにするよう奏上した〔注:霍光が薨じたとき、宣帝(せんてい)は使者を遣わして手厚く喪を取り仕切らせ、立派な棺や葬具を賜り、近衛の兵士に葬送させた〕。太后はいずれも聞き入れず、ただ白蓋(はくがい、白い覆いの車)に騎馬二人を添えるのみとし、門生(もんせい、弟子)たちが棺の綱を引いて見送った。のちに、帝の師であった重みから、西平の都郷(ときょう)を分け、広徳の弟の甫徳(ほとく)を都郷侯(ときょうこう)に封じた。

二年
春，護羌校尉龐參以恩信招誘諸羌，號多等帥衆降；參遣詣闕，賜號多侯印，遣之。參始還治令居，通河西道。
零昌分兵寇益州，遣中郎將尹就討之。
夏，四月，丙午，立貴人滎陽閻氏爲皇后。后性妬忌，後宮李氏生皇子保，后鴆殺李氏。
五月，京師旱，河南及郡國十九蝗。
六月，丙戌，太尉司馬苞薨。
秋，七月，辛巳，以太僕泰山馬英爲太尉。
八月，遼東鮮卑圍無慮；九月，又攻夫犂營，殺縣令。
壬午晦，日有食之。
尹就擊羌黨呂叔都等，蜀人陳省、羅橫應募刺殺叔都，皆封侯，賜錢。
詔屯騎校尉班雄屯三輔。雄，超之子也。以左馮翊司馬鈞行征西將軍，督關中諸郡兵八千餘人。龐參將羌、胡兵七千餘人，與鈞分道並擊零昌。參兵至勇士東，爲杜季貢所敗，引退。鈞等獨進，攻拔丁奚城，杜季貢率衆僞逃。鈞令右扶風仲光等收羌禾稼，光等違鈞節度，散兵深入，羌乃設伏要擊之，鈞在城中，怒而不救。冬，十月，乙未，光等兵敗，並沒，死者三千餘人，鈞乃遁還。龐參旣失期，稱病引還，皆坐徵，下獄，鈞自殺。時度遼將軍梁慬亦坐事抵罪。校書郎中扶風馬融上書稱參、慬智能，宜宥過責效。詔赦參等，以馬賢代參領護羌校尉，復以任尚爲中郎將，代班雄屯三輔。
懷令虞詡說尚曰：「兵法：弱不攻強，走不逐飛，自然之勢也。今虜皆馬騎，日行數百里，來如風雨，去如絕弦，以步追之，勢不相及，所以雖屯兵二十餘萬，曠日而無功也。爲使君計，莫如罷諸郡兵，各令出錢數千，二十人共市一馬，以萬騎之衆，逐數千之虜，追尾掩截，其道自窮。便民利事，大功立矣！」尚卽上言，用其計，遣輕騎擊杜季貢於丁奚城，破之。
太后聞虞詡有將帥之略，以爲武都太守。羌衆數千遮詡於陳倉崤谷，詡卽停軍不進，而宣言「上書請兵，須到當發。」羌聞之，乃分鈔傍縣。詡因其兵散，日夜進道，兼行百餘里，令吏士各作兩竈，日增倍之，羌不敢逼。或問曰：「孫臏減竈而君增之；兵法日行不過三十里，以戒不虞，而今日且二百里：何也？」詡曰：「虜衆多，吾兵少，徐行則易爲所及，速進則彼所不測。虜見吾竈日增，必謂郡兵來迎，衆多行速，必憚追我。孫臏見弱，吾今示強，勢有不同故也。」
旣到郡，兵不滿三千，而羌衆萬餘，攻圍赤亭數十日。詡乃令軍中，强弩勿發，而潛發小弩；羌以爲矢力弱，不能至，幷兵急攻。詡於是使二十强弩共射一人，發無不中，羌大震，退。詡因出城奮擊，多所傷殺。明日，悉陳其兵衆，令從東郭門出，北郭門入，貿易衣服，回轉數周；羌不知其數，更相恐動。詡計賊當退，乃潛遣五百餘人於淺水設伏，候其走路；虜果大奔，因掩擊，大破之，斬獲甚衆，賊由是敗散。詡乃占相地勢，築營壁百八十所，招還流亡，假賑貧民，開通水運。詡始到郡，穀石千，鹽石八千，見戶萬三千；視事三年，米石八十，鹽石四百，民增至四萬餘戶，人足家給，一郡遂安。
十一月，庚申，郡國十地震。
十二月，武陵澧中蠻反，州郡討平之。
己酉，司徒夏勤罷。
庚戌，以司空劉愷爲司徒，光祿勳袁敞爲司空。敞，安之子也。
前虎賁中郎將鄧弘卒。弘性儉素，治歐陽《尚書》，授帝禁中。有司奏贈弘驃騎將軍，位特進，封西平侯。太后追弘雅意，不加贈位、衣服，但賜錢千萬，布萬匹；兄騭等復辭不受。詔封弘子廣德爲西平侯。將葬，有司復奏發五營輕車騎士，禮儀如霍光故事。太后皆不聽，但白蓋雙騎，門生輓送。後以帝師之重，分西平之都鄕，封廣德弟甫德爲都鄕侯。
