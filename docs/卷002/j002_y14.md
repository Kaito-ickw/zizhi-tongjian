# 卷002 周紀二 — 顯王十七年

> 巻 2 / 294 ・ 周紀二 ・ 年号: 顯王十七年 ・ 西暦: 352 BCE

[← 巻インデックス](README.md)

顯王十七年〔注:己巳(きし)の年、紀元前三五二年〕。

秦の大良造(だいりょうぞう)〔注:この大良造とは衞鞅(えいおう)のこと〕が魏を攻めた。

諸侯が魏の襄陵(じょうりょう)を包囲した。

---

<details>
<summary>原文を表示</summary>

十七年
秦大良造伐魏。
諸侯圍魏襄陵。

</details>
