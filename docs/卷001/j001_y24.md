# 卷001 周紀一 — 安王二十六年

安王二十六年〔注:乙巳(いっし)の年、紀元前三七六年〕。

(安)王が崩じ、子の烈王(れつおう)・喜(き)が立った。

魏・韓・趙が、ともに晋の靖公(せいこう)を廃して一介の庶人とし、その領地を分け取った〔注:(これで)唐叔(とうしゅく)への祭祀は絶えた〕。

---

<details>
<summary>原文を表示</summary>

二十六年
王崩，子烈王喜立。
魏、韓、趙共廢晉靖公爲家人而分其地。
