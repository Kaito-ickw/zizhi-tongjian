# 卷003 周紀三 — 赧王上二年

二年〔注:戊申(つちのえさる)の年、紀元前三一三年〕。

秦の右更(ゆうこう)の疾(しつ)〔注:右更は秦の爵位の第十四等である〕が趙を討ち、藺(りん)を攻め落とし、趙の将である莊豹(そうひょう)を捕らえた。

秦王は齊を討とうとしたが、齊と楚が合従(がっしょう)で親しく結んでいるのを気にかけ、そこで張儀(ちょうぎ)を楚に遣わして楚王を説いて言わせた。「大王がまことにわたくしの言うことを聞き入れ、関を閉ざして齊との盟約を断ってくださるなら〔注:関を閉ざすとは、(齊の)使者を拒んで通さないことである〕、わたくしは商於(しょうお)の地六百里を献上し、秦の女性を大王の身辺のお世話をする妾(しょう)として差し上げ〔注:箕帚の妾とは、掃除のお世話をする者というほどの意である〕、秦と楚はたがいに娘を嫁がせ嫁を娶って、末永く兄弟の国となりましょう。」楚王は喜んでこれを承知した。群臣はみな祝賀したが、陳軫(ちんしん)ひとりが弔意を表した。王が怒って言った。「わたしは兵を起こすこともなく六百里の地を得たのだ。なぜ弔うのか。」陳軫が答えた。「そうではありません。わたくしの見るところ、商於の地は手に入らず、かえって齊と秦が結びつくでしょう。齊と秦が結びつけば、必ず災いが訪れます。」王が言った。「その根拠があるのか。」答えて言った。「そもそも秦が楚を重んじるのは、楚に齊という(味方の)国があるからです〔注:楚は東北で齊に接し、西で秦に接している〕。いま関を閉ざして齊との盟約を断てば、楚は孤立します。秦がどうして孤立した国を欲しがって、商於の地六百里を与えましょうか。張儀は秦に戻れば、必ず王を裏切るでしょう。そうなれば王は北は齊との交わりを断ち、西は秦との間に災いを生み、両国の兵がそろって攻めてくることになります。王のためにはかるなら、内々には(齊と)結んだまま、表向きだけ齊と絶交し、人を張儀に随行させて、もし秦が約束どおり地を与えてくれてから齊と絶交しても遅くはありません。」王が言った。「どうか陳子(=陳軫)は口を閉じ、二度と言わぬように。わたしが地を手に入れるのを待っていてくれ。」そして宰相の印を張儀に授け、厚く褒美を与えた。こうして関を閉ざして齊との盟約を断ち、一人の将軍を張儀に随行させて秦に向かわせた。

張儀は(秦に着くと)車から落ちたふりをして〔注:詳は佯(いつわる)と読み、ふりをすることである〕、三か月のあいだ朝廷に出仕しなかった。楚王はこれを聞いて言った。「張儀は、わたしの齊との絶交がまだ十分でないと思っているのだろうか。」そこで勇士の宋遺(そうい)に宋の割符を借りさせ、北へ行って(齊に乗り込み)齊王を罵らせた。齊王は大いに怒り、節を曲げて秦に仕えた。こうして齊と秦の交わりが結ばれた。張儀はそこでようやく朝廷に出てきて、楚の使者に会って言った。「あなたはなぜ地を受け取らないのか。某(なにがし)の地から某の地まで、広さはわずか六里ですぞ〔注:東西の広がりを廣、南北の広がりを袤という〕。」使者は怒り、帰って楚王に報告した。楚王は大いに怒り、兵を出して秦を攻めようとした。陳軫が言った。「わたくしはもう口を開いてよろしいでしょうか。秦を攻めるよりは、いっそ名のある都市ひとつを秦に賄賂として贈り、秦と力を合わせて齊を攻めるほうがよいでしょう。そうすれば、われらが秦に失った地を、齊から取り返すことになります。いま王はすでに齊と絶交したうえ、秦に欺(あざむ)きの責めを問おうとなさる。これでは齊と秦の交わりを結びつけ、天下の兵を招き寄せることになり、国は必ず大きく傷つきましょう。」しかし楚王は聞き入れず、屈匄(くつかい)に兵を率いさせて秦を討たせた。秦もまた兵を出し、庶長(しょちょう)の章(しょう)にこれを攻撃させた。

二年
秦右更疾伐趙，拔藺，虜其將莊豹。
秦王欲伐齊，患齊、楚之從親，乃使張儀至楚，說楚王曰：「大王誠能聽臣，閉關絕約於齊，臣請獻商於之地六百里，使秦女得爲大王箕帚之妾，秦、楚嫁女娶婦，長爲兄弟之國。」楚王說而許之。羣臣皆賀，陳軫獨弔。王怒曰：「寡人不興師而得六百里地，何弔也？」對曰：「不然。以臣觀之，商於之地不可得而齊、秦合，齊、秦合則患必至矣。」王曰：「有說乎？」對曰：「夫秦之所以重楚者，以其有齊也。今閉關絕約於齊則楚孤，秦奚貪夫孤國而與之商於之地六百里！張儀至秦，必負王。是王北絕齊交，西生患於秦也，兩國之兵必俱至。爲王計者，不若陰合而陽絕於齊，使人隨張儀，苟與吾地，絕齊未晚也。」王曰：「願陳子閉口，毋復言，以待寡人得地！」乃以相印授張儀，厚賜之。遂閉關絕約於齊，使一將軍隨張儀至秦。
張儀詳墮車，不朝三月。楚王聞之，曰：「儀以寡人絕齊未甚邪？」乃使勇士宋遺借宋之符，北罵齊王。齊王大怒，折節以事秦，齊、秦之交合。張儀乃朝，見楚使者曰：「子何不受地？從某至某，廣袤六里。」使者怒，還報楚王。楚王大怒，欲發兵而攻秦。陳軫曰：「軫可發口言乎？攻之不如因賂之以一名都，與之幷力而攻齊，是我亡地於秦，取償於齊也。今王已絕於齊而責欺於秦，是吾合齊、秦之交而來天下之兵也，國必大傷矣！」楚王不聽，使屈匄帥師伐秦。秦亦發兵使庶長章擊之。
