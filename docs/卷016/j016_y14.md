三年〔注:庚子(かのえね)の年、紀元前一四一年〕。

冬、十月。日食と月食がともに起き、(空が)五日のあいだ赤かった。

十二月の晦(みそか=月末)、雷が鳴った。太陽が紫色に見えた。五星(=五つの惑星)が逆行して太微(たいび)に留まり、月が天廷(=太微)の中を貫いた〔注:『晋書天文志』によると、太微は天子の朝廷であり、五帝の座、十二諸侯の府にあたる。その外まわりの星は九卿、南のへりの中央の二星のあいだを端門(たんもん)といい、東を左執法(=廷尉になぞらえる)、西を右執法(=御史大夫になぞらえる)とし、周囲の星々をそれぞれ朝廷の官位に配する〕。

春、正月。詔して言った。「農は天下の本(もと)である。黄金・珠・玉は、飢えても食べられず、寒くても着られない。貨幣として用いられているが〔注:師古によると、幣(へい)は有る無しを通わせ、貴い賤しいを取り引きするためのもの〕、(その値打ちの)始めも終わりも見定めがたい。ここ数年、ときに実りの悪い年があるのは、思うに末(=商工)に走る者が多く、農民が少ないからであろう。郡国に命じて、つとめて農耕と養蚕を勧め、ますます樹木を植えさせよ。さすれば衣食の物が得られる。役人が民を徴発し、あるいは人を雇って〔注:韋昭によると、發民は民を使役すること、取庸はその費えを出して人を雇うこと〕黄金・珠・玉を採取させた場合は、贓罪(ぞうざい=不正蓄財の罪)に問い、盗みと同じく扱う。二千石の官でこれを黙認した者は、同罪とする」。

甲寅(きのえとら)の日、皇太子が元服(げんぷく)の冠(かんむり)の礼を行った〔注:冠(かん)〕。

甲子(きのえね)の日、帝(=景帝)が未央宮(びおうきゅう)で崩じた〔注:臣瓚によると、享年四十八〕。太子が皇帝の位に即(つ)いた。十六歳であった。皇太后(=竇〈とう〉氏)を尊んで太皇太后とし、皇后(=王氏)を皇太后とした。

二月、癸酉(みずのととり)の日、孝景皇帝を陽陵(ようりょう)に葬った〔注:臣瓚によると、崩御から埋葬まで、あわせて十日であった〕。

三月。皇太后(=王氏)の同母弟である田蚡(でんふん)を武安侯(ぶあんこう)に〔注:『漢書地理志』によると、武安県は魏郡に属す。田蚡は武安に封ぜられたが、その食邑(しょくゆう)は清河郡の鄃(ゆ)であった〕、(田)勝(しょう)を周陽侯(しゅうようこう)に封じた〔注:『史記正義』によると、絳州(こうしゅう)聞喜県(ぶんきけん)の東二十九里に周陽の故城がある〕。

班固(はんこ)の賛(=論評)にいう。孔子はこう言った。「いまのこの民も、(夏・殷・周の)三代がまっすぐな道(=正しい政道)を行うことができた、その同じ民である」〔注:師古によると、これは『論語』に載る孔子の言葉。いまの時代の人も、夏・殷・周が治め導いたのと同じ民であり、政(まつりごと)の教化が淳(あつ)く一(いつ)であったからこそ、まっすぐな道を行えた、という意。今がそうでないことを嘆いたのである〕。まことにその通りである。周の末や秦の世の弊害は、法網が細かく、条文が厳しかったのに、それでも悪事をはたらく者を取り締まりきれなかった〔注:師古によると、(不勝は)「取り締まりきれない」の意〕。漢が興ると、わずらわしく苛酷な(法を)一掃し、民とともに休息した。孝文帝(=文帝)に至って、これに恭(うやうや)しさと倹約を加えた。孝景帝(=景帝)はその事業を受け継いだ。五、六十年のあいだに、ついには風俗を改め変えるまでになり、民は淳朴(じゅんぼく)で人情に厚くなった〔注:師古によると、黎(れい)は衆(=おおぜいの民)、醇(じゅん)は混じり気のないこと〕。周では成王・康王(の治世)を称え、漢では文帝・景帝(の治世)を称える。みごとなことである。

漢が興ったとき、秦の(疲弊の)あとを受け、なすべき仕事は多くて財は乏しく、天子でさえ毛色のそろった四頭立ての馬車(=鈞駟〈きんし〉)を仕立てられず〔注:四頭が同じ毛色の馬を鈞駟という〕、将軍や宰相には牛車に乗る者さえあり〔注:師古によると、牛に車を引かせること。漢代は牛車を賤(いや)しいものとしたが、魏・晋以降は王公がこぞって牛車に乗るようになった〕、庶民(=齊民〈せいみん〉)には蓄えとてなかった〔注:蘇林によると、(藏蓋とは)蔵(しま)い蓋(おお)っておく物が無いこと〕。天下が平定されると、高祖は商人に対し、絹(きぬ)を着ることと車に乗ることを禁じ、租税を重くして、これを苦しめ辱(はずかし)めた〔注:賈(こ=商人)。衣(い=着る)〕。孝惠帝(=恵帝)・高后(=呂后)の時には、天下が落ち着いたばかりだというので、ふたたび商人への法を緩めた。しかし市井(=商人)の子孫は、やはり官に仕えて役人になることはできなかった。役人の俸禄を量(はか)り、官府の費用を見積もって、その分を民に賦課した〔注:師古によると、ちょうど足りるだけ取った。量(りょう)、度(たく)〕。そして山川・園池(えんち=苑や池)・市井からの租税収入は、天子から封君(=領地をもつ諸侯)の湯沐邑(とうもくゆう=私領)に至るまで、それぞれが私的な暮らしの費(ついえ)とし、国家の常の経費には組み入れなかった〔注:師古によると、それぞれが自分の取った賦税で自分の暮らしをまかない、朝廷の倉や蔵には入れなかった、ということ。経(けい)は常(つね)の意〕。山東(=函谷関以東)の穀物を水運で都に運んで中都官(=都の諸官府)に供給したが〔注:師古によると、中都官は都の諸々の官府のこと〕、年に数十万石(こく)を超えなかった。続く孝文帝・孝景帝は、清浄(=無為)で恭しく倹約につとめ、天下を安んじ養った。七十余年のあいだ、国家には大事(おおごと)がなく、水害や干ばつの災いに遭わないかぎり、民は誰もが満ち足り、家々は豊かであった。都(みやこ)も地方も、米倉(=廩〈りん〉・庾〈ゆ〉)はみな満ち、官府の蔵には財貨が余った。都の銭(ぜに)は巨万(きょまん)の額に積み上がり〔注:師古によると、累巨萬は数百万万(=数えきれぬほどの巨額)のこと。校(こう)は数えること〕、(銭をつなぐ)紐(ひも)が朽ちて数えることもできぬほどであった。太倉(たいそう=都の大穀倉)の穀物は、古いもの(=陳〈ちん〉)の上に古いものが重なり〔注:師古によると、陳は古いということ〕、あふれて外に野積みされ、ついには腐って食べられなくなるほどであった。庶民の町なかにも馬がおり、田畑のあぜ道(=阡陌〈せんぱく〉)のあいだには馬が群れをなした〔注:師古によると、阡陌は田中のあぜ道〕。子を産んだ牝馬(めうま=字牝〈じひん〉)に乗る者は、(寄り合いの場から)はじき出されて加われなかった〔注:孟康によると、皆が牡馬(おうま)に乗るところへ牝馬が混じると蹴り合い噛み合うので、追い出されて同席できなかった。師古は、当時は豊かで牝馬に乗るのを恥としたまでで、蹴り合いのせいとは限らないとする〕。里の門番(=閭閻〈りょえん〉を守る者)でさえ上等な穀物と肉を食べた。役人となった者は同じ職に長く留まって子や孫の代に及び〔注:如淳によると、当時は事(おおごと)がなく役人がたびたび転任しなかったので、子孫が生まれ育つまで同じ職にあった。長(ちょう)〕、官にある者はその官名をそのまま姓や呼び名とした〔注:如淳によると、『史記貨殖伝』の倉氏(そうし)・庫氏(こし)がそれである〕。それゆえ人々はみな身を大切にして軽々しく法を犯さず、義(=正しい行い)を先にし、恥となること(=詘辱〈くつじょく〉)を後にした〔注:師古によると、義を行うことを第一とし、恥辱を受けるのを退けた、の意〕。この時代には、法網はゆるやかでありながら民は富んでいた。(が、やがて)財に頼って驕(おご)りあふれ、ある者は土地や富を兼ね併(あわ)せ、豪族・徒党(=豪黨)の輩(やから)は、力ずくで郷里(=鄕曲)を取り仕切るようになった〔注:師古によると、その豪富をたのんで、ほしいままに私的な制裁を行ったということ。斷(だん)〕。領地をもつ宗室(=皇族)〔注:師古によると、国の宗姓で封邑(=領地)を受けた者をいう〕、公・卿・大夫から下の者まで、こぞって奢侈(しゃし=ぜいたく)を競い、住まいや車・衣服は上(=身分の上の者)をしのぎ、限度というものがなかった。物事は盛んになれば衰える。それは必然の移り変わりである。これより後、孝武帝(=武帝)は、内(うち)には奢侈(しゃし)を窮(きわ)め、外(そと)には夷狄(いてき=異民族)を討ち払い、天下はものさびしく荒(すさ)み、財力は使い果たされてしまった。

三年
冬，十月，日月皆食，赤五日。
十二月晦，雷；日如紫；五星逆行守太微；月貫天廷中。
春，正月，詔曰：「農，天下之本也。黃金、珠、玉，飢不可食，寒不可衣，以爲幣用，不識其終始。間歲或不登，意爲末者衆，農民寡也。其令郡國務勸農桑，益種樹，可得衣食物。吏發民若取庸采黃金、珠、玉者，坐贓爲盜。二千石聽者，與同罪。」
甲寅，皇太子冠。
甲子，帝崩于未央宮。太子卽皇帝位，年十六。尊皇太后爲太皇太后，皇后爲皇太后。
二月，癸酉，葬孝景皇帝于陽陵。
三月，封皇太后同母弟田蚡爲武安侯，勝爲周陽侯。
班固贊曰：孔子稱：「斯民也，三代之所以直道而行也。」信哉！周、秦之敝，罔密文峻，而姦軌不勝。漢興，掃除煩苛，與民休息；至于孝文，加之以恭儉；孝景遵業。五六十載之間，至於移風易俗，黎民醇厚。周云成、康，漢言文、景，美矣！
漢興，接秦之弊，作業劇而財匱，自天子不能具鈞駟，而將相或乘牛車，齊民無藏蓋。天下已平，高祖乃令賈人不得衣絲、乘車，重租稅以困辱之。孝惠、高后時，爲天下初定，復弛商賈之律；然市井之子孫，亦不得仕宦爲吏。量吏祿，度官用，以賦於民。而山川、園池、市井租稅之入，自天子以至於封君湯沐邑，皆各爲私奉養焉，不領於天子之經費。漕轉山東粟以給中都官，歲不過數十萬石。繼以孝文、孝景，清淨恭儉，安養天下，七十餘年之間，國家無事，非遇水旱之災，民則人給家足。都鄙廩庾皆滿，而府庫餘貨財；京師之錢累鉅萬，貫朽而不可校；太倉之粟陳陳相因，充溢露積於外，至腐敗不可食。衆庶街巷有馬，而阡陌之間成羣，乘字牝者擯而不得聚會。守閭閻者食粱肉；爲吏者長子孫，居官者以爲姓號。故人人自愛而重犯法，先行義而後詘辱焉。當此之時，罔疏而民富，役財驕溢，或至兼幷、豪黨之徒，以武斷於鄕曲。宗室有土、公、卿、大夫以下，爭于奢侈，室廬、輿服僭于上，無限度。物盛而衰，固其變也；自是之後，孝武內窮侈靡，外攘夷狄，天下蕭然，財力耗矣！
