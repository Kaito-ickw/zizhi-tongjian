# 卷003 周紀三 — 愼靚王三年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 愼靚王三年 ・ 西暦: 318 BCE

[← 巻インデックス](README.md)

---

三年〔注:癸卯(きぼう)の年、紀元前三一八年〕。

楚・趙・魏・韓・燕の五国がそろって秦を攻め、函谷関(かんこくかん)に攻めかかった。秦の側は兵を出して迎え撃ち、五国の軍はことごとく敗れて逃げ去った。

宋が初めて王を称した。

---

三年
楚、趙、魏、韓、燕同伐秦，攻函谷關。秦人出兵逆之，五國之師皆敗走。
宋初稱王。
