# 卷030 漢紀二十二 — 孝成皇帝上之上二年

二年〔注:戊戌(つちのえいぬ)の年、紀元前二三年〕。

春、三月、天下に大赦を行った。

御史大夫(ぎょしたいふ)の張忠(ちょうちゅう)が卒(しゅっ)した。

夏、四月、丁卯(ひのとう)の日、侍中(じちゅう)・太僕(たいぼく)の王音(おういん)を御史大夫とした。こうして王氏はますます盛んになり、郡国の太守(たいしゅ)・国相(こくしょう)や刺史(しし)は、みな王氏の門下から出るようになった〔注:王氏の家来であった者がみな高官に就いた、という意〕。五侯(ごこう)やその兄弟たちは、競って奢侈(しゃし)にふけり、賄賂(わいろ)や珍宝(ちんぽう)が四方から贈り集まった。彼らはみな世故(せこ)に通じてそつがなく、士(し)を好み賢者を養い、財をなげうって人に施しては、たがいに評判の高さを張り合った。客は門に満ち、争って彼らの名声をはやし立てた。劉向(りゅうきょう)は陳湯(ちんとう)にこう語った。「いま災異(さいい)がこのありさまであるのに、外戚(がいせき)は日ごとに盛んになっている。この調子では、必ずや劉氏(りゅうし)を危うくするだろう。私は幸いにも、同姓の末(すえ)の縁につらなる者として、代々漢の厚い恩を受けてきた。わが身は宗室(そうしつ)の生き残りの老人として、三代の主(しゅ)にお仕えしてきた〔注:三主とは、宣帝(せんてい)・元帝(げんてい)・成帝(せいてい)〕。主上は私を先帝以来の旧臣として、参内(さんだい)するたびにいつも格別の礼をもって遇してくださる。この私が言わずして、いったい誰が言うべきだろうか」。そこで封事(ふうじ)を奉り、口を極めてこう諫(いさ)めた。

「臣(しん)が聞きますに、君主たる者で安泰を望まぬ者はおりませんが、それでいてしばしば危うくなり、存続を望まぬ者はおりませんが、それでいてしばしば滅びます。これは臣下を御(ぎょ)する術(すべ)を失うからです。そもそも大臣が権柄(けんぺい)を握り、国政を専(もっぱ)らにして、害をなさなかったためしはありません。ゆえに『書経(しょきょう)』には『臣下が威(い)を作(な)し福を作す(=賞罰の権をほしいままにする)ことがあれば、汝(なんじ)の家を害し、汝の国に凶(きょう)をもたらす』とあります〔注:『周書・洪範(こうはん)』の言葉である。「而」は「汝」のこと。ただ君主のみが威福を作してよく、臣下がこれを行えば凶害を招く、という意〕。孔子も『禄(ろく)が公室(こうしつ)を去り、政(まつりごと)が大夫(たいふ)に及ぶ』のは、危亡(きぼう)の兆(きざ)しだと申しました〔注:臣瓚(しんさん)いわく、政が君主から出ず、下って大夫に及ぶこと。師古いわく、『論語』に孔子が「禄が公室を去って五世、政が大夫に及んで四世、ゆえに三桓(さんかん)の子孫は衰えた」と言った〕。いま王氏という一姓で、朱(あか)い車輪に飾り立てた轂(こしき)の車に乗る者が二十三人もおり、青や紫の綬(じゅ)、貂(てん)や蟬(せみ)の冠飾(かんしょく)を帯びた者が御帳(みちょう)のうちに満ちあふれ、天子の左右に魚の鱗(うろこ)のごとく居並んでおります〔注:漢の制度では、列侯は紫の綬、二千石は青の綬。侍中・中常侍はみな銀の璫(とう)に、左に貂、金の蟬を付けた。師古いわく、天子の左右にあって、次々と魚の鱗のように連なっていることをいう〕。大将軍(王鳳)が政務を取りしきって権をふるい、五侯はおごり高ぶって分(ぶん)を越えた盛んさを誇り、こぞって威福をほしいままにし、人を打ち裁くのも思うがままで、行いは汚れているのに政(まつりごと)を司(つかさど)るふりをし、私心を抱きながら公(おおやけ)に名を借り〔注:師古いわく、内では汚れた私的なふるまいをし、外では公の政道(せいどう)を司るふりをすること〕、東宮(とうぐう、太后)の尊厳に寄りかかり〔注:東宮とは太后のお住まい。漢の制度では太后はおおむね長楽宮(ちょうらくきゅう)に住まわれ、未央宮(びおうきゅう)の東にあるので「東宮」という〕、甥(おい)と舅(おじ)という(天子との)縁を笠(かさ)に着て、その威勢を重からしめております。尚書(しょうしょ)・九卿(きゅうけい)・州牧(しゅうぼく)・郡守(ぐんしゅ)はみな王氏の門から出て、政(まつりごと)の枢機(すうき)を握り、徒党を組んで馴(な)れ合っております。王氏を褒(ほ)めたたえる者は引き立てられ、逆らい恨む者は誅(ちゅう)せられ傷つけられます。遊説(ゆうぜい)の徒は王氏のために弁じ、政を執る者は王氏のために口をきく。宗室(そうしつ)を押しのけ退け、皇族を孤立させ弱らせ、その中に知恵や才のある者がいれば、ことさらにそしり貶(おとし)めて登用せず、宗室を官職から遠ざけ、朝廷や役所での勤めにも就かせません。自分たちと権を分かち合うことを恐れているのです。たびたび燕王(えんおう)や蓋主(がいしゅ)の例を持ち出しては主上の心に宗室への疑いを抱かせ〔注:燕王・蓋主の事は『昭帝紀(しょうていき)』に見える。宗室は近しい身内でありながら反逆することもある、と示すためである〕、その一方で呂氏(りょし)・霍氏(かくし)のことには触れようとしません〔注:呂氏の事は『呂后紀(りょこうき)』に、霍氏の事は『宣帝紀(せんていき)』に見える。呂・霍の両家はいずれも専横の罪で誅滅(ちゅうめつ)されたので、王氏のためにその名をはばかって口にしないのである〕。内には管叔(かんしゅく)・蔡叔(さいしゅく)のような反逆の芽を秘めながら、外には周公(しゅうこう)を引き合いに出して忠臣ぶりを取りつくろい、兄弟が要職を占め、一族がからみ合って盤踞(ばんきょ)しております。上古から秦・漢に至るまで、外戚で分を越えて尊貴(そんき)を極めた者で、王氏ほどの例はありません。そもそも物事が極まって盛んになれば、必ず尋常でない異変が前ぶれとして現れ、その人に下る天意の微(かす)かな兆候(ちょうこう)となるものです〔注:微小なものに伏(ふ)し、形となって現れる、という意〕。孝昭帝(こうしょうてい)のとき、冠(かんむり)のような石が泰山(たいざん)に立ち、倒れていた柳が上林苑(じょうりんえん)で起き上がり、そののち孝宣帝(こうせんてい)が即位なさいました〔注:この事は第二十三巻、昭帝の元鳳(げんぽう)三年に見える〕。いま王氏の先祖の墳墓(ふんぼ)が済南(せいなん)にありますが〔注:王氏はもと済南の東平陵(とうへいりょう)の人で、武帝のとき、繡衣御史(しゅういぎょし)の王賀(おうが)が免官されてのち、魏郡(ぎぐん)の元城(げんじょう)に移り住んだ〕、その梓(あずさ)の柱が枝葉(しよう)を生じ、こんもりと茂って屋根を突き抜け、根は地中に垂れ下がっております〔注:「𠚏」は「垂(すい)」の字。『漢書』は「根垂地中」に作る〕。あの(昭帝のときの)石が立ち柳が起きた異変も、この明らかさには及びません。物事の勢いは二つながらに大きくはなれず、王氏と劉氏もまた並び立つことはできません。下(王氏)に泰山のような安泰があれば、上(劉氏)には積み上げた卵、すなわち累卵(るいらん)のような危うさがあるのです。陛下は先帝の子孫として宗廟(そうびょう)をお守りになる身でありながら、国の命運を外戚に移してしまい、ご自身は賤(いや)しい下僕(げぼく)の身に落ちようとしておられます〔注:師古いわく、皁隷(そうれい)とは卑賤(ひせん)の者。『春秋左氏伝』に、大夫は士を臣とし、士は皁を臣とし、皁は輿(よ)を臣とし、輿は隷(れい)を臣とする、とある〕。たとえご自身のためでなくとも、宗廟をどうなさるおつもりですか。婦人は夫の家を内(うち)とし、父母の家を外(そと)とするものです。王氏のこの専横は、じつは皇太后にとっても福ではありません〔注:如淳いわく、「内」とは親しむこと。ところが皇太后は逆に夫の家(劉氏)を外になさっている〕。孝宣皇帝が舅(おじ)の平昌侯(へいしょうこう)に権を与えなかったのは、まさにその身を全(まっと)うし安んじるためでした〔注:平昌侯王無故(おうむこ)は、宣帝の舅である〕。そもそも聡明(そうめい)な者は、福をまだ形のないうちに起こし、わざわいを起こる前に消すものです。よろしく明らかな詔(みことのり)を発し、徳のあるお言葉を述べ、宗室を引き寄せて近づけ、親しんで信を置かれ〔注:師古いわく、「援」は引くこと。引き上げて近くに付けること〕、外戚を退けて遠ざけ、政を授けず〔注:師古いわく、「遠」とは、疎(うと)んじて離すこと〕、みな官を解いて屋敷に下がらせ〔注:「弟」は「第」に同じ。邸宅のことである〕、もって先帝のなさったことに倣(なら)い、外戚を手厚く安んじてその一族を全うなさるべきです。それこそまことに東宮(太后)のお心にかない、外戚の家の福ともなりましょう。王氏は永く存続してその爵禄(しゃくろく)を保ち、劉氏は長く安泰でその社稷(しゃしょく)を失わずにすみます。これこそ、内外の姓、すなわち劉氏と王氏をともに称(たた)え睦(むつ)ませ、子々孫々(ししそんそん)かぎりない安泰の計(はかりごと)というものです。もしこの策を行わなければ、斉(せい)を奪った田氏(でんし)が今の世にふたたび現れ、晋(しん)を分けた六卿(りっけい)が必ずや漢に起こり、後の世継ぎの憂(うれ)いとなることは、火を見るよりも明らかです〔注:田氏は斉を簒奪(さんだつ)し、六卿は晋を分けた。漢にもこうしたわざわいが起ころう、という意〕。どうか陛下、深く聖慮(せいりょ)をめぐらしてくださいませ」。この上書が奉られると、天子は劉向を召し出して引見し、嘆息してその思いを痛ましく思い、こう言った。「そなた、ひとまず休むがよい。私はよく考えてみよう」。しかし、ついにその言葉を用いることはできなかった。

秋、関東(かんとう)で大水(おおみず)があった。

八月、甲申(きのえさる)の日、定陶(ていとう)の共王(きょうおう)康(こう、劉康)が薨(こう)じた。

この年、信都王(しんとおう)の興(こう、劉興)を移して中山王(ちゅうざんおう)とした。

二年
春，三月，大赦天下。
御史大夫張忠卒。
夏，四月，丁卯，以侍中、太僕王音爲御史大夫。於是王氏愈盛，郡國守相、刺史皆出其門下。五侯羣弟爭爲奢侈，賂遺珍寶，四面而至，皆通敏人事，好士養賢，傾財施予以相高尚；賓客滿門，競爲之聲譽。劉向謂陳湯曰：「今災異如此，而外家日盛，其漸必危劉氏。吾幸得以同姓末屬，累世蒙漢厚恩，身爲宗室遺老，歷事三主。上以我先帝舊臣，每進見，常加優禮。吾而不言，孰當言者！」遂上封事極諫曰：「臣聞人君莫不欲安，然而常危；莫不欲存，然而常亡；失御臣之術也。夫大臣操權柄，持國政，未有不爲害者也。故《書》曰︰『臣之有作威作福，害于而家，凶于而國。』孔子曰：『祿去公室，政逮大夫，』危亡之兆也。今王氏一姓，乘朱輪華轂者二十三人，青、紫、貂、蟬充盈幄內，魚鱗左右。大將軍秉事用權，五侯驕奢僭盛，並作威福，擊斷自恣，行汙而寄治，身私而託公，依東宮之尊，假甥舅之親，以爲威重。尚書、九卿、州牧、郡守皆出其門，管執樞機，朋黨比周；稱譽者登進，忤恨者誅傷；游談者助之說，執政者爲之言。排擯宗室，孤弱公族，其有智能者，尤非毀而不進，遠絕宗室之任，不令得給事朝省，恐其與己分權；數稱燕王、蓋主以疑上心，避諱呂、霍而弗肯稱。內有管、蔡之萌，外假周公之論，兄弟據重，宗族盤互，歷上古至秦、漢，外戚僭貴未有如王氏者也。物盛必有非常之變先見，爲其人微象。孝昭帝時，冠石立於泰山，仆柳起於上林，而孝宣帝卽位。今王氏先祖墳墓在濟南者，其梓柱生枝葉，扶疏上出屋，根𠚏地中，雖立石起柳，無以過此之明也。事勢不兩大，王氏與劉氏亦且不並立，如下有泰山之安，則上有累卵之危。陛下爲人子孫，守持宗廟，而令國祚移於外親，降爲皁隸，縱不爲身，柰宗廟何！婦人內夫家而外父母家，此亦非皇太后之福也。孝宣皇帝不與舅平昌侯權，所以全安之也。夫明者起福於無形，銷患於未然，宜發明詔，吐德音，援近宗室，親而納信，黜遠外戚，毋授以政，皆罷令就弟，以則效先帝之所行，厚安外戚，全其宗族，誠東宮之意，外家之福也。王氏永存，保其爵祿，劉氏長安，不失社稷，所以褒睦外內之姓，子子孫孫無疆之計也。如不行此策，田氏復見於今，六卿必起於漢，爲後嗣憂，昭昭甚明。唯陛下深留聖思！」書奏，天子召見向，歎息悲傷其意，謂曰：「君且休矣，吾將思之！」然終不能用其言。
秋，關東大水。
八月，甲申，定陶共王康薨。
是歲，徙信都王興爲中山王。
