# 卷007 秦紀二 — 始皇帝下三十六年

> 巻 7 / 294 ・ 秦紀二 ・ 年号: 始皇帝下三十六年 ・ 西暦: 211 BCE

三十六年〔注:庚寅(かのえとら)の年、紀元前二一一年〕。

東郡(とうぐん)〔注:東郡はもともと衛(えい)の地であったが、秦が衛を野王(やおう)へ移し、その旧地に東郡を設けた〕に隕石(いんせき)が落ちた。何者かがその石に「始皇死して地は分かれん」と刻んだ。始皇帝は御史(ぎょし)を派遣してかたっぱしから取り調べさせたが、誰も認めなかった。そこで石の周辺に住む者をことごとく捕らえて誅殺し、その石を焼き払った〔注:「燔」は焼く・燃やす意〕。

河北(かほく)の榆中(ゆちゅう)〔注:「河北」とは黄河(北河)の北岸の地のこと〕に三万戸を移住させ、爵位を一級ずつ授けた。

三十六年
有隕石于東郡。或刻其石曰：「始皇死而地分。」始皇使御史逐問，莫服；盡取石旁居人誅之，燔其石。
遷河北榆中三萬家；賜爵一級。
