# 卷006 秦紀一 — 始皇帝上五年

五年〔注:己未(きび)の年、紀元前二四二年〕。

蒙驁(もうごう)が魏を伐(う)って、酸棗(さんそう)・燕(えん)・虛(きょ)・長平(ちょうへい)・雍丘(ようきゅう)・山陽(さんよう)など三十の城を取り、はじめて東郡(とうぐん)を置いた。

これより前のこと。劇辛(げきしん)は趙にいたころ龐煖(ほうけん)と仲がよかったが、のちに燕に仕えた。燕王は、趙がたびたび秦に苦しめられ、廉頗(れんぱ)が去って龐煖が将となっているのを見て、その疲弊につけ込んで趙を攻めようと考え、劇辛に意見を求めた。劇辛は答えた。「龐煖など、たやすく料理できる相手にすぎません。」燕王は劇辛を将として趙を伐たせた。趙では龐煖がこれを迎え撃ち、劇辛を殺して、燕の軍二万を奪い取った。

諸侯は、秦の攻め伐つことに終わるときがないのを憂えた。

五年
蒙驁伐魏，取酸棗、燕、虛、長平、雍丘、山陽等三十城；初置東郡。
初，劇辛在趙與龐煖善，已而仕燕。燕王見趙數困於秦，廉頗去而龐煖爲將，欲因其敝而攻之，問於劇辛，對曰︰「龐煖易與耳！」燕王使劇辛將而伐趙。趙龐煖禦之，殺劇辛，取燕師二萬。
諸侯患秦攻伐無已時。
