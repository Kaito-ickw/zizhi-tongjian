# 卷054 漢紀四十六 — 孝桓皇帝上之下四年

四年〔注:辛丑(かのとうし)の年、紀元一六一年〕。

春、正月、辛酉(かのととり)の日、南宮(なんきゅう)の嘉徳殿(かとくでん)が火災に遭った。戊子(つちのえね)の日、丙署(へいしょ)が火災に遭った〔注:『百官志』によれば、丙署の長は七人、秩(ちつ、官位)は四百石、黄綬(こうじゅ)で、宦官(かんがん)がこれに任じられ、中宮(=後宮)の別処(=別の建物)を管理した〕。

大いに疫病(疫)が流行した。

二月、壬辰(みずのえたつ)の日、武庫(ぶこ、武器庫)が火災に遭った。

司徒の盛允(せいいん)が免ぜられ、大司農の种暠(ちゅうこう)を司徒とした〔注:司馬光の考証——『袁紀』はこれを前年に置くが、祝恬(しゅくてん)の薨去ののちに盛允があり、その盛允が免ぜられて种暠が司徒となるまで半年を隔てており、『袁紀』の誤りである。今は范曄(はんよう)の『後漢書』に従う〕。

三月、太尉の黄瓊(こうけい)が免ぜられた。夏、四月、太常で沛国(はいこく)の劉矩(りゅうく)を太尉とした。これより先、劉矩は雍丘(ようきゅう)の令(れい、県の長官)であったとき〔注:雍丘は陳留(ちんりゅう)郡に属し、もとの杞国(きこく)である〕、礼譲(れいじょう)をもって民を教化した。訴訟をする者があると、いつもこれを前へ引き寄せ、耳をつかんで訓(さと)し告げ、「怒り(忿恚)は堪え忍ぶべきもの、県の役所(県官)に(訴え出て)立ち入るべきではない」と説き、(家へ)帰らせて改めて考え直させた。訴える者は感じ入って、そのたびにそれぞれ(訴えを)やめて去っていった。

甲寅(きのえとら)の日、河間孝王(かかんこうおう)の子で参戸亭侯(さんこていこう)の博(はく)を任城王(にんじょうおう)に封じ、(任城)孝王の後嗣(後)を継がせた〔注:任城孝王(=名は尚(しょう))は元嘉(げんか)元年に薨(こう)じ、子がなく国は絶えていた。今、博を継嗣に立てて封じた。河間孝王は名を開(かい)、任城孝王は名を尚という〕。

五月、辛酉(かのととり)の日、彗星(すいせい、孛星)が心(しん、心宿)に現れた〔注:『晋書・天文志』によれば、心宿の三星のうち、中央の星を明堂といい天子の位、前の星は太子、後ろの星は庶子(にあたる)〕。

丁卯(ひのとう)の日、原陵(げんりょう)の長寿門(ちょうじゅもん)が火災に遭った〔注:原陵は光武帝の陵(みささぎ)〕。

己卯(つちのとう)の日、京師(けいし、都)に雹(ひょう)が降った。

六月、京兆(けいちょう)・扶風(ふふう)および涼州(りょうしゅう)で地震があった。

庚子(かのえね)の日、岱山(たいざん)と博(はく)の尤来山(ゆうらいざん)がともに崩れ裂けた〔注:岱山は博県の西北にある。尤来山は徂来山(そらいざん)の別名。博はいまの博城県(である)〕。

己酉(つちのととり)の日、天下を赦(ゆる)した。

司空の虞放(ぐほう)が免ぜられ、前の太尉である黄瓊を司空とした。

犍為属国(けんいぞっこく)の夷(い、異民族)が民(百姓)を侵し略奪した〔注:犍為属国は(安帝の)永初元年に、犍為南部都尉を犍為属国都尉に改めて設けたもので、朱提(しゅてい)・漢陽の二県を領する〕。益州刺史の山昱(さんいく)がこれを撃ち破った。

零吾(れいご)の羌が先零(せんれい)の諸種族とともに反乱を起こし、三輔(さんぽ、長安周辺の地)を侵した。

秋、七月、京師で雩(う、雨乞いの祭り)を行った〔注:『公羊伝』に、雩は旱(ひでり)のときの祭りである、とある〕。

公卿以下の俸禄(奉=俸)を減らし、王侯から(その封地の)租(=収入)の半分を借り上げ(貣)〔注:「貣(とく)」は借りる意で、仮借(かしゃく)の用法である〕、関内侯(かんだいこう)・虎賁(こほん)・羽林(うりん)の緹騎(ていき)・営士(えいし)・五大夫(ごたいふ)(の位)を、それぞれ値(あたい)に等差をつけて(額を)定めて売った。

九月、司空の黄瓊が免ぜられ、大鴻臚(だいこうろ)で東萊(とうらい)の劉寵(りゅうちょう)を司空とした。劉寵はかつて会稽(かいけい)太守であったとき、煩雑で苛酷な(政令)を取り除き、不法を禁じ取り締まり、郡中はよく治まった。(その後)召されて将作大匠(しょうさくだいしょう)となった。山陰県(さんいんけん)に五、六人の老人(老叟)がいて、若邪(じゃくや)の山谷の間から出てきて〔注:若邪は、いまの越州会稽県の東南にある〕、めいめい百銭を持参して劉寵を見送り、こう言った。「山谷の田舎者(鄙生)で、これまで郡の役所(郡朝)を見知ったこともございません〔注:郡の政庁を郡朝といい、公府の政庁を府朝という〕。ほかの太守のときは、役人が(取り立てに)民間へ出向いて求め、夜になっても絶えることなく、あるいは犬が一晩中吠えて、民は安んじることができませんでした。あなたさま(明府)が御着任(下車)されてからは、犬は夜に吠えず、民は役人(の姿)を見ることもございません。年老いて聖明(=ご立派なお方)の御代に出会えましたのに、いま、(われらを)お見捨てになって去られると聞きましたので、こうして(老体を)支えあって(自ら)お見送りに参ったしだいです」。劉寵は言った。「私の政(まつりごと)が、どうしてあなたがた(公)の(お褒めの)言葉に及びましょうか。父老(=ご老人がた)にご苦労をおかけしました」。(そして劉寵は)一人ごとに大銭(=価値の高い銭)を一枚ずつ選んで(これを)受け取った。

冬、先零・沈氐(しんてい)の羌が諸種族の羌とともに并(へい)・涼(りょう)の二州を侵し、校尉の段熲(だんけい)が湟中(こうちゅう)の義従(ぎじゅう)〔注:湟中には義従胡がいる。すなわち小月氏(しょうげっし)の胡(=北方の異民族)である〕を率いてこれを討った。涼州刺史の郭閎(かくこう)は、(段熲と)その功(てがら)を共にしようと欲ばって、段熲の軍を引き留め〔注:稽固(けいこ)とは、足止め(=停留)のことである〕、進ませなかった。義従は従軍が長引いて故郷を恋しがり、みなことごとく叛(そむ)いて帰ってしまった。郭閎は(その)罪を段熲になすりつけ、段熲はその罪に問われて召し出されて獄に下され、左校(さこう、土木の労役)に送られた。済南の相(しょう、国の長官)の胡閎(ここう)を代わりに校尉とした。胡閎には威厳も計略もなく、羌はそのまま跳梁(ちょうりょう、のさばり暴れること)し、(漢軍の)営塢(えいう、とりで)〔注:『説文』に、塢は小さな防壁(障)、一説に低い城をいう、とある〕を陥落(覆没)させ、次々に(仲間を)招き集めて結び、諸郡へ突入(=不意に押し入る)し、賊の害(寇患)はいよいよ激しくなった。泰山太守の皇甫規(こうほき)が上疏(じょうそ)して言った。「いま狡猾(こうかつ)な賊(=泰山の賊)は滅び、泰山はおおむね平定されましたが、ふたたび羣羌(ぐんきょう、もろもろの羌)がそろって反逆したと聞きました。臣は邠(ひん)・岐(き)の地に生まれ育ち、年は五十九。昔、郡の吏(=役人)であったとき、二度にわたって羌の叛乱に出くわし、あらかじめその事を策(はか)って、はからずも的中した言葉がございました〔注:(これは皇甫規が)馬賢(ばけん)が必ず敗れると見抜いたことをいう。事は五十二巻、順帝の永和五年(の記事)に見える〕。臣はもともと痼疾(こしつ、持病)があり、犬馬(けんば、わが身の謙称)の齢(よわい)が尽きて(死んで)、大恩に報いられぬことを恐れます。願わくは冗官(じょうかん、閑職)を乞い、一台の車(単車)に一人(一介)の使者として、三輔(の地)を慰労(=ねぎらい慰める)し、国家の威光と恩沢を宣(の)べ広め、(臣が)習熟した地形と兵勢(=軍事の情勢)をもって諸軍を助けたく存じます。臣は窮(きわ)まり孤(こ)危(あや)うい中に身を置き、郡将(ぐんしょう、郡守=太守)(の失政)を座視すること、すでに数十年。鳥鼠(ちょうそ、山の名)から東の岱(たい、=泰山)に至るまで、その病(=弊害の根)は一つでございます〔注:李賢いわく、郡将は郡守のこと。鳥鼠は山の名で、いまの渭州の西、すなわち先零の羌が侵略略奪した所。東岱は泰山で、叔孫無忌(しゅくそんむき)が反乱した所。みな郡守が(民を)綏撫(すいぶ、なだめ慰める)しなかったために反乱を招いた、その病は一つである〕。力ずくで猛(たけ)き敵を(攻め)求めるのは、清(きよ)く平らかな(政治をする)に及ばず、孫(子)・呉(子)(の兵法)に努め通じるのは、法を奉(たてまつ)り守る(=郡守が法を遵守する)に及びません〔注:李賢いわく、猛敵を(攻め)求めるよりは清平の政で(民を)撫(なで)るほうがよく、兵書に明るいよりは郡守が法を奉じて(民を)反乱させぬほうがよい、という意〕。先の変事(=羌の反乱)はまだ遠い昔のことではなく、臣はまことにこれを憂えております〔注:李賢いわく、戚(せき)は憂うの意。前変とは羌の反乱をいう〕。それゆえ、職分を越えて、この区々(くく、ささやかな真心)を尽くすしだいです」。(帝は)詔(みことのり)して、皇甫規を中郎将とし、節(せつ)を持たせて関西の兵を監(かん)せしめ、零吾らを討たせた。十一月、皇甫規は羌を撃ち、これを破り、八百級(=八百人の首)を斬った。先零の諸種族の羌は皇甫規の威信を慕い、(たがいに)勧め合って降伏する者が十万人あまりにのぼった。

四年
春，正月，辛酉，南宮嘉德殿火；戊子，丙署火。
大疫。
二月，壬辰，武庫火。
司徒盛允免，以大司農种暠爲司徒。
三月，太尉黃瓊免；夏，四月，以太常沛國劉矩爲太尉。
初，矩爲雍丘令，以禮讓化民；有訟者，常引之於前，提耳訓告，以爲忿恚可忍，縣官不可入，使歸更思。訟者感之，輒各罷去。
甲寅，封河間孝王子參戶亭侯博爲任城王，奉孝王後。
五月，辛酉，有星孛于心。
丁卯，原陵長壽門火。
己卯，京師雨雹。
六月，京兆、扶風及涼州地震。
庚子，岱山及博尤來山並穨裂。
己酉，赦天下。
司空虞放免，以前太尉黃瓊爲司空。
犍爲屬國夷寇鈔百姓。益州刺史山昱擊破之。
零吾羌與先零諸種反，寇三輔。
秋，七月，京師雩。
減公卿已下奉，貣王侯半租，占賣關內侯、虎賁、羽林緹騎、營士、五大夫錢各有差。
九月，司空黃瓊免，以大鴻臚東萊劉寵爲司空。
寵嘗爲會稽太守，簡除煩苛，禁察非法，郡中大治；徵爲將作大匠。山陰縣有五六老叟，自若邪山谷間出，人齎百錢以送寵曰：「山谷鄙生，未嘗識郡朝，他守時，吏發求民間，至夜不絕，或狗吠竟夕，民不得安。自明府下車以來，狗不夜吠，民不見吏；年老遭值聖明，今聞當見棄去，故自扶奉送。」寵曰：「吾政何能及公言邪！勤苦父老！」爲人選一大錢受之。
冬，先零、沈氐羌與諸種羌寇幷、涼二州，校尉段熲將湟中義從討之。涼州刺史郭閎貪共其功，稽固熲軍，使不得進；義從役久戀鄕舊，皆悉叛歸。郭閎歸罪於熲，熲坐徵下獄，輸作左校，以濟南相胡閎代爲校尉。胡閎無威略，羌遂陸梁，覆沒營塢，轉相招結，唐突諸郡，寇患轉盛。泰山太守皇甫規上疏曰：「今猾賊就滅，泰山略平，復聞羣羌並皆反逆。臣生長邠岐，年五十有九，昔爲郡吏，再更叛羌，豫籌其事，有誤中之言。臣素有痼疾，恐犬馬齒窮，不報大恩，願乞宂官，備單車一介之使，勞來三輔，宣國威澤，以所習地形兵勢佐助諸軍。臣窮居孤危之中，坐觀郡將已數十年，自鳥鼠至于東岱，其病一也。力求猛敵，不如清平；勤明孫、吳，未若奉法。前變未遠，臣誠戚之，是以越職盡其區區。」詔以規爲中郎將，持節監關西兵討零吾等。十一月，規擊羌，破之，斬首八百級。先零諸種羌慕規威信，相勸降者十餘萬。
