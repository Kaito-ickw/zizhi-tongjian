# 卷006 秦紀一 — 始皇帝上十九年

十九年〔注:癸酉(きゆう)の年、紀元前二二八年〕。

王翦(おうせん)が趙軍を攻撃して大いに破り、趙葱(ちょうそう)を討ち取った。顏聚(がんしゅう)は逃亡し、ついに秦は邯鄲(かんたん)を攻め落として、趙王遷(ちょうおうせん)を捕虜とした〔注:趙はこの年に滅亡した〕。秦王は邯鄲へ赴き、かつて母の実家と恨みを抱(いだ)き合っていた者たちを、ことごとく殺した〔注:王の母はもと邯鄲の美女で、その経緯は前巻の周の赧王(なんおう)五十八年の条に見える〕。そして帰路につき、太原(たいげん)・上郡(じょうぐん)を通って秦へ戻った。

太后(たいこう)が薨(こう)じた。

王翦は中山(ちゅうざん)に駐屯して、燕(えん)に圧力をかけた〔注:中山は春秋の鮮虞(せんぐ)で、戦国時代には中山国となり、趙が滅ぼしてその地を中山郡とした〕。趙の公子嘉(こうしか)は、一族数百人を率いて代(だい)へ逃れ、みずから立って代王(だいおう)となった。趙が滅んだのち、趙の大夫(たいふ)たちは少しずつ嘉のもとに身を寄せていった。嘉は燕と兵を合わせ、上谷(じょうこく)に布陣した〔注:上谷は燕の地で、のちに秦が上谷郡を置いた〕。

楚(そ)の幽王(ゆうおう)が薨じ、国人(こくじん)はその弟の郝(かく)を立てた。三月、郝の庶兄(しょけい)である負芻(ふすう)が郝を殺して、みずから王に立った。

魏(ぎ)の景湣王(けいびんおう)が薨じ、その子の假(か)が立った。

燕の太子丹(たいしたん)は秦王を恨み〔注:かつて秦で人質となったとき礼遇されなかったことを恨んだのである〕、報復しようと考えて、その傅(ふ。守り役)である鞠武(きくぶ)に相談した。鞠武はこう答えた。西は三晉(さんしん。韓・魏・趙の三国)と盟約を結び、南は齊(せい)・楚と連携し、北は匈奴(きょうど)と和睦して、そのうえで秦を討つ策をはかるべきです、と。太子は言った。「太傅(たいふ)の策は、いたずらに時日(じじつ)を費やして長くかかりすぎ、人の心を悶々(もんもん)とさせます。とてもそれを待ってなどいられません。」

しばらくして、将軍樊於期(はんおき)が罪を得て、燕へ逃げてきた。太子はこれを受け入れ、館(やかた)に住まわせた。鞠武は諫(いさ)めて言った。「そもそも秦王のあの残虐(ざんぎゃく)さで、燕に対して怒りを積もらせているのです。それだけでも背筋が凍る思いだというのに、ましてや樊将軍がここにいると秦に知れたら、どうなりましょうか。これでは、飢えた虎の通り道に肉を投げ置くようなものです。どうか太子よ、急いで樊将軍を匈奴へ送り出してしまわれますように。」太子は言った。「樊将軍は天下のどこにも身の置きどころを失い、この丹(たん)を頼って身を寄せてこられたのだ。たとえそれで丹の命が尽きることになろうとも、見捨てるわけにはいかぬ〔注:"命卒"とは命が尽きること。丹は、樊将軍が窮して身を寄せてきた以上、死力を尽くしてもかくまうべきだ、と言ったのである〕。どうか別の策を考えてくれ。」鞠武は言った。「そもそも、危険を冒して安泰を求め、災いの種をまいて福としようとなさる。はかりごとは浅はかなのに、秦への恨みばかりは深い。たった一人との新しい交わりにこだわって、国家の重大な禍(わざわい)を顧みようとなさらない。これこそ、わざわざ恨みを買い、災いを助長するというものです。」だが太子は聞き入れなかった。

太子は、衞(えい)の人である荊軻(けいか)が賢者だと聞き〔注:荊軻の「荊」は楚の旧称で、楚がまだ国号を改める前に受けた姓だという〕、言葉を低くし手厚い礼をもって、面会を求めた。そして荊軻に語った。「いま秦はすでに韓王を捕虜とし、さらに兵を起こして南は楚を討ち、北は趙に迫っている。趙が秦を支えきれなければ、災いは必ずこの燕にも及ぶだろう。燕は小さく弱く、たびたび戦(いくさ)で苦しめられてきた。とても秦に立ち向かえるものではない。諸侯はみな秦に服従し、もはや誰も合従(がっしょう)しようとはしない。丹のひそかな策は愚かなものかもしれぬが、こう考えている――もし天下の勇士を得て、その者を秦への使者として送り込み、秦王を脅(おど)して、諸侯から奪い取った土地をことごとく返させることができたなら、と。かつて曹沫(そうばつ)が齊の桓公(かんこう)に迫ったように、それができれば最上だ。だが、もしそれがかなわなければ、その場で秦王を刺し殺してしまう。そうなれば、秦の大将は国外で兵権をほしいままにしている一方、国内には乱が生じ、君臣は互いに疑心暗鬼に陥る。その隙(すき)に乗じて諸侯が合従できれば、秦を打ち破ることは間違いない。どうか荊卿(けいけい)、この件に心を留めてくだされ。」荊軻はこれを引き受けた。こうして荊卿を上等の館に住まわせ、太子は毎日その門下に通った。荊軻をもてなし養うのに、ありとあらゆる心づくしを尽くした。

やがて王翦が趙を滅ぼすと、太子はそれを聞いて恐れ、荊軻を秦へ発(た)たせようとした。荊軻は言った。「いま、何の手土産(=信用の証(あかし))もなく行ったところで、秦王に近づくことはできません。もし樊将軍の首と、燕の督亢(とくこう)の地図とを手に入れ〔注:督亢は燕の肥沃(ひよく)な土地〕、秦王に献上できれば、秦王は必ず喜んで私に会うでしょう。そうなって初めて、私は太子のご恩に報いることができるのです。」太子は言った。「樊将軍は窮地に追われて丹を頼ってこられたのだ。その首を求めるなど、丹にはとてもできぬ。」

そこで荊軻は、ひそかに樊於期に会って言った。「秦が将軍を遇したやり方は、まことにむごいものでした。ご両親も一族も、みな殺されるか、官に没入(ぼつにゅう)されてしまった。いま聞けば、秦は将軍の首に懸賞をかけ、黄金千斤(きん)と一万戸の邑(ゆう)を出すという。これをどうなさるおつもりか。」樊於期は深いため息をつき、涙を流して言った。「いったい、どんな策があるというのだ。」荊卿は言った。「将軍の首を頂戴して秦王に献上したいのです。さすれば秦王は必ず喜んで私を引見しましょう。そのとき私は、左手で王の袖をつかみ、右手でその胸を刺し貫きます〔注:"揕(ちん)"とは、剣でその胸を刺すことをいう〕。そうすれば、将軍の(ご一族の)仇(あだ)は報われ、燕が秦に辱(はずかし)められてきた恥もすすがれるのです。」樊於期は言った。「それこそ、この私が日夜、歯を食いしばり、胸を焦がして願ってきたことだ。」そう言うと、みずから首をはねて死んだ〔注:"切齒腐心"とは、歯ぎしりするほどの怒りと、はらわたの煮えくり返る思いをいう〕。太子はこれを聞いて駆けつけ、亡骸(なきがら)に伏して泣いたが、もはやどうすることもできず、ついに樊将軍の首を箱に納めた。

太子はあらかじめ、天下に名だたる鋭い匕首(あいくち)を手に入れ、職人に命じて毒薬を刃に焼き付けさせた〔注:"焠(さい)"とは、毒薬を剣の刃に染み込ませることをいう〕。それを人で試したところ、流れ出る血が糸筋(いとすじ)を濡(ぬ)らすほどのかすり傷でも、その場で即死しない者はいなかった。こうして支度(したく)を整えて荊軻を送り出すことにし、燕の勇士秦舞陽(しんぶよう)を副使(ふくし)として付け、秦へ入らせた。

十九年
王翦擊趙軍，大破之，殺趙葱，顏聚亡，遂克邯鄲，虜趙王遷。王如邯鄲，故與母家有仇怨者皆殺之。還，從太原、上郡歸。
太后薨。
王翦屯中山以臨燕。趙公子嘉帥其宗數百人奔代，自立爲代王。趙之亡，大夫稍稍歸之，與燕合兵，軍上谷。
楚幽王薨，國人立其弟郝。三月，郝庶兄負芻殺之，自立。
魏景湣王薨，子假立。
燕太子丹怨王，欲報之，以問其傅鞠武。鞠武請西約三晉，南連齊、楚，北媾匈奴以圖秦。太子曰︰「太傅之計，曠日彌久，令人心惽然，恐不能須也。」頃之，將軍樊於期得罪，亡之燕，太子受而舍之。鞠武諫曰︰「夫以秦王之暴而積怒於燕，足爲寒心，又況聞樊將軍之所在乎！是謂委肉當餓虎之蹊也。願太子疾遣樊將軍入匈奴！」太子曰︰「樊將軍窮困於天下，歸身於丹，是固丹命卒之時也。願更慮之！」鞠武曰︰「夫行危以求安，造禍以爲福，計淺而怨深，連結一人之後交，不顧國家之大害，所謂資怨而助禍矣。」太子不聽。
太子聞衞人荊軻之賢，卑辭厚禮而請見之。謂軻曰︰「今秦已虜韓王，又舉兵南伐楚，北臨趙；趙不能支秦，則禍必至於燕，燕小弱，數困於兵，何足以當秦！諸侯服秦，莫敢合從。丹之私計愚，以爲誠得天下之勇士使於秦，劫秦王，使悉反諸侯侵地。若曹沫之與齊桓公，則大善矣；□不可，則因而刺殺之。彼大將擅兵於外而內有亂，則君臣相疑，以其間，諸侯得合從，其破秦必矣。唯荊卿留意焉！」荊軻許之。於是舍荊卿於上舍，太子日造門下，所以奉養荊軻，無所不至。及王翦滅趙，太子聞之懼，欲遣荊軻行。荊軻曰︰「今行而無信，則秦未可親也。誠得樊將軍首與燕督亢之地圖，奉獻秦王，秦王必說見臣，臣乃有以報。」太子曰︰「樊將軍窮困來歸丹，丹不忍也！」荊軻乃私見樊於期曰︰「秦之遇將軍，可謂深矣，父母宗族皆爲戮沒！今聞購將軍首，金千斤，邑萬家，將柰何？」於期太息流涕曰︰「計將安出？」荊卿曰︰「願得將軍之首以獻秦王，秦王必喜而見臣，臣左手把其袖，右手揕其胸，則將軍之仇報而燕見陵之愧除矣！」樊於期曰︰「此臣之日夜切齒腐心也！」遂自刎。太子聞之，奔往伏哭，然已無柰何，遂以函盛其首。太子豫求天下之利匕首，使工以藥焠之，以試人，血濡縷，人無不立死者。乃裝爲遣荊軻，以燕勇士秦舞陽爲之副，使入秦。
