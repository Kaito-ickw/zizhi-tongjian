# 卷002 周紀二 — 顯王五年

[← 巻インデックス](README.md)

顯王五年〔注:丁巳(ていし)の年、紀元前三六四年〕。

秦の獻公(けんこう)が三晉(さんしん)の軍を石門(せきもん)で打ち破り、六万の首を斬った。周の王は獻公に黼黻(ほふつ)の服〔注:斧の形や、二つの「己」の字を背中合わせにした文様を刺繍した礼装の衣〕を下賜した。

---

<summary>原文を表示</summary>

五年
秦獻公敗三晉之師于石門，斬首六萬。王賜以黼黻之服。
