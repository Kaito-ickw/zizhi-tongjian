# 卷005 周紀五 — 赧王下五十八年

> 巻 5 / 294 ・ 周紀五 ・ 年号: 赧王下五十八年 ・ 西暦: 257 BCE

五十八年〔注:甲辰(こうしん)の年、紀元前二五七年〕。

十月、(秦王は)武安君(ぶあんくん=白起〈はくき〉)の官爵(かんしゃく)を取り上げて士伍(しご=一兵卒の身分)に落とし〔注:法によれば、罪を得て官爵を失うことを「士伍」と称した。爵を奪い、一般の士卒の列に身を置かせるという意である〕、陰密(いんみつ)へ移すことにした。十二月、さらに兵を送って汾城(ふんじょう)のほとりに駐屯(ちゅうとん)させた〔注:汾城は王齕の軍の後ろ盾とするため、秦が兵を屯(たむろ)させた地である〕。武安君は病(やまい)で、まだ(陰密へ)出立していなかった。諸侯が王齕を攻め、王齕はたびたび退(しりぞ)き、使者が日ごとにやって来た。そこで王は人を遣わして武安君を追い立て、咸陽(かんよう)の中にとどまることを許さなかった。武安君は咸陽の西門を出て十里、杜郵(とゆう)に至った。王は應侯(おうこう)や群臣と謀(はか)って言った。「白起は左遷(させん)されてもなお、不満げで、言いたいことがある様子だ。」そこで王は使者を遣わして剣を賜(たまわ)り、武安君はついに自殺した。秦の人々は彼を憐(あわ)れみ、村々や町々でみな彼を祭った。

魏の公子無忌(むき=信陵君〈しんりょうくん〉)は、邯鄲(かんたん)の城下で秦軍を大いに破り、王齕は邯鄲の囲みを解いて逃げ去った。(秦の将)鄭安平(ていあんぺい)は趙軍に追いつめられ、二万の兵を率いて趙に降(くだ)った。應侯はこのことで罪に問われた〔注:鄭安平はかつて范雎(はんしょ)をかくまって王稽(おうけい)に引き合わせ、その縁で范雎は秦に入って宰相となった。だから范雎は鄭安平の身元を引き受けて取り立てた。いま鄭安平が趙に降ったので、范雎はそのために罪を得たのである。秦の法では、身元を引き受けて推挙した者が期待にそむけば、推挙された者と同じ罪に問われた〕。

公子無忌は、趙を救って存続させたのち、魏へ帰ることをあえてせず、賓客とともに趙にとどまり、(魏から従えてきた)将に命じてその軍を魏へ返した。趙王は平原君(へいげんくん)と相談し、五城をもって公子を封じることにした。趙王はみずから道を掃き清めて公子を迎え、主人(あるじ)としての礼をとり、公子を(客の座る)西の階(きざはし)へと案内した。公子は横へ身を引いてそれを辞退し、(主人の側の)東の階から上がり〔注:『礼記(らいき)』曲礼(きょくらい)に、主人は東の階につき、客は西の階につく、とある。客がへりくだる場合は、主人の側の階につく〕、みずから「自分には罪過(ざいか)がある」と言い、魏には背(そむ)いてしまい、趙には何の功もない、と述べた。趙王は公子と日暮れまで酒を酌(く)み交わしたが、(公子があまりに)へりくだるので、とうとう五城を差し出すと言い出せなかった。趙王は鄗(こう)を公子の湯沐邑(とうもくゆう=その税収を入浴・身支度の費えに充てる領地)とした。魏もまた改めて信陵(しんりょう)の地を公子に奉(たてまつ)った。公子は、趙に毛公(もうこう)という処士(しょし=仕官せぬ賢人)が博打(ばくち)打ちの仲間に身を隠し、薛公(せっこう)という処士が酒売りの家に身を隠していると聞き、これに会おうとした。だが二人は会おうとしない。そこで公子は、こっそり徒歩で出かけて、彼らと交わった。平原君はそれを聞いて、(公子の行いを)よくないとそしった。公子は言った。「私は平原君が賢いと聞いて、それゆえ魏に背いてまで趙を救ったのだ。ところが平原君が交わっている相手ときたら、ただ豪族(ごうぞく)に推挙された者ばかりで〔注:「豪挙」とは、豪族・有力者に推挙された者をいう〕、本物の士を求めてはいない。この無忌(=公子)が、あの二人と交わるのでさえ、彼らに相手にしてもらえぬのではと案じているほどなのに、平原君はそれを恥だと言うのか。」そう言って旅支度をととのえ、立ち去ろうとした。平原君は冠(かんむり)を脱いで詫(わ)びたので、公子はとどまった。

平原君は魯連(ろれん=魯仲連〈ろちゅうれん〉)を封じようとした〔注:魯仲連が新垣衍(しんえんえん)の「秦を帝とせよ」という説をくじいた功による〕。使者が三度も往復したが、魯連はとうとう受けようとしない。そこで(平原君は)千金をもって魯連の長寿を祝おうとした。魯連は笑って言った。「天下の士が尊ばれるわけは、人のために憂(うれ)いを払い、難儀を解き、もつれを解きほぐしながら、その見返りを取らぬ点にあります。もし見返りを取るなら、それは商人(あきんど)のすること。」こうして平原君に別れを告げて去り、生涯ふたたび平原君に会うことはなかった。

秦の太子の妃(きさき)は華陽夫人(かようふじん)といったが、子がなかった。夏姬(かき)が子の異人(いじん)を生んだ。異人は趙に人質となっていた。秦がたびたび趙を伐(う)ったので、趙の人々は異人を礼遇しなかった。異人は庶(しょ)の身分の孫として諸侯のもとに人質となっており、車や乗り物、暮らしの費(つい)えにもゆとりがなく、起居(ききょ)も苦しく、思うにまかせなかった。

陽翟(ようてき)の大商人(おおあきんど)、呂不韋(りょふい)が、たまたま邯鄲にやって来て異人を見かけ、言った。「これは掘り出し物(奇貨〈きか〉)だ。買い込んでおく値打ちがある。」そこで異人を訪ねて、説いて言った。「私はあなたの家の門を大きくしてさしあげられます。」異人は笑って言った。「まずはご自分の家の門を大きくなさるがよい。」呂不韋は言った。「あなたはお分かりでない。私の家の門は、あなたの家の門が大きくなるのを待ってこそ大きくなるのです。」異人は、その言わんとするところを心に悟り、彼を引き入れて座らせ、深く語り合った。呂不韋は言った。「秦王はもうお年を召しておられる。太子は華陽夫人を愛しておられるが、夫人には子がない。あなたの兄弟は二十余人。なかでも子傒(しけい)が秦の国を継ぐ有力な立場にあり、士倉(しそう)がこれを補佐しております。あなたは中ほどの(目立たぬ)立場にあって、さほど寵愛(ちょうあい)も受けず、長らく諸侯のもとに人質となっておられる。太子が即位なされたら、あなたは(兄弟と)世継ぎの座を争えますまい。」異人は言った。「それでは、どうすればよいのか。」呂不韋は言った。「嫡子(ちゃくし)を立てられるのは、ただ華陽夫人おひとり。この不韋、貧しい身ではありますが、千金をもって、あなたのために西(=秦)へおもむき、あなたを世継ぎに立てさせましょう。」異人は言った。「もしあなたの策どおりになれば、秦の国を分けて、あなたとともに治めようではないか。」呂不韋はそこで五百金を異人に与えて賓客と縁を結ばせ、さらに五百金で珍しい品々や愛玩(あいがん)の物を買い込み、それらを携(たずさ)えて西へ向かった。まず華陽夫人の姉に会い、珍しい品々を夫人に献じ、そのついでに「子の異人さまは賢く、賓客は天下に広く、いつも日夜、太子さまと夫人さまを慕って泣いておられ、『異人は夫人さまを天と頼んでおります』と申しておられます」とほめそやした。夫人は大いに喜んだ。呂不韋はそこで夫人の姉を通じて、夫人を説かせて言った。「そもそも美貌(びぼう)で人に仕える者は、色香(いろか)が衰えれば寵愛も薄れるもの。いま夫人さまは愛されてはいても子がございません。花の盛りの今のうちに、早く子(=太子の子ら)のうちから賢く孝行な者を選んで自分と結びつけ、これを推して嫡子になさらなければ、色香が衰えて寵愛が薄れたとき、ひとことものを言おうとしても、もはやかないましょうか。いま子の異人さまは賢く、ご自分が中ほどの子で嫡子にはなれぬとわきまえておられます。夫人さまがこの折に異人さまを引き立ててやれば、異人さまは(継ぐべき)国がないのに国を持ち、夫人さまは子がないのに子を持つことになり、生涯、秦で寵愛を受けられましょう。」夫人はもっともだと思い、折を見て太子に言った。「子の異人は、たいそう賢く、行き来する者はみな彼をほめております。」そして涙ながらに言った。「私は不幸にして子がございません。どうか子の異人を世継ぎに立て、わが身を託させてくださいませ。」太子はこれを承知し、夫人と玉の符(ふ)に刻みを入れて、異人を世継ぎとすることを約束した。そして異人に手厚く贈り物を届け、呂不韋に異人の傅(ふ=もり役)を頼んだ。異人の名声は諸侯の間で高まった。

呂不韋は、邯鄲の姫(おんな)のうち飛び切りの美女をめとって同居していたが、彼女が身ごもっているのを知っていた。異人が呂不韋と酒を飲んだとき、その女を見て、(自分に)くれと求めた。呂不韋はうわべは怒ってみせたが、まもなく彼女を異人に献じた。(女は)身ごもって一年で子の政(せい)を生み〔注:およそ身ごもって十二か月で生まれたという。子の政が、すなわち(後の)始皇帝(しこうてい)である〕、異人はこの女を夫人とした。邯鄲が(秦に)囲まれたとき、趙の人々は異人を殺そうとした。異人は呂不韋とともに、見張りの者に黄金六百斤(きん)を渡し、抜け出して逃げ、秦軍のもとへ駆け込んで、ようやく(秦へ)帰り着いた。異人は楚(そ)の装(よそお)いをして華陽夫人にまみえた〔注:楚服とは、楚の人の服のこと〕。夫人は言った。「私は楚の人。あなたを我が子としよう。」そう言って、異人の名を改めて「楚(そ)」とした。

==五十八年==
十月，免武安君爲士伍，遷之陰密。十二月，益發卒軍汾城旁。武安君病，未行，諸侯攻王齕，齕數卻，使者日至，王乃使人遣武安君，不得留咸陽中。武安君出咸陽西門十里，至杜郵。王與應侯、羣臣謀曰：「白起之遷，意尚怏怏有餘言。」王乃使使者賜之劍，武安君遂自殺。秦人憐之，鄕邑皆祭祀焉。
魏公子無忌大破秦師於邯鄲下，王齕解邯鄲圍走。鄭安平爲趙所困，將二萬人降趙，應侯由是得罪。
公子無忌旣存趙，遂不敢歸魏，與賓客留居趙，使將將其軍還魏。趙王與平原君計，以五城封公子。趙王掃除自迎，執主人之禮，引公子就西階。公子側行辭讓，從東階上，自言辠過，以負於魏，無功於趙。趙王與公子飲至暮，口不忍獻五城，以公子退讓也。趙王以鄗爲公子湯沐邑。魏亦復以信陵奉公子。公子聞趙有處士毛公隱於博徒，薛公隱於賣漿家，欲見之，兩人不肯見；公子乃間步從之游。平原君聞而非之。公子曰：「吾聞平原君之賢，故背魏而救趙。今平原君所與遊，徒豪舉耳，不求士也。以無忌從此兩人遊，尚恐其不我欲也，平原君乃以爲羞乎！」爲裝欲去。平原君免冠謝，乃止。
平原君欲封魯連，使者三返，終不肯受。又以千金爲魯連壽，魯連笑曰：「所貴於天下士，爲人排患釋難解紛亂而無取也。卽有取，是商賈之事也！」遂辭平原君而去，終身不復見。
秦太子之妃曰華陽夫人，無子；夏姬生子異人。異人質於趙；秦數伐趙，趙人不禮之。異人以庶孽孫質於諸侯，車乘進用不饒，居處困不得意。
陽翟大賈呂不韋適邯鄲，見之，曰：「此奇貨可居！」乃往見異人，說曰：「吾能大子之門！」異人笑曰：「且自大君之門！」不韋曰：「子不知也，吾門待子門而大。」異人心知所謂，乃引與坐，深語。不韋曰：「秦王老矣。太子愛華陽夫人，夫人無子，子之兄弟二十餘人，子傒有秦國之業，士倉又輔之。子居中，不甚見幸，久質諸侯。太子卽位，子不得爭爲嗣矣。」異人曰：「然則柰何？」不韋曰：「能立適嗣者，獨華陽夫人耳。不韋雖貧，請以千金爲子西遊，立子爲嗣。」異人曰：「必如君策，請得分秦國與君共之。」不韋乃以五百金與異人，令結賓客。復以五百金買奇物玩好，自奉而西，見華陽夫人之姊，而以奇物獻於夫人，因譽子異人之賢，賓客徧天下，常日夜泣思太子及夫人，曰：「異人也以夫人爲天！」夫人大喜。不韋因使其姊說夫人曰：「夫以色事人者，色衰則愛弛。今夫人愛而無子，不以繁華時蚤自結於諸子中賢孝者，舉以爲適，卽色衰愛弛，雖欲開一言，尚可得乎！今子異人賢，而自知中子，不得爲適，夫人誠以此時拔之，是子異人無國而有國，夫人無子而有子也，則終身有寵於秦矣。」夫人以爲然，承間言於太子曰：「子異人絕賢，來往者皆稱譽之。」因泣曰：「妾不幸無子，願得子異人立以爲子以託妾身！」太子許之，與夫人刻玉符，約以爲嗣，因厚餽遺異人，而請呂不韋傅之。異人名譽盛於諸侯。
呂不韋娶邯鄲諸姬絕美者與居，知其有娠，異人從不韋飲，見而請之。不韋佯怒，旣而獻之，孕期年而生子政，異人遂以爲夫人。邯鄲之圍，趙人欲殺之，異人與不韋行金六百斤予守者，脫亡赴秦軍，遂得歸。異人楚服而見華陽夫人，夫人曰：「吾楚人也，當自子之。」因更其名曰楚。
