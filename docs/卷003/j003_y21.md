# 卷003 周紀三 — 赧王上十五年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上十五年 ・ 西暦: 300 BCE

[← 巻インデックス](README.md)

---

十五年〔注:辛酉(かのととり)の年、紀元前三〇〇年〕。

①秦の涇陽君(けいようくん)が齊へ人質となった。

②秦の華陽君(かようくん)が楚を伐ち、楚軍を大いに破って三万を斬首し、その将軍である景缺(けいけつ)を殺して、楚の襄城(じょうじょう)を取った。楚王は恐れ、太子を齊へ人質に送って和を請うた。

③秦の樗里疾(ちょりしつ)が卒(しゅつ)した。趙人の樓緩(ろうかん)を丞相(じょうしょう)とした。

④趙の武靈王は末子の何(か)を可愛がり、自分が生きているうちにこれを(後継ぎに)立てようと思った。

十五年
秦涇陽君爲質於齊。
秦華陽君伐楚，大破楚師，斬首三萬，殺其將景缺，取楚襄城。楚王恐，使太子爲質於齊以請平。
秦樗里疾卒，以趙人樓緩爲丞相。
趙武靈王愛少子何，欲及其生而立之。
