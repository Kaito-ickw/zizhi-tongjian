# 卷001 周紀一 — 烈王四年

> 巻 1 / 294 ・ 周紀一 ・ 年号: 烈王四年 ・ 西暦: 372 BCE

[← 巻インデックス](README.md)

烈王四年〔注:己酉(きゆう)の年、紀元前三七二年〕。

趙が衛を攻め、都鄙(とひ)七十三を奪い取った〔注:「都鄙」は卿(けい)・大夫(たいふ)の領邑(采邑)をいう〕。

魏が趙の軍を北藺(ほくりん)で破った。

---

<details>
<summary>原文を表示</summary>

四年
趙伐衞，取都鄙七十三。
魏敗趙師于北藺。
