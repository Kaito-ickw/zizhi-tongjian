# 卷002 周紀二 — 顯王元年

顯王元年〔注:癸丑(きちゅう)の年、紀元前三六八年〕。

齊が魏を攻め、觀津(かんしん)を奪い取った。

趙が齊に侵攻し、長城(ちょうじょう)を奪い取った。

顯王
元年
齊伐魏，取觀津。
趙侵齊，取長城。
