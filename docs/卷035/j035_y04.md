# 卷035 漢紀二十七 — 孝平皇帝上二年

二年〔注:壬戌(みずのえいぬ)の年、紀元二年〕。

春、黄支国(こうしこく)が犀(さい)を献上した。黄支は南海の中にあり、都から三万里離れている〔注:黄支国は日南(にちなん)のさらに南にある〕。王莽は漢の威徳を輝かせようとして、その王に手厚く贈り物をし、使者を遣わして貢ぎ物を献上させたのである。

越巂郡(えつすいぐん)が、黄色い竜が川の中を泳いだと報告すると、太師の孔光、大司徒の馬宮らは、みな「王莽の功徳は周公に匹敵します。宗廟に告げ祭るべきです」と唱えた。大司農の孫宝(そんぽう)は言った。「周公は最高の聖人、召公は大いなる賢人でありながら、なお互いに快く思わぬことがあり、それが経典に記されてはいても、二人の名声を損なってはおりません〔注:説は「悦」と読む〕。いまは風雨が時候どおりでなく、民は窮乏しております。何か一事あるごとに群臣が口をそろえる、このありさまでは〔注:雷同(らいどう)して阿(おもね)り、みだりに吉兆を言い立てることをいう〕、そのほめていることが、かえってほめるに値しないものなのではありませんか」〔注:ほめている対象が、実はほめるべきものでない、ということ〕。このとき、大臣たちはみな顔色を失った。甄邯はただちに、太后の制(みことのり)を承(う)けたとして、この議論を打ち切らせた。おりしも孫宝は役人を遣わして母を都へ迎えさせたが、母は道中で病になり、弟の家にとどまって、孫宝のもとには妻子だけが来た。司直(しちょく)の陳崇(ちんすう)が孫宝を弾劾して奏上し、事は三公に下されて、その場で取り調べられた〔注:就いて問いただした、ということ〕。孫宝は答えて言った。「私は年七十、もうろくして心が惑い、母を供養する恩愛の心も衰えて、つい妻子のことばかりにかまけました。弾劾状のとおりです」〔注:誖は「惑う」、眊は耄(もうろく)に同じ。孫宝は自ら、自分は年老いて心が乱れ惑い、母を供養する恩愛も衰えた、と述べ、すべて弾劾状のとおりだと認めたのである〕。孫宝は罪に問われて免ぜられ、家で生涯を終えた。

帝は名を衎(かん)と改めた。

三月、癸酉(みずのととり)の日、大司空の王崇は病と称して免ぜられた。王莽を避けるためであった。

夏、四月、丁酉(ひのととり)の日、左将軍の甄豊を大司空とし、右将軍の孫建を左将軍とし、光禄勲の甄邯を右将軍とした。

代孝王(だいこうおう)の玄孫(やしゃご)の子である劉如意(りゅうにょい)を広宗王(こうそうおう)に、江都易王(こうといおう)の孫である盱台侯(くいこう)劉宮(りゅうきゅう)を広川王(こうせんおう)に、広川恵王(こうせんけいおう)の曾孫である劉倫(りゅうりん)を広徳王(こうとくおう)に立てた〔注:代孝王劉参(りゅうしん)の孫の劉義(りゅうぎ)は清河(せいが)に改封され、国は孫の劉年(りゅうねん)に伝わったが、宣帝の地節四年に罪で廃された。いま劉如意を封じて孝王の後を継がせた。江都易王劉非(りゅうひ)の国は子の劉建(りゅうけん)に伝わったが、武帝の元狩二年に謀反して自殺した。いま劉宮を立てて易王の後を継がせた。広川恵王の国は、子孫の代に罪で廃された。いま劉倫を立てて恵王の後を継がせた。これらはみな、王莽が政を執って、絶えた家を継がせることで衆人を惑わそうとしたものである〕。漢が興(おこ)って以来の大功臣の子孫である周恭(しゅうきょう)らを継いで封じ、みな列侯および関内侯とした〔注:周恭は絳侯(こうこう)周勃(しゅうぼつ)の玄孫である。共は「恭」と読む〕。あわせて百十七人であった。

諸郡国で大干ばつと蝗(いなご)の害があり、青州(せいしゅう)がとりわけひどく〔注:青州は平原・千乗(せんじょう)・済南・斉・北海・東萊(とうらい)などの郡や、甾川(しせん)・膠東(こうとう)・高密(こうみつ)などの王国を管轄する〕、民は流亡した。王莽は太后に申し出た。「質素な無地の絹をお召しになり〔注:繒練(そうれん)とは、模様のない絹のこと〕、お食事もいくらか減らして、天下に範をお示しになるべきです」。王莽はそこで上書して、銭百万を出し、田三十頃を献じて大司農に渡し、貧しい民を救済する助けにしたいと願い出た。そこで公卿たちはみなこれを慕ってまねをし、田や宅を献じた者は全部で二百三十人にのぼった。献じられた田宅は、人数に応じて貧しい民に分け与えられた〔注:人数を数えて、それに応じて田宅を与えたのである〕。また長安の城内に五つの里(り=居住区)を造り〔注:里とは民の住む区画〕、二百戸の宅を設けて、貧しい民を住まわせた。王莽は群臣を率いて太后に奏上して言った。「幸いに陛下の御徳のおかげで、このごろ風雨は時候どおりとなり、甘露(かんろ)が降り、霊妙な芝(しば)が生え、蓂莢(べいきょう)・朱草(しゅそう)・嘉禾(かか=みごとな稲)など、めでたいしるしが時を同じくしてそろって現れました〔注:休は「めでたい」、徴は「しるし」の意〕。どうか陛下、帝王の常の御服に戻られ、太官(たいかん)の正式な御膳を復され、臣下がそれぞれ喜びの心を尽くし、お世話を全うできるようにしてくださいませ」。王莽はまた太后に詔を下させて、この願いを許さないことにした。水害や干ばつがあるたびに、王莽はそのつど精進の食事をとり〔注:素食とは菜食で、肉を断つこと〕、左右の者がそれを太后に告げた。太后は使者を遣わして詔した。「聞けば、公(=王莽)は菜食をして、民を深く憂えているとのこと。今年の秋は幸い実りがよい。公はおりにかなって肉を食べ、国のために身をいたわるがよい」。

六月、鉅鹿(きょろく)に隕石(いんせき)が二つ落ちた。

光禄大夫で楚国の龔勝(きょうしょう)、太中大夫で琅邪(ろうや)の邴漢(へいかん)は、王莽が政を専(ほしいまま)にしているのを見て、ともに骸骨を乞うた。王莽は太后に冊書で詔を下させて言った。「朕(ちん)は、官職の務めで大夫(たいふ)を煩(わずら)わせることを心苦しく思う。大夫よ、身を修め道を守って、長寿をまっとうするがよい」。いずれも手厚い礼遇を加えて郷里へ送り出した。

梅福(ばいふく)は、王莽が必ず漢の帝位を簒奪(さんだつ)すると見抜き、ある朝、妻子を捨てて立ち去り、行方知れずとなった。その後、会稽(かいけい)で梅福を見かけたという者があり、梅福は姓名を変えて、呉(ご)の市場の門番になっていたという〔注:会稽郡は当時、呉県(ごけん)を治所(ちしょ)としていた〕。

秋、九月、戊申(つちのえさる)の晦日(みそか)、日食があった。天下の囚人を赦した。

執金吾(しつきんご)の候(こう=属官)である陳茂(ちんも)を遣わして、江湖(こうこ=長江・湖沼)の賊である成重(せいちょう)ら二百余人を説き諭させると、みな自首した。彼らの家族を、それぞれの本籍地に送って賦役の務めに就かせた〔注:賊二百余人はみな別々の県の出身者なので、自首した後、その家族をそれぞれの本属の県邑に送って賦役に従わせたのである〕。首領の成重は雲陽(うんよう)に移され、官有の田宅を賜った〔注:成重は賊の頭目であったので、これを移したのである〕。

王莽は、漢の威徳がきわめて盛んで、以前とは違うことを示して太后を喜ばせようとし、単于にそれとなく示して、王昭君(おうしょうくん)の娘である須卜居次雲(すぼくきょしうん)を遣わして都に来させ、太后に侍(はべ)らせた。そのためにこれに賜るものは、たいそう手厚かった。

車師後王国(しゃしこうおうこく)に、玉門関(ぎょくもんかん)に通じる新しい道ができ〔注:車師後王国は務塗谷(むとこく)を治所とし、長安から八千九百五十里離れている〕、往来がいくらか近くなったので、戊己校尉(ぼきこうい)の徐普(じょふ)はこれを開こうとした。車師後王の姑句(こく)は、その新道に当たって漢の使者に物資を供給する負担を負うことを、内心不都合に思っていた。徐普はまずその境界を明らかにし、そのうえで朝廷に奏上しようとして、姑句を召して境界を確かめさせようとしたが、姑句は応じなかったので、これを捕らえてつないだ。その妻の股紫陬(こしすう)は姑句に言った。「以前、車師前王は都護(とご)の司馬(しば)に殺されました〔注:これは以前のことをいう。車師前王は交河城(こうがじょう)を治所とし、長安から八千百五十里離れている〕。いま長くつながれていれば、あなたも必ず殺されます。匈奴に降(くだ)ったほうがましです」。姑句はすぐさま馬を駆って高昌壁(こうしょうへき)を突き抜けて逃れ、匈奴に入った〔注:高昌は、その地が高く開けて人や物が栄えていることから名づけられたとも、漢代の高昌の砦(とりで)にちなむともいう〕。

また、去胡来王(きょこらいおう)の唐兜(とうとう)は、赤水(せきすい)の羌(きょう)としばしば攻め合ったが〔注:去胡来王は婼羌国(じゃくきょうこく)の王の号。胡戎(こじゅう)を去って漢に帰服したという意。赤水に住む羌は大きな部族で、婼羌と隣り合っている〕、勝てず、都護に急を告げた。だが都護の但欽(たんきん)は、時を移さずには救援しなかった〔注:但欽は人の姓名〕。唐兜は窮地に追い詰められ、但欽を怨み、東のかた玉門関に拠った。だが玉門関は受け入れなかった。そこで唐兜は妻子や民千余人を引き連れて逃げ、匈奴に降った。単于はこれを受け入れて、左谷蠡(さろくり)王の地に置き〔注:左谷蠡王の住む地である〕、使者を遣わして上書し、その次第を述べて言った。「臣(=単于)、つつしんで、すでに受け入れました」。漢は詔して中郎将の韓隆(かんりゅう)らを匈奴に遣わし、単于を責めた。単于は頭を地につけて謝罪し、二人の降った者を捕らえて、使者に返し渡した〔注:二人の者とは姑句と唐兜〕。漢は詔して、中郎将の王萌(おうぼう)を西域の悪都奴(あくとぬ)の境界で待たせた〔注:悪都奴は西域の谷の名〕。単于は使者を遣わして二人を送り届け、あわせてその罪を赦すよう請うた〔注:二人のために漢に願い出て、その背いた罪を赦してもらおうとした〕。使者がこのことを朝廷に奏聞すると、王莽は聞き入れず、西域の諸国の王を集めるよう詔し、軍を陳列して姑句・唐兜を斬り、これを諸王に見せつけた〔注:後への戒めとして、二度と背かせぬためである〕。そして四か条の取り決めを新たに設けた〔注:あらためてこの制を作ったのである〕。すなわち、中国人で匈奴に逃げ込んだ者、烏孫(うそん)から逃げて匈奴に降った者、西域諸国で中国の印綬を帯びながら匈奴に降った者、烏桓(うがん)で匈奴に降った者は、いずれも匈奴は受け入れてはならぬ、と。中郎将の王駿(おうしゅん)・王昌(おうしょう)、副校尉の甄阜(しんふ)・王尋(おうじん)を匈奴に遣わし、四か条を単于に頒布(はんぷ)した。これを璽書(じしょ)と同じ函(はこ)に封じ込め〔注:璽書と同じ一つの函に入れて封じた〕、単于に渡して、奉じ行わせた。あわせて、もと宣帝が匈奴と結んだ取り決めを封じた函を回収して漢に持ち帰った〔注:宣帝は匈奴と「長城より南は漢が、長城より北は匈奴が領有し、降る者があっても受け入れない」と約束していた。いま王莽は、その取り決めが明確でないとして、四か条を頒布し、旧来の取り決めを回収したのである〕。このとき王莽は、中国では二字の名を用いてはならぬよう奏上して命じており〔注:『公羊春秋伝』が二字の名をそしっているので、王莽はこれにならったのである〕、そこで使者を遣わして単于にそれとなく示し、「上書して漢の教化を慕い、二字の名を改めて一字の名にすれば、漢は必ず手厚い賞を加えるだろう」と伝えさせた。単于はこれに従い、上書して言った。「幸いに藩臣(はんしん)の末に連なり、ひそかに太平の聖なる制度を楽しんでおります。臣のもとの名は囊知牙斯(のうちがし)ですが、いまつつしんで名を改めて『知(ち)』といたします」。王莽は大いに喜び、太后に申し出て、使者を遣わして返答を告げ、手厚く賞賜を与えた。

王莽は、自分の娘を帝に配して皇后とし、もって自分の権を固めようとして、奏上して言った。「皇帝が即位されて三年になりますが、長秋宮(ちょうしゅうきゅう=皇后)はまだ立てられず〔注:秋は実りの時、長は永久の意。それゆえ皇后の宮の名とした〕、後宮の媵(よう=付き添いの女)もまだ満ちておりません〔注:媵とは、嫁に付き添って嫁ぐ女。いにしえ、諸侯が一国から女を嫁がせると、九国が媵を添えた〕。さきの国家の難、すなわち世継ぎが絶えたことは、もともと世継ぎがなかったことに発し、后(きさき)を娶(めと)るのが正しくなかったことによります〔注:取は「娶る」と読む〕。どうか『五経』を考究して、后を娶る礼を定め、十二女(じゅうにじょ)の義を正し〔注:いにしえ、天子は一度に九人の女を娶った。『公羊伝』に、諸侯は一度に九人の女を聘(へい)するとあるから、周の天子にはもとより十二女の礼があったのである〕、もって世継ぎを広く得られるようにし、殷・周二王の後裔(こうえい)、および周公・孔子の家系、長安にいる列侯の嫡子(ちゃくし)の娘を、広く選び集めてください」〔注:二王の後とは殷・周の子孫。周公・孔子の世とは、周公・孔子の子孫で嫡流を継ぐ者。適は「嫡」と読み、正妻の生んだ子のこと〕。事は担当の役人に下され、多くの女たちの名が候補として奉られた。王氏すなわち王莽の一族の娘で選に入っている者が多かったので、王莽は彼女らが自分の娘と后の座を争うことを恐れ、すぐに上言した。「私には徳がなく、私の娘も器量が劣っております。ほかの女たちと並べて選ばれるのはふさわしくありません」。太后はこれを誠心からの言葉だと思い、詔を下して言った。「王氏の娘は、朕の母方の一族である。これを選ぶでない」。庶民・諸生(しょせい=学生)・郎吏(ろうり)以上で、闕(けつ=宮門)に詰めて上書する者が日に千余人にのぼり、公卿大夫の中には朝廷に出向く者もあれば、省(しょう=役所)の戸口に伏す者もあって、みな口々に言った。「安漢公の盛んな勲功は、これほど堂々たるものです。いま后を立てるにあたって、どうして公の娘だけを除くのですか。天下は誰に命運を託せばよいのでしょうか。どうか公の娘を天下の母としていただきたい」。王莽は長史(ちょうし)以下の属官を遣わし、手分けして公卿や諸生を説き諭し、上書をやめさせようとした〔注:曉止とは、説き諭してやめさせること〕。だが上書する者はいよいよ増えた。太后はやむなく、公卿が王莽の娘を選ぶことを聞き入れた。王莽はまた自ら申し立てた。「私の娘に限らず、ほかの女たちを広く選ぶべきです」。公卿は言い争って言った。「ほかの女たちを選んで、正統(=皇后の地位)を二分するべきではありません」〔注:皇后の位は、まさに王莽の娘にあるべきだ、ということ〕。王莽はそこで申し出た。「私の娘をご覧いただきたい」。

二年
春，黃支國獻犀牛。黃支在南海中，去京師三萬里。王莽欲燿威德，故厚遺其王，令遣使貢獻。
越巂郡上黃龍游江中，太師光、大司徒宮等咸稱「莽功德比周公，宜告祠宗廟。」大司農孫寶曰：「周公上聖，召公大賢，尚猶有不相說，著於經典，兩不相損。今風雨未時，百姓不足，每有一事，羣臣同聲，得無非其美者？」時大臣皆失色。甄邯卽時承制罷議者。會寶遣吏迎母，母道病，留弟家，獨遣妻子。司直陳崇劾奏寶，事下三公卽訊。寶對曰：「年七十，誖眊，恩衰共養，營妻子，如章。」寶坐免，終於家。
帝更名衎。
三月，癸酉，大司空王崇謝病免，以避王莽。
夏，四月，丁酉，左將軍甄豐爲大司空，右將軍孫建爲左將軍，光祿勳甄邯爲右將軍。
立代孝王玄孫之子如意爲廣宗王，江都易王孫盱台侯宮爲廣川王，廣川惠王曾孫倫爲廣德王。紹封漢興以來大功臣之後周共等皆爲列侯及關內侯，凡百一十七人。
郡國大旱、蝗，青州尤甚，民流亡。王莽白太后：宜衣繒練，頗損膳，以示天下。莽因上書願出錢百萬，獻田三十頃，付大司農助給貧民。於是公卿皆慕效焉，凡獻田宅者二百三十人，以口賦貧民。又起五里於長安城中，宅二百區，以居貧民。莽帥羣臣奏太后言：「幸賴陛下德澤，間者風雨時，甘露降，神芝生，蓂莢、朱草、嘉禾，休徵同時並至。願陛下遵帝王之常服，復太官之法膳，使臣子各得盡驩心，備共養！」莽又令太后下詔，不許。每有水旱，莽輒素食，左右以白太后。太后遣使者詔曰：「聞公菜食，憂民深矣。今秋幸孰，公以時食肉，愛身爲國！」
六月，隕石于鉅鹿二。
光祿大夫楚國龔勝、太中大夫琅邪邴漢以王莽專政，皆乞骸骨，莽令太后策詔之曰：「朕愍以官職之事煩大夫，大夫其脩身守道，以終高年。」皆加優禮而遣之。
梅福知王莽必篡漢祚，一朝棄妻子去，不知所之。其後，人有見福於會稽者，變姓名爲吳市門卒云。
秋，九月，戊申晦，日有食之，赦天下徒。
遣執金吾候陳茂諭說江湖賊成重等二百餘人皆自出，送家在所收事。重徙雲陽，賜公田宅。
王莽欲悅太后以威德至盛，異於前，乃風單于令遣王昭君女須卜居次云入侍太后，所以賞賜之甚厚。
車師後王國有新道通玉門關，往來差近，戊己校尉徐普欲開之。車師後王姑句以當道供給使者，心不便也。普欲分明其界，然後奏之，召姑句使證之；不肯，繫之。其妻股紫陬謂姑句曰：「前車師前王爲都護司馬所殺，今久繫必死，不如降匈奴！」卽馳突出高昌壁，入匈奴。又去胡來王唐兜與赤水羌數相寇，不勝，告急都護，都護但欽不以時救助。唐兜困急，怨欽，東守玉門關；玉門關不內，卽將妻子、人民千餘人亡降匈奴；單于受置左谷蠡地，遣使上書言狀曰：「臣謹已受。」詔遣中郎將韓隆等使匈奴，責讓單于；單于叩頭謝罪，執二虜還付使者。詔使中郎將王萌待於西域惡都奴界上。單于遣使送，因請其罪；使者以聞。莽不聽，詔會西域諸國王，陳軍斬姑句、唐兜以示之；乃造設四條，中國人亡入匈奴者，烏孫亡降匈如者，西域諸國佩中國印綬降匈奴者，烏桓降匈奴者，皆不得受。遣中郎將王駿、王昌、副校尉甄阜、王尋使匈奴，班四條與單于，雜函封，付單于，令奉行；因收故宣帝所爲約束封函還。時莽奏令中國不得有二名，因使使者以風單于，宜上書慕化，爲一名，漢必加厚賞。單于從之，上書言：「幸得備藩臣，竊樂太平聖制。臣故名囊知牙斯，今謹更名曰知。」莽大說，白太后，遣使者答諭，厚賞賜焉。
莽欲以女配帝爲皇后以固其權，奏言：「皇帝卽位三年，長秋宮未建，掖庭媵未充。乃者國家之難，本從無嗣，配取不正，請考論《五經》，定取后禮，正十二女之義，以廣繼嗣，博采二王後及周公、孔子世、列侯在長安者適子女。」事下有司，上衆女名，王氏女多在選中者，莽恐其與己女爭，卽上言：「身無德，子材下，不宜與衆女並采。」太后以爲至誠，乃下詔曰：「王氏女，朕之外家，其勿采。」庶民、諸生、郎吏以上守闕上書者日千餘人，公卿大夫或詣廷中，或伏省戶下，咸言：「安漢公盛勳堂堂若此，今當立后，獨柰何廢公女，天下安所歸命！願得父女爲天下母！」莽遣長史以下分部曉止公卿及諸生，而上書者愈甚。太后不得已，聽公卿采莽女。莽復自白：「宜博選衆女。」公卿爭曰：「不宜采諸女以貳正統。」莽乃白：「願見女。」
