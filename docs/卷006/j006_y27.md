# 卷006 秦紀一 — 始皇帝上十八年

十八年〔注:壬申(じんしん)の年、紀元前二二九年〕。

王翦(おうせん)が上地(じょうち)の兵を率いて井陘(せいけい)へ下り、端和(たんわ)が河内(かだい)の兵を率いて、ともに趙(ちょう)を攻めた〔注:端和は楊端和(ようたんわ)のことで、本文では「楊」の字が脱(ぬ)け落ちている〕。趙では李牧(りぼく)と司馬尚(しばしょう)がこれを防いだ。秦は趙王の寵臣(ちょうしん)である郭開(かくかい)に多額の金を贈り、李牧と司馬尚を讒言(ざんげん)させて、この二人が謀反(むほん)をたくらんでいると言わせた。趙王は、趙葱(ちょうそう)と齊(せい)の将軍顏聚(がんしゅう)を遣わして、李牧らに代えようとした。だが李牧はこの命に従わなかったので、趙はこれを捕らえて殺し、司馬尚を罷免(ひめん)した。

十八年
王翦將上地兵下井陘，端和將河內兵共伐趙。趙李牧、司馬尚禦之。秦人多與趙王嬖臣郭開金，使毀牧及尚，言其欲反。趙王使趙葱及齊將顏聚代之。李牧不受命，趙人捕而殺之，廢司馬尚。
