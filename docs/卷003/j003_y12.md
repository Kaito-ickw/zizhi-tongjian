# 卷003 周紀三 — 赧王上六年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上六年 ・ 西暦: 309 BCE

[← 巻インデックス](README.md)

---

六年〔注:壬子(みずのえね)の年、紀元前三〇九年〕。

秦がはじめて丞相(じょうしょう)の官を置き、樗里疾(ちょりしつ)を右丞相とした〔注:応劭(おうしょう)によれば、丞は「承(う)ける」、相は「助ける」の意である。荀悦(じゅんえつ)によれば、秦はもともと次国(じこく)であって任じられる卿は二人であったため、左右の丞相を置き、三公の官はなかった。樗里疾は秦の惠王の弟である。高誘(こうゆう)によれば、疾は渭水(いすい)の南岸の陰郷(いんきょう)に住み、その里に大きな樗(おうち)の木があったので、樗里子と号した〕。

六年
