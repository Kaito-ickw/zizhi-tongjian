# 卷004 周紀四 — 赧王中四十年

赧王四十年〔注:丙戌(へいじゅつ)の年、紀元前二七五年〕。

秦の相国(しょうこく)である穰侯(じょうこう)が魏を攻めた。韓の暴鳶(ぼうえん)が魏を救おうとしたが、穰侯はこれを大いに破り、四万の首を斬った。暴鳶は開封(かいほう)へ逃げ、魏は八つの城を差し出して和を結んだ。穰侯はふたたび魏を攻め、芒卯(ぼうぼう)を敗走させ、北宅(ほくたく)に攻め入った〔注:北宅とは、すなわち宅陽(たくよう)のことである〕。〔注:校勘注によれば、章本など一部の版では、この下の「魏」の字の上に「遂圍大梁(ついに大梁を囲んだ)」の四字があるという〕。魏の人々は温(おん)の地を割いて和を結んだ〔注:温は、春秋時代の温邑(おんゆう)で、晋に属していた地である〕。

四十年
秦相國穰侯伐魏。韓暴鳶救魏，穰侯大破之，斬首四萬。暴鳶走開封。魏納八城以和。穰侯復伐魏，走芒卯，入北宅。魏人割溫以和。
