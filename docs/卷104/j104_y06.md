# 卷104 晉紀二十六 — 烈宗孝武皇帝上之中六年

①春、正月、帝は初めて仏法を信奉し、宮殿内に精舎(しょうじゃ、僧の修行の場)を建て、諸々の沙門(しゃもん、仏僧)を招いてそこに住まわせた。尚書左丞の王雅(おうが)が上表して諫めたが、聞き入れられなかった。王雅は、王肅(おうしゅく)の曾孫である。

②丁酉の日、尚書の謝石(しゃせき)を僕射(ぼくや)とした。

③二月、東夷と西域の六十二国が秦に入貢した。

④夏、六月、庚子の朔(ついたち)、日食があった。

⑤秋、七月、甲午の日、交趾(こうし)太守の杜瑗(とえん)が李遜(りそん)を斬り、交州は平定された。

⑥冬、十月、故(もと)武陵王の晞(き)が新安(しんあん)で卒(しゅっ)した。追って新寧郡王に封じ、その子の遵(じゅん)を跡継ぎとするよう命じた。

⑦十一月、己亥の日、前(さき)の会稽内史の郗愔(ちいん)を司空とした。だが郗愔は固辞して(職に)就かなかった。

⑧秦の荊州刺史の都貴(とき)は、その司馬(しば)の閻振(えんしん)と中兵参軍の呉仲(ごちゅう)を遣わし、兵二万を率いて竟陵(きょうりょう)に攻め込ませた。桓沖(かんちゅう)は南平太守の桓石虔(かんせきけん)、衛軍参軍の桓石民(かんせきみん)らを遣わし、水陸あわせて二万の兵を率いてこれを防がせた。桓石民は、桓石虔の弟である。十二月、甲辰の日、桓石虔は閻振・呉仲を襲撃し、これを大いに破った。閻振・呉仲は退いて管城(かんじょう)を守った。桓石虔がさらに進んでこれを攻め、癸亥の日、管城を落とし、閻振・呉仲を捕らえ、首級七千を挙げ、一万人を捕虜とした。詔(みことのり)して桓沖の子の桓謙(かんけん)を宜陽侯に封じ、桓石虔を河東太守を兼ねさせた。

⑨この年、江東は大飢饉に見舞われた。

六年
春，正月，帝初奉佛法，立精舍於殿內，引諸沙門居之。尚書左丞王雅表諫，不從。雅，肅之曾孫也。
丁酉，以尚書謝石爲僕射。
二月，東夷、西域六十二國入貢于秦。
夏，六月，庚子朔，日有食之。
秋，七月，甲午，交趾太守杜瑗斬李遜，交州平。
冬，十月，故武陵王晞卒于新安，追封新寧郡王，命其子遵爲嗣。
十一月，己亥，以前會稽內史郗愔爲司空；愔固辭不起。
秦荊州刺史都貴遣其司馬閻振、中兵參軍吳仲帥衆二萬寇竟陵，桓沖遣南平太守桓石虔、衞軍參軍桓石民等帥水陸二萬拒之。石民，石虔之弟也。十二月，甲辰，石虔襲擊振、仲，大破之，振、仲退保管城。石虔進攻之，癸亥，拔管城，獲振、仲，斬首七千級，俘虜萬人。詔封桓沖子謙爲宜陽侯。以桓石虔領河東太守。
是歲，江東大饑。
