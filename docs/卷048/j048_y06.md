# 卷048 漢紀四十 — 孝和皇帝下九年

九年〔注:丁酉(ひのととり)の年、紀元九七年〕。

春、三月、庚辰(かのえたつ)の日、隴西(ろうせい)で地震があった。

癸巳(みずのとみ)の日、済南安王(さいなんあんおう)の康(こう)が薨(こう)じた。

西域長史(せいいきちょうし)の王林(おうりん)が車師後王(しゃしこうおう)を攻め、これを斬った〔注:後王とは涿鞮(たくてい)である〕。

夏、四月、丁卯(ひのとう)の日、楽成王(らくせいおう)であった劉黨(りゅうとう)の子、巡(じゅん)を封じて、あらためて楽成王とした。

五月、皇后の父で屯騎校尉(とんきこうい)の陰綱(いんこう)を封じて呉房侯(ごぼうこう)とし、特進(とくしん、特別の高位)の位をもって自邸に退かせた〔注:呉房県は汝南(じょなん)郡に属する〕。

六月、旱(ひでり)と蝗害(こうがい)があった。

秋、八月、鮮卑(せんぴ)が肥如(ひじょ)を侵した。遼東太守(りょうとうたいしゅ)の祭参(さいさん)は、敗戦を招いた罪(沮敗)に問われ、獄に下されて死んだ〔注:肥如県は遼西(りょうせい)郡に属する。鮮卑は遼西の肥如を侵し、そのまま遼東郡の境内に攻め入ったのである〕。

閏月(うるうづき)、辛巳(かのとみ)の日、皇太后の竇氏(とうし)が崩じた。はじめ、梁貴人(りょうきじん)が亡くなったのち〔注:この事は第四十六巻、章帝の建初八年に見える〕、宮中(宮省)の事は秘され、帝が梁氏の生んだ子であることを知る者はいなかった。舞陰公主(ぶいんこうしゅ)の子、梁扈(りょうこ)は、従兄(いとこ)の梁䄠(りょうせん)を遣わして三府(さんぷ、三公の役所)に上申書(奏記)を奉り〔注:扈は梁松(りょうしょう)の子である。帝の母である梁貴人は幼くして母を失い、伯母の舞陰公主に養われた〕、こう述べさせた。「漢家の旧典(きゅうてん、古いきまり)では、天子の貴い母の一族を尊崇するもの。ところが梁貴人はみずから聖躬(せいきゅう、天子=帝)をお生み育てになりながら、尊号を受けておられません。どうかこの議をあらためて立てていただきたい」と。太尉(たいい)の張酺(ちょうほ)がこの旨を奏上すると、帝はしばらく感じ入って慟哭(どうこく)し、やがて言った。「そなたの考えではどうか」。張酺は、(梁貴人に)追って尊号を奉り、母方の伯叔父たち(諸舅)を取り立てて世話するよう請うた。帝はこれに従った。おりしも、梁貴人の姉で、南陽(なんよう)の樊調(はんちょう)の妻である嫕(えい)が、上書して自らの訴え(自訟)を述べた。「わたくしの父、竦(しょう、梁竦)は冤罪(えんざい)のうちに牢獄で死に、その骸骨(がいこつ)は埋葬もされておりません。母はすでに七十を過ぎ、弟の棠(とう)らは遠く絶域(ぜついき、はての地)にあって、生死もわかりません。どうか竦の朽ちた骨を引き取り、母と弟を本郡(故郷の郡)に帰らせていただきたいのです」。帝は嫕を引見し、こうして梁貴人が無実のうちに死んだ次第(枉歿之状)を知った。三公が上奏した。「光武帝(こうぶてい)が呂太后(りょたいごう)を退けた先例にならい、竇太后の尊号を貶(おとし)め、先帝と合葬すべきではありません」と〔注:呂太后を退けた事は第四十五巻、光武の中元元年に見える。胡三省いわく——だがこれは光武の過ちであって、それを故事として引いてよいものか〕。百官にも上言する者が多かった。帝は手ずから詔(みことのり)を書いて言った。「竇氏は法度に従わなかったとはいえ、太后はつねに自らを控えめにしてこられた(自減損)。朕は十年お仕えし〔注:即位から今に至るまで十年〕、深く大義を思うに、礼には、臣下や子が尊い上の方を貶める定め(文)はない。恩のうえでは離れるに忍びず、義のうえでは(その尊号を)欠くに忍びない。前代を按(かんが)みれば、上官太后(じょうかんたいごう)もまた降格・廃黜(はいちゅつ)されなかった〔注:上官桀(じょうかんけつ)父子が誅されても、その累(るい)は上官太后に及ばなかったことをいう。事は第二十二巻、昭帝の元鳳元年に見える〕。この議、二度と論ずるな」と。丙申(ひのえさる)の日、章徳皇后(しょうとくこうごう、竇太后)を葬った。

焼唐(しょうとう)羌〔注:異本では「唐」を「当」に作る(=焼当羌)〕の迷唐(めいとう)が、衆八千人を率いて隴西を侵し、塞内(さいない、辺境内)の諸種(しょしゅ)の羌を脅して、歩兵・騎兵あわせて三万人を糾合し、隴西の兵を撃ち破り、大夏(たいか)の長(ちょう、県の長官)を殺した〔注:大夏県は隴西郡に属する〕。詔(みことのり)して、行征西将軍(こうせいせいしょうぐん、征西将軍代行)の劉尚(りゅうしょう)を遣わし、越騎校尉(えっきこうい)の趙世(ちょうせい)をその副(ふく、副官)とし〔注:『考異』は『西羌伝』が「趙代」に作ると記すが、いま『帝紀』に従う。唐の太宗の諱(いみな)が世民であったため、李賢が范曄(はんよう)の『後漢書』に注するさい「世」を「代」に改めたものである〕、漢の兵・羌・胡あわせて三万人を率いてこれを討たせた。劉尚は狄道(てきどう)に、趙世は枹罕(ふかん)に駐屯した〔注:狄道・枹罕の二県はともに隴西郡に属する〕。劉尚は司馬(しば)の寇盱(こうく)を遣わして諸郡の兵を監督させ、四方から同時に集結させた。迷唐は懼(おそ)れ、老弱(ろうじゃく)を棄てて、臨洮(りんとう)の南(の山)へ逃げ込んだ。劉尚らは追って高山(こうざん)に至り、これを大いに破って、千余人を斬り捕らえた。迷唐は兵を引いて去り、漢の兵も死傷者が多く、もはや追えなかったので、引き揚げた。

九月、庚申(かのえさる)の日、司徒(しと)の劉方(りゅうほう)が策書(さくしょ)によって免じられ、自殺した。

甲子(きのえね)の日、梁貴人を追尊(ついそん)して皇太后とし、諡(おくりな)を恭懐(きょうかい)とし、葬礼の制(喪制)をあらためて整えなおした(追復)。冬、十月、乙酉(きのととり)の日、梁太后とその姉の大貴人(だいきじん)を西陵(せいりょう)に改葬した〔注:西陵——その地が敬陵(けいりょう)の西にあるので西陵と称する。李賢いわく、もとの埋葬が不備であったため改葬したのである〕。樊調を擢(ぬ)きんでて羽林左監(うりんさかん)とした。皇太后の父、竦(梁竦)を追って封じ、諡して褒親愍侯(ほうしんびんこう)とし〔注:諡法に「国に在りて難に逢うを愍という」とある〕、使者を遣わしてその喪(亡骸)を迎え、恭懐皇后の陵のかたわらに葬った。竦の妻子を呼び戻し、子の棠を封じて楽平侯(らくへいこう)、棠の弟の雍(よう)を乗氏侯(じょうしこう)、雍の弟の翟(てき)を單父侯(ぜんほこう)とした。位はみな特進、賜り物は巨万(きょまん)を数え、その寵遇(ちょうぐう)は当世に光(かがや)き、梁氏はこのときから栄えた。

清河王(せいがおう)の慶(けい)は、このときはじめて、亡き母である宋貴人(そうきじん)の墓に詣でて祭ることを願い出た〔注:宋貴人の墓は雒陽城の北、樊濯聚(はんたくしゅう)にある〕。帝はこれを許し、太官(たいかん、宮中の食膳役所)に命じて四季(四時)の祭具を給(たま)わせた。慶は涙を流して言った。「生きておられるうちは養い仕えることがかなわなかったが、ついに祭祀をお捧げできた。わたしの私的な願いはこれで足りた」。慶は祠堂(しどう、みたまや)を作りたいと願ったが、(亡き母を)恭懐梁后(きょうかいりょうこう)と同列に並べるとの嫌疑(けんぎ)を招くことを恐れ、ついに口に出せず、つねに左右(側近)に向かって泣き、これを終生(沒齒)の恨みとした。のちに上言して言った。「わたしの外祖母(母方の祖母)である王(おう)は年老いております。どうか雒陽に来て病を療(りょう)じることをお許しください」。そこで詔して宋氏をことごとく京師(けいし)に帰らせ〔注:宋氏が故郡に帰された事は第四十六巻、章帝の建初七年に見える〕、慶の母方の伯叔父(舅)である衍(えん)・俊(しゅん)・蓋(がい)・暹(せん)らをみな郎(ろう、宮中の近侍官)とした。

十一月、癸卯(みずのとう)の日、光禄勲(こうろくくん)で河南(かなん)の呂蓋(りょがい)を司徒とした。

十二月、丙寅(ひのえとら)の日、司空(しくう)の張奮(ちょうふん)が罷(や)めさせられた。壬申(みずのえさる)の日、太僕(たいぼく)の韓稜(かんりょう)を司空とした。

西域都護(せいいきとご)で定遠侯(ていえんこう)の班超(はんちょう)は、掾(えん、属官)の甘英(かんえい)を遣わして大秦(たいしん、はるか西方の大国)・条支(じょうし)へ使者として赴かせた〔注:大秦国は西海の西にあり、その人民は背が高く端正で中国に似ているため大秦と称する。いまの拂菻国(ふつりんこく)がこれである〕。甘英は西海(せいかい)の果てまで窮(きわ)めた。いずれも前代の人々が至ったことのない地で、その風土をことごとく調べ、その珍奇(珍怪)なものごとを伝えなかったものはない。安息(あんそく、西方の大国)の西の境に至り、大海を臨んで渡ろうとすると、船人(ふなびと)が甘英に言った。「海水は広大です。行き来する者は、よい順風に逢えば三月で渡れますが、もし遅い風に遭えば二年かかることもあります。ですから海に入る者は、みな三年分の食糧を携えます。海上では人はとかく故郷を思い慕(した)わしくなり、しばしば死ぬ者が出ます」。甘英はそこで(渡るのを)思いとどまった。

九年
春，三月，庚辰，隴西地震。
癸巳，濟南安王康薨。
西域長史王林擊車師後王，斬之。
夏，四月，丁卯，封樂成王黨子巡爲樂成王。
五月，封皇后父屯騎校尉陰綱爲吳房侯，以特進就第。
六月，旱，蝗。
秋，八月，鮮卑寇肥如，遼東太守祭參坐沮敗，下獄死。
閏月，辛巳，皇太后竇氏崩。初，梁貴人旣死，宮省事祕，莫有知帝爲梁氏出者。舞陰公主子梁扈遣從兄䄠奏記三府，以爲「漢家舊典，崇貴母氏，而梁貴人親育聖躬，不蒙尊號，求得申議。」太尉張酺言狀，帝感慟良久，曰：「於君意若何？」酺請追上尊號，存錄諸舅。帝從之。會貴人姊南陽樊調妻嫕上書自訟曰：「妾父竦冤死牢獄，骸骨不掩；母氏年踰七十，及弟棠等遠在絕域，不知死生。願乞收竦朽骨，使母、弟得歸本郡。」帝引見嫕，乃知貴人枉歿之狀。三公上奏，「請依光武黜呂太后故事，貶竇太后尊號，不宜合葬先帝，」百官亦多上言者。帝手詔曰：「竇氏雖不遵法度，而太后常自減損。朕奉事十年，深惟大義：禮，臣子無貶尊上之文，恩不忍離，義不忍虧。按前世，上官太后亦無降黜，其勿復議！」丙申，葬章德皇后。
燒唐羌迷唐率衆八千人寇隴西，脅塞內諸種羌合步騎三萬人擊破隴西兵，殺大夏長。詔遣行征西將軍劉尚、越騎校尉趙世副之，將漢兵、羌、胡共三萬人討之。尚屯狄道，世屯枹罕；尚遣司馬寇盱監諸郡兵，四面並會。迷唐懼，棄老弱，奔入臨洮南。尚等追至高山，大破之，斬虜千餘人。迷唐引去，漢兵死傷亦多，不能復追，乃還。
九月，庚申，司徒劉方策免，自殺。
甲子，追尊梁貴人爲皇太后，諡曰恭懷，追復喪制。冬，十月，乙酉，改葬梁太后及其姊大貴人于西陵。擢樊調爲羽林左監。追封諡皇太后父竦爲褒親愍侯，遣使迎其喪，葬於恭懷皇后陵旁。徵還竦妻子；封子棠爲樂平侯，棠弟雍爲乘氏侯，雍弟翟爲單父侯，位皆特進，賞賜以巨萬計，寵遇光於當世，梁氏自此盛矣。
清河王慶始敢求上母宋貴人冢，帝許之，詔太官四時給祭具。慶垂涕曰：「生雖不獲供養，終得奉祭祀，私願足矣！」欲求作祠堂，恐有自同恭懷梁后之嫌，遂不敢言，常泣向左右，以爲沒齒之恨。後上言：「外祖母王年老，乞詣雒陽療疾，」於是詔宋氏悉歸京師，除慶舅衍、俊、蓋、暹等皆爲郎。
十一月，癸卯，以光祿勳河南呂蓋爲司徒。
十二月，丙寅，司空張奮罷。壬申，以太僕韓稜爲司空。
西域都護定遠侯班超遣掾甘英使大秦、條支，窮西海，皆前世所不至，莫不備其風土，傳其珍怪焉。及安息西界，臨大海，欲渡，船人謂英曰：「海水廣大，往來者逢善風，三月乃得渡，若遇遲風，亦二歲者；故入海，人皆齎三歲糧，海中善使人思土戀慕，數有死亡者。」英乃止。
