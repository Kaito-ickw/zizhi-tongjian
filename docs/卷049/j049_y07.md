# 卷049 漢紀四十一 — 孝安皇帝上六年

六年〔注:壬子(みずのえね)の年、紀元一一二年〕。

春、正月、甲寅(きのえとら)の日、詔(みことのり)して言った。「およそ献上される初物(はつもの)の珍味は、多くがその季節のものではない。あるものは(温室で)蒸し育てて無理に熟させ〔注:土の室(むろ)に火を蓄え、土の気を蒸らして育て、強引に熟させることをいう。『前漢書』召信臣伝(しょうしんしんでん)に、太官(たいかん、宮中の食膳役)の園では冬に葱(ねぎ)・韮(にら)・野菜のたぐいを植え、小屋で覆い、昼夜たえず火をくすぶらせ、温気で生育させた、とある〕、あるものは芽生えのうちに掘り出すので、味が十分に乗らぬうちに生育を途中で絶たれてしまう。これがどうして時節に順(したが)って物を育てるやり方であろうか。古い伝(でん、『論語』)にも『その時季のものでなければ食べない』とある〔注:李賢(りけん)いわく、『論語』に「不時不食(時ならざるものは食べず)」とあり、季節外れの物は食べないという意。『前漢書』に、召信臣が、季節はずれの物は人を害するので供え物にすべきでない、と言ったとある〕。今より後、陵廟(りょうびょう、陵墓と宗廟)の祭祀に供える物や、天子に差し上げる物は、すべて季節を待ってから献上せよ」。こうして取りやめになったものは全部で二十三種であった。

三月、十の州で蝗(いなご)の害があった。

夏、四月、乙丑(きのとうし)の日、司空(しくう)の張敏(ちょうびん)が罷免された。己卯(つちのとう)の日、太常(たいじょう)の劉愷(りゅうがい)を司空とした。

詔して、建武(けんぶ)年間の創業の功臣である二十八将に、みなその封爵を子孫に継がせた。

五月、旱(ひでり)があった。

丙寅(ひのえとら)の日、詔して、中二千石(ちゅうにせんせき)から下は黄綬(こうじゅ、黄色の印の組み紐の官)に至るまで、すべて俸禄の等級(秩)をもとに戻させた〔注:董巴(とうは)の『輿服志(よふくし)』に、中二千石は青綬(せいじゅ)、四百石・三百石・二百石は黄綬、とある。四年に百官の俸禄を減らしたが、今これをもとに戻したのである〕。

六月、壬辰(みずのえたつ)の日、豫章(よしょう)の員谿原山(いんけいげんざん)が崩れた〔注:豫章郡は洛陽の南二千七百里にあり、揚州(ようしゅう)に属する〕。

辛巳(かのとみ)の日、天下を赦した。

侍御史(じぎょし)の唐喜(とうき)が漢陽(かんよう)の賊である王信(おうしん)を討ち、これを破って斬った。杜季貢(としこう)は逃げて滇零(てんれい)に従った。この年、滇零が死に、子の零昌(れいしょう)が後を継いだが、まだ年が若かったので、同じ部族の狼莫(ろうばく)が彼のために計略をめぐらし、杜季貢を将軍として、別に丁奚城(ていけいじょう)に駐(とど)めた〔注:『東観記』によれば、丁奚城は北地(ほくち)郡の霊州県(れいしゅうけん)にある〕。

六年
春，正月，甲寅，詔曰：「凡供薦新味，多非其節，或鬱養強孰，或穿掘萌芽，味無所至而夭折生長，豈所以順時育物乎！《傳》曰：『非其時不食。』自今當奉祠陵廟及給御者，皆須時乃上。」凡所省二十三種。
三月，十州蝗。
夏，四月，乙丑，司空張敏罷。己卯，以太常劉愷爲司空。
詔建武元功二十八將皆紹封。
五月，旱。
丙寅，詔令中二千石下至黃綬，一切復秩。
六月，壬辰，豫章員谿原山崩。
辛巳，赦天下。
侍御史唐喜討漢陽賊王信，破斬之。杜季貢亡，從滇零。是歲，滇零死，子零昌立，年尚少，同種狼莫爲其計策，以季貢爲將軍，別居丁奚城。
