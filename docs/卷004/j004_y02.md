# 卷004 周紀四 — 赧王中十九年

> 巻 4 / 294 ・ 周紀四 ・ 年号: 赧王中十九年 ・ 西暦: 296 BCE

赧王十九年〔注:乙丑(いっちゅう)の年、紀元前二九六年〕。

楚の懷王(かいおう)が病を発し、秦で薨じた。秦人はその亡骸を(楚へ)送り返した。楚の人々はみな懷王を哀れみ、まるで肉親を悼むように悲しんだ。諸侯はこの一件によって、秦を非道とみなすようになった。

齊・韓・魏・趙・宋が連合して秦を攻め、鹽氏(えんし)まで進んで引きあげた〔注:鹽氏は、塩池をつかさどる官にちなんで呼ばれた地名である〕。秦は韓に武遂(ぶすい)を、魏に封陵(ほうりょう)を返して和睦した〔注:(赧王)十二年に秦が魏から封陵を、韓から武遂を奪い取っていたが、今これらをみな返して和睦したのである〕。

趙の主父(しゅほ)が新たに獲得した地を巡視し〔注:趙が新たに奪い取った中山(ちゅうざん)の地である〕、そのまま代(だい)へ抜けた。西方の西河(せいか)で樓煩(ろうはん)の王と会い、その軍勢を(自軍に)組み入れた。

魏の襄王(じょうおう)が薨じ、その子の昭王(しょうおう)が立った。

韓の襄王が薨じ、その子の釐王(きおう)咎(きゅう)が立った。

十九年
楚懷王發病，薨於秦，秦人歸其喪。楚人皆憐之，如悲親戚。諸侯由是不直秦。
齊、韓、魏、趙、宋同擊秦，至鹽氏而還。秦與韓武遂、與魏封陵以和。
趙主父行新地，遂出代；西遇樓煩王于西河而致其兵。
魏襄王薨，子昭王立。
韓襄王薨，子釐王咎立。
