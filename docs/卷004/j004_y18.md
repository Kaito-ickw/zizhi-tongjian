赧王三十五年〔注:辛巳(しんし)の年、紀元前二八〇年〕。

秦の白起(はくき)が趙軍を破り、二万の首を斬って、代(だい)の光狼城(こうろうじょう)を攻め取った。また司馬錯(しばさく)に隴西(ろうせい)の兵を発(おこ)させ〔注:隴山(ろうざん)から西は、もとは戎(じゅう)や氐(てい)・羌(きょう)の地であったが、秦が代々攻め広げて、その地に隴西郡を置いた〕、蜀(しょく)を経て楚の黔中(けんちゅう)を攻め、これを攻め落とした〔注:このとき秦兵は蜀から巴郡(はぐん)の枳県(しけん)の道を通って、楚の黔中を攻め取った〕。楚は漢水(かんすい)以北および上庸(じょうよう)の地を献じた〔注:漢北とは、漢水より北の宛(えん)・葉(しょう)・樊(はん)・鄧(とう)・隨(ずい)・唐(とう)などの地をいう。上庸は、のちの曹魏の新城、唐の房陵郡(ぼうりょうぐん)の地である〕。

三十五年
秦白起敗趙軍，斬首二萬，取代光狼城。又使司馬錯發隴西兵，因蜀攻楚黔中，拔之。楚獻漢北及上庸地。
