# 卷003 周紀三 — 赧王上十二年

> 巻 3 / 294 ・ 周紀三 ・ 年号: 赧王上十二年 ・ 西暦: 303 BCE

[← 巻インデックス](README.md)

---

十二年〔注:戊午(つちのえうま)の年、紀元前三〇三年〕。

①彗星(すいせい)が現れた。

②秦は魏の蒲阪(ほはん)・晉陽(しんよう)・封陵(ほうりょう)を取り、さらに韓の武遂(ぶすい)を取った。

③齊・韓・魏は、楚が合従(がっしょう)の親しい盟約に背いたとして〔注:九年に楚は齊・韓と合従を結んでいたが、これに背いたのである〕、兵を合わせて楚を伐った。楚王は太子の橫(おう)を秦へ人質に送って救援を請うた。秦の客卿(かくけい)である通(とう)が兵を率いて楚を救うと、三国は兵を引いて去った。

十二年
彗星見。
秦取魏蒲阪、晉陽、封陵；又取韓武遂。
齊、韓、魏以楚負其從親，合兵伐楚。楚王使太子橫爲質於秦以請救。秦客卿通將兵救楚，三國引兵去。
