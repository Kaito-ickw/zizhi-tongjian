# 卷045 漢紀三十七 — 顯宗孝明皇帝下八年

八年〔注:乙丑(きのとうし)の年、紀元六五年〕。

春、正月、己卯(つちのとう)の日。司徒の范遷(はんせん)が薨(こう)じた。

三月、辛卯(かのとう)の日。太尉の虞延(ぐえん)を司徒とし、衞尉(えいい)の趙熹(ちょうき)に太尉の職務を代行させた。

越騎(えっき)の司馬(しば)である鄭衆(ていしゅう)が、北匈奴(きたきょうど)への使者として遣わされた〔注:越騎校尉(えっきこうい)のもとには司馬が一人置かれ、俸禄は千石(せきえ)であった〕。単于(ぜんう)は鄭衆を拝礼させようとしたが、鄭衆は屈しなかった。単于は彼を取り囲んで監禁し、水も火(=煮炊き)も与えなかった。鄭衆が刀を抜いて死を誓うと〔注:死をもって誓い、単于には屈しないという覚悟であった〕、単于は恐れて手を引き、あらためて使者を立て、鄭衆に従わせて都へ帰した。

これより先、大司農(だいしのう)の耿國(こうこく)がこう進言していた。「度遼(どりょう)将軍を置いて五原(ごげん)に駐屯させ、南匈奴(みなみきょうど)が逃げ出すのを防ぐべきです」。だが朝廷はこれを採らなかった。南匈奴の須卜骨都侯(しょぼくこつとこう)らは、漢が北匈奴と使者を交わしているのを知り、内心に不満と恨みを抱いて背こうとし〔注:骨都侯は匈奴の異姓(いせい)の大臣で、左・右の骨都侯がいた。また異姓には呼衍(こえん)氏・須卜(しょぼく)氏・丘林(きゅうりん)氏・蘭(らん)氏があり、いずれも匈奴国中の名族で、常に単于と婚姻を結んでいた〕、ひそかに人を北匈奴のもとへ遣わして、兵を出して自分たちを迎えに来させようとした。鄭衆は塞(さい)を出たとき、何か異変があると疑い、様子をうかがっていたところ、はたして須卜の使者を捕らえた。そこで上奏した。「あらためて大将を置き、南北二つの匈奴が通じ合うのを防ぐべきです」。これによって初めて度遼営(どりょうえい)が設けられ、中郎将(ちゅうろうしょう)の吳棠(ごとう)に度遼将軍の職務を代行させ、黎陽(れいよう)の虎牙営(こがえい)の兵を率いて五原の曼柏(まんぱく)に駐屯させた〔注:後漢の光武帝は幽州・冀州の兵で天下を平定したので、黎陽に営を立て、謁者(えっしゃ)に監督させ、騎兵千人を置いた。前漢の昭帝が范明友(はんめいゆう)を度遼将軍に任じて以来、ここで再び設置されたのである。曼柏県は勝州(しょうしゅう)銀城県の境にあった〕。

秋。十四の郡国(ぐんこく)で大水が出た。

冬、十月。北宮(ほっきゅう)が完成した。

丙子(ひのえね)の日。死罪で獄につながれた囚人を募って度遼営へ送り、罪を犯して逃亡していた者には、それぞれ程度に応じて罪をあがなわせた。楚王(そおう)の英(えい)は、黄色の縑(けん)と白の紈(がん)を国相(こくしょう)のもとへ届けて言った〔注:前漢の成帝のとき、王国の内史(ないし)を廃し、相に民を治めさせた。その職務は太守と同じで、俸禄は二千石。紈は今でいう絹(けん)、縑は糸を二本合わせて織った絹である〕。「藩(はん)の守りとして取り立てていただきながら、過ちと悪が積み重なってしまいました。大いなる恩を喜び、絹を献上して、罪をあがないたく存じます」。国相がこれを奏上すると、詔(みことのり)が下って答えた。「楚王は黄帝・老子(ろうし)の奥深い教えを誦(とな)え、浮屠(ふと=仏)の慈悲を尊び、三月のあいだ身を清めて斎戒(さいかい)し〔注:「齊」は「齋(=斎戒)」と読む〕、神に誓いを立てている。何の疑い、何のはばかりがあろう、悔いや咎(とが)などあろうはずがない。あがないの品は返し、伊蒲塞(いほそく)・桑門(そうもん=出家者)への盛大な供応(きょうおう)の足しにするがよい」。

これより先、帝は西域(せいいき)に神があり、その名を仏(ぶつ)というと聞き、そこで天竺(てんじく)へ使者を遣わしてその教えを求め、その経典と沙門(しゃもん)を得て連れ帰らせていた。その経典はおおむね虚無(きょむ)を根本とし、慈悲と不殺生(ふせっしょう)を尊んでいた。人は死んでも精神は滅びず、ふたたび形を受けて生まれ変わる。生前の善悪にはすべて報い(報応)があるとし、それゆえ精神を修め鍛えて、ついには仏となることを貴ぶ。彼らは雄大で壮麗な言葉を巧みに用いて、無知な世俗の人々を勧め誘った。その道に通じた者は沙門と呼ばれた。こうして中国に初めてその教えが伝わり、その姿を絵に描いた。王公や貴人のなかでも、楚王の英ひとりが真っ先にこれを好んだ〔注:浮屠とは仏のことで、漢語では「覚」、すなわち人々を目覚めさせる意である。沙門は漢語では「息」、欲を去って無為に帰すことをいう。伝えによれば、明帝が金色の大きな人の夢を見て群臣に問うたところ、「西方に仏という神がおります」と答える者があり、そこで天竺へ使者を遣わして教えを問い、その姿を写させた。使者は白馬に経典を負わせて帰り、洛陽(らくよう)城の雍関(ようかん)の西に白馬寺(はくばじ)を建てた。これが中国に仏道が広まる始まりであった〕。

壬寅(みずのえとら)の晦(みそか=月末)の日。日食があり、皆既(かいき)であった〔注:「旣」は「尽きる」の意である〕。帝は詔して、もろもろの役人に職務に励むよう命じ、はばかることなく思うところを言い尽くせと求めた。そこで官にある者はみな封事(ふうじ=封をした上奏)を奉り、それぞれ政の得失を述べた。帝は上奏文を読み、深く自らを責め、提出された文書を百官に配って示した。詔して言った。「諸官の言うことは、みな朕(ちん)の過ちである。民の冤(えん=無実の訴え)を晴らすことができず、ずる賢い役人を取り締まることもできぬ。そのうえ軽々しく民の力を使い、宮殿を修繕(しゅうぜん)し、出入りに節度なく、喜怒の度を越えてしまった。昔の戒めをつくづく顧みると、ぞっとして身がすくむ思いだ。ただ、わが徳の薄さゆえに、時が経つうちに怠惰に陥りはせぬかと、それだけが恐ろしいのだ」。

北匈奴は使者を遣わして貢ぎ物を納めてはいたが、侵略や略奪はやまず〔注:「鈔」は略奪する意〕、辺境の城は昼でも門を閉ざしていた。帝が、北匈奴の使者に答礼の使者を遣わそうと議(はか)ると、鄭衆が上疏(じょうそ)して諫(いさ)めた。「私が聞きますに、北の単于が漢の使者を求めてしきりに招こうとするのは、南の単于の配下を離間させ、西域三十六国(さんじゅうろっこく)の心を匈奴へ繋ぎ止めようとするためです〔注:武帝が西域を開いたとき、もともと三十六国があった。「心を堅める」とは、彼らを匈奴ひとすじに従わせようとする意である〕。さらに漢が和親(わしん)したと喧伝(けんでん)して近隣の敵に誇示し、西域で漢に帰服(きふく)しようとする者をためらわせ疑わせ、漢の地を慕う者を中国に絶望させようというのです。漢の使者がひとたび着けば、単于は思い上がって増長します。もしまた使者を遣わせば、匈奴はきっと「してやったり」と思い込み、(漢への帰服を)諫める群臣たちは二度と口を開けなくなりましょう〔注:「駁議」とは、単于に漢へ帰服するよう勧める者たちのことである〕。そうなれば南単于の朝廷は動揺し、烏桓(うがん)にも漢から離れる心が生じます〔注:南単于の朝廷は西河(せいが)の美稷(びしょく)にあった。「動揺」とは、塞を出て北へ去ろうとすることをいう。烏桓はもともと匈奴に従っていたが、漢は校尉を置いて監督・保護し、匈奴と通じ合わせないようにしていた。「離心」とは、その心が漢に親しまず匈奴へ二心(ふたごころ)を抱くことをいう〕。南単于は長く漢の地に住み、地勢や情勢をすっかり知り尽くしております。もしばらばらに離反すれば、たちまち辺境の害となりましょう。今は幸い度遼の軍勢が北の辺りで威を示しております。たとえ答礼せずとも、敵は害をなそうとはいたしますまい」。だが帝は従わなかった。

帝はあらためて鄭衆を遣わそうとした。鄭衆はそこで上奏した。「私は先に使者を奉じたとき、匈奴に拝礼しませんでした。単于は怒り恨んで兵を出し、私を取り囲みました。いままた命を受ければ、必ずや辱められ屈服を迫られましょう。私はとても、大漢の使節の節(しるし)を持ちながら、毛皮をまとった者ども(=匈奴)に向かってひとり拝礼するなど忍びません〔注:『前漢書・匈奴伝』に、君長から下々まで皆が家畜の肉を食らい、その皮を着、毛氈(もうせん)や毛皮をまとう、とある。「旃」は「氈」と同じである〕。もし匈奴がそうやって私を屈服させられたとなれば、大漢の威勢を損なうことになります」。だが帝は聞き入れなかった。鄭衆はやむをえず出立したが、道中でも続けて上書し、頑なに(遣使の取りやめを)言い争った。帝は詔して鄭衆を厳しく責め、呼び戻して廷尉(ていい)に下し獄(ごく)につないだが、ちょうど恩赦に当たって、家に帰された。その後、帝は匈奴から来た者に会い、鄭衆が単于と礼をめぐって争ったありさまを聞いて、あらためて鄭衆を召し出し、軍司馬(ぐんしば)とした〔注:漢の制度では、大将軍の営は五つの部(ぶ)に分かれ、部ごとに校尉(こうい)一人(俸禄は比二千石)、軍司馬一人(比千石)が置かれた。校尉を置かない部には、軍司馬一人だけを置いた。帝が鄭衆を軍司馬としたのは、馬廖(ばりょう)とともに車師(しゃし)を攻めさせるためであった〕。

八年
春，正月，己卯，司徒范遷薨。
三月，辛卯，以太尉虞延爲司徒，衞尉趙熹行太尉事。
越騎司馬鄭衆使北匈奴，單于欲令衆拜，衆不爲屈。單于圍守，閉之不與水火；衆拔刀自誓，單于恐而止，乃更發使，隨衆還京師。
初，大司農耿國言：「宜置度遼將軍屯五原，以防南匈奴逃亡，」朝廷不從。南匈奴須卜骨都侯等知漢與北虜交使，內懷嫌怨，欲畔，密使人詣北虜，令遣兵迎之。鄭衆出塞，疑有異；伺候，果得須卜使人。乃上言：「宜更置大將，以防二虜交通。」由是始置度遼營，以中郎將吳棠行度遼將軍事，將黎陽虎牙營士屯五原曼柏。
秋，郡國十四大水。
冬，十月，北宮成。
丙子，募死罪繫囚詣度遼營；有罪亡命者，令贖罪各有差。楚王英奉黃縑、白紈詣國相曰：「託在藩輔，過惡累積，歡喜大恩，奉送縑帛，以贖愆罪。」國相以聞，詔報曰：「楚王誦黃、老之微言，尚浮屠之仁慈，潔齊三月，與神爲誓，何嫌何疑，當有悔吝！其還贖，以助伊蒲塞、桑門之盛饌。」
初，帝聞西域有神，其名曰佛，因遣使之天竺求其道，得其書及沙門以來。其書大抵以虛無爲宗，貴慈悲不殺；以爲人死，精神不滅，隨復受形；生時所行善惡，皆有報應，故所貴修煉精神，以至爲佛。善爲宏闊勝大之言，以勸誘愚俗。精於其道者，號曰沙門。於是中國始傳其術，圖其形像，而王公貴人，獨楚王英最先好之。
壬寅晦，日有食之，旣。詔羣司勉脩職事，極言無諱。於是在位者皆上封事，各言得失；帝覽章，深自引咎，以所上班示百官。詔曰：「羣僚所言，皆朕之過。民冤不能理，吏黠不能禁；而輕用民力，繕脩宮宇，出入無節，喜怒過差。永覽前戒，竦然兢懼；徒恐薄德，久而致怠耳！」
北匈奴雖遣使入貢，而寇鈔不息，邊城晝閉。帝議遣使報其使者，鄭衆上疏諫曰：「臣聞北單于所以要致漢使者，欲以離南單于之衆，堅三十六國之心也；又當揚漢和親，誇示鄰敵，令西域欲歸化者局足狐疑，懷土之人絕望中國耳。漢使旣到，便偃蹇自信；若復遣之，虜必自謂得謀，其羣臣駁議者不敢復言。如是，南庭動搖，烏桓有離心矣。南單于久居漢地，具知形勢，萬分離析，旋爲邊害。今幸有度遼之衆揚威北垂，雖勿報答，不敢爲患。」帝不從。復遣衆往，衆因上言：「臣前奉使，不爲匈奴拜，單于恚恨，遣兵圍臣；今復銜命，必見陵折，臣誠不忍持大漢節對氈裘獨拜。如令匈奴遂能服臣，將有損大漢之強。」帝不聽。衆不得已，旣行，在路連上書固爭之上；詔切責衆，追還，繫廷尉，會赦，歸家。其後帝見匈奴來者，聞衆與單于爭禮之狀，乃復召衆爲軍司馬。
