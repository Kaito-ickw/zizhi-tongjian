# 卷104 晉紀二十六 — 烈宗孝武皇帝上之中四年

①春、正月、辛酉の日、大赦を行った。

②秦の長楽公苻丕(ふひ)らは詔を受け取ると恐れおののき、そこで諸軍に力を合わせて襄陽を攻めさせた。秦王苻堅は自ら軍を率いて襄陽を攻めようとし、陽平公苻融(ふゆう)に関東六州の兵を率いて寿春(じゅしゅん)に集結させ、梁熙には河西の兵をもって後詰めとせよと詔した。陽平公苻融は諫めて言った――「陛下が江南を取ろうとされるなら、当然、広く謀りをめぐらし熟慮すべきで、慌てて事を起こしてはなりません。もしただ襄陽を取るだけなら、どうしてわざわざ陛下御自らの御駕(みくるま)をわずらわせるほどのことでありましょうか。天下の兵をこぞって動かして、たった一つの城のために事を起こすなど、これまで例のないことです。いわゆる『随侯(ずいこう)の珠(たま)をもって、はるか高くの雀を撃つ』〔注:『呂氏春秋』に、随侯の名玉で高空の雀を撃てば、世は必ずこれを笑う、用いるものが重く、得るものが軽いからだ、とある〕というものです」。梁熙も諫めて言った――「晋主の暴虐は、いまだ(かつての呉の)孫晧(そんこう)ほどではなく、その山河は険しく堅固で、守りやすく攻めにくいものです。陛下がどうしても江南を平定しようとされるなら、将帥に手分けして命じ、関東の兵を率いて南のかた淮水(わいすい)・泗水(しすい)に臨ませ、梁州・益州の兵を下して東のかた巴(は)・峡(きょう)から出させればよいだけのこと。なぜ必ずしも御自ら鸞輅(らんろ、天子の車)をかがめ、遠く湿地の地へ出向かれる必要がありましょうか。昔、漢の光武帝は公孫述(こうそんじゅつ)を誅し、晋の武帝は孫晧を捕虜にしましたが、この二帝が自ら全軍を統べ、御自ら太鼓の桴(ばち)を執って、矢や石の飛ぶ戦場に身をさらしたとは聞いておりません」。苻堅はそこで(親征を)取りやめた。

冠軍将軍・南郡相の劉波(りゅうは)に兵八千を率いて襄陽を救えと詔したが、劉波は秦を畏れ、あえて進もうとしなかった。朱序はたびたび出撃して秦の兵を破り、秦軍が退いてやや遠ざかると、朱序は備えを怠るようになった。二月、襄陽の督護の李伯護(りはくご)がひそかにその子を遣わして秦に降参の意を通じ、内応することを願い出た。長楽公苻丕は諸軍に命じてこれに進攻させた。戊午の日、襄陽を陥落させ、朱序を捕らえて長安に送った。秦王苻堅は、朱序が(晋への)節義を守り通したことを評価して、度支尚書(たくししょうしょ)に任じた。一方、李伯護は不忠であるとして、斬った。

秦の将軍慕容越(ぼようえつ)は順陽(じゅんよう)を陥落させ、太守で譙国(しょうこく)の丁穆(ていぼく)を捕らえた。苻堅は彼を官職につけようとしたが、丁穆は固辞して受けなかった。苻堅は中壘将軍の梁成(りょうせい)を荊州刺史とし、兵一万を配して襄陽に鎮させ、才能や人望のある者を選んで、礼遇して用いた。

桓沖は襄陽が陥落したことを受けて、上疏して官印と符節を返上し、解任を願い出たが、許されなかった。詔を下して劉波の官を免じたが、まもなくまた冠軍将軍とした。

③秦は前将軍の張蚝(ちょうこう)を幷州刺史とした。

④兗州刺史の謝玄(しゃげん)は兵一万余を率いて彭城(ほうじょう)を救援し、泗口(しこう)に布陣したが、密使を遣わして戴𨔵(たいとん)に連絡しようとしても、できなかった。部曲(ぶきょく、配下部隊)の将である田泓(でんおう)が、水にもぐって密かに彭城へ赴きたいと願い出たので、謝玄は彼を送り出した。田泓は秦人に捕らえられ、手厚く賄賂を渡されて、「南軍(晋軍)はすでに敗れた」と(城中に)言うよう仕向けられた。田泓は偽ってこれを承知したが、いざ(城の近くに来ると)城中に向かって叫んだ――「南軍はまもなく到着する。私は単身で報せに来て、賊に捕らえられた。しっかり持ちこたえよ!」。秦人は彼を殺した。彭超は輜重(しちょう、軍需物資)を留城(りゅうじょう)に置いていたが、謝玄はわざと噂を広めて、後軍将軍の何謙(かけん)を留城に向かわせた。彭超はこれを聞くと、彭城の包囲を解いて兵を引き返し、輜重を守った。戴𨔵は彭城の兵を率いて何謙に従い、謝玄のもとへ逃れた。彭超はそこで彭城を占拠し、兗州治中の徐褒(じょほう)を留めて守らせ、南のかた盱眙(くい)を攻めた。俱難(くなん)は淮陰(わいいん)を陥落させ、邵保(しょうほ)を留めて守備させた。

⑤三月、壬戌の日、詔を下した――「国境に憂いが多く、その年の作物も実らない。天子の御用に供するものは、事を倹約に従わせよ。九族への供給や、諸官の俸禄は、当分の間、半分に減らしてよい。およそ諸々の労役・経費は、軍事・国政の要務でないものは、すべて停止・省略すべし」。

⑥癸未の日、右将軍の毛虎生(もうこせい)に兵三万を率いて巴中(はちゅう)を攻撃させ、魏興(ぎこう)を救援させた。前鋒督護の趙福(ちょうふく)らが巴西(はせい)に至ったところ、秦の将張紹(ちょうしょう)らに敗れ、七千余人を失った。毛虎生は退いて巴東(はとう)に駐屯した。蜀の人・李烏(りう)が兵二万を集め、成都を包囲して毛虎生に呼応したが、秦王苻堅は破虜将軍の呂光(りょこう)にこれを撃たせて滅ぼした。夏、四月、戊申の日、韋鍾(いしょう)が魏興を陥落させると、吉挹(きつゆう)は刀を抜いて自殺しようとしたが、側近たちがその刀を奪った。ちょうどそこへ秦人が到着して彼を捕らえたが、吉挹は口をきかず、食事もとらずに死んだ。秦王苻堅は嘆じて言った――「周孟威(しゅうもうい、周虓)はかつて屈することなく、丁彦遠(ていげんえん、丁穆)は後に身の潔白を守り、吉祖沖(きつそちゅう、吉挹)は口を閉ざして死んだ。なんと晋には忠臣が多いことか!」。吉挹の参軍の史穎(しえい)は逃げ帰ることができ、吉挹が臨終に手ずから書いた上奏文を持ち帰った。詔を下して吉挹に益州刺史の官を追贈した。

⑦秦の毛当(もうとう)・王顕(おうけん)は兵二万を率いて襄陽から東進し、俱難・彭超と合流して淮南を攻めた。五月、乙丑の日、俱難・彭超が盱眙を陥落させ、高密内史の毛璪之(もうそうし)を捕らえた。秦兵六万が幽州刺史の田洛(でんらく)を三阿(さんあ)に包囲したが、そこは広陵(こうりょう)からわずか百里の地であった。朝廷は大いに動揺し、長江沿いに守備の陣を並べ、征虜将軍の謝石(しゃせき)を遣わして水軍を率いさせ、涂中(じょちゅう)に駐屯させた。謝石は謝安の弟である。

右衛将軍の毛安之(もうあんし)らが兵四万を率いて堂邑(どうゆう)に駐屯した。秦の毛当・毛盛(もうせい)が騎兵二万を率いて堂邑を襲うと、毛安之らは驚いて総崩れになった。兗州刺史の謝玄は広陵から三阿を救援した。丙子の日、俱難・彭超は戦いに敗れ、退いて盱眙を守った。六月、戊子の日、謝玄と田洛が兵五万を率いて進んで盱眙を攻めると、俱難・彭超はまたも敗れ、退いて淮陰に駐屯した。謝玄は何謙らを遣わし、水軍で潮に乗って川をさかのぼらせ、夜のうちに(秦が架けた)淮水の橋を焼いた。邵保は戦死し、俱難・彭超は退いて淮北に駐屯した。謝玄は何謙・戴𨔵・田洛とともにこれを追撃し、君川(くんせん)で戦って、またも大いにこれを打ち破った。俱難・彭超は北へ逃げ、かろうじて我が身一つで逃れた。謝玄は広陵に引き上げ、詔によって冠軍将軍の称号を進められ、さらに徐州刺史を兼ねることになった。

秦王苻堅はこれを聞いて大いに怒った。秋、七月、彭超を檻車(おりぐるま)で召還して廷尉に下すと、彭超は自殺した。俱難は爵位を削られて庶民に落とされた。

毛当を徐州刺史として彭城に鎮させ、毛盛を兗州刺史として湖陸(ころく)に鎮させ、王顕を揚州刺史として下邳(かひ)を守らせた。

謝安が宰相となり、秦人がたびたび侵攻して辺境の軍が敗れることが続いても、謝安はいつも穏やかで落ち着いた態度でこれを鎮めた。政治を執るにあたっては、大きな筋道を立てることに努め、細かな詮索はしなかった。当時の人々は謝安を王導になぞらえ、その文雅の風は王導に勝ると評した。

⑧八月、丁亥の日、左将軍の王蘊(おううん)を尚書僕射(しょうしょぼくや)とし、しばらくして丹陽尹に遷した。王蘊は自分が皇室の外戚(皇后の父)であることから、中央にとどまることを望まず、しきりに地方勤めを求めたので、あらためて都督浙江東五郡諸軍事・会稽内史とした。

⑨この年、秦は大飢饉に見舞われた。

四年
春，正月，辛酉，大赦。
秦長樂公丕等得詔惶恐，乃命諸軍幷力攻襄陽。秦王堅欲自將攻襄陽，詔陽平公融以關東六州之兵會壽春，梁熙以河西之兵爲後繼。陽平公融諫曰：「陛下欲取江南，固當博謀熟慮，不可倉猝。若止取襄陽，又豈足親勞大駕乎！未有動天下之衆而爲一城者，所謂『以隨侯之珠彈千仞之雀』也！」梁熙諫曰：「晉主之暴，未如孫晧，江山險固，易守難攻。陛下必欲廓清江表，亦不過分命將帥，引關東之兵，南臨淮、泗，下梁、益之卒，東出巴、峽，又何必親屈鸞輅，遠幸沮澤乎！昔漢光武誅公孫述，晉武帝擒孫晧，未聞二帝自統六師，親執枹鼓，蒙矢石也。」堅乃止。
詔冠軍將軍南郡相劉波帥衆八千救襄陽，波畏秦，不敢進。朱序屢出戰，破秦兵，引退稍遠，序不設備。二月，襄陽督護李伯護密遣其子送款於秦，請爲內應；長樂公丕命諸軍進攻之。戊午，克襄陽，執朱序，送長安。秦王堅以序能守節，拜度支尚書；以李伯護爲不忠，斬之。
秦將軍慕容越拔順陽，執太守譙國丁穆。堅欲官之，穆固辭不受。堅以中壘將軍梁成爲荊州刺史，配兵一萬，鎭襄陽，選其才望，禮而用之。
桓沖以襄陽陷沒，上疏送章節，請解職；不許。詔免劉波官，俄復以爲冠軍將軍。
秦以前將軍張蚝爲幷州刺史。
兗州刺史謝玄帥衆萬餘救彭城，軍于泗口，欲遣間使報戴𨔵而不可得；部曲將田泓請沒水潛行趣彭城，玄遣之。泓爲秦人所獲，厚賂之，使云南軍已敗；泓僞許之，旣而告城中曰：「南軍垂至，我單行來報，爲賊所得，勉之！」秦人殺之。彭超置輜重於留城，謝玄揚聲遣後軍將軍何謙向留城。超聞之，釋彭城圍，引兵還保輜重。戴𨔵帥彭城之衆，隨謙奔玄，超遂據彭城，留兗州治中徐褒守之，南攻盱眙。俱難克淮陰，留邵保戍之。
三月，壬戌，詔以「疆埸多虞，年穀不登，其供御所須，事從儉約；九親供給，衆官廩俸，權可減半。凡諸役費，自非軍國事要，皆宜停省。」
癸未，使右將軍毛虎生帥衆三萬擊巴中，以救魏興。前鋒督護趙福等至巴西，爲秦將張紹等所敗，亡七千餘人。虎生退屯巴東。蜀人李烏聚衆二萬，圍成都以應虎生，秦王堅使破虜將軍呂光擊滅之。夏，四月，戊申，韋鍾拔魏興，吉挹引刀欲自殺，左右奪其刀；會秦人至，執之，挹不言不食而死。秦王堅歎曰：「周孟威不屈於前，丁彥遠潔己於後，吉祖沖閉口而死，何晉氏之多忠臣也！」挹參軍史穎得歸，得挹臨終手疏，詔贈益州刺史。
秦毛當、王顯帥衆二萬自襄陽東會俱難、彭超攻淮南。五月，乙丑，難、超拔盱眙，執高密內史毛璪之。秦兵六萬圍幽州刺史田洛于三阿，去廣陵百里；朝廷大震，臨江列戍，遣征虜將軍謝石帥舟師屯涂中。石，安之弟也。
右衞將軍毛安之等帥衆四萬屯堂邑。秦毛當、毛盛帥騎二萬襲堂邑，安之等驚潰。兗州刺史謝玄自廣陵救三阿。丙子，難、超戰敗，退保盱眙。六月，戊子，玄與田洛帥衆五萬進攻盱眙，難、超又敗，退屯淮陰。玄遣何謙等帥舟師乘潮而上，夜，焚淮橋。邵保戰死，難、超退屯淮北。玄與何謙、戴𨔵、田洛共追之，戰于君川，復大破之，難、超北走，僅以身免。謝玄還廣陵，詔進號冠軍將軍，加領徐州刺史。
秦王堅聞之，大怒。秋，七月，檻車徵超下廷尉，超自殺。難削爵爲民。
以毛當爲徐州刺史，鎭彭城；毛盛爲兗州刺史，鎭湖陸；王顯爲揚州刺史，戍下邳。
謝安爲宰相，秦人屢入寇，邊兵失利，安每鎭之以和靜。其爲政，務舉大綱，不爲小察。時人比安於王導，而謂其文雅過之。
八月，丁亥，以左將軍王蘊爲尚書僕射，頃之，遷丹陽尹。蘊自以國姻，不欲在內，苦求外出，復以爲都督浙江東五郡諸軍事、會稽內史。
是歲，秦大饑。
