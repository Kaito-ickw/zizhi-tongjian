# 卷006 秦紀一 — 始皇帝上二年

二年〔注:丙辰(へいしん)の年、紀元前二四五年〕。

麃公(ひょうこう)が兵を率いて卷(けん)を攻め、首を三万斬った〔注:麃(ひょう)は邑(むら)の名。麃公は史書にその姓名が伝わらない。卷も邑の名である〕。

趙は廉頗(れんぱ)を假相國(かりのしょうこく)に任じ、魏を伐(う)って繁陽(はんよう)を取った。趙の孝成王(こうせいおう)が薨(こう)じ、子の悼襄王(とうじょうおう)が位に即(つ)いた。悼襄王は武襄君(ぶじょうくん)の樂乘(がくじょう)を遣わして廉頗と交代させようとした。廉頗は怒り、武襄君に攻めかかった。武襄君は逃げ去った。廉頗もまた(趙を)出奔して魏に身を寄せたが、長らく魏は彼を信用して用いることができなかった。趙の軍はたびたび秦に苦しめられたので、趙王はまた廉頗を取り戻したいと思い、廉頗のほうもまた趙にふたたび用いられたいと願っていた。趙王は使者を遣わし、廉頗がまだ使いものになるかどうかを見させた。廉頗の仇敵(きゅうてき)である郭開(かくかい)は、使者に多くの金を握らせ、廉頗を悪く言わせようとした。廉頗は使者に会うと、一度の食事で米を一斗(いっと)、肉を十斤(じっきん)平らげ、甲(よろい)をつけて馬に乗ってみせ、まだ使えるところを示した。だが使者は帰って報告した。「廉将軍は老いてはおりますが、まだ達者によく召し上がります。しかし私と座っているあいだに、ほんのしばらくのうちに三度も厠(かわや)へ立たれました〔注:矢(し)は糞(くそ)のことである〕。」趙王は廉頗を老いぼれたと思い、ついに召し出さなかった〔注:郭開が廉頗を陥れたのは、彼が仇であったからである。のちに郭開が讒言(ざんげん)して李牧(りぼく)を殺させたのは、賄賂(わいろ)に目がくらんだだけのことだ。讒(ざん)する者の悪辣(あくらつ)さにはきりがなく、その国を傾ける禍(わざわい)は、とても言い尽くせるものではない〕。楚の人がひそかに使いを出して廉頗を迎え入れた。廉頗は一たび楚の将となったが、これという手柄もなく、「わしはやはり趙の兵を率いたい」と言った。そしてついに壽春(じゅしゅん)で死んだ〔注:壽春県は、漢では九江郡に属し、唐では壽州の治所となった。始皇六年に楚はちょうど都を壽春に遷(うつ)しており、史書はここで廉頗の事を語り終えるのである〕。

二年
麃公將卒攻卷，斬首三萬。
趙以廉頗爲假相國，伐魏，取繁陽。趙孝成王薨，子悼襄王立，使武襄君樂乘代廉頗。廉頗怒，攻武襄君；武襄君走。廉頗出奔魏；久之，魏不能信用。趙師數困於秦，趙王思復得廉頗，廉頗亦思復用於趙。趙王使使者視廉頗尚可用否。廉頗之仇郭開多與使者金，令毀之。廉頗見使者，一飯斗米，肉十斤，被甲上馬，以示可用，使者還報曰︰「廉將軍雖老，尚善飯；然與臣坐，頃之三遺矢矣。」趙王以爲老，遂不召。楚人陰使迎之。廉頗一爲楚將，無功，曰︰「我思用趙人！」卒死於壽春。
