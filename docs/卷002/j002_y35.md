# 卷002 周紀二 — 顯王四十三年

顯王四十三年〔注:乙未(きのとひつじ)の年、紀元前三二六年〕。

趙の肅侯が薨じ、子の武靈王(ぶれいおう)が立った〔注:武靈王、名は雍(よう)〕。武靈王は博聞師(はくぶんし=見聞の広い顧問)三人と、左右の司過(しか=君主の過ちを正す役)三人を置き、まず先君に仕えた重臣・肥義(ひぎ)に意見を求め、その俸禄を加増した。

---

<summary>原文を表示</summary>

四十三年
趙肅侯薨，子武靈王立；置博聞師三人，左、右司過三人，先問先君貴臣肥義，加其秩。
